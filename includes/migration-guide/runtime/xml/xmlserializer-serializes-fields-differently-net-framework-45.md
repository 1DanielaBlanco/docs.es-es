### <a name="xmlserializer-serializes-fields-differently-in-net-framework-45"></a>XmlSerializer serializa los campos de manera diferente en .NET Framework 4.5

|   |   |
|---|---|
|Detalles|Los cambios en <xref:System.Xml.Serialization.XmlSerializer?displayProperty=name> en .NET Framework 4.5 han hecho que los campos tengan un formato distinto en el XML serializado.|
|Sugerencia|Este comportamiento se ha corregido en una actualización de servicio de .NET Framework 4.5. Para corregir este problema, actualice .NET Framework 4.5 o actualice a la versión 4.5.1 o posterior de .NET Framework. Como alternativa, la siguiente opción de configuración revertirá al comportamiento 4.0 de <xref:System.Xml.Serialization.XmlSerializer?displayProperty=name>:<pre><code>&lt;system.xml.serialization&gt;&#13;&#10;&lt;xmlSerializer useLegacySerializerGeneration=&quot;true&quot; /&gt;&#13;&#10;&lt;/system.xml.serialization&gt;&#13;&#10;</code></pre>|
