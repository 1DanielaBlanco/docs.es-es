### <a name="linq-to-ef-join-throws-argumentoutofrangeexception"></a>La combinación LINQ to EF produce una excepción ArgumentOutOfRangeException

|   |   |
|---|---|
|Detalles|Las consultas o enumerables de LINQ que llaman a "Join" en entidades de Entity Framework pueden provocar una excepción <xref:System.ArgumentOutOfRangeException?displayProperty=name> en .NET Framework 4.5.|
|Sugerencia|Este problema se corrigió en una actualización de servicio. Para corregir este problema, actualice .NET Framework 4.5 o actualice a la versión 4.5.1 o posterior de .NET Framework.|
