### <a name="wpf-aero2-and-aerolite-themes-toggle-button-doesnt-show-any-state"></a>El botón de alternancia de temas Aero2 y AeroLite de WPF no muestra ningún estado

|   |   |
|---|---|
|Detalles|En .NET Framework 4.5, los temas Aero2 y AeroLite de WPF no muestran ninguna diferencia entre el estado "activado" y "desactivado" de un botón de alternancia.|
|Sugerencia|Este problema se corrigió en una actualización de servicio. Para corregir este problema, actualice .NET Framework 4.5 o actualice a la versión 4.5.1 o posterior de .NET Framework. Como alternativa, se puede aplicar explícitamente un estilo a los botones de alternancia para solucionar este problema.|
|Ámbito|Major|
|Versión|4.5|
|Tipo|Tiempo de ejecución|
