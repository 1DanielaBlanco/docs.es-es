<span data-ttu-id="bf778-191">Las plataformas de destino se especifican en el archivo del proyecto.</span><span class="sxs-lookup"><span data-stu-id="bf778-191">Target frameworks are specified in your project file.</span></span> <span data-ttu-id="bf778-192">Cuando especifique una única plataforma de destino, use el elemento **TargetFramework**.</span><span class="sxs-lookup"><span data-stu-id="bf778-192">When a single target framework is specified, use the **TargetFramework** element.</span></span> <span data-ttu-id="bf778-193">El siguiente archivo de proyecto de aplicación de consola muestra cómo elegir como destino .NET Core 2.0:</span><span class="sxs-lookup"><span data-stu-id="bf778-193">The following console app project file demonstrates how to target .NET Core 2.0:</span></span>

```xml
    <TargetFrameworks>netstandard1.4;net40;net45</TargetFrameworks>

    <Reference Include="System.Threading.Tasks" />
  </ItemGroup>

```
