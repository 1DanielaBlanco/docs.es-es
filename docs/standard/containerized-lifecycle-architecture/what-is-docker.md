<span data-ttu-id="a40eb-116">Para ejecutar [contenedores de Windows](https://msdn.microsoft.com/virtualization/windowscontainers/about/about_overview), hay dos tipos de tiempos de ejecución:</span><span class="sxs-lookup"><span data-stu-id="a40eb-116">To run [Windows Containers](https://msdn.microsoft.com/virtualization/windowscontainers/about/about_overview), there are two types of runtimes:</span></span>

-   <span data-ttu-id="a40eb-119">**Contenedor de Hyper-V** esta opción se expande el aislamiento que ofrecen los contenedores de Windows Server mediante la ejecución de cada contenedor en una máquina virtual altamente optimizada.</span><span class="sxs-lookup"><span data-stu-id="a40eb-119">**Hyper-V Container** This expands on the isolation provided by Windows Server Containers by running each container in a highly optimized VM.</span></span> <span data-ttu-id="a40eb-120">En esta configuración, el kernel del host del contenedor no se comparte con los contenedores de Hyper-V, lo que proporciona un mejor aislamiento.</span><span class="sxs-lookup"><span data-stu-id="a40eb-120">In this configuration, the kernel of the container host is not shared with the Hyper-V Containers, providing better isolation.</span></span>

<span data-ttu-id="a40eb-121">Las imágenes de estos contenedores se crean de la misma manera y funcionan del mismo.</span><span class="sxs-lookup"><span data-stu-id="a40eb-121">The images for these containers are created in the same way and function the same.</span></span> <span data-ttu-id="a40eb-122">La diferencia radica en cómo se crea el contenedor de la imagen, la ejecución de un contenedor de Hyper-V requiere un parámetro adicional.</span><span class="sxs-lookup"><span data-stu-id="a40eb-122">The difference is in how the container is created from the image—running a Hyper-V Container requires an extra parameter.</span></span> <span data-ttu-id="a40eb-123">Para más información, vea [Contenedores de Hyper-V](https://msdn.microsoft.com/virtualization/windowscontainers/about/about_overview).</span><span class="sxs-lookup"><span data-stu-id="a40eb-123">For details, see [Hyper-V Containers](https://msdn.microsoft.com/virtualization/windowscontainers/about/about_overview).</span></span>
