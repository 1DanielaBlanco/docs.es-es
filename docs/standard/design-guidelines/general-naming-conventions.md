## <a name="word-choice"></a><span data-ttu-id="4d7e7-104">Elección de palabras</span><span class="sxs-lookup"><span data-stu-id="4d7e7-104">Word Choice</span></span>

 <span data-ttu-id="4d7e7-106">Por ejemplo, una propiedad denominada `HorizontalAlignment` es inglés-lee mejor que `AlignmentHorizontal`.</span><span class="sxs-lookup"><span data-stu-id="4d7e7-106">For example, a property named `HorizontalAlignment` is more English-readable than `AlignmentHorizontal`.</span></span>  
  
 <span data-ttu-id="4d7e7-107">**✓ HACER** preferible la legibilidad a la mayor brevedad.</span><span class="sxs-lookup"><span data-stu-id="4d7e7-107">**✓ DO** favor readability over brevity.</span></span>  
  
 <span data-ttu-id="4d7e7-108">El nombre de propiedad `CanScrollHorizontally` es mejor que `ScrollableX` (una referencia oculta al eje x).</span><span class="sxs-lookup"><span data-stu-id="4d7e7-108">The property name `CanScrollHorizontally` is better than `ScrollableX` (an obscure reference to the X-axis).</span></span>

 <span data-ttu-id="4d7e7-110">**X DO NOT** utilizar la notación húngara.</span><span class="sxs-lookup"><span data-stu-id="4d7e7-110">**X DO NOT** use Hungarian notation.</span></span>  
  
 <span data-ttu-id="4d7e7-111">**X evitar** utilizando identificadores que están en conflicto con las palabras clave de ampliamente utilizar lenguajes de programación.</span><span class="sxs-lookup"><span data-stu-id="4d7e7-111">**X AVOID** using identifiers that conflict with keywords of widely used programming languages.</span></span>  
  
 <span data-ttu-id="4d7e7-112">Según la regla 4 de Common Language Specification (CLS), todos los lenguajes compatibles con deben proporcionar un mecanismo que permite el acceso a elementos con nombre que utiliza una palabra clave de ese idioma como un identificador.</span><span class="sxs-lookup"><span data-stu-id="4d7e7-112">According to Rule 4 of the Common Language Specification (CLS), all compliant languages must provide a mechanism that allows access to named items that use a keyword of that language as an identifier.</span></span> <span data-ttu-id="4d7e7-113">C#, por ejemplo, utiliza el @ inicio de sesión como un mecanismo de escape en este caso.</span><span class="sxs-lookup"><span data-stu-id="4d7e7-113">C#, for example, uses the @ sign as an escape mechanism in this case.</span></span> <span data-ttu-id="4d7e7-114">Sin embargo, todavía es una buena idea para evitar palabras clave comunes porque es mucho más difícil de usar un método con la secuencia de escape que uno sin él.</span><span class="sxs-lookup"><span data-stu-id="4d7e7-114">However, it is still a good idea to avoid common keywords because it is much more difficult to use a method with the escape sequence than one without it.</span></span>
