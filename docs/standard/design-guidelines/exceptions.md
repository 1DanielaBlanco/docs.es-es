# Instrucciones de dise&#241;o para excepciones
Control de excepciones presenta muchas ventajas sobre informes de errores basado en el valor devuelto. Diseño buen marco ayuda a los desarrolladores de aplicaciones aprovechar las ventajas de las excepciones. Esta sección explica las ventajas de las excepciones y presenta las directrices para usar de forma eficaz.
