# <a name="getting-started"></a>Introducción

Hay varias maneras de empezar a trabajar con .NET.  Como .NET es una plataforma masiva, existen varios artículos en esta documentación que muestran cómo puede comenzar con. NET, cada uno desde una perspectiva distinta.

## <a name="getting-started-using-net-languages"></a>Introducción al uso de lenguajes .NET

* Los artículos de [introducción a C#](../csharp/getting-started/index.md) y de [tutoriales de C#](../csharp/tutorials/index.md) ofrecen varias maneras de comenzar con el enfoque puesto en C#.

* Los tutoriales de [introducción a F#](../fsharp/tutorials/getting-started/index.md) proporcionan tres formas principales en que puede usar F#: con Visual Studio, Visual Studio Code o herramientas de la línea de comandos.

* Los artículos de [introducción a Visual Basic](../visual-basic/getting-started/index.md) proporcionan guías sobre el uso de Visual Basic en Visual Studio, e incluyen otros diversos recursos de aprendizaje.

## <a name="getting-started-using-net-core"></a>Introducción al uso de .NET Core

* En [Introducción a .NET Core](../core/getting-started.md) se proporciona un resumen de artículos que muestran cómo empezar a trabajar con .NET Core en diferentes sistemas operativos y el uso de diferentes herramientas.

* En los [tutoriales de .NET Core](../core/tutorials/index.md) se describen varias maneras de comenzar con .NET Core con el sistema operativo y las herramientas de su elección.

## <a name="getting-started-using-docker-on-net-framework"></a>Introducción al uso de Docker en .NET Framework

En [Docker en .NET Framework](../framework/docker/index.md) se muestra cómo puede utilizar .NET Framework en contenedores de Docker de Windows.
