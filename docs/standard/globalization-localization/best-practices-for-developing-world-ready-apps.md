6.  <span data-ttu-id="d50c1-122">Cuando sea posible, trate las cadenas como cadenas enteras en lugar de como una serie de caracteres individuales.</span><span class="sxs-lookup"><span data-stu-id="d50c1-122">Whenever possible, handle strings as entire strings instead of as a series of individual characters.</span></span> <span data-ttu-id="d50c1-123">Esto es especialmente importante al ordenar o buscar subcadenas.</span><span class="sxs-lookup"><span data-stu-id="d50c1-123">This is especially important when sorting or searching for substrings.</span></span> <span data-ttu-id="d50c1-124">De esta forma evitará problemas asociados con el análisis de caracteres combinados.</span><span class="sxs-lookup"><span data-stu-id="d50c1-124">This will prevent problems associated with parsing combined characters.</span></span>

## <a name="localization-best-practices"></a><span data-ttu-id="d50c1-132">Prácticas recomendadas de localización</span><span class="sxs-lookup"><span data-stu-id="d50c1-132">Localization Best Practices</span></span>

9. <span data-ttu-id="d50c1-147">Use Visual Studio para crear cuadros de diálogo de Windows Forms, de manera que se puedan localizar mediante el [Editor de recursos de Windows Forms (Winres.exe)](../../../docs/framework/tools/winres-exe-windows-forms-resource-editor.md).</span><span class="sxs-lookup"><span data-stu-id="d50c1-147">Use Visual Studio to create Windows Forms dialog boxes, so that they can be localized using the [Windows Forms Resource Editor (Winres.exe)](../../../docs/framework/tools/winres-exe-windows-forms-resource-editor.md).</span></span> <span data-ttu-id="d50c1-148">No programe los cuadros de diálogo de Windows Forms manualmente.</span><span class="sxs-lookup"><span data-stu-id="d50c1-148">Do not code Windows Forms dialog boxes by hand.</span></span>  
  
10. <span data-ttu-id="d50c1-149">Organice una localización profesional (traducción).</span><span class="sxs-lookup"><span data-stu-id="d50c1-149">Arrange for professional localization (translation).</span></span>
