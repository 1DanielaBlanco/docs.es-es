# Manipular cadenas en .NET Framework
.NET Framework proporciona un amplio conjunto de rutinas que le permiten crear, comparar y modificar cadenas de forma eficaz, así como analizar rápidamente grandes cantidades de texto y de datos para buscar, quitar y reemplazar patrones de texto.  
  
## En esta sección  
 [Procedimientos recomendados para el uso de cadenas](../../../docs/standard/base-types/best-practices-strings.md)  
 Examina los métodos de ordenación, comparación y uso de mayúsculas y minúsculas en cadenas de .NET Framework, además de proporcionar recomendaciones para seleccionar un método de control de cadenas.  
  
 [Expresiones regulares de .NET Framework](../../../docs/standard/base-types/regular-expressions.md)  
 Proporciona información detallada sobre las expresiones regulares de .NET Framework, como elementos de lenguaje, comportamiento de las expresiones regulares y ejemplos.  
  
 [Operaciones básicas de cadenas](../../../docs/standard/base-types/basic-string-operations.md)  
 Describe las operaciones de cadena proporcionadas por las clases <xref:System.Text.StringBuilder?displayProperty=fullName> y <xref:System.String?displayProperty=fullName>, como la creación de nuevas cadenas a partir de matrices de bytes, la comparación de valores de cadena y la modificación de cadenas existentes.
