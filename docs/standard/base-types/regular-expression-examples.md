# Ejemplos de expresiones regulares
Esta sección contiene ejemplos de código que ilustran el uso de expresiones regulares aplicaciones comunes.  
  
> [!NOTE]
>  El espacio de nombres <xref:System.Web.RegularExpressions> contiene varios objetos de expresiones regulares que implementan los modelos de expresiones regulares predefinidos para el análisis de cadenas de documentos HTML, XML y ASP.NET.  Por ejemplo, la clase <xref:System.Web.RegularExpressions.TagRegex> identifica las etiquetas iniciales de una cadena y la clase <xref:System.Web.RegularExpressions.CommentRegex> identifica los comentarios de ASP.NET de una cadena.

 Ofrece un ejemplo que muestra cómo se reemplazan fechas con el formato mm\/dd\/aa por fechas con el formato dd\-mm\-aa.

 Ofrece un ejemplo que muestra cómo se extrae un protocolo y un número de puerto de una cadena que contiene una dirección URL.  Por ejemplo, "http:\/\/www.contoso.com:8080\/cartas\/readme.html" devuelve "http:8080".
