 <span data-ttu-id="90434-108">Las ventajas de tener los datos disponibles en dos vistas son las siguientes:</span><span class="sxs-lookup"><span data-stu-id="90434-108">The benefits of having data available in two views are:</span></span>  
  
-   <span data-ttu-id="90434-109">La parte estructurada de un documento XML puede asignarse a un conjunto de datos y ser almacenada, indizada y buscada de forma eficaz.</span><span class="sxs-lookup"><span data-stu-id="90434-109">The structured portion of an XML document can be mapped to a dataset, and be efficiently stored, indexed, and searched.</span></span>
