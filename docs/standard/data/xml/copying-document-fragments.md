# Copiar fragmentos de documento
Se puede crear un nodo **XmlDocumentFragment** y, después, agregar nodos en niveles inferiores.  Al insertar el nodo **XmlDocumentFragment** mediante el método **InsertNode**, no se copia el nodo **XmlDocumentFragment**, pero se insertan sus nodos secundarios en DOM.
