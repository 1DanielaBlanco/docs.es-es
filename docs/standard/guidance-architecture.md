# <a name="net-architecture-guidance"></a>Guía de arquitectura de .NET

## <a name="containerized-docker-application-lifecycle-with-the-microsoft-platform-and-toolscontainerized-lifecycle-architectureindexmd"></a>[Ciclo de vida de aplicación de Docker en contenedores con las herramientas y plataforma de Microsoft](./containerized-lifecycle-architecture/index.md)

Esta guía es una introducción a los procesos de recomendadas del ciclo de vida de extremo a extremo que usará para desarrollar, validar e implementar aplicaciones en contenedores de Docker mediante Visual Studio y Microsoft Azure.

## <a name="modernize-existing-net-applications-with-azure-cloud-and-windows-containersmodernize-with-azure-and-containersindexmd"></a>[Modernizar las aplicaciones .NET existentes con la nube de Azure y los contenedores de Windows](./modernize-with-azure-and-containers/index.md)

Esta guía es una introducción a las estrategias que se debe migrar las aplicaciones web existentes para la nube de Azure y otros contenedores de Windows. Obtendrá información sobre las estrategias de código, la migración de datos, orchestrators y procesa los elementos de configuración/CD.

## <a name="architect-modern-web-applications-with-aspnet-core-and-azuremodern-web-apps-azure-architectureindexmd"></a>[Diseñar aplicaciones web modernas con ASP.NET Core y Azure](modern-web-apps-azure-architecture/index.md)

Esta guía es una introducción a la arquitectura recomendada, el diseño y los procesos de implementación que se usará para las aplicaciones de ASP.NET y ASP.NET Core de compilación y hospedar las aplicaciones de Azure.

## <a name="architecting-container-and-microservice-based-applicationsmicroservices-architectureindexmd"></a>[Diseñar la arquitectura de aplicaciones basadas en contenedor y microservicio](microservices-architecture/index.md)

Esta guía es una introducción al desarrollo de aplicaciones basadas en microservicios y a su administración mediante contenedores. En ella se trata el diseño de la arquitectura y los métodos de implementación con .NET Core y contenedores de Docker.
