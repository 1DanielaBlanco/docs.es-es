# <a name="building-documentation-with-mdoc"></a>Documentación para crear mdoc

mdoc es un sistema de compilación de documentación para documentos de referencia administrados. Tomando los ensamblados como referencia, mdoc crea el formato básico de la documentación de referencia administrada, que se complementa mediante contenido creado.
