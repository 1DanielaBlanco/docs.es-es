-   <span data-ttu-id="75bb5-107">**Entorno de desarrollo/pruebas**: una VM en la nube es ideal para desarrollo y pruebas en la nube.</span><span class="sxs-lookup"><span data-stu-id="75bb5-107">**Dev/test environment**: A VM in the cloud is perfect for development and testing in the cloud.</span></span> <span data-ttu-id="75bb5-108">Puede crear rápidamente o detener el entorno según sus necesidades.</span><span class="sxs-lookup"><span data-stu-id="75bb5-108">You can rapidly create or stop the environment depending on your needs.</span></span>
