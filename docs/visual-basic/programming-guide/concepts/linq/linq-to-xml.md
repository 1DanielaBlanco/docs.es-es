# <a name="linq-to-xml-visual-basic"></a>LINQ to XML (Visual Basic)
LINQ to XML proporciona una interfaz de programación XML en memoria que aprovecha el marco Integrated Query (LINQ). LINQ to XML utiliza las últimas capacidades de lenguaje de .NET Framework y es comparable a una interfaz de programación de Document Object Model (DOM) XML actualizada y rediseñada.  
  
 La familia LINQ de tecnologías proporciona una experiencia de consultas coherente para objetos (LINQ to Objects), bases de datos relacionales (LINQ to SQL) y XML (LINQ to XML).
