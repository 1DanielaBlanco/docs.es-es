# <a name="concatenation-operations-visual-basic"></a>Operaciones de concatenación (Visual Basic)
Concatenación hace referencia a la operación de anexar una secuencia a otra.  
  
 La siguiente ilustración muestra una operación de concatenación de dos secuencias de caracteres.  
  
 ![Gráfico mostrando la concatenación de dos secuencias. ](../../../../csharp/programming-guide/concepts/linq/media/concat.png "Concat")
