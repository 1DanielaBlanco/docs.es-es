# <a name="serializing-xml-trees-visual-basic"></a>Serializar árboles XML (Visual Basic)
Serializar un árbol XML significa generar XML a partir de un árbol XML. Puede serializar a un archivo, para una implementación concreta de la <xref:System.IO.TextWriter>(clase), o para una implementación concreta de un <xref:System.Xml.XmlWriter>.</xref:System.Xml.XmlWriter> </xref:System.IO.TextWriter>  
  
 Puede controlar diversos aspectos del proceso de serialización. Por ejemplo, puede decidir si desea agregar una sangría al XML serializado y si desea escribe una declaración XML.
