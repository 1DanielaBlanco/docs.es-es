# <a name="introduction-to-linq-visual-basic"></a>Introducción a LINQ (Visual Basic)
Language-Integrated Query (LINQ) es que una innovación introducida en la versión 3.5 de .NET Framework que une la brecha entre el mundo de los objetos y el mundo de los datos.  
  
 Tradicionalmente, las consultas con datos se expresan como cadenas sencillas, sin comprobación de tipos en tiempo de compilación ni compatibilidad con IntelliSense. Además, tiene que aprender un lenguaje de consulta diferente para cada tipo de origen de datos: SQL bases de datos, documentos XML, varios servicios Web y así sucesivamente. LINQ permite una *consulta* una construcción de lenguaje de primera clase en Visual Basic. Escribir consultas en colecciones fuertemente tipadas de objetos mediante palabras clave del lenguaje y operadores familiar.  
  
 Puede escribir consultas LINQ en Visual Basic para las bases de datos SQL Server, documentos XML, conjuntos de datos de ADO.NET y cualquier colección de objetos que admite <xref:System.Collections.IEnumerable>o genérica <xref:System.Collections.Generic.IEnumerable%601>interfaz.</xref:System.Collections.Generic.IEnumerable%601> </xref:System.Collections.IEnumerable> Compatibilidad con LINQ también se proporciona por parte de terceros para muchos servicios Web y otras implementaciones de la base de datos.  
  
 Puede usar consultas LINQ en proyectos nuevos o junto con las consultas de LINQ no en proyectos existentes. El único requisito es que el proyecto como destino .NET Framework 3.5 o posterior.
