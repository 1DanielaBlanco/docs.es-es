# <a name="linq-and-reflection-visual-basic"></a>LINQ y reflexión (Visual Basic)
Las API de reflexión de biblioteca de clases de .NET Framework se pueden utilizar para examinar los metadatos en un ensamblado .NET y crear colecciones de tipos, miembros de tipo, parámetros etc. que se encuentran en ese ensamblado. Dado que estas colecciones admiten la interfaz genérica `IEnumerable` interfaz, se pueden consultar mediante LINQ.
