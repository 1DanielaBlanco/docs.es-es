# <a name="generation-operations-visual-basic"></a>Operaciones de generación (Visual Basic)
Generación hace referencia a la creación de una nueva secuencia de valores.  
  
 Los métodos de operador de consulta estándar que realizan la generación se enumeran en la sección siguiente.  
  
## <a name="methods"></a>Métodos  
  
|Nombre del método|Descripción|Sintaxis de expresiones de consulta de Visual Basic|Más información|  
|-----------------|-----------------|------------------------------------------|----------------------|  
|DefaultIfEmpty|Reemplaza una colección vacía con una colección de singleton con valores predeterminados.|No es aplicable.|<xref:System.Linq.Enumerable.DefaultIfEmpty%2A?displayProperty=fullName></xref:System.Linq.Enumerable.DefaultIfEmpty%2A?displayProperty=fullName><br /><br /> <xref:System.Linq.Queryable.DefaultIfEmpty%2A?displayProperty=fullName></xref:System.Linq.Queryable.DefaultIfEmpty%2A?displayProperty=fullName>|  
|Empty|Devuelve una colección vacía.|No es aplicable.|<xref:System.Linq.Enumerable.Empty%2A?displayProperty=fullName></xref:System.Linq.Enumerable.Empty%2A?displayProperty=fullName>|  
|Intervalo|Genera una colección que contiene una secuencia de números.|No es aplicable.|<xref:System.Linq.Enumerable.Range%2A?displayProperty=fullName></xref:System.Linq.Enumerable.Range%2A?displayProperty=fullName>|  
|Repeat|Genera una colección que contiene un valor repetido.|No es aplicable.|<xref:System.Linq.Enumerable.Repeat%2A?displayProperty=fullName></xref:System.Linq.Enumerable.Repeat%2A?displayProperty=fullName>|
