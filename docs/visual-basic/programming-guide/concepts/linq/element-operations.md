# <a name="element-operations-visual-basic"></a>Operaciones de elemento (Visual Basic)
Operaciones de elementos devuelven un único elemento específico de una secuencia.  
  
 Los métodos de operador de consulta estándar que realizan operaciones de elementos se muestran en la sección siguiente.
