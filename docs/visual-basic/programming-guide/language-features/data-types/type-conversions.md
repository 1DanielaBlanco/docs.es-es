# Conversiones de tipos en Visual Basic
[!INCLUDE[vs2017banner](../../../../csharp/includes/vs2017banner.md)]

El proceso de cambiar un valor de un tipo de datos a otro tipo se denomina *conversión*.  Las conversiones son de *ampliación* o de *conversión*, en función de las capacidades de datos de los tipos involucrados.  También pueden ser *implícitas* o *explícitas*, según la sintaxis del código fuente.
