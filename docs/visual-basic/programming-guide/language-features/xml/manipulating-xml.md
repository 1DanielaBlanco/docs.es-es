# Manipular XML en Visual Basic
[!INCLUDE[vs2017banner](../../../../csharp/includes/vs2017banner.md)]

Puede usar *literales XML* para cargar XML desde un origen externo, como una cadena, un archivo o una secuencia.  A continuación, podrá usar [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] para manipular el código XML y [!INCLUDE[vbteclinqext](../../../../csharp/getting-started/includes/vbteclinqext_md.md)] para consultarlo.
