# Crear XML en Visual Basic
[!INCLUDE[vs2017banner](../../../../visual-basic/developing-apps/includes/vs2017banner.md)]

[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] permite usar *literales XML* directamente en el código.  La sintaxis de los literales XML representa objetos de [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq-md.md)] y es similar a la sintaxis de XML 1.0.  Esto facilita la creación mediante programación de elementos, documentos y fragmentos XML porque el código tiene la misma estructura que el último XML.
