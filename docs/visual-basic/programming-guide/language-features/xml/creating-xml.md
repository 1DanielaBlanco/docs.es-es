# <a name="creating-xml-in-visual-basic"></a>Crear XML en Visual Basic
[!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]permite utilizar *literales XML* directamente en el código. La sintaxis de literales XML representa [!INCLUDE[sqltecxlinq](../../../../csharp/programming-guide/concepts/linq/includes/sqltecxlinq_md.md)] objetos y es similar a la sintaxis XML 1.0. Esto facilita la creación de elementos, documentos y fragmentos XML mediante programación porque el código tiene la misma estructura que el XML final.
