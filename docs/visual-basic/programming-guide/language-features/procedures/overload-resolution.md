# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="de113-102">Resolución de sobrecarga (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="de113-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="de113-103">Cuando el compilador de Visual Basic encuentra una llamada a un procedimiento que se define en varias versiones sobrecargadas, el compilador debe decidir cuál de las sobrecargas para llamar a.</span><span class="sxs-lookup"><span data-stu-id="de113-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="de113-104">Para ello, siga estos pasos:</span><span class="sxs-lookup"><span data-stu-id="de113-104">It does this by performing the following steps:</span></span>  
  
1.  <span data-ttu-id="de113-105">**Accesibilidad.**</span><span class="sxs-lookup"><span data-stu-id="de113-105">**Accessibility.**</span></span> <span data-ttu-id="de113-106">Elimina cualquier sobrecarga con un nivel de acceso que impide que el código de llamada de llamarlo.</span><span class="sxs-lookup"><span data-stu-id="de113-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2.  <span data-ttu-id="de113-107">**Número de parámetros.**</span><span class="sxs-lookup"><span data-stu-id="de113-107">**Number of Parameters.**</span></span> <span data-ttu-id="de113-108">Elimina cualquier sobrecarga que define un número diferente de parámetros que se suministran en la llamada.</span><span class="sxs-lookup"><span data-stu-id="de113-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3.  <span data-ttu-id="de113-109">**Tipos de datos de parámetro.**</span><span class="sxs-lookup"><span data-stu-id="de113-109">**Parameter Data Types.**</span></span> <span data-ttu-id="de113-110">El compilador da preferencia de los métodos de instancia a través de métodos de extensión.</span><span class="sxs-lookup"><span data-stu-id="de113-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="de113-111">Si se encuentra ningún método de instancia que requiere solo las conversiones para que coincida con la llamada al procedimiento de ampliación, se quitan todos los métodos de extensión y el compilador sigue con solo los candidatos de método de instancia.</span><span class="sxs-lookup"><span data-stu-id="de113-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="de113-112">Si no se encuentra ningún método de instancia como ese, continúa con la instancia y métodos de extensión.</span><span class="sxs-lookup"><span data-stu-id="de113-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="de113-113">En este paso, elimina cualquier sobrecarga para el que no se puede convertir los tipos de datos de los argumentos de llamada a los tipos de parámetro definidos en la sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="de113-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4.  <span data-ttu-id="de113-114">**Conversiones de restricción.**</span><span class="sxs-lookup"><span data-stu-id="de113-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="de113-115">Elimina cualquier sobrecarga que requiere una conversión de restricción de los tipos de argumento que realiza la llamada a los tipos de parámetro definidas.</span><span class="sxs-lookup"><span data-stu-id="de113-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="de113-116">Esto es cierto si la comprobación de tipo modificador ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) es `On` o `Off`.</span><span class="sxs-lookup"><span data-stu-id="de113-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5.  <span data-ttu-id="de113-117">**Ampliación mínima.**</span><span class="sxs-lookup"><span data-stu-id="de113-117">**Least Widening.**</span></span> <span data-ttu-id="de113-118">El compilador considera las sobrecargas restantes en parejas.</span><span class="sxs-lookup"><span data-stu-id="de113-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="de113-119">Para cada par, compara los tipos de datos de los parámetros definidos.</span><span class="sxs-lookup"><span data-stu-id="de113-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="de113-120">Si los tipos en una de las sobrecargas todas se amplían a los tipos correspondientes en el otro, el compilador elimina la última.</span><span class="sxs-lookup"><span data-stu-id="de113-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="de113-121">Es decir, conserva la sobrecarga que requiere la menor cantidad de ampliación.</span><span class="sxs-lookup"><span data-stu-id="de113-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6.  <span data-ttu-id="de113-122">**Candidato único.**</span><span class="sxs-lookup"><span data-stu-id="de113-122">**Single Candidate.**</span></span> <span data-ttu-id="de113-123">Sigue considerando las sobrecargas en parejas hasta solo una sobrecarga permanece y resuelve la llamada a esa sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="de113-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="de113-124">Si el compilador no puede reducir las sobrecargas a un único candidato, genera un error.</span><span class="sxs-lookup"><span data-stu-id="de113-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
