# <a name="overload-resolution-visual-basic"></a><span data-ttu-id="bfa8f-102">Resolución de sobrecarga (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="bfa8f-102">Overload Resolution (Visual Basic)</span></span>
<span data-ttu-id="bfa8f-103">Cuando el compilador de Visual Basic encuentra una llamada a un procedimiento que está definido en varias versiones sobrecargadas, el compilador debe decidir cuál de las sobrecargas para llamar a.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-103">When the Visual Basic compiler encounters a call to a procedure that is defined in several overloaded versions, the compiler must decide which of the overloads to call.</span></span> <span data-ttu-id="bfa8f-104">Para ello, ejecute los pasos siguientes:</span><span class="sxs-lookup"><span data-stu-id="bfa8f-104">It does this by performing the following steps:</span></span>  
  
1.  <span data-ttu-id="bfa8f-105">**Accesibilidad.**</span><span class="sxs-lookup"><span data-stu-id="bfa8f-105">**Accessibility.**</span></span> <span data-ttu-id="bfa8f-106">Elimina cualquier sobrecarga con un nivel de acceso que impide que el código de llamada llamarlo.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-106">It eliminates any overload with an access level that prevents the calling code from calling it.</span></span>  
  
2.  <span data-ttu-id="bfa8f-107">**Número de parámetros.**</span><span class="sxs-lookup"><span data-stu-id="bfa8f-107">**Number of Parameters.**</span></span> <span data-ttu-id="bfa8f-108">Elimina cualquier sobrecarga que define un número diferente de parámetros que se suministran en la llamada.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-108">It eliminates any overload that defines a different number of parameters than are supplied in the call.</span></span>  
  
3.  <span data-ttu-id="bfa8f-109">**Tipos de datos de parámetro.**</span><span class="sxs-lookup"><span data-stu-id="bfa8f-109">**Parameter Data Types.**</span></span> <span data-ttu-id="bfa8f-110">El compilador da preferencia de métodos de instancia a través de métodos de extensión.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-110">The compiler gives instance methods preference over extension methods.</span></span> <span data-ttu-id="bfa8f-111">Si se encuentra cualquier método de instancia que requiere solo conversiones de ampliación que coincida con la llamada de procedimiento, se quitan todos los métodos de extensión y el compilador sigue con solo los candidatos de método de instancia.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-111">If any instance method is found that requires only widening conversions to match the procedure call, all extension methods are dropped and the compiler continues with only the instance method candidates.</span></span> <span data-ttu-id="bfa8f-112">Si no se encuentra ningún método de instancia de este tipo, continúa con la instancia y los métodos de extensión.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-112">If no such instance method is found, it continues with both instance and extension methods.</span></span>  
  
     <span data-ttu-id="bfa8f-113">En este paso, elimina cualquier sobrecarga para que los tipos de datos de los argumentos de llamada no se puede convertir a los tipos de parámetro definidos en la sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-113">In this step, it eliminates any overload for which the data types of the calling arguments cannot be converted to the parameter types defined in the overload.</span></span>  
  
4.  <span data-ttu-id="bfa8f-114">**Conversiones de restricción.**</span><span class="sxs-lookup"><span data-stu-id="bfa8f-114">**Narrowing Conversions.**</span></span> <span data-ttu-id="bfa8f-115">Elimina cualquier sobrecarga que requiera una conversión de restricción de los tipos de argumento que realiza la llamada a los tipos de parámetro definidos.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-115">It eliminates any overload that requires a narrowing conversion from the calling argument types to the defined parameter types.</span></span> <span data-ttu-id="bfa8f-116">Esto es cierto si la comprobación de tipo modificador ([Option Strict (instrucción)](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) es `On` o `Off`.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-116">This is true whether the type checking switch ([Option Strict Statement](../../../../visual-basic/language-reference/statements/option-strict-statement.md)) is `On` or `Off`.</span></span>  
  
5.  <span data-ttu-id="bfa8f-117">**Ampliación mínima.**</span><span class="sxs-lookup"><span data-stu-id="bfa8f-117">**Least Widening.**</span></span> <span data-ttu-id="bfa8f-118">El compilador considera que las sobrecargas restantes en parejas.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-118">The compiler considers the remaining overloads in pairs.</span></span> <span data-ttu-id="bfa8f-119">Para cada pareja, compara los tipos de datos de los parámetros definidos.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-119">For each pair, it compares the data types of the defined parameters.</span></span> <span data-ttu-id="bfa8f-120">Si los tipos en una de las sobrecargas todos los amplían los tipos correspondientes en la otra, el compilador elimina la última.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-120">If the types in one of the overloads all widen to the corresponding types in the other, the compiler eliminates the latter.</span></span> <span data-ttu-id="bfa8f-121">Es decir, conserva la sobrecarga que requiere la menor cantidad de ampliación.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-121">That is, it retains the overload that requires the least amount of widening.</span></span>  
  
6.  <span data-ttu-id="bfa8f-122">**Candidata única.**</span><span class="sxs-lookup"><span data-stu-id="bfa8f-122">**Single Candidate.**</span></span> <span data-ttu-id="bfa8f-123">Continúa Considerando las sobrecargas por partes hasta que solo una sobrecarga permanece y resuelve la llamada a esa sobrecarga.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-123">It continues considering overloads in pairs until only one overload remains, and it resolves the call to that overload.</span></span> <span data-ttu-id="bfa8f-124">Si el compilador no puede reducir las sobrecargas a una sola candidata, genera un error.</span><span class="sxs-lookup"><span data-stu-id="bfa8f-124">If the compiler cannot reduce the overloads to a single candidate, it generates an error.</span></span>
