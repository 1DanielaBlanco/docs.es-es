# Constantes y enumeraciones en Visual Basic
[!INCLUDE[vs2017banner](../../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Las constantes son una forma de utilizar nombres significativos en lugar de valores que no se modifican.  Las constantes almacenan valores que, como su nombre indica, permanecen constantes durante la ejecución de una aplicación.  Puede utilizar constantes para hacer más legible el código, puesto que se utilizan nombres significativos en lugar de números.  
  
 Las enumeraciones proporcionan una forma cómoda de trabajar con conjuntos de constantes relacionadas y de asociar valores de constantes con nombres.  Por ejemplo, se puede declarar una enumeración para un conjunto de constantes de tipo entero asociadas con los días de la semana, y después utilizar los nombres de los días en el código en lugar de sus valores enteros.
