# Las instrucciones &#39;Inherits&#39; deben preceder a todas las declaraciones de una interfaz.
Una instrucción `Inherits` debe ser la primera línea no vacía sin comentarios en una definición de clase.  
  
 **Identificador de error:** BC30357  
  
### Para corregir este error  
  
-   Mueva la instrucción `Inherits` para que preceda a todas las declaraciones.
