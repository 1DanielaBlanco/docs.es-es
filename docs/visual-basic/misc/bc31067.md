# <a name="method-in-a-structure-cannot-be-declared-39protected39-or-39protected-friend39"></a>No se puede declarar el método de una estructura &#39; Proteger &#39; o &#39; Protected Friend &#39;
Se ha declarado un método de una estructura como `Protected` o `Protected Friend`.  
  
 **Identificador de error:** BC31067  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare los métodos incluidos en estructuras como `Public` o `Private`.
