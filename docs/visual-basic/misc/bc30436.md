# <a name="members-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Los miembros de un módulo no se pueden declarar &#39; &lt;especificador&gt;&#39;
Usó un especificador que no es válido en un miembro dentro de una instrucción `Module` . Los módulos no admiten la creación de instancias ni la herencia y tampoco pueden implementar interfaces.  
  
 **Identificador de error:** BC30436  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador.
