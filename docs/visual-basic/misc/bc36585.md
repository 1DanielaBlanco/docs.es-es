# <a name="ltelementnamegt-is-not-a-parameter-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>'&lt;elementname&gt;'no es un parámetro de método de extensión'&lt;methodname&gt;'definido en'&lt;typename&gt;'
Una llamada a procedimiento ofrece un argumento adicional que no está definido por el método de extensión.  
  
 **Identificador de error:** BC36585  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el argumento superfluo de la lista de argumentos.
