# <a name="access-denied-to-39ltnamegt39"></a>Acceso denegado a &#39; &lt;nombre&gt;&#39;
No se puede acceder a un dispositivo, archivo o disco porque no los permisos de seguridad del equipo son insuficientes.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que tiene los permisos necesarios.
