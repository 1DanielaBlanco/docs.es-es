# Los m&#233;todos no se pueden declarar como &#39;Static&#39;.
El modificador `Static` no se puede aplicar a procedimientos completos.  
  
 **Identificador de error:** BC30810  
  
### Para corregir este error  
  
-   Utilice el modificador `Static` para declarar elementos individuales dentro de un procedimiento como estático.
