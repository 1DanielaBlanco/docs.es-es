# No se pueden recibir eventos del objeto porque el objeto ya est&#225; desencadenando eventos hasta el n&#250;mero m&#225;ximo de receptores de eventos que admite.
Se ha superado el número máximo de receptores de eventos admitidos por el objeto.  
  
### Para corregir el error  
  
-   Reduzca el número de receptores de eventos.
