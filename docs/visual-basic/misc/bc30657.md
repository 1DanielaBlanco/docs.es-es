# &#39;&lt;procedurename&gt;&#39; tiene un tipo de valor devuelto o unos tipos de par&#225;metros que no se admiten.
Se declara un procedimiento con parámetros o un tipo de valor devuelto que [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] no puede representar.  
  
 **Identificador de error:** BC30657  
  
### Para corregir este error  
  
-   Defina los tipos de devolución o los parámetros como un tipo compatible.  
  
-   Desarrolle un nuevo tipo que describa los datos usados por este procedimiento.
