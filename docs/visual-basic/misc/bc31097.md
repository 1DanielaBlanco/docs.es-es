# <a name="member-39ltmembernamegt39-cannot-be-found-in-class-39ltclassnamegt39"></a>Miembro '&lt;membername&gt;'no se encuentra en la clase'&lt;classname&gt;'
Miembro '\<membername >' no se encuentra en la clase\<classname >'. Esta situación suele ser consecuencia de un 'Microsoft.VisualBasic.dll' discordante.  
  
 No se pudo encontrar un miembro definido.  
  
 **Identificador de error:** BC31097  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Vuelva a compilar el programa para ver si el error se repite.  
  
2.  Si el error se repite, guarde el trabajo y reinicie [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
3.  Si el error persiste, reinstale [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
4.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
