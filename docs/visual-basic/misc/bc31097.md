# No se encuentra el miembro &#39;&lt;nombreDeMiembro&gt;&#39; en la clase &#39;&lt;nombreDeClase&gt;&#39;
No se encuentra el miembro '\<nombreDeMiembro\>' en la clase '\<nombreDeClase\>'. Esta situación suele ser consecuencia de un 'Microsoft.VisualBasic.dll' discordante.  
  
 No se pudo encontrar un miembro definido.  
  
 **Identificador de error:** BC31097  
  
### Para corregir este error  
  
1.  Vuelva a compilar el programa para ver si el error se repite.  
  
2.  Si el error se repite, guarde el trabajo y reinicie [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)].  
  
3.  Si el error persiste, reinstale [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
4.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
