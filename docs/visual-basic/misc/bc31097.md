# <a name="member-39ltmembernamegt39-cannot-be-found-in-class-39ltclassnamegt39"></a>Miembro &#39; &lt;membername&gt; &#39; no se encuentra en la clase &#39; &lt;classname&gt;&#39;
Miembro '\<nombreDeMiembro >' no se encuentra en la clase\<classname >'. Esta situación suele ser consecuencia de un 'Microsoft.VisualBasic.dll' discordante.  
  
 No se pudo encontrar un miembro definido.  
  
 **Identificador de error:** BC31097  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Vuelva a compilar el programa para ver si el error se repite.  
  
2.  Si el error se repite, guarde su trabajo y reinicie Visual Studio.  
  
3.  Si el error persiste, vuelva a instalar Visual Basic.  
  
4.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
