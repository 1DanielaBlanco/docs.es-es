## <a name="to-correct-this-error"></a><span data-ttu-id="3cd82-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="3cd82-106">To correct this error</span></span>

```vb  
Sub RunQuery(ByVal collection As List(Of Integer), _  
             ByRef filterValue As Integer)  
    Dim fv = filterValue  
    Dim queryResult = From num In collection _  
                      Where num < fv  
End Sub  
```
