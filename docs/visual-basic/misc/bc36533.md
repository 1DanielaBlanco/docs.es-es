## <a name="to-correct-this-error"></a><span data-ttu-id="822fb-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="822fb-106">To correct this error</span></span>

    Dim queryResult = From num In collection _  
                      Where num < fv
