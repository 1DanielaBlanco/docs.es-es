# <a name="reference-required-to-module-ltmodulenamegt-containing-the-base-class-ltclassnamegt"></a>Necesaria una referencia al módulo '&lt;modulename&gt;'que contiene la clase base'&lt;classname&gt;'
Necesaria una referencia al módulo '\<NombreMódulo >' que contiene la clase base\<classname >'. Agregue una al proyecto.  
  
 La clase se define en un módulo al que no se hace referencia directamente en el proyecto. El compilador de Visual Basic requiere una referencia para evitar la ambigüedad en caso de la clase está definida en más de un módulo.  
  
 **Identificador de error:** BC30008  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Incluya el nombre del módulo al que no se hace referencia en las referencias del proyecto.
