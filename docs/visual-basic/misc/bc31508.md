# <a name="cannot-find-net-framework-directory-lterrorgt"></a>No se puede encontrar el directorio de .NET Framework: &lt;error&gt;
El directorio donde el [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] se almacenan los ensamblados no se pudo encontrar.  
  
 **Id. de error:** BC31508  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Vuelva a compilar el programa para ver si el error se repite.  
  
2.  Si el error continúa, guarde el trabajo y reinicie [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
3.  Si el error se repite, reinstale [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
4.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
