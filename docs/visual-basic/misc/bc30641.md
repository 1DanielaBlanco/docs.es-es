# <a name="39byval39-and-39byref39-cannot-be-combined"></a>&#39;ByVal&#39; y &#39;ByRef&#39; no se pueden combinar
Las palabras clave `ByVal` y `ByRef` se excluyen mutuamente.  
  
 **Identificador de error:** BC30641  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Especifique `ByVal` o `ByRef`, pero no ambos.
