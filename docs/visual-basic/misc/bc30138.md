# <a name="unable-to-create-temp-file-in-path-39ltfilenamegt39-lterror-messagegt"></a>No se puede crear archivo temporal en ruta de acceso &#39; &lt;filename&gt;&#39;: &lt;mensaje de error&gt;
El compilador de Visual Basic llama a la herramienta Assembly Linker (Al.exe, también denominado Alink) para generar un ensamblado con un manifiesto. El vinculador notificó un error al crear un archivo o escribir un recurso en memoria en este. Podría tratarse de un problema de configuración.  
  
 **Identificador de error:** BC30138  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Examine el mensaje de error citado y consulte el tema [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) para obtener una explicación y consejos.  
  
2.  Si el error persiste, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
