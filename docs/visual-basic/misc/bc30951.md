# <a name="wrong-number-of-type-arguments-passed-to-method-ltgenericprocedurenamegt"></a>Número incorrecto de argumentos de tipo pasados al método '&lt;nombredeprocedimientogenérico&gt;'
Se llamó a un procedimiento genérico con un número de argumentos de tipo que no coincide con el número de parámetros de tipo con el que se definió.  
  
 **Identificador de error:** BC30951  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione un argumento de tipo para cada parámetro de tipo definido para el procedimiento genérico.  
  
     O bien  
  
-   Llame al procedimiento genérico sin ningún argumento de tipo y deje que el compilador intente deducir los argumentos de tipo.
