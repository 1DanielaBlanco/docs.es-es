# No se permiten nombres de tipo no intr&#237;nseco en las expresiones de compilaci&#243;n condicional.
La expresión de compilación condicional hace referencia a un tipo no intrínseco, pero se permiten solo tipos intrínsecos.  
  
 Para más información sobre el tipo intrínseco en [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)], vea [Resumen de tipos de datos](../../visual-basic/language-reference/keywords/data-types-summary.md).  
  
 **Identificador de error:** BC31426  
  
### Para corregir este error  
  
-   Reemplace el tipo no intrínseco por un tipo intrínseco.
