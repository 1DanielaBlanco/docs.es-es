# El m&#233;todo &#39;&lt;methodname&gt;&#39; no se puede declarar como &#39;Partial&#39; porque solo puede haber un m&#233;todo &#39;&lt;methodname&gt;&#39; marcado como &#39;Partial&#39;
No puede tener dos declaraciones de método parcial que tengan el mismo nombre y la misma firma.  
  
 **Id. de error:** BC31433  
  
### Para corregir este error  
  
1.  Cambie el nombre o la firma de una de las declaraciones de método parcial.
