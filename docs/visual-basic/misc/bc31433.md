# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Método '\<methodname >' no se puede declarar como 'Partial' porque sólo un método '\<methodname >' puede marcarse como 'Partial'
No puede tener dos declaraciones de método parcial que tengan el mismo nombre y la misma firma.  
  
 **Identificador de error:** BC31433  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Cambie el nombre o la firma de una de las declaraciones de método parcial.
