# <a name="method-ltmethodnamegt-cannot-be-declared-partial-because-only-one-method-ltmethodnamegt-can-be-marked-partial"></a>Método '&lt;methodname&gt;' no se puede declarar como 'Partial' porque sólo un método '&lt;methodname&gt;' pueden marcarse como 'Partial'
No puede tener dos declaraciones de método parcial que tengan el mismo nombre y la misma firma.  
  
 **Identificador de error:** BC31433  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Cambie el nombre o la firma de una de las declaraciones de método parcial.
