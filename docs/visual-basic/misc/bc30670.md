# <a name="39redim39-statements-require-a-parenthesized-list-of-the-new-bounds-of-each-dimension-of-the-array"></a>&#39; ReDim &#39; instrucciones requieren una lista entre paréntesis de los nuevos límites de cada dimensión de la matriz
Debe especificar el nuevo tamaño de una matriz como parte de una instrucción `ReDim` .  
  
 **Identificador de error:** BC30670  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione el tamaño de cada rango de la matriz; por ejemplo:  
  
    ```  
    ReDim arr(5, 6)  
    ```
