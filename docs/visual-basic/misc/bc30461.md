# Los clases no se pueden declarar como &#39;&lt;especificador&gt;&#39;
Ha intentado declarar una `Class` o un `Module` con un especificador no válido.  
  
 **Identificador de error:** BC30461  
  
### Para corregir este error  
  
-   Quite el especificador.
