# <a name="classes-cannot-be-declared-39ltspecifiergt39"></a>No se pueden declarar clases &#39; &lt;especificador&gt;&#39;
Ha intentado declarar una `Class` o un `Module` con un especificador no válido.  
  
 **Identificador de error:** BC30461  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador.
