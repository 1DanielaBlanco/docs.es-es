# &#39;New&#39; no se puede usar en la clase &#39;&lt;nombreDeClase&gt;&#39; porque contiene un miembro &#39;MustOverride&#39; que no est&#225; reemplazado
Intentó usar `New` en una clase que contiene un miembro `MustOverride` que no se reemplazó.  
  
 **Identificador de error:** BC30376  
  
### Para corregir este error  
  
-   Quite la instrucción `New`.
