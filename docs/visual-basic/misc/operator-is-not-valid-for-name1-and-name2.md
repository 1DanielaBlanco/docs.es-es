# <a name="operator-is-not-valid-for-name1-and-name2"></a>Operador no es válido para '\<nombre1 >' y '\<nombre2 >'
Un operador no es adecuado para los tipos en los que opera.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que el operador es adecuado para los tipos en los que opera.
