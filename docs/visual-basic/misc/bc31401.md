# La variable local est&#225;tica &#39;&lt;nombreDeVariable&gt;&#39; ya est&#225; declarada
Ya se declaró otra variable local estática con el mismo nombre.  
  
 **Identificador de error:** BC31401  
  
### Para corregir este error  
  
1.  Quite las declaraciones locales estáticas redundantes.  
  
2.  Asigne un nombre único a cada variable local estática.
