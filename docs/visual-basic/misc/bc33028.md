# <a name="conversion-operators-cannot-convert-to-object"></a>Los operadores de conversión no se pueden convertir a Object.
Un operador de conversión está declarado con un tipo de valor devuelto de [Object Data Type](../../visual-basic/language-reference/data-types/object-data-type.md).  
  
 En el tiempo de compilación, [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] considera que existe una conversión predefinida desde cualquier tipo de referencia a cualquier tipo de su jerarquía de herencia, es decir, cualquier tipo del que deriva o derivado de este. `Object` es el tipo de datos universal de [!INCLUDE[dnprdnshort](~/includes/dnprdnshort-md.md)], por lo que cada tipo se deriva de `Object`.  
  
 Dado que el compilador considera que esta conversión ya está definida, no permite la redefinición.  
  
 **Identificador de error:** BC33028  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la definición de este operador completamente. Ya está predefinido.
