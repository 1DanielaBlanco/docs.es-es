# <a name="project-ltprojectnamegt-cannot-reference-project-ltprojectnamegt-because-ltprojectnamegt-directly-or-indirectly-references-ltprojectnamegt"></a>Proyecto '&lt;projectname&gt;'no puede hacer referencia de proyecto'&lt;projectname&gt;'porque'&lt;projectname&gt;'directa o indirectamente hace referencia a'&lt;projectname &gt;'
Un proyecto incluye una referencia a un segundo proyecto. A su vez, el segundo proyecto incluye una referencia al primer proyecto.  
  
 **Identificador de error:** BC30761  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite una de las referencias.
