# <a name="type-39lttypenamegt39-is-not-defined-or-the-module-containing-it-is-not-loaded-in-the-debugging-session"></a>Tipo de &#39; &lt;typename&gt; &#39; no está definido o el módulo que lo contiene no está cargado en la sesión de depuración
Se ha hecho referencia a un tipo que no está disponible o que no está definido.  
  
 **Identificador de error:** BC30743  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe que el nombre del tipo está escrito correctamente.  
  
2.  Defina el tipo dentro de la clase o módulo actual.
