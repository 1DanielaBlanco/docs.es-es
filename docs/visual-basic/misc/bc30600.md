# <a name="39withevents39-variable-does-not-raise-any-instance-events-that-are-accessible-to-39ltcontainernamegt39"></a>&#39; WithEvents &#39; variable no provoca eventos de instancia que son accesibles para &#39; &lt;containername&gt;&#39;
El objeto al que hace referencia no genera eventos accesibles.  
  
 **Identificador de error:** BC30600  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Modifique los modificadores de acceso para el evento o elija un evento que sea accesible en el contexto actual.
