# <a name="unable-to-open-response-file-ltfilenamegt"></a>No se puede abrir el archivo de respuesta '&lt;filename&gt;'
El archivo que al está intentando tener acceso no está disponible.  
  
 **Identificador de error:** BC2011  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que la conexión de red está todavía activa.  
  
-   Compruebe sus permisos de acceso al archivo.
