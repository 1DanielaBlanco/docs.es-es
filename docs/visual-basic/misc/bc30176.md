# <a name="only-one-of-39public39-39private39-39protected39-39friend39-or-39protected-friend39-can-be-specified"></a>Sólo uno de los &#39; Público &#39; &#39; Privada &#39; &#39; Proteger &#39; &#39; Friend &#39; o &#39; Protected Friend &#39; se puede especificar
La accesibilidad en conflicto se produce en una declaración de elemento. Se puede especificar solo un tipo de acceso.  
  
 **Identificador de error:** BC30176  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Elija un tipo de acceso y quite las demás palabras clave de la declaración.
