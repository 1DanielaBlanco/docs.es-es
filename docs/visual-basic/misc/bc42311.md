# No se puede crear el archivo de documentaci&#243;n XML &#39;&lt;nombre&gt;&#39;: &lt;mensaje&gt;
No se puede crear el archivo de documentación XML debido a un error.  
  
 **Identificador de error:** BC42311  
  
 Los mensajes siguientes se pueden mostrar con este error:  
  
-   [Un bloque de comentario XML no se puede asociar con ningún elemento de lenguaje que admita la aplicación de comentarios de documentación XML](../../visual-basic/misc/bc42312.md).  
  
-   [No se permite la etiqueta de comentario XML 'returns' en una propiedad 'WriteOnly'](../../visual-basic/misc/bc42313.md).  
  
-   [El comentario XML no se puede aplicar más de una vez en un \<tipo\> parcial.](../../visual-basic/misc/bc42314.md).  
  
-   [No se permite la etiqueta de comentario XML 'returns' en un elemento de lenguaje 'declare sub'](../../visual-basic/misc/bc42315.md).  
  
-   [Error de análisis de documentación XML: la etiqueta inicial '\<etiqueta\>' no tiene la correspondiente etiqueta de cierre.](../../visual-basic/misc/bc42316.md).  
  
-   [El parámetro \<parámetro\> de tipo de comentario XML no coincide con un parámetro de tipo de la instrucción \<palabra clave\> correspondiente.](../../visual-basic/misc/bc42317.md).  
  
-   [El parámetro de tipo de comentario XML debe tener un atributo 'name'](../../visual-basic/misc/bc42318.md).  
  
-   [La excepción del comentario XML debe tener un atributo 'cref'](../../visual-basic/language-reference/error-messages/xml-comment-exception-must-have-a-cref-attribute.md).  
  
### Para corregir este error  
  
-   Consulte el mensaje proporcionado para obtener recomendaciones.
