# <a name="39lttypename1gt39-cannot-be-converted-to-39lttypename2gt39"></a>&#39;&lt;NombreTipo1&gt; &#39; no se puede convertir a &#39; &lt;nombredetipo2&gt;&#39;
Se intentó una conversión de tipo entre dos tipos incompatibles.  
  
 **Identificador de error:** BC30741  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe los tipos que se van a convertir.
