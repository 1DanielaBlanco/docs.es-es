# <a name="set-parameter-cannot-be-declared-ltspecifiergt"></a>El parámetro 'Set' no se puede declarar '&lt;especificador&gt;'
Se especificó un modificador no válido con un parámetro en un procedimiento de propiedad `Set` .  
  
 **Identificador de error:** BC31065  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite los modificadores no válidos como, por ejemplo, `ByRef`, `ParamArray`y `Optional`.
