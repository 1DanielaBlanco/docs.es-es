# <a name="operator-39ltoperatornamegt39-is-not-defined-for-types-39lttype1gt39-and-39lttype2gt39"></a>Operador &#39; &lt;operatorname&gt; &#39; no está definido para los tipos de &#39; &lt;type1&gt; &#39; y &#39; &lt;type2&gt;&#39;
Ha intentado utilizar un operador binario para realizar una operación que no es válida para los dos elementos de código que devuelven valores implicados.  
  
 **Id. de error:** BC30452  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe los dos elementos y asegúrese de que sean compatibles.
