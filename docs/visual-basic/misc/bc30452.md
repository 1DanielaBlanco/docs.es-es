# <a name="operator-ltoperatornamegt-is-not-defined-for-types-lttype1gt-and-lttype2gt"></a>Operador '&lt;operatorname&gt;'no está definido para los tipos'&lt;type1&gt;'y'&lt;type2&gt;'
Ha intentado utilizar un operador binario para realizar una operación que no es válida para los dos elementos de código que devuelven valores implicados.  
  
 **Identificador de error:** BC30452  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe los dos elementos y asegúrese de que sean compatibles.
