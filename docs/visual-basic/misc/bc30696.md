# &#39;&lt;membername1&gt;&#39; y &#39;&lt;membername2&gt;&#39; no se pueden sobrecargar el uno al otro, porque solo se diferencian en los tipos de los par&#225;metros opcionales
Los miembros sobrecargados deben tener tipos de datos diferentes para uno o varios parámetros requeridos.  
  
 **Id. de error:** BC30696  
  
### Para corregir este error  
  
-   Defina al menos un parámetro requerido como un tipo de datos diferente.
