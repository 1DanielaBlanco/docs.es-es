# <a name="methods-declared-39overrides39-cannot-be-declared-39overridable39-because-they-are-implicitly-overridable"></a>Los métodos declarados &#39;invalida&#39; no se pueden declarar &#39;Overridable&#39; porque son implícitamente reemplazables
A diferencia de la mayoría de los métodos, los métodos marcados con el modificador `Overrides` se pueden reemplazar de forma predeterminada.  
  
 **Id. de error:** BC30730  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Ignore el modificador `Overridable` de los métodos marcados con el modificador `Overrides` .
