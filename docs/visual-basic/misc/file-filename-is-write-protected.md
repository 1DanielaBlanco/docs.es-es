# <a name="file-ltfilenamegt-is-write-protected"></a>Archivo '&lt;filename&gt;' está protegido contra escritura
El archivo que está intentando escribir está protegido contra escritura y no se puede cambiar.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Cambie los permisos en el archivo a lectura y escritura o escritura en un archivo diferente.
