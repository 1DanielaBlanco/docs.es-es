# <a name="39sub-new39-cannot-be-declared-39ltmodifiergt39"></a>&#39;Sub New&#39; no se pueden declarar &#39; &lt;modificador&gt;&#39;
Ha intentado declarar un elemento `Sub New` con un modificador, como `Overrides`, que no es válido.  
  
 **Id. de error:** BC30364  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el modificador.
