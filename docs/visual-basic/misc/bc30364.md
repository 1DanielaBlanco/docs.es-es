# <a name="sub-new-cannot-be-declared-ltmodifiergt"></a>No se pueden declarar 'Sub New' '&lt;modificador&gt;'
Ha intentado declarar un elemento `Sub New` con un modificador, como `Overrides`, que no es válido.  
  
 **Identificador de error:** BC30364  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el modificador.
