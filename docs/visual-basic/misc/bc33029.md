# <a name="conversion-operators-cannot-convert-from-an-interface-type"></a>Los operadores de conversión no pueden convertir de un tipo de interfaz
Un operador de conversión se declara con un tipo de interfaz para el parámetro.  
  
 En tiempo de compilación, [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] considera una conversión predefinida que existe desde cualquier interfaz a cualquier tipo de referencia. Este tipo de conversión puede producir un error en tiempo de ejecución, pero el compilador no puede predecir los resultados en tiempo de ejecución, por lo que permite que estas conversiones se compilen.  
  
 Dado que el compilador considera que esta conversión ya está definida, no permite la redefinición.  
  
 **Identificador de error:** BC33029  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite completamente esta definición de operador. Ya está predefinido.
