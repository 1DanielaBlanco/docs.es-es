# <a name="39strconvlinguisticcasing39-requires-39strconvlowercase39-or-39strconvuppercase39"></a>&#39; StrConv.LinguisticCasing &#39; requiere &#39; StrConv.LowerCase &#39; o &#39; StrConv.UpperCase &#39;
Ha intentado usar `StrConv.LinguisticCasing`, que solo es válido junto con `StrConv.LowerCase` o `StrConv.UpperCase`.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Use `StrConv.LowerCase` o `StrConv.UpperCase` junto con `StrConv.LinguisticCasing`.
