# <a name="strconvlinguisticcasing-requires-strconvlowercase-or-strconvuppercase"></a>'StrConv.LinguisticCasing' requiere 'StrConv.LowerCase' o 'StrConv.UpperCase'
Ha intentado usar `StrConv.LinguisticCasing`, que solo es válido junto con `StrConv.LowerCase` o `StrConv.UpperCase`.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Use `StrConv.LowerCase` o `StrConv.UpperCase` junto con `StrConv.LinguisticCasing`.
