# <a name="39let39-and-39set39-assignment-statements-are-no-longer-supported"></a>&#39; Permiten &#39; y &#39; Conjunto de &#39; ya no se admiten instrucciones de asignación
No se puede usar`Let` y `Set` en asignaciones.  
  
 **Identificador de error:** BC30807  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Omita `Let` y `Set` en las instrucciones de asignación; por ejemplo:  
  
     `X = 2`
