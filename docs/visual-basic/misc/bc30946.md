# <a name="stop-request-is-pending"></a>Hay una solicitud de detención pendiente.
En el depurador de Visual Studio, una expresión especifica una llamada a procedimiento, pero hay una solicitud de detención del subproceso.  
  
 El depurador no intenta llamar a un procedimiento de un subproceso que no está activo.  
  
 **Identificador de error:** BC30946  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Si es posible, determine el origen de la solicitud para detener el subproceso, para poder evitar que vuelva a producirse.  
  
-   Finalice y reinicie la depuración.
