# <a name="lib-expected"></a>Se esperaba 'Lib'
Una instrucción `Declare` especifica un nombre de biblioteca sin usar la palabra clave `Lib` .  
  
 **Identificador de error:** BC30218  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue la palabra clave `Lib` antes del nombre de la biblioteca.
