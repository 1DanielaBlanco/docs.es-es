# <a name="file-ltfilenamegt-cannot-be-deleted-because-it-is-open"></a>Archivo '&lt;filename&gt;' no se puede eliminar porque está abierto
Se ha intentado una eliminación en un archivo abierto.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cierre el archivo antes de eliminarlo.
