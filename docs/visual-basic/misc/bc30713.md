# <a name="39exit39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Salir &#39; instrucciones no son válidas en la ventana Inmediato.
Las instrucciones`Exit` solo pueden usarse en el código fuente.  
  
 **Id. de error:** BC30713  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `Exit` del código de depuración.
