# <a name="shared-cannot-be-combined-with-ltspecifiergt-on-a-property-declaration"></a>'Shared' no se puede combinar con '&lt;especificador&gt;' en una declaración de propiedad
Ha intentado combinar el modificador `Shared` con un especificador que no es válido en una combinación de este tipo en una declaración de propiedad.  
  
 **Identificador de error:** BC30502  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador.
