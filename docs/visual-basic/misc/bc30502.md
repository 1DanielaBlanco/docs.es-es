# <a name="39shared39-cannot-be-combined-with-39ltspecifiergt39-on-a-property-declaration"></a>&#39; Compartido &#39; no se puede combinar con &#39; &lt;especificador&gt;&#39; en una declaración de propiedad
Ha intentado combinar el modificador `Shared` con un especificador que no es válido en una combinación de este tipo en una declaración de propiedad.  
  
 **Identificador de error:** BC30502  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador.
