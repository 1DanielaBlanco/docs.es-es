# <a name="array-initializer-is-missing-ltnumbergt-elements"></a>Falta el inicializador de matriz &lt;número&gt; elementos
No ha facilitado suficientes elementos en el inicializador de la matriz.  
  
 **Identificador de error:** BC30567  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Facilite el argumento o los argumentos que faltan.  
  
-   Use `ReDim` para cambiar el tamaño de la matriz.
