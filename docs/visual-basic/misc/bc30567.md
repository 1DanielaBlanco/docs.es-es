# Faltan &lt;number&gt; elementos en el inicializador de matriz
No ha facilitado suficientes elementos en el inicializador de la matriz.  
  
 **Id. de error:** BC30567  
  
### Para corregir este error  
  
-   Facilite el argumento o los argumentos que faltan.  
  
-   Use `ReDim` para cambiar el tamaño de la matriz.
