# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<nombre >' no puede exponer el tipo \<type1 > en \<especificador > \<type2 > '\<typename >' a través de \<nombre >
Ha intentado exponer un tipo privado fuera de la clase pública `MyClass`.  
  
 **Identificador de error:** BC30508  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare el tipo `Public`.
