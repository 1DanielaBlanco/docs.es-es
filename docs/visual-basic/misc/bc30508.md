# <a name="ltnamegt-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-lttypenamegt-through-ltnamegt"></a>'&lt;nombre&gt;' no puede exponer el tipo &lt;type1&gt; en &lt;especificador&gt; &lt;type2&gt; '&lt;typename&gt;' a través de &lt;nombre&gt;
Ha intentado exponer un tipo privado fuera de la clase pública `MyClass`.  
  
 **Identificador de error:** BC30508  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare el tipo `Public`.
