# <a name="39ltnamegt39-cannot-expose-type-lttype1gt-in-ltspecifiergt-lttype2gt-39lttypenamegt39-through-ltnamegt"></a>&#39;&lt;nombre&gt; &#39; no se puede exponer el tipo &lt;type1&gt; en &lt;especificador&gt; &lt;type2&gt; &#39; &lt;typename&gt; &#39; a través de &lt;nombre&gt;
Ha intentado exponer un tipo privado fuera de la clase pública `MyClass`.  
  
 **Id. de error:** BC30508  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare el tipo `Public`.
