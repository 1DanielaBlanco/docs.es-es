# <a name="unrecognized-option-optionname-ignored"></a>Opción no reconocida \<optionname >; se ha omitido
Se ha especificado una opción que no es válida desde la línea de comandos.  
  
 **Identificador de error:** BC2007  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe las opciones especificadas para encontrar la que no es válida.
