# <a name="39custom39-modifier-can-only-be-used-immediately-before-an-39event39-declaration"></a>&#39;Custom&#39; modificador solo se puede usar inmediatamente antes un &#39;eventos&#39; declaración
La palabra clave `Custom` debe preceder solo a la palabra clave `Event` en una declaración de evento personalizado.  
  
 **Identificador de error:** BC31140  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Mueva la palabra clave `Custom` para que preceda a la palabra clave `Event` .  
  
     -O bien-  
  
-   Elimine la palabra clave `Custom` .
