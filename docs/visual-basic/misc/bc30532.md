# <a name="conversion-from-39date39-to-39double39-requires-calling-the-39datetooadate39-method"></a>Conversión de &#39; fecha &#39; a &#39; doble &#39; requiere llamar a la &#39; fecha. ToOADate &#39; método
Ha intentado convertir una `Date` valor a un `Double` valor, que no se puede realizar sin usar el <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> método.  
  
 **Identificador de error:** BC30532  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use la <xref:System.DateTime.ToOADate%2A?displayProperty=nameWithType> método para convertir el valor.
