# <a name="lttype1gt-lttypenamegt-and-lttype2gt-lttypenamegt-declared-in-ltnamegt-conflict-in-ltnamespacegt-ltnamespacenamegt"></a>&lt;tipo1&gt; '&lt;typename&gt;' y &lt;type2&gt; '&lt;typename&gt;', declarados en'&lt;nombre&gt;', entran en conflicto en &lt;espacio de nombres &gt; '&lt;namespacename&gt;'
Dos elementos declarados, como una clase y un módulo, tienen el mismo nombre en el mismo espacio de nombres.  
  
 **Identificador de error:** BC30175  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Mueva la declaración de uno de los elementos a otro espacio de nombres.  
  
-   Cambie el nombre de uno de los elementos.
