# <a name="event-lteventnamegt-cannot-be-found"></a>Evento '&lt;eventname&gt;' no se encuentra
Ha intentado controlar un evento que el compilador no puede encontrar.  
  
 **Identificador de error:** BC30590  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que se ha escrito correctamente el nombre del evento.
