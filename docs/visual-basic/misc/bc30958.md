# <a name="object-created-by-edit-and-continue-is-not-yet-available-for-evaluation"></a>El objeto creado por Edit y Continue todavía no está disponible para la evaluación
En el depurador de Visual Studio, se crea un nuevo objeto, pero todavía no se puede tener acceso a él.  
  
 Las causas posibles de la falta de disponibilidad de un objeto incluyen el error de asignación de memoria y el estado no sincronizado del objeto. Estas son condiciones internas temporales y no suelen ser el resultado de la entrada en el depurador.  
  
 **Identificador de error:** BC30958  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Finalice y reinicie la depuración.
