# <a name="argument-39ltargumentnamegt39-is-nothing-or-empty"></a>Argumento &#39; &lt;argumentname&gt;&#39; tiene un valor o está vacío
Un nombre de argumento no es válido porque se inicializó en `Nothing` o está vacío. Un argumento debe tener un tipo de datos específico y un nombre válido.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue un nombre de argumento válido.
