# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-sub-function-or-operator-with-a-non-empty-body"></a>&#39; System.Runtime.InteropServices.DllImportAttribute &#39; no se puede aplicar a un Sub, Function u Operator con un cuerpo no vacío
El atributo `DllImportAttribute` se aplicó a un `Sub` `Function`, o `Operator` que no está vacío.  
  
 **Identificador de error:** BC31522  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite todo el código de `Sub`, `Function`o `Operator` para usar este atributo.
