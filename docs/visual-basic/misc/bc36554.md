## <a name="to-correct-this-error"></a><span data-ttu-id="8193f-111">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="8193f-111">To correct this error</span></span>  
  
-   <span data-ttu-id="8193f-112">Si la matriz de parámetros no especifica el tipo de datos que desea extender, agregue un nuevo primer parámetro que especifique este tipo.</span><span class="sxs-lookup"><span data-stu-id="8193f-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```  
    <Extension()>  
    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
-   <span data-ttu-id="8193f-113">Si la matriz de parámetros especifica el tipo de datos que desea extender, considere la posibilidad de cambiar a una matriz normal, que requiere un argumento en lugar de una matriz de parámetros.</span><span class="sxs-lookup"><span data-stu-id="8193f-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="8193f-114">Las matrices normales se pueden extender.</span><span class="sxs-lookup"><span data-stu-id="8193f-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>  
    Public Function Sum(ByVal ints() As Integer) As Integer  
        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```
