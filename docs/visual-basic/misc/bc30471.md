# La expresi&#243;n no es una matriz ni un m&#233;todo y no puede tener una lista de argumentos
Ha facilitado una lista de argumentos para una expresión que no es una matriz ni un método. Solo las matrices y los métodos pueden tener listas de argumentos.  
  
 **Id. de error:** BC30471  
  
### Para corregir este error  
  
-   Quite la lista de argumentos.
