# <a name="variable-cannot-be-initialized-with-non-array-type-ltelementnamegt"></a>No se puede inicializar la variable con tipo que no son de matriz '&lt;elementname&gt;'
Una variable que se declara como una matriz se debe inicializar con un valor de matriz.  
  
```  
' Not valid.  
' The following line causes this error when executed with Option Strict off.  
' Dim arrayVar1() = 10  
```  
  
 **Identificador de error:** BC36536  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Inicialice la variable de matriz con un valor de matriz:  
  
    ```
    ' With Option Strict off.  
    Dim arrayVar2() = {1, 2, 3}  
    ' With Option Strict on.  
    Dim arrayVar3() As Integer = {1, 2, 3}  
    ```
