# <a name="39addressof39-expressions-are-not-valid-in-debug-windows"></a>&#39; AddressOf &#39; expresiones no son válidas en ventanas de depuración.
Las instrucciones`AddressOf` solo pueden usarse en el código fuente.  
  
 **Identificador de error:** BC30731  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite las instrucciones `AddressOf` del código de depuración.
