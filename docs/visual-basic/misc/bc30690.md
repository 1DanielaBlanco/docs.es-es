# <a name="structure-ltstructurenamegt-cannot-be-indexed-because-it-has-no-default-property"></a>Estructura '&lt;structurename&gt;' no se puede indizar porque no tiene ninguna propiedad predeterminada
Se intentó usar la sintaxis de la propiedad predeterminada con una estructura que no tiene ninguna propiedad predeterminada.  
  
 **Identificador de error:** BC30690  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use la sintaxis de propiedades estándar para tener acceso a las propiedades de la estructura.
