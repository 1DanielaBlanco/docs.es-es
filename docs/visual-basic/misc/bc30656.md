# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Campo &#39; &lt;fieldname&gt;&#39; es un tipo no admitido
Un campo está declarado como un tipo que no se puede representar por [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
 **Identificador de error:** BC30656  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Defina el campo como un tipo compatible.  
  
2.  Desarrolle un nuevo tipo que describa los datos en el campo.
