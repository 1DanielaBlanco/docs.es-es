# <a name="field-39ltfieldnamegt39-is-of-an-unsupported-type"></a>Campo '&lt;fieldname&gt;' es de un tipo no admitido
Un campo se declara como un tipo que no se puede representar por [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
 **Identificador de error:** BC30656  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Defina el campo como un tipo compatible.  
  
2.  Desarrolle un nuevo tipo que describa los datos en el campo.
