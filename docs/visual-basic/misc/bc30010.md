# <a name="reference-required-to-module-ltmodulenamegt-containing-the-implemented-interface-ltinterfacenamegt"></a>Necesaria una referencia al módulo '&lt;modulename&gt;'que contiene la interfaz implementada'&lt;interfacename&gt;'
Necesaria una referencia al módulo '\<NombreMódulo >' que contiene la interfaz implementada '\<interfacename >'. Agregue una al proyecto.  
  
 La interfaz está definida en un módulo al que no se hace referencia directamente en el proyecto. El compilador de Visual Basic requiere una referencia para evitar la ambigüedad en caso de la interfaz se define en más de un módulo.  
  
 **Identificador de error:** BC30010  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Incluya el nombre del módulo al que no se hace referencia en las referencias del proyecto.
