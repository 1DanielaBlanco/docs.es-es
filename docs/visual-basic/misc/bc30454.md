# <a name="expression-is-not-a-method"></a>La expresión no es un método.
El destino de la llamada de método no es un método.  
  
 **Identificador de error:** BC30454  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que no escribe incorrectamente el nombre del método.
