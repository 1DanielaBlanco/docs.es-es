# <a name="option-ltoptionnamegt-requires-ltargumentgt"></a>Opción &lt;optionname&gt; requiere &lt;argumento&gt;
Se ha omitido un argumento requerido.  
  
 **Identificador de error:** BC2006  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Facilite el argumento que falta.
