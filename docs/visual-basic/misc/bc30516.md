# <a name="overload-resolution-failed-because-no-accessible-39ltmethodgt39-accepts-this-number-of-arguments"></a>Error de resolución de sobrecarga no accesible &#39; &lt;método&gt; &#39; tiene acceso acepta este número de argumentos
Ha realizado una llamada a un método sobrecargado, pero el compilador no encuentra ningún método con una lista de argumentos coincidente.  
  
 **Identificador de error:** BC30516  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe la lista de argumentos proporcionada.
