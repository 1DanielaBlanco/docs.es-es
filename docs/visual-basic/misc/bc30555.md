# <a name="default-member-ltmembernamegt-is-not-a-property"></a>Miembro predeterminado '&lt;membername&gt;' no es una propiedad
Intentó declarar un miembro `Default`. Sin embargo, la palabra clave `Default` se usa para identificar propiedades predeterminadas.  
  
 **Identificador de error:** BC30555  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe que está identificando una propiedad con `Default`, o bien quite la palabra clave `Default` .
