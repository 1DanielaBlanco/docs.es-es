# <a name="39systemruntimeinteropservicesdllimportattribute39-cannot-be-applied-to-a-39get39-or-39set39"></a>&#39; System.Runtime.InteropServices.DllImportAttribute &#39; no se puede aplicar a un &#39; Get &#39; o &#39; Conjunto de &#39;
El atributo `DllImportAttribute` se aplicó a un procedimiento de propiedad `Get` o `Set` .  
  
 **Id. de error:** BC31524  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite `DllImportAttribute` de los procedimientos de propiedad `Get` y `Set` .
