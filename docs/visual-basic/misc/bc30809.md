# <a name="39wend39-statements-are-no-longer-supported-use-39end-while39-statements-instead"></a>&#39; Wend &#39; ya no se admiten instrucciones; Usar &#39; End While &#39; las instrucciones en su lugar
La construcción `End While` reemplaza a la palabra clave `Wend` .  
  
 **Identificador de error:** BC30809  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use `End While` para finalizar un bloque `While` .
