# <a name="type-of-39ltvariablenamegt39-cannot-be-inferred-from-an-expression-containing-39ltvariablenamegt39"></a><span data-ttu-id="fd7c9-102">Tipo de &#39; &lt;variablename&gt;&#39; no se pueden inferir a partir de una expresión que contenga &#39;&lt; VariableName&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="fd7c9-102">Type of &#39;&lt;variablename&gt;&#39; cannot be inferred from an expression containing &#39;&lt;variablename&gt;&#39;</span></span>
<span data-ttu-id="fd7c9-103">El compilador no puede inferir el tipo de datos de una variable si la variable se usa para establecer su valor inicial en la declaración.</span><span class="sxs-lookup"><span data-stu-id="fd7c9-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="fd7c9-104">Por ejemplo, si se establece `Option Infer` en `On`, no se compilan los ejemplos siguientes:</span><span class="sxs-lookup"><span data-stu-id="fd7c9-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
-   <span data-ttu-id="fd7c9-105">Declaraciones</span><span class="sxs-lookup"><span data-stu-id="fd7c9-105">Declarations</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
-   <span data-ttu-id="fd7c9-106">Bucle`For` </span><span class="sxs-lookup"><span data-stu-id="fd7c9-106">`For` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
-   <span data-ttu-id="fd7c9-107">Bucle`For Each` </span><span class="sxs-lookup"><span data-stu-id="fd7c9-107">`For Each` loop</span></span>  
  
    ```  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="fd7c9-108">**Identificador de error:** BC30980</span><span class="sxs-lookup"><span data-stu-id="fd7c9-108">**Error ID:** BC30980</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fd7c9-109">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="fd7c9-109">To correct this error</span></span>  
  
-   <span data-ttu-id="fd7c9-110">Si las dos variables estaban previstas para hacer referencia a valores diferentes, cambie el nombre de la variable que declara.</span><span class="sxs-lookup"><span data-stu-id="fd7c9-110">If the two variables were intended to refer to different values, change the name of the variable that you are declaring.</span></span>  
  
-   <span data-ttu-id="fd7c9-111">Use un valor literal en lugar del nombre de variable en el valor inicial, en el lado derecho de la asignación.</span><span class="sxs-lookup"><span data-stu-id="fd7c9-111">Use a literal value instead of the variable name in the initial value, on the right side of the assignment.</span></span>  
  
-   <span data-ttu-id="fd7c9-112">Use una cláusula `As` para especificar el tipo de la variable que declara.</span><span class="sxs-lookup"><span data-stu-id="fd7c9-112">Use an `As` clause to specify the type of the variable you are declaring.</span></span>
