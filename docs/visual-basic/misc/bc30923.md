# <a name="ltnamegt-cannot-expose-type-lttypegt-used-in-the-underlying-delegate-type-ltdelegatetypegt-of-the-event-it-is-implementing-to-ltspecifier1gt-lttype1gt-through-ltspecifier2gt-lttype2gt"></a>'&lt;nombre&gt;'no puede exponer el tipo'&lt;tipo&gt;'utilizado en el tipo delegado subyacente'&lt;delegatetype&gt;' del evento que implementa en &lt;specifier1&gt; '&lt;type1&gt;' a través de &lt;specifier2&gt; '&lt;type2&gt;'
El código expone un tipo privado fuera de la clase pública.  
  
 **Identificador de error:** BC30923  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare el tipo `Public` o use otro tipo.
