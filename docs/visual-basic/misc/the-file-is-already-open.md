# El archivo ya est&#225; abierto
Se intentó abrir un archivo que ya está abierto.  
  
### Para corregir este error  
  
-   Cierre el archivo antes de intentar abrirlo.  
  
     O bien  
  
-   Quite la instrucción `FileOpen` redundante del código.
