# <a name="39ltnamegt39-is-already-declared-as-a-parameter-of-this-method"></a>&#39; &lt;nombre&gt;&#39; ya se declaró como un parámetro de este método
El nombre de parámetro especificado ya se ha declarado para este método.  
  
 **Identificador de error:** BC30734  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el nombre del parámetro para que sea único.
