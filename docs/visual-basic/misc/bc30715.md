# <a name="39catch39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Catch &#39; instrucciones no son válidas en la ventana Inmediato.
Las instrucciones`Catch` solo pueden usarse en el código fuente.  
  
 **Id. de error:** BC30715  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `Catch` del código de depuración.
