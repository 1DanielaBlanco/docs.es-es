# <a name="expected-beginning-39lt39-for-an-xml-tag"></a>Se esperaba la apertura &#39; &lt; &#39; para una etiqueta XML
A la etiqueta XML de un literal XML le falta el carácter de apertura '<' necesario.  
  
 **Identificador de error:** BC31165  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue el carácter '<' al principio de la etiqueta XML para el literal XML.
