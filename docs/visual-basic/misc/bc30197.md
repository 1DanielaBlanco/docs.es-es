# <a name="39as39-expected"></a>&#39;Como&#39; esperado
Se produce una declaración que no proporciona una cláusula `As` para uno o varios de los elementos declarados.  
  
 **Identificador de error:** BC30197  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Facilite la cláusula `As` según sea necesario o bien active `Option Strict Off`.
