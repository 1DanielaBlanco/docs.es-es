# <a name="properties-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>Las propiedades de un módulo no se pueden declarar &#39; &lt;especificador&gt;&#39;
Ha intentado usar un especificador como, por ejemplo, `MustOverride`, `NotOverridable`o `Overridable`, en una propiedad de un módulo.  
  
 **Identificador de error:** BC30503  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador.
