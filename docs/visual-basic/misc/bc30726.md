# <a name="39with39-contexts-and-statements-are-not-valid-in-debug-windows"></a>&#39;Con&#39; contextos e instrucciones no son válidas en ventanas de depuración.
Las instrucciones`With` solo pueden usarse en el código fuente.  
  
 **Identificador de error:** BC30726  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `With` del código de depuración.
