# No se permite el car&#225;cter &#39;character&#39; (0x&lt;hexadecimal&gt;) en un literal XML
Un carácter no admitido se usó en un literal XML.  
  
 **Identificador de error:** BC31176  
  
### Para corregir este error  
  
-   Quite el carácter no admitido.
