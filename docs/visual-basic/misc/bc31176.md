# <a name="character-character-0xlthexadecimalgt-is-not-allowed-in-xml-literal"></a>Carácter 'carácter' (0 x&lt;hexadecimal&gt;) no se permite en XML literal
Un carácter no admitido se usó en un literal XML.  
  
 **Identificador de error:** BC31176  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el carácter no admitido.
