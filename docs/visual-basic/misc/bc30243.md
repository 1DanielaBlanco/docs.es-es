# <a name="ltmodifiergt-is-not-valid-on-an-event-declaration"></a>'&lt;modificador&gt;' no es válido en una declaración de evento
Una instrucción `Event` contiene una palabra clave no válida, como `ReadOnly`.  
  
 **Identificador de error:** BC30243  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite la palabra clave no válida de la instrucción `Event` .
