 <span data-ttu-id="56199-104">Cuando se divide la definición de una clase en varias declaraciones parciales, el compilador trata a la clase como la unión de todas sus declaraciones parciales.</span><span class="sxs-lookup"><span data-stu-id="56199-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="56199-105">Esto se aplica no solo a los miembros, sino también a la implementación, la herencia y el nivel de acceso.</span><span class="sxs-lookup"><span data-stu-id="56199-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="56199-106">Para invalidar un procedimiento o una propiedad, una clase debe heredar dicho procedimiento o propiedad de una clase base.</span><span class="sxs-lookup"><span data-stu-id="56199-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="56199-107">Por lo tanto, para especificar `MustOverride` para un procedimiento o una propiedad en una clase base, se debe especificar `MustInherit` para la clase.</span><span class="sxs-lookup"><span data-stu-id="56199-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="56199-108">Dado que son mutuamente contradictorios, no es posible especificar ambos, `MustInherit` y `NotInheritable` , para la misma clase.</span><span class="sxs-lookup"><span data-stu-id="56199-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>
