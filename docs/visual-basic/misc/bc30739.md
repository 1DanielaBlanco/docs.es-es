# <a name="39raiseevent39-statements-are-not-valid-in-the-immediate-window"></a>&#39; RaiseEvent &#39; instrucciones no son válidas en la ventana Inmediato.
Las instrucciones`RaiseEvent` no se pueden usar en la ventana **Inmediato** .  
  
 **Identificador de error:** BC30739  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Pruebe las instrucciones `RaiseEvent` desde el código del **Editor de código**.
