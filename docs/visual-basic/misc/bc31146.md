# <a name="xml-name-expected"></a>Se esperaba un nombre XML
Se especificó un literal XML o una propiedad de eje XML sin nombre.  
  
 **Identificador de error:** BC31146  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue un nombre para el literal XML o la propiedad de eje XML.
