# <a name="property-ltpropertynamegt-with-no-parameters-cannot-be-found"></a>Propiedad '&lt;propertyname&gt;' sin parámetros no se encuentra
El atributo usado tiene una propiedad que necesita uno o más parámetros.  
  
 **Identificador de error:** BC30658  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Especifique los parámetros para la propiedad.
