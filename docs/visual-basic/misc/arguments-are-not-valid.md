# <a name="arguments-are-not-valid"></a>Los argumentos no son válidos.
Se han pasado argumentos que no son válidos para una función o subrutina.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe los argumentos para determinar por qué no son válidos.  
  
2.  Si se facilitan argumentos mediante un cálculo, compruebe los valores que procesa el cálculo.
