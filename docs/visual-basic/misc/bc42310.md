# La etiqueta de comentario XML &#39;include&#39; debe tener atributo &#39;&lt;atributo&gt;&#39;.
La etiqueta de comentario XML 'include' debe tener atributo \<atributo\>. Se omitirá el comentario XML.  
  
 Falta un atributo `file` o `path`, pasado a la etiqueta `<`include`>` o está incompleto.  
  
 **Identificador de error:** BC42310  
  
### Para corregir este error  
  
-   Agregue o complete el atributo necesario en la etiqueta \<include\>.
