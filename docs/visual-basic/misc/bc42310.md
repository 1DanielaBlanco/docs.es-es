# <a name="xml-comment-tag-39include39-must-have-a-39ltattributegt39-attribute"></a>Etiqueta de comentario XML &#39;incluyen&#39; debe tener un &#39; &lt;atributo&gt; &#39; atributo
Etiqueta de comentario XML 'include' debe tener un \<atributo > atributos. Se omitirá el comentario XML.  
  
 Falta un atributo `file` o `path` , pasado a la etiqueta `<`include`>` o está incompleto.  
  
 **Identificador de error:** BC42310  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue o complete el atributo necesario en la \<incluyen > etiqueta.
