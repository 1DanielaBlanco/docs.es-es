# <a name="xml-comment-tag-include-must-have-a-ltattributegt-attribute"></a>Etiqueta de comentario XML 'include' debe tener un '&lt;atributo&gt;' atributo
Etiqueta de comentario XML 'include' debe tener un \<atributo > atributos. Se omitirá el comentario XML.  
  
 Falta un atributo `file` o `path` , pasado a la etiqueta `<`include`>` o está incompleto.  
  
 **Identificador de error:** BC42310  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue o complete el atributo necesario en el \<incluyen > etiqueta.
