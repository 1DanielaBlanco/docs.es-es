# <a name="ltnamegt-is-already-declared-as-ltdeclarationgt-in-this-ltdeclarationspacegt"></a>'&lt;nombre&gt;'ya se declaró como'&lt;declaración&gt;' en este &lt;declarationspace&gt;
Se han declarado dos elementos de programación con el mismo nombre en el mismo espacio de declaración (una clase, un módulo, una interfaz o una estructura).  
  
 **Identificador de error:** BC30260  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Mueva una de las declaraciones a un espacio de declaración diferente o cambie el nombre de uno de los elementos declarados.
