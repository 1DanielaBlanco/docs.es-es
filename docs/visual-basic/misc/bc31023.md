# <a name="unable-to-parse-xml-lterrorgt"></a>No se puede analizar XML: &lt;error&gt;
Existe un error en el archivo XML que se está usando.  
  
 **Identificador de error:** BC31023  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe que el código XML esté bien formado, que sea válido y que el archivo no esté dañado.
