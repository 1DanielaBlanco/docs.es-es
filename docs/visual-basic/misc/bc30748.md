# <a name="cannot-convert-to-lttypegt"></a>No se puede convertir a '&lt;tipo&gt;'
Se intentó realizar una conversión no válida.  
  
 **Identificador de error:** BC30748  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Elija un tipo de conversión compatible.
