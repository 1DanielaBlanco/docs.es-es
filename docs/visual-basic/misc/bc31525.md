# El tipo &#39;&lt;typename&gt;&#39; se importa de diferentes versiones de ensamblado &#39;&lt;assemblyname&gt;&#39;
El tipo '\<typename\>' se importa de diferentes versiones de ensamblado '\<assemblyname\>'. No se pueden usar distintas versiones del mismo ensamblado en el mismo proyecto.  
  
 Se importó un tipo de diferentes versiones del mismo ensamblado, lo que crea ambigüedad.  
  
 **Identificador de error:** BC31525  
  
### Para corregir este error  
  
-   Quite las referencias a varias versiones del mismo ensamblado.
