# <a name="type-39lttypenamegt39-is-imported-from-different-versions-of-assembly-39ltassemblynamegt39"></a>Tipo '&lt;typename&gt;'se importa de diferentes versiones de ensamblado'&lt;assemblyname&gt;'
Tipo '\<typename >' se importa de diferentes versiones de ensamblado '\<assemblyname >'. No se pueden usar distintas versiones del mismo ensamblado en el mismo proyecto.  
  
 Se importó un tipo de diferentes versiones del mismo ensamblado, lo que crea ambigüedad.  
  
 **Identificador de error:** BC31525  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las referencias a varias versiones del mismo ensamblado.
