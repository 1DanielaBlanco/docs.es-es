# <a name="39type39-statements-are-no-longer-supported-use-39structure39-statements-instead"></a>&#39;Tipo&#39; ya no se admiten instrucciones; usar &#39;estructura&#39; instrucciones en su lugar
La palabra clave `Type` ya no puede usarse para definir los tipos definidos por el usuario.  
  
 **Identificador de error:** BC30802  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use la instrucción `Structure` para definir un nuevo tipo.
