# No se puede calcular el n&#250;mero de per&#237;odos con los argumentos proporcionados.
Una llamada a la función `NPer`no contiene todos los argumentos necesarios.  
  
### Para corregir este error  
  
-   Asegúrese de que los valores `Rate`, `Prnt` y `PV` están incluidos en la llamada de función.
