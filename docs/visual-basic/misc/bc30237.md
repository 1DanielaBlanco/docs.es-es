# <a name="parameter-already-declared-with-name-39ltargumentnamegt39"></a>Ya se declaró el parámetro con nombre &#39; &lt;argumentname&gt;&#39;
Una declaración de procedimiento define dos argumentos con el mismo nombre. Todos los nombres de argumento de una declaración de procedimiento deben ser únicos.  
  
 **Identificador de error:** BC30237  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el nombre de uno de los argumentos.
