# <a name="39ltimplementsclausegt39-cannot-implement-39lttypenamegt39-because-39lttypenamegt39-is-a-reserved-name"></a>&#39;&lt;implementsclause&gt; &#39; no se puede implementar &#39; &lt;typename&gt; &#39; porque &#39; &lt;typename&gt; &#39; es un nombre reservado
Una instrucción `Implements` especificó un nombre reservado.  
  
 **Id. de error:** BC31415  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el nombre del miembro de interfaz que no se puede implementar.
