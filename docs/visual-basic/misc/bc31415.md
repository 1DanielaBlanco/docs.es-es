# <a name="ltimplementsclausegt-cannot-implement-lttypenamegt-because-lttypenamegt-is-a-reserved-name"></a>'&lt;implementsclause&gt;'no puede implementar'&lt;typename&gt;'porque'&lt;typename&gt;' es un nombre reservado
Una instrucción `Implements` especificó un nombre reservado.  
  
 **Identificador de error:** BC31415  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el nombre del miembro de interfaz que no se puede implementar.
