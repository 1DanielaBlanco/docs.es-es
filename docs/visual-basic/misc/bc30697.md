# <a name="39ltmembername1gt39-cannot-override-39ltmembername2gt39-because-they-differ-by-the-types-of-optional-parameters"></a>&#39; &lt;nombredemiembro1&gt;&#39; no se puede invalidar &#39;&lt; nombredemiembro2&gt;&#39; porque se diferencian en los tipos de parámetros opcionales
Los miembros sobrecargados deben tener tipos de datos diferentes para uno o varios parámetros requeridos.  
  
 **Identificador de error:** BC30697  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Defina al menos un parámetro requerido como un tipo de datos diferente.
