# <a name="cannot-inherit-interface-ltinterfacename1gt-because-it-could-be-identical-to-interface-ltinterfacename2gt-from-which-the-interface-ltinterfacename3gt-inherits-for-some-type-arguments"></a><span data-ttu-id="0a046-102">No se puede heredar la interfaz '&lt;interfacename1&gt;'porque podría ser idéntica a la interfaz'&lt;interfacename2&gt;' desde el que la interfaz '&lt;nombreinterfaz3&gt;' hereda para algunos argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="0a046-102">Cannot inherit interface '&lt;interfacename1&gt;' because it could be identical to interface '&lt;interfacename2&gt;' from which the interface '&lt;interfacename3&gt;' inherits for some type arguments</span></span>
<span data-ttu-id="0a046-103">Una interfaz genérica hereda de dos o más interfaces genéricas, y dos de las herencias podrían entrar en conflicto para ciertos valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="0a046-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="0a046-104">Las instrucciones siguientes pueden generar este error.</span><span class="sxs-lookup"><span data-stu-id="0a046-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="0a046-105">Si `derivedInterface` se construye o se implementa al proporcionar el mismo tipo tanto a `t1` como a `t2`, debe heredar dos versiones de `interfaceA` con argumentos de tipo idénticos.</span><span class="sxs-lookup"><span data-stu-id="0a046-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="0a046-106">Esto produciría ambigüedad sobre a qué versión se debe acceder.</span><span class="sxs-lookup"><span data-stu-id="0a046-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="0a046-107">**Identificador de error:** BC32123</span><span class="sxs-lookup"><span data-stu-id="0a046-107">**Error ID:** BC32123</span></span>
