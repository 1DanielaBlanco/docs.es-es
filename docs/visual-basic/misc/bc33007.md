# <a name="39end-operator39-must-be-preceded-by-a-matching-39operator39"></a>&#39; End Operator &#39; debe ir precedida de una coincidencia &#39; operador &#39;
Aparece una instrucción `End Operator` sin la correspondiente declaración `Operator` delante de ella.  
  
 **Identificador de error:** BC33007  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la instrucción `End Operator` si es redundante.  
  
-   Facilite el procedimiento `Operator` si falta alguno.  
  
-   Traslade la instrucción `End Operator` al lugar adecuado del código.
