# <a name="reference-required-to-assembly-39ltassemblynamegt39-containing-the-type-39ltmembernamegt39"></a>Necesaria una referencia al ensamblado &#39; &lt;assemblyname&gt;&#39; que contiene el tipo de &#39;&lt; MemberName&gt;&#39;
Necesaria una referencia al ensamblado '\<assemblyname >' que contiene el tipo '\<nombreDeMiembro >'. Agregue una al proyecto.  
  
 El proyecto contiene un tipo que depende de otro tipo de un ensamblado que no está incluido en las referencias del proyecto.  
  
 **Identificador de error:** BC30652  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue una referencia de proyecto para los tipos dependientes en el proyecto.
