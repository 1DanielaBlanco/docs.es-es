# No se puede cargar la biblioteca a la que se hace referencia &#39;&lt;nombreArchivo&#39;: &lt;error&gt;
No se pudo cargar una biblioteca a la que este proyecto hace referencia.  
  
 **Identificador de error:** BC31011  
  
### Para corregir este error  
  
1.  Compruebe que el archivo está en la ubicación especificada y no está bloqueado por otra aplicación.
