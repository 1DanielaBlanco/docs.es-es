# <a name="unable-to-load-referenced-library-39ltfilenamegt39-lterrorgt"></a>No se puede cargar la biblioteca que se hace referencia &#39; &lt;filename&gt;&#39;: &lt;error&gt;
No se pudo cargar una biblioteca a la que este proyecto hace referencia.  
  
 **Identificador de error:** BC31011  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe que el archivo está en la ubicación especificada y no está bloqueado por otra aplicación.
