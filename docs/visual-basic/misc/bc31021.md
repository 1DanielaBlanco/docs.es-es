# <a name="visual-basic-compiler-is-unable-to-recover-from-the-following-error-lterrorgt"></a>Compilador de Visual Basic es no se puede recuperar del error siguiente: &lt;error&gt;
El [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador es no se puede recuperar del error siguiente: `error`>.  
  
 Se han deshabilitado algunas características del IDE.  
  
 Se ha producido un error interno en el [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador.  
  
 **Identificador de error:** BC31021  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Guarde su trabajo y reinicie [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
2.  Si el error se repite, reinstale [!INCLUDE[vsprvs](../../csharp/includes/vsprvs_md.md)].  
  
3.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
