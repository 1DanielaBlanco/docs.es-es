# Las expresiones de consulta no son v&#225;lidas en ventanas de depuraci&#243;n.
Se ha escrito una consulta LINQ en la ventana **Inspección** o **Inmediato** durante una sesión de depuración. Las consultas no están disponibles en ventanas de depuración.  
  
 **Identificador de error:** BC36609
