# <a name="ltstatementnamegt-statement-requires-an-array"></a>'&lt;nombredeinstrucción&gt;' instrucción requiere una matriz
Las instrucciones `Erase` y `ReDim` requieren tipos de datos de matriz. El tipo proporcionado no es una matriz.  
  
 **Identificador de error:** BC30049  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el tipo de datos por un tipo de matriz.
