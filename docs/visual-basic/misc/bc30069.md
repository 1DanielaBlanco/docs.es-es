# <a name="for-loop-control-variable-ltvariablenamegt-already-in-use-by-an-enclosing-for-loop"></a>La variable de control de bucle '&lt;variablename&gt;' ya está en uso por un bucle For envolvente
Los bucles `For` anidados deben usar todos variables de control únicas.  
  
 **Identificador de error:** BC30069  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie uno de los nombres de variable de control.
