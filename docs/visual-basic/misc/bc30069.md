# <a name="for-loop-control-variable-39ltvariablenamegt39-already-in-use-by-an-enclosing-for-loop"></a>Variable de control del bucle for &#39; &lt;variablename&gt; &#39; ya está en uso por un bucle For envolvente
Los bucles `For` anidados deben usar todos variables de control únicas.  
  
 **Identificador de error:** BC30069  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie uno de los nombres de variable de control.
