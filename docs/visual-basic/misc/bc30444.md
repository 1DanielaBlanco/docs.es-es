# <a name="39set39-is-already-declared"></a>&#39;Establecer&#39; ya se ha declarado
Una instrucción `Set` aparece más de una vez en un bloque de código. La instrucción `Set` declara un procedimiento de propiedad `Set` que se utiliza para asignar un valor a una propiedad.  
  
 **Id. de error:** BC30444  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la instrucción redundante `Set` .
