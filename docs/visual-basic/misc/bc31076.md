# <a name="39ltmodulenamegt39-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>&#39;&lt;ModuleName&gt; &#39; es un módulo y no se puede hacer referencia como un ensamblado
Se intentó usar un módulo como un ensamblado.  
  
 **Identificador de error:** BC31076  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie la referencia a un ensamblado.
