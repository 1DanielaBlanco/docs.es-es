 <span data-ttu-id="a9862-105">El código siguiente muestra el error en conversiones.</span><span class="sxs-lookup"><span data-stu-id="a9862-105">The following code illustrates failed conversions.</span></span> <span data-ttu-id="a9862-106">El delegado es `FunDel`.</span><span class="sxs-lookup"><span data-stu-id="a9862-106">The delegate is `FunDel`.</span></span>

Delegate Function FunDel(ByVal i As Integer, ByVal d As Double) As Integer

Function ExampleMethod2(ByVal m As Integer, ByVal aDouble As Double) As Date

    ' are not compatible.
