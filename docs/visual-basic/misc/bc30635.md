# <a name="39readonly39-and-39writeonly39-cannot-be-combined"></a>&#39;ReadOnly&#39; y &#39;WriteOnly&#39; no se pueden combinar
Los modificadores `ReadOnly` y `WriteOnly` se excluyen mutuamente.  
  
 **Id. de error:** BC30635  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Especifique `ReadOnly` o `WriteOnly`, pero no ambos.
