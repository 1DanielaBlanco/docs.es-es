# Error al insertar el manifiesto Win32: &lt;manifiesto&gt;
No se puede insertar el archivo especificado para la opción del compilador `/win32manifest`. Esto puede deberse a falta de privilegios de seguridad suficientes para el archivo de manifiesto.  
  
 **Identificador de error:** BC31191  
  
### Para corregir este error  
  
1.  Asegúrese de que la identidad que ejecuta el comando del compilador de Visual Basic tiene acceso al archivo de manifiesto Win32.
