# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39; EndIf &#39; ya no se admiten instrucciones; Usar &#39; End If &#39; en su lugar
Ya no se admite la instrucción `EndIf` .  
  
 **Identificador de error:** BC30826  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use `End If` para terminar las instrucciones `If` .
