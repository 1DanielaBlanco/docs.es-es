# <a name="39ltassemblynamegt39-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>&#39; &lt;assemblyname&gt;&#39; es un ensamblado y no se puede hacer referencia como un módulo
Se intentó usar un ensamblado como un módulo.  
  
 **Identificador de error:** BC31077  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie la referencia a un módulo.
