# <a name="ltassemblynamegt-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>'&lt;assemblyname&gt;' es un ensamblado y no puede hacer referencia como un módulo
Se intentó usar un ensamblado como un módulo.  
  
 **Identificador de error:** BC31077  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie la referencia a un módulo.
