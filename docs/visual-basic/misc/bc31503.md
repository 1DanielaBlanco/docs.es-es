# <a name="39lttypenamegt39-cannot-be-used-as-an-attribute-because-it-is-not-a-class"></a>&#39; &lt;typename&gt;&#39; no se puede usar como atributo porque no es una clase
Se intentó usar como atributo un tipo que no es una clase.  
  
 **Identificador de error:** BC31503  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Defina atributos personalizados como clases que derivan de `System.Attribute`.
