# <a name="property-39ltpropertynamegt39-is-39writeonly39"></a>Propiedad &#39; &lt;propertyname&gt; &#39; es &#39;WriteOnly&#39;
Intentó recuperar un valor de una propiedad declarada `WriteOnly`.  
  
 **Identificador de error:** BC30524  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador `WriteOnly` de declaración de propiedad.
