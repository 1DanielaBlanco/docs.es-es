# <a name="ltinterfacename1gt-cannot-implement-ltmethodnamegt-because-there-is-no-matching-ltmethodgt-on-interface-ltinterfacename2gt"></a>'&lt;interfacename1&gt;'no puede implementar'&lt;methodname&gt;' porque no hay ninguna coincidencia &lt;método&gt; en la interfaz '&lt;interfacename2&gt;'
Intentó implementar un método en una interfaz cuando no hay ningún método coincidente en la interfaz de la que se derivó la primera interfaz.  
  
 **Identificador de error:** BC30401  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de haber escrito correctamente el nombre del método o proporcione el método que falta.
