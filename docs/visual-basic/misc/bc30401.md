# <a name="39ltinterfacename1gt39-cannot-implement-39ltmethodnamegt39-because-there-is-no-matching-ltmethodgt-on-interface-39ltinterfacename2gt39"></a>&#39; &lt;interfacename1&gt;&#39; no se puede implementar &#39;&lt; MethodName&gt;&#39; porque no hay ninguna coincidencia &lt;método&gt; de interfaz &#39;&lt; interfacename2&gt;&#39;
Intentó implementar un método en una interfaz cuando no hay ningún método coincidente en la interfaz de la que se derivó la primera interfaz.  
  
 **Identificador de error:** BC30401  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de haber escrito correctamente el nombre del método o proporcione el método que falta.
