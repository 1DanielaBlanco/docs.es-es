# <a name="39new39-is-not-valid-in-this-context"></a>&#39;Nueva&#39; no es válido en este contexto
La palabra clave `New` aparece donde se requiere un nombre de tipo. El contexto llama a un tipo de valor que no se puede combinar con `New`.  
  
 **Id. de error:** BC30200  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la palabra clave `New` .
