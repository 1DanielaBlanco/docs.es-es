# &#39;New&#39; no es v&#225;lido en este contexto
La palabra clave `New` aparece donde se requiere un nombre de tipo. El contexto llama a un tipo de valor que no se puede combinar con `New`.  
  
 **Id. de error:** BC30200  
  
### Para corregir este error  
  
-   Quite la palabra clave `New`.
