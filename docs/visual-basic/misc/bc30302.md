# <a name="type-character-39ltcharactergt39-cannot-be-used-in-a-declaration-with-an-explicit-type"></a>Carácter de tipo &#39; &lt;caracteres&gt;&#39; no se puede usar en una declaración con un tipo explícito
Usó un carácter de tipo, como `$`, al especificar explícitamente un tipo.  
  
 **Identificador de error:** BC30302  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el carácter de tipo o quite la cláusula `As <Type>` de la declaración.
