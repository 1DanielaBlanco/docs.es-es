# <a name="assembly-39ltassemblynamegt39-cannot-be-created-because-its-path-is-longer-than-259-characters"></a>Ensamblado &#39; &lt;assemblyname&gt; &#39; no se puede crear porque su ruta de acceso tiene más de 259 caracteres
La ruta de acceso de este ensamblado supera la longitud máxima de 259 caracteres.  
  
 **Identificador de error:** BC31518  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cree una ruta de acceso más corta. Para ello, use nombres más cortos o seleccione una jerarquía de archivos menos profunda.
