# <a name="argument-39ltargumentnamegt39-is-not-valid-for-the-array"></a>Argumento &#39; &lt;argumentname&gt;&#39; no es válido para la matriz
Una matriz contiene un argumento no válido.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe la ortografía de los argumentos de la expresión. Un nombre de variable mal escrito puede crear implícitamente una variable numérica que se inicialice en cero.
