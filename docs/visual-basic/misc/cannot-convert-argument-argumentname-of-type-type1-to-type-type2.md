# <a name="cannot-convert-argument-ltargumentnamegt-of-type-lttype1gt-to-type-lttype2gt"></a>No se puede convertir el argumento '&lt;nombredeargumento&gt;'de tipo'&lt;type1&gt;' al tipo'&lt;type2&gt;'
Una operación de conversión intenta convertir un argumento a un tipo incompatible.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que el tipo al que está intentando convertir el argumento sea válido.
