# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>Argumento &#39;acceso&#39; no es válido. Los valores válidos para el modo Input son &#39;OpenAccess.Read&#39; y &#39;OpenAccess.Default&#39;
Un valor de acceso no es válido en el modo `Input` .  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el valor a `OpenAccess.Read` u `OpenAccess.Default`.
