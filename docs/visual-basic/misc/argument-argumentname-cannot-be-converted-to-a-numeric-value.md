# <a name="argument-39ltargumentnamegt39-cannot-be-converted-to-a-numeric-value"></a>Argumento &#39; &lt;argumentname&gt;&#39; no se puede convertir en un valor numérico
Una conversión intentó convertir una variable, como una cadena, en un valor numérico.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Busque una conversión que sea significativa.
