# <a name="local-variable-ltvariablenamegt-cannot-be-referred-to-before-it-is-declared"></a>Variable local '&lt;variablename&gt;' no se puede hacer referencia a antes de declararla
Una variable dentro de un procedimiento se usa antes de su declaración.  
  
 **Identificador de error:** BC32000  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Mueva la declaración antes de la primera referencia a la variable.
