# <a name="character-sequence-39--39-is-not-allowed-in-an-xml-comment"></a>Secuencia de caracteres &#39;--&#39; no se permite en un comentario XML
Un literal de comentario XML contiene la secuencia de caracteres '--', que no se admite.  
  
 **Identificador de error:** BC31171  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la secuencia de caracteres '--' del literal de comentario XML.
