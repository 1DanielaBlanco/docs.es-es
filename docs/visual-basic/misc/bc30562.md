# <a name="39ltnamegt39-is-ambiguous-between-declarations-in-modules-39ltmodulename1gt39-and-39ltmodulename2gt39"></a>'&lt;nombre&gt;'es ambiguo entre las declaraciones de módulos'&lt;modulename1&gt;'y'&lt;modulename2&gt;'
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador no tiene reglas para solucionar conflictos; debe eliminar nombres.  
  
 **Identificador de error:** BC30562  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use el nombre completo.
