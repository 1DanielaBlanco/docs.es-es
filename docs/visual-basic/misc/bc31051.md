# <a name="namespace-or-type-39ltnamegt39-has-already-been-imported"></a>Namespace o tipo &#39; &lt;nombre&gt; &#39; ya se ha importado.
Se intentó importar un espacio de nombres más de una vez.  
  
 **Identificador de error:** BC31051  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `Imports` redundantes.
