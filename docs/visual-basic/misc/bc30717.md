# <a name="39try39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Try &#39; instrucciones no son válidas en la ventana Inmediato.
Las instrucciones`Try` solo pueden usarse en el código fuente.  
  
 **Id. de error:** BC30717  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `Try` del código de depuración.
