# <a name="39ltderivedtypenamegt39-cannot-inherit-from-lttypegt-39ltconstructedbasetypenamegt39-because-it-expands-the-access-of-type-39ltinternaltypenamegt39-to-ltregiongt-39ltregionnamegt39"></a><span data-ttu-id="2bc29-102">&#39; &lt;nombretipoderivado&gt;&#39; no puede heredar de &lt;tipo&gt; &#39;&lt; nombretipobaseconstruido&gt;&#39; porque amplía el acceso del tipo &#39;&lt; nombretipointerno&gt;&#39; a &lt;región&gt; &#39;&lt; nombreregión&gt;&#39;</span><span class="sxs-lookup"><span data-stu-id="2bc29-102">&#39;&lt;derivedtypename&gt;&#39; cannot inherit from &lt;type&gt; &#39;&lt;constructedbasetypename&gt;&#39; because it expands the access of type &#39;&lt;internaltypename&gt;&#39; to &lt;region&gt; &#39;&lt;regionname&gt;&#39;</span></span>
<span data-ttu-id="2bc29-103">Una clase o interfaz derivada intenta ampliar el nivel de acceso de un tipo interno. Para ello, lo usa como un argumento de tipo para una clase base o una interfaz.</span><span class="sxs-lookup"><span data-stu-id="2bc29-103">A derived class or interface attempts to expand the access level of an internal type by using it as a type argument to a base class or interface.</span></span>  
  
 <span data-ttu-id="2bc29-104">El código siguiente puede generar este error.</span><span class="sxs-lookup"><span data-stu-id="2bc29-104">The following code can generate this error.</span></span>  
  
```  
Public Class containingClass  
    Public Class baseClass(Of t)  
    End Class  
    Friend Class derivedClass  
        Inherits baseClass(Of internalStructure)  
    End Class  
    Private Structure internalStructure  
        Dim firstMember As Integer  
    End Structure  
End Class  
```  
  
 <span data-ttu-id="2bc29-105">El código situado fuera de `containingClass` no tiene permitido acceder a `internalStructure`.</span><span class="sxs-lookup"><span data-stu-id="2bc29-105">Code outside `containingClass` is not allowed to access `internalStructure`.</span></span> <span data-ttu-id="2bc29-106">Sin embargo, se puede acceder a `derivedClass` desde cualquier código del mismo ensamblado.</span><span class="sxs-lookup"><span data-stu-id="2bc29-106">However, `derivedClass` can be accessed from any code in the same assembly.</span></span> <span data-ttu-id="2bc29-107">Por lo tanto, `derivedClass` no puede heredar `baseClass` si usa `internalStructure` como un argumento de tipo, porque podría exponer `internalStructure` en toda la región de código de definición.</span><span class="sxs-lookup"><span data-stu-id="2bc29-107">Therefore, `derivedClass` cannot inherit `baseClass` if it uses `internalStructure` as a type argument, because that could expose `internalStructure` throughout the defining code region.</span></span>  
  
 <span data-ttu-id="2bc29-108">**Identificador de error:** BC30921</span><span class="sxs-lookup"><span data-stu-id="2bc29-108">**Error ID:** BC30921</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="2bc29-109">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="2bc29-109">To correct this error</span></span>  
  
-   <span data-ttu-id="2bc29-110">Ajuste los niveles de acceso de las clases o las interfaces para que el tipo derivado no amplíe el nivel de acceso del tipo interno.</span><span class="sxs-lookup"><span data-stu-id="2bc29-110">Adjust the access levels of the classes or interfaces so that the derived type does not expand the access level of the internal type.</span></span>  
  
     <span data-ttu-id="2bc29-111">O bien</span><span class="sxs-lookup"><span data-stu-id="2bc29-111">-or-</span></span>  
  
-   <span data-ttu-id="2bc29-112">Si no puede ajustar los niveles de acceso, no use el tipo interno como argumento de tipo al construir la clase base o la interfaz.</span><span class="sxs-lookup"><span data-stu-id="2bc29-112">If you cannot adjust the access levels, then do not use the internal type as a type argument when constructing the base class or interface.</span></span>
