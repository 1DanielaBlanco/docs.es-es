# <a name="expected-closing-gt-for-embedded-expression"></a>Se esperaba el cierre ' %&gt;' para la expresión insertada
Faltan los caracteres `%>` de cierre necesarios en una expresión insertada de un literal XML.  
  
 **Identificador de error:** BC31159  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue los caracteres `%>` de cierre necesarios al final de la expresión insertada.
