# &#39;Custom Event&#39; debe terminar con su &#39;End Event&#39; correspondiente
Una declaración `Custom Event` debe terminar con una instrucción `End Event`.  
  
 **Identificador de error:** BC31114  
  
### Para corregir este error  
  
-   Compruebe que la declaración `Custom Event` finaliza con una instrucción `End Event`.
