# El m&#233;todo &#39;&lt;nombreM&#233;todo&gt;&#39; ya est&#225; declarado en la interfaz &#39;&lt;nombreInterfaz&gt;&#39;
Se ha declarado un método de interfaz otra vez, duplicándolo.  
  
 **Identificador de error:** BC30377  
  
### Para corregir este error  
  
-   Quite la declaración duplicada.
