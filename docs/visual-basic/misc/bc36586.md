# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-extension-method-ltmethodnamegt-defined-in-lttypenamegt"></a>Argumento no especificado para el parámetro '&lt;parametername&gt;'del método de extensión'&lt;methodname&gt;'definido en'&lt;typename&gt;'
No se ha proporcionado un argumento para un parámetro necesario en un método de extensión.  
  
 **Identificador de error:** BC36586  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione un argumento para el parámetro especificado.
