# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Argumento no especificado para el parámetro &#39; &lt;parametername&gt;&#39; del método de extensión &#39;&lt; MethodName&gt;&#39; definido en &#39;&lt; TypeName&gt;&#39;
No se ha proporcionado un argumento para un parámetro necesario en un método de extensión.  
  
 **Identificador de error:** BC36586  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione un argumento para el parámetro especificado.
