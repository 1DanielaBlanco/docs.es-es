# <a name="39ltfunctiongt39-is-not-declared"></a>&#39;&lt;función&gt; &#39; no se ha declarado
'\<función >' no se ha declarado. La función se movió al espacio de nombres Microsoft.VisualBasic.  
  
 **Identificador de error:** BC30818  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use la función adecuada del espacio de nombres `Microsoft.VisualBasic` .
