# &#39;End Interface&#39; debe ir precedida de la instrucci&#243;n &#39;Interface&#39; correspondiente
Una instrucción `End Interface` aparece sin una instrucción `Interface` correspondiente.`End Interface` debe ir precedida de una instrucción `Interface` correspondiente.  
  
 **Identificador de error:** BC30252  
  
### Para corregir este error  
  
1.  Compruebe si una instrucción `Interface` anterior está mal escrita o no es válida.  
  
2.  Compruebe que los miembros `Interface` tengan el formato correcto.
