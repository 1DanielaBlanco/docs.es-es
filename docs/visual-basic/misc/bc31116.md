# <a name="removehandler-declaration-must-end-with-a-matching-end-removehandler"></a>La declaración 'RemoveHandler' debe terminar con la declaración 'End RemoveHandler' correspondiente
Una declaración `RemoveHandler` debe terminar con una instrucción `End RemoveHandler` .  
  
 **Identificador de error:** BC31116  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe que la declaración `RemoveHandler` finaliza con una instrucción `End RemoveHandler` .
