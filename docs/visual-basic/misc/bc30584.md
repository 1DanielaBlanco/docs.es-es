# <a name="39lttypegt39-cannot-be-inherited-more-than-once"></a>&#39;&lt;tipo de&gt; &#39; no se puede heredar más de una vez
Duplicó una instrucción `Inherits` .  
  
 **Identificador de error:** BC30584  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la instrucción `Inherits` redundante.
