# <a name="automation-object-does-not-have-a-default-value"></a>El objeto de automatización no tiene un valor predeterminado
Visual Basic no se puede determinar al miembro predeterminado para el objeto especificado.  
  
### <a name="to-correct-the-error"></a>Para corregir el error  
  
-   Compruebe la documentación del objeto e incluya una especificación explícita para la propiedad o el método.
