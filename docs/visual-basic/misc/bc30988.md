<span data-ttu-id="ec486-103">Cuando se declara una instancia de una clase, la palabra clave `New` debe ir seguida por un nombre de tipo o por `With`.</span><span class="sxs-lookup"><span data-stu-id="ec486-103">When you declare an instance of a class, the `New` keyword must be followed by a type name or by `With`.</span></span> <span data-ttu-id="ec486-104">Por ejemplo, las instrucciones siguientes declaran `client` como una instancia de la clase `Customer` .</span><span class="sxs-lookup"><span data-stu-id="ec486-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="ec486-105">El nombre del tipo `Customer` sigue a `New`.</span><span class="sxs-lookup"><span data-stu-id="ec486-105">The type name `Customer` follows `New`.</span></span>  
  
```

' The next declaration uses an object initializer.  
Dim client As New Customer() With {.Name = "Litware, Inc."}  
```  
  
 <span data-ttu-id="ec486-106">A partir de [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], puede declarar un objeto como una instancia de un tipo anónimo, en cuyo caso no se especifica un tipo de datos.</span><span class="sxs-lookup"><span data-stu-id="ec486-106">Beginning with [!INCLUDE[vb_orcas_long](~/includes/vb-orcas-long-md.md)], you can declare an object to be an instance of an anonymous type, in which case you do not specify a data type.</span></span> <span data-ttu-id="ec486-107">En declaraciones de tipos anónimos, la palabra clave `With` sigue a `New`.</span><span class="sxs-lookup"><span data-stu-id="ec486-107">In anonymous type declarations, the keyword `With` follows `New`.</span></span>
