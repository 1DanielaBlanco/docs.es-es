# <a name="value-valuename1-cannot-be-converted-to-valuename2"></a>Valor '\<nombredevalor1 >' no se puede convertir a '\<nombredevalor2 >'
Se intentó una conversión de tipo entre dos tipos incompatibles.  
  
 **Identificador de error:** BC30742  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe los tipos que se van a convertir.
