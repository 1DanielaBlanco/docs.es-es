# <a name="39ltnamegt39-cannot-expose-the-underlying-delegate-type-39ltdelegatetypegt39-of-the-event-it-is-implementing-outside-the-project-through-ltspecifiergt-39lttypegt39"></a>&#39; &lt;nombre&gt;&#39; no se puede exponer el tipo delegado subyacente &#39;&lt; delegateType&gt;&#39; del evento que implementa fuera del proyecto a través de &lt;especificador&gt; &#39;&lt; tipo de&gt;&#39;
El código expone un tipo privado fuera de la clase pública.  
  
 **Id. de error:** BC30915  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare el tipo `Public`.  
  
     O bien  
  
-   Utilice otro tipo.
