# <a name="xml-literals-and-xml-axis-properties-are-not-available-during-this-debugging-session-because-they-are-not-used-in-this-assembly"></a>Los literales XML y las propiedades de eje XML no están disponibles durante esta sesión de depuración porque no se utilizan en este ensamblado
Un literal XML o la propiedad de eje XML se ha hecho referencia en el **inspección** o **inmediato** ventana durante una sesión de depuración en el XML en [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] características no están disponibles. Este es el caso de un ensamblado que no use el XML en [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] características o es una versión de lanzamiento.  
  
 **Id. de error:** BC31196  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Inicie una nueva sesión de depuración con la compilación de depuración del ensamblado.
