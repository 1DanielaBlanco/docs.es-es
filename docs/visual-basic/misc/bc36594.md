# <a name="definition-of-method-ltmethodgt-is-not-accessible-in-this-context"></a>Definición de método &lt;método&gt; no es accesible en este contexto
Una función de agregado que no existe se incluyó con el operador `Into` .  
  
 **Identificador de error:** BC36594  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe la ortografía del nombre de función de agregado y realice las correcciones necesarias.  
  
-   Agregue el método como una función de agregado personalizada (método de extensión). Para obtener más información acerca de cómo crear una función de agregado personalizada, vea [Aggregate (cláusula)](../../visual-basic/language-reference/queries/aggregate-clause.md).
