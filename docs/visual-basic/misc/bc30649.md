# <a name="lttypenamegt-is-an-unsupported-type"></a>'&lt;typename&gt;' es un tipo no admitido
El tipo especificado no puede representarse mediante Visual Studio.  
  
 **Identificador de error:** BC30649  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use un tipo compatible.  
  
-   Desarrolle un tipo nuevo.
