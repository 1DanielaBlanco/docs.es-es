# &#39;&lt;nombreTipo&gt;&#39; es un tipo no compatible.
[!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] no puede representar el tipo especificado.  
  
 **Identificador de error:** BC30649  
  
### Para corregir este error  
  
-   Use un tipo compatible.  
  
-   Desarrolle un tipo nuevo.
