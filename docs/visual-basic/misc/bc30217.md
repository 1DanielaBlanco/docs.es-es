# Se esperaba una constante de cadena
Un carácter que no es de comillas dobles \("\) aparece en un lugar en que el contexto requiere una constante de cadena. Una posible causa es que falten las comillas en la cadena.  
  
 **Identificador de error:** BC30217  
  
### Para corregir este error  
  
1.  Asegúrese de que la cadena esté debidamente rodeada de comillas dobles.  
  
2.  Proporcione una constante de cadena.
