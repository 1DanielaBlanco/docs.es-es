# <a name="39end39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Terminar &#39; instrucciones no son válidas en la ventana Inmediato.
Las instrucciones `Stop` y `End` suspenden la ejecución y no están permitidas en un contexto de depuración.  
  
 **Identificador de error:** BC30123  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   No emita una instrucción `End` o `Stop` en la ventana **Inmediato** .
