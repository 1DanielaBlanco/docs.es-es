# <a name="39module39-statement-must-end-with-a-matching-39end-module39"></a>&#39;Módulo&#39; instrucción debe terminar con una coincidencia &#39;End Module&#39;
Los bloques de módulo deben terminar con una instrucción `End``Module` .  
  
 **Identificador de error:** BC30625  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue una instrucción `End Module` al final del bloque `Module` .
