# <a name="39class39-constraint-cannot-be-specified-multiple-times-for-the-same-type-parameter"></a>&#39;Clase&#39; restricción no se puede especificar varias veces para el mismo parámetro de tipo
Una lista de restricciones incluye la [(Visual Basic) de la clase](https://msdn.microsoft.com/library/0777c6e6-46bc-451b-ad70-57b49d4ef4f7) restricción más de una vez.  
  
 Una lista de restricciones en un parámetro de tipo puede especificar que el argumento de tipo pasado a ese parámetro de tipo debe ser un tipo de valor (con la [estructura (Visual Basic)](https://msdn.microsoft.com/library/263ce115-ac36-4c05-8cb7-0e0eead5c6d0) restricción) o debe ser un tipo de referencia (con la `Class` restricción) . No se pueden especificar ambas restricciones en el mismo parámetro de tipo y ninguna de ellas se puede especificar más de una vez.  
  
 Identificador de error: BC32101  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las palabras clave `Class` redundantes. Debe tener solo una en la lista de restricciones.
