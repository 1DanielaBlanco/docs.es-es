# <a name="named-argument-ltargumentnamegt-specified-multiple-times"></a>Argumento con nombre '&lt;argumentname&gt;' se especificó varias veces
Se especificó un argumento con nombre más de una vez.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite todas las especificaciones salvo una.
