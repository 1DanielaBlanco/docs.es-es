# El argumento &#39;&lt;argumentname&gt;&#39; debe ser mayor o igual a 1.
Un argumento tiene un valor que es menor o igual a cero.  
  
### Para corregir este error  
  
-   Proporcione un argumento que sea mayor que cero.
