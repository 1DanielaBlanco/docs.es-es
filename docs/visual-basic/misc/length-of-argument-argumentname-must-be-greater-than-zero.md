# <a name="length-of-argument-39ltargumentnamegt39-must-be-greater-than-zero"></a>Longitud del argumento &#39; &lt;argumentname&gt;&#39; debe ser mayor que cero.
Un argumento tiene un valor que es menor o igual a cero.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione un argumento que sea mayor que cero.
