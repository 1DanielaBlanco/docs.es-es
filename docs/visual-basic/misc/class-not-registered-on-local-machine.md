# <a name="class-not-registered-on-local-machine"></a>La clase no está registrada en el equipo local.
Normalmente se encuentra este error como resultado de un archivo dañado. Puede que necesite volver a instalar [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Vuelva a instalar [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].
