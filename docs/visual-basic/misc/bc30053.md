# <a name="arrays-cannot-be-declared-with-39new39"></a><span data-ttu-id="8fd0a-102">Las matrices no se pueden declarar con &#39; Nuevo &#39;</span><span class="sxs-lookup"><span data-stu-id="8fd0a-102">Arrays cannot be declared with &#39;New&#39;</span></span>
<span data-ttu-id="8fd0a-103">La palabra clave `New` solo puede aparecer en la parte de la inicialización de una declaración de matriz.</span><span class="sxs-lookup"><span data-stu-id="8fd0a-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="8fd0a-104">Esto significa que `New` debe estar en el lado derecho del signo igual (`=`) para que pueda crear un nuevo tipo de matriz, que se asignará a la variable de matriz.</span><span class="sxs-lookup"><span data-stu-id="8fd0a-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="8fd0a-105">El acceso directo para la inicialización de clase no está disponible para las matrices.</span><span class="sxs-lookup"><span data-stu-id="8fd0a-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="8fd0a-106">Las siguientes dos líneas de código son ambas válidas y equivalentes, porque inicializan un objeto desde una clase.</span><span class="sxs-lookup"><span data-stu-id="8fd0a-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="8fd0a-107">Sin embargo, la inicialización de matriz no puede usar el mismo método abreviado como la inicialización de la clase.</span><span class="sxs-lookup"><span data-stu-id="8fd0a-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="8fd0a-108">Tenga en cuenta que la cláusula `New` para una matriz debe contener ambos paréntesis, `()`, y las llaves, `{}`.</span><span class="sxs-lookup"><span data-stu-id="8fd0a-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="8fd0a-109">Los paréntesis especifican que el nuevo tipo es una matriz, y las llaves proporcionan los valores de inicialización.</span><span class="sxs-lookup"><span data-stu-id="8fd0a-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="8fd0a-110">El compilador requiere las llaves aunque estén vacías, es decir, aunque no esté inicializando ninguno de los valores de matriz.</span><span class="sxs-lookup"><span data-stu-id="8fd0a-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>  
  
 <span data-ttu-id="8fd0a-111">**Identificador de error:** BC30053</span><span class="sxs-lookup"><span data-stu-id="8fd0a-111">**Error ID:** BC30053</span></span>
