# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>No se puede especificar un modificador que acepte valores NULL en una variable y su tipo
El modificador de tipo que acepta valores NULL (?) se incluyó para una variable declarada y su tipo declarado.  
  
 **Identificador de error:** BC33100  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite el modificador de tipo que acepta valores NULL (?) de la variable o su tipo declarado.
