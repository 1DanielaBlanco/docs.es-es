# <a name="39ltnamegt39-is-declared-in-project-39ltprojectname1gt39-which-is-not-referenced-by-project-39ltprojectname2gt39"></a>&#39;&lt;nombre&gt; &#39; se declara en el proyecto &#39; &lt;projectname1&gt;&#39;, que no se hace referencia al proyecto &#39; &lt;projectname2&gt;&#39;
Un elemento de programación se define en un proyecto al que el proyecto actual no tiene una referencia.  
  
 **Identificador de error:** BC32004  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   En el proyecto actual, agregue una referencia al proyecto que contiene el elemento deseado.
