# <a name="events-in-a-module-cannot-be-declared-specifier"></a>Los eventos de un módulo no se pueden declarar '\<especificador >'
Se ha utilizado un especificador que no es válido en un evento dentro de una instrucción `Module` . Los módulos no admiten la creación de instancias ni la herencia y tampoco pueden implementar interfaces.  
  
 **Identificador de error:** BC30434  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el especificador.
