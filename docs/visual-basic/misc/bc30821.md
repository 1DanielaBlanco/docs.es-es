# <a name="39ltrsetstmtgt39-is-not-declared"></a>&#39;&lt;rsetstmt&gt; &#39; no se ha declarado
'\<rsetstmt >' no se ha declarado. Ya no se admiten instrucciones RSet; use Microsoft.VisualBasic.RSeten su lugar.  
  
 Varias funciones que eran intrínsecas a Visual Basic en versiones anteriores se han movido a la <xref:Microsoft.VisualBasic?displayProperty=nameWithType> espacio de nombres. Esto hace que sus características estén disponibles de manera más general para todos los lenguajes de programación.  
  
 **Identificador de error:** BC30821  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use la función `RSet` en el espacio de nombres `Microsoft.VisualBasic` en su lugar.
