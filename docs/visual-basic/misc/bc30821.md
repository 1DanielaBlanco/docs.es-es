# <a name="39ltrsetstmtgt39-is-not-declared"></a>'&lt;rsetstmt&gt;' no se ha declarado
'\<rsetstmt >' no se ha declarado. Ya no se admiten instrucciones RSet; use Microsoft.VisualBasic.RSeten su lugar.  
  
 Varias funciones que eran intrínsecas a [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] en versiones anteriores se han movido a la <xref:Microsoft.VisualBasic?displayProperty=fullName>espacio de nombres.</xref:Microsoft.VisualBasic?displayProperty=fullName> Esto hace que sus funciones estén disponibles de manera más general para todos los lenguajes de programación.  
  
 **Identificador de error:** BC30821  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use la función `RSet` en el espacio de nombres `Microsoft.VisualBasic` en su lugar.
