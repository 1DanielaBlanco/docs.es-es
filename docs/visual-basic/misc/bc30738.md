# <a name="sub-main-is-declared-more-than-once-in-ltnamespacegt-ltmessagegt"></a>'Sub Main' se declaró más de una vez en '&lt;espacio de nombres&gt;': &lt;mensaje&gt;
`Sub Main` solo puede declararse una vez en un espacio de nombres.  
  
 **Identificador de error:** BC30738  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que hay un solo procedimiento `Sub Main` en el proyecto.
