# <a name="39sub-main39-is-declared-more-than-once-in-39ltnamespacegt39-ltmessagegt"></a>&#39;Sub Main&#39; está declarado más de una vez en &#39; &lt;espacio de nombres&gt;&#39;: &lt;mensaje&gt;
`Sub Main` solo puede declararse una vez en un espacio de nombres.  
  
 **Identificador de error:** BC30738  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que hay un solo procedimiento `Sub Main` en el proyecto.
