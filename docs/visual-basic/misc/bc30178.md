# <a name="specifier-is-duplicated"></a>El especificador está duplicado.
Una palabra clave aparece más de una vez en la misma declaración.  
  
 **Identificador de error:** BC30178  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las repeticiones de la palabra clave de la declaración.
