# <a name="39lttypenamegt39-is-a-type-in-39ltcontainernamegt39-and-cannot-be-used-as-an-expression"></a>&#39;&lt;TypeName&gt; &#39; es un tipo de &#39; &lt;containername&gt; &#39; y no se puede usar como una expresión.
Uno de los nombres que se va a usar ya se ha declarado como el nombre de un tipo y no se puede usar en una expresión.  
  
 **Identificador de error:** BC30691  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Nombre único del miembro.  
  
-   Usar corchetes alrededor del nombre del miembro.
