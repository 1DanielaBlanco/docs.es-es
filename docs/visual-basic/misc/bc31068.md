# <a name="delegate-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>No se puede declarar un delegado de una interfaz &#39; &lt;especificador&gt;&#39;
Se especificó un modificador no válido con un delegado declarado en una interfaz.  
  
 **Identificador de error:** BC31068  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite los modificadores como, por ejemplo, `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`y `Overridable`.
