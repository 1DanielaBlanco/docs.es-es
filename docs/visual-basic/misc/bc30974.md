# <a name="comment-statements-cannot-be-evaluated-not-in-toc"></a>No se pueden evaluar instrucciones de comentario (no en la tabla de contenido)
Se escribió una instrucción de comentario en la ventana Inmediato durante una sesión de depuración. Las instrucciones de comentario en la ventana Inmediato no se evalúan.  
  
 **Identificador de error:** BC30974
