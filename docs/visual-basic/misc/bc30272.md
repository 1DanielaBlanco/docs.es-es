# <a name="ltargumentnamegt-is-not-a-parameter-of-ltmethoddeclarationgt"></a>'&lt;nombredeargumento&gt;'no es un parámetro de'&lt;methoddeclaration&gt;'
Una llamada a procedimiento facilita un argumento adicional que no lo define el procedimiento.  
  
 **Identificador de error:** BC30272  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el argumento superfluo de la lista de argumentos.
