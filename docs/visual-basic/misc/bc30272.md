# <a name="39ltargumentnamegt39-is-not-a-parameter-of-39ltmethoddeclarationgt39"></a>&#39;&lt;argumentName&gt; &#39; no es un parámetro de &#39; &lt;methoddeclaration&gt;&#39;
Una llamada a procedimiento facilita un argumento adicional que no lo define el procedimiento.  
  
 **Id. de error:** BC30272  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el argumento superfluo de la lista de argumentos.
