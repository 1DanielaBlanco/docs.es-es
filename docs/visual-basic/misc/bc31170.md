# <a name="character-character-0xlthexadecimalgt-is-not-allowed-in-an-xml-name"></a>Carácter 'carácter' (0 x&lt;hexadecimal&gt;) no se permite en un nombre XML
Un carácter no admitido se incluye en el nombre de un literal XML o una propiedad de eje XML.  
  
 **Identificador de error:** BC31170  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el carácter no admitido del nombre del literal XML o la propiedad de eje XML.
