# <a name="enum-in-an-interface-cannot-be-declared-specifier"></a>No se puede declarar una enumeración de una interfaz '\<especificador >'
Se especificó un modificador no válido con una enumeración declarada en una interfaz.  
  
 **Identificador de error:** BC31069  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite los modificadores como, por ejemplo, `Public`, `Private`, `Shared`, `Friend`, `Protected`, `Protected Friend`u `Overridable`.
