# <a name="array-initializer-cannot-be-specified-for-a-non-constant-dimension-use-the-empty-initializer-3939"></a><span data-ttu-id="388a3-102">No se puede especificar un inicializador de matriz para una dimensión no constante; Utilice al inicializador vacío&#39;{}&#39;</span><span class="sxs-lookup"><span data-stu-id="388a3-102">Array initializer cannot be specified for a non constant dimension; use the empty initializer &#39;{}&#39;</span></span>
<span data-ttu-id="388a3-103">Una matriz inicializa una dimensión que no se conoce en tiempo de compilación.</span><span class="sxs-lookup"><span data-stu-id="388a3-103">An array initializes a dimension that is not known at compile time.</span></span>  
  
 <span data-ttu-id="388a3-104">El código siguiente genera este error.</span><span class="sxs-lookup"><span data-stu-id="388a3-104">The following code generates this error.</span></span>  
  
```  
Dim j As Integer  
Dim intArray As Integer = New Integer(1, j) {{0, 100}, {1,101}}  
```  
  
 <span data-ttu-id="388a3-105">El código siguiente evita el error.</span><span class="sxs-lookup"><span data-stu-id="388a3-105">The following code avoids the error.</span></span>  
  
```  
Dim intArray As Integer = New Integer(1, j) {}  
For i As Integer = 0 To j  
    intArray(0, i) = i  
    intArray(1, i) = 100 + i  
Next i  
```  
  
 <span data-ttu-id="388a3-106">**Identificador de error:** BC30949</span><span class="sxs-lookup"><span data-stu-id="388a3-106">**Error ID:** BC30949</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="388a3-107">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="388a3-107">To correct this error</span></span>  
  
-   <span data-ttu-id="388a3-108">Si es posible, especifique una dimensión constante en la declaración de matriz.</span><span class="sxs-lookup"><span data-stu-id="388a3-108">If possible, specify a constant dimension in the array declaration.</span></span>  
  
-   <span data-ttu-id="388a3-109">Si no se puede especificar una dimensión constante, debe inicializar la matriz mediante un bucle cuando se conoce la dimensión no constante.</span><span class="sxs-lookup"><span data-stu-id="388a3-109">If you cannot specify a constant dimension, then you must initialize the array using a loop when the nonconstant dimension becomes known.</span></span>
