# <a name="argument-39access39-is-not-valid-valid-values-for-output-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>Argumento &#39; Acceder al &#39; no es válido. Los valores válidos para el modo Output son #39;OpenAccess.Write &#39; y &#39;OpenAccess.Default &#39;
Un valor de acceso no es válido en el modo Output.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Cambie el valor a `OpenAccess.Write` u `OpenAccess.Default`.
