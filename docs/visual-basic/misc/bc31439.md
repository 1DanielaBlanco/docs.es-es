# <a name="partial-method-39ltmethodnamegt39-cannot-use-the-39implements39-keyword"></a>Método parcial &#39; &lt;methodname&gt;&#39; no se puede usar la &#39; implementa &#39; (palabra clave)
Una definición de método parcial no puede implementar una interfaz o un miembro de interfaz.  
  
 **Identificador de error:** BC31439  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Elimine la instrucción `Implements` de la definición de método parcial.
