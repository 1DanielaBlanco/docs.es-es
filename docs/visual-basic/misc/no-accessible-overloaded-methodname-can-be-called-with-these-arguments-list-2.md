# <a name="no-accessible-overloaded-39ltmethodnamegt39-can-be-called-with-these-arguments-without-a-narrowing-conversion-ltlistgt"></a>No accesible sobrecargado &#39; &lt;methodname&gt;&#39; se puede llamar con estos argumentos sin una conversión de restricción: &lt;lista&gt;
Se ha llamado a un método sobrecargado, pero el método no coincide con la lista de argumentos proporcionada sin una conversión de restricción.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Especifique `Option``Strict` `Off`.  
  
2.  Cambie los argumentos para que coincida con una firma del método sobrecargado.
