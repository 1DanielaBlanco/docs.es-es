# <a name="ltmodifiergt-is-not-valid-on-a-local-constant-declaration"></a>'&lt;modificador&gt;' no es válido en una declaración de constante local
Una instrucción `Const` dentro de un procedimiento contiene una palabra clave no válida, como `Overloads`.  
  
 **Identificador de error:** BC30246  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite la palabra clave no válida de la instrucción `Const` .
