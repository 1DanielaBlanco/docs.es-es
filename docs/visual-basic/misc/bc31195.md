# <a name="operator-operator-is-not-defined-for-types-type1-and-type2"></a>El operador 'operator' no está definido para los tipos 'type1' y 'type2'
El operador 'operator' no está definido para los tipos 'type1' y 'type2'. Puede usar la propiedad 'Value' para obtener el valor de cadena del primer elemento de '\<Elementoprimario >'.  
  
 Se usó un literal XML en una comparación binaria o de tipo `Boolean` .  
  
 **Identificador de error:** BC31195  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use la propiedad `Value` del literal XML para la comparación binaria o de tipo `Boolean` .
