# &#39;End Using&#39; debe ir precedida de la instrucci&#243;n &#39;Using&#39; correspondiente.
Una instrucción `End Using` aparece sin ninguna declaración `Using` anterior.  
  
 **Identificador de error:** BC36007  
  
### Para corregir este error  
  
-   Quite la instrucción `End Using` si es redundante.  
  
-   Proporcione el [Using \(Instrucción\)](../../visual-basic/language-reference/statements/using-statement.md) en caso de que falte.  
  
-   Inserte la instrucción `End Using` en el lugar adecuado del código.
