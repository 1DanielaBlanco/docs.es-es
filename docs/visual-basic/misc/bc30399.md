# <a name="39mybase39-cannot-be-used-with-method-39ltmethodnamegt39-because-it-is-declared-39mustoverride39"></a>&#39;MyBase&#39; no se puede usar con el método &#39; &lt;methodname&gt; &#39; porque se declaró &#39;MustOverride&#39;
Ha intentado usar `MyBase` con un método que se ha declarado `MustOverride`.  
  
 **Identificador de error:** BC30399  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la declaración `MustOverride`
