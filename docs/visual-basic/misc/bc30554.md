# <a name="39ltnamespacenamegt39-is-ambiguous"></a>'&lt;namespacename&gt;' es ambiguo
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador no tiene ninguna regla de resolución de conflictos; debe eliminar nombres.  
  
 **Identificador de error:** BC30554  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use el nombre completo.
