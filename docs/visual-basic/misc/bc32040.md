# <a name="the-39ltkeywordgt39-keyword-is-used-to-overload-inherited-members-do-not-use-the-39ltkeywordgt39-keyword-when-overloading-39sub-new39"></a>El &#39; &lt;palabra clave&gt; &#39; palabra clave se usa para sobrecargar los miembros heredados; no utilice la &#39; &lt;palabra clave&gt; &#39; palabra clave cuando sobrecargue un procedimiento &#39;Sub New&#39;
Se ha declarado un constructor con la palabra clave `Overloads` .  
  
 Visual Basic no admite la herencia ni la sobrecarga de constructores.  
  
 **Identificador de error:** BC32040  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la palabra clave `Overloads` de todas las declaraciones de constructor.
