# <a name="argument-39ltargumentnamegt39-must-be-greater-than-0-or-equal-to--1"></a>Argumento &#39; &lt;argumentname&gt;&#39; debe ser mayor que 0 o igual a -1
El valor del argumento es menor que 0, pero no es igual a -1.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que el valor del argumento está dentro del intervalo válido.  
  
2.  Si se proporciona el argumento mediante un cálculo, compruebe los valores de los procesos de cálculo.
