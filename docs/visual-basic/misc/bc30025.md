# <a name="property-missing-end-property"></a>Falta 'End Property' en Property.
Una instrucción `End` debe coincidir con el bloque que termina. Una definición `Property` no se terminó correctamente.  
  
 **Identificador de error:** BC30025  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue una instrucción `End Property` al final de la definición de propiedad.
