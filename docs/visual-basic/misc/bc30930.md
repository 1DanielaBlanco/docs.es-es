# <a name="39ltnamegt39-necessary-for-compiling-this-construct-cannot-be-found"></a>&#39;&lt;nombre&gt;&#39;, necesario para compilar esta construcción, no se encuentra
No se encuentra un elemento necesario para la compilación.  
  
 **Identificador de error:** BC30930  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Proporcione el elemento auxiliar necesario.  
  
2.  Asegúrese de que el código sea código administrado válido.
