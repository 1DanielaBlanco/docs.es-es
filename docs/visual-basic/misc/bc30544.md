# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>El método no puede contener tanto un &#39; Try &#39; instrucción y un &#39; en caso de Error &#39; o &#39; Reanudar &#39; instrucción
Combinó una instrucción `Try` con `On Error` o `Resume`.  
  
 **Identificador de error:** BC30544  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Quite `On Error` o `Resume`.
