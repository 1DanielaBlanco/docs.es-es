# <a name="39option39-statements-must-precede-any-declarations-or-39imports39-statements"></a>&#39;Opción&#39; instrucciones deben preceder a cualquier declaración o &#39;importaciones&#39; instrucciones
Las instrucciones`Option` deben ser las primeras instrucciones sin comentarios del código fuente.  
  
 **Identificador de error:** BC30627  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Mueva las instrucciones `Option` al principio del código fuente, inmediatamente antes de las instrucciones `Imports` y `Namespace` .
