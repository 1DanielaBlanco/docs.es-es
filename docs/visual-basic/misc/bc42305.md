# <a name="xml-comment-tag-39lttaggt39-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>Etiqueta de comentario XML &#39; &lt;etiqueta&gt; &#39; aparece con atributos idénticos más de una vez en el mismo bloque de comentario XML
Etiqueta de comentario XML \<tag > aparece con atributos idénticos más de una vez en el mismo bloque de comentario XML. Se omitirá el comentario XML.  
  
 El comentario XML contiene más de una etiqueta con atributos idénticos.  
  
 **Id. de error:** BC42305  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la etiqueta duplicada.
