# Los m&#233;todos de un m&#243;dulo no se pueden declarar &#39;&lt;especificador&gt;&#39;.
Se ha usado un especificador que no es válido en un método dentro de una instrucción `Module`. Nunca se pueden crear instancias de módulos, no admiten herencia y no pueden implementar interfaces.  
  
 **Identificador de error:** BC30433  
  
### Para corregir este error  
  
-   Quite el especificador.
