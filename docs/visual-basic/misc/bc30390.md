# <a name="39ltqualifiedmethodgt39-is-not-accessible-in-this-context-because-it-is-39ltmodifiergt39"></a>&#39; &lt;qualifiedmethod&gt;&#39; no es accesible en este contexto porque es &#39;&lt; modificador&gt;&#39;
Ha intentado obtener acceso a un método que no es accesible en este contexto debido a un modificador que se le ha aplicado.  
  
 **Identificador de error:** BC30390  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el modificador.
