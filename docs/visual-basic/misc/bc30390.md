# <a name="ltqualifiedmethodgt-is-not-accessible-in-this-context-because-it-is-ltmodifiergt"></a>'&lt;qualifiedmethod&gt;'no es accesible en este contexto porque es'&lt;modificador&gt;'
Ha intentado obtener acceso a un método que no es accesible en este contexto debido a un modificador que se le ha aplicado.  
  
 **Identificador de error:** BC30390  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el modificador.
