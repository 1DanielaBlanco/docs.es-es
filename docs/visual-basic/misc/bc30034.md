# <a name="bracketed-identifier-is-missing-closing-3939"></a>Identificador entre corchetes Falta cierre &#39;] &#39;
Los corchetes de un nombre con escape deben aparecer entre pares coincidentes.  
  
 **Identificador de error:** BC30034  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Coloque un corchete de cierre al final del nombre con escape.
