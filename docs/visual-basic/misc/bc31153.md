# <a name="required-attribute-39version39-missing-from-xml-declaration"></a>Requiere atributo &#39; versión &#39; falta de declaración XML
Falta el atributo `version` para un literal de documento XML.  
  
 **Identificador de error:** BC31153  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue el atributo `version` a la declaración de documento XML, como en el ejemplo siguiente: `<?xml version="1.0"?>`
