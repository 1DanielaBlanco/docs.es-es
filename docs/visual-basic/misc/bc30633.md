# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39; Conjunto de &#39; la instrucción debe terminar con una coincidencia &#39; Conjunto final &#39;
Los procedimientos de propiedad`Set` deben terminar con una instrucción `End Set` .  
  
 **Identificador de error:** BC30633  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que el procedimiento de propiedad `Set` termine con una construcción `End Set` .
