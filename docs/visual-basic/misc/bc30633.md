# <a name="39set39-statement-must-end-with-a-matching-39end-set39"></a>&#39;Establecer&#39; instrucción debe terminar con una coincidencia &#39;End Set&#39;
Los procedimientos de propiedad`Set` deben terminar con una instrucción `End Set` .  
  
 **Identificador de error:** BC30633  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que el procedimiento de propiedad `Set` termine con una construcción `End Set` .
