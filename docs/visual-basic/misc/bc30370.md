<span data-ttu-id="6ed49-103">Una lista de restricciones o un inicializador de matriz no termina de manera válida.</span><span class="sxs-lookup"><span data-stu-id="6ed49-103">An array initializer or a constraint list has not been ended in a valid fashion.</span></span>
