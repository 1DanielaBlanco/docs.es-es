# <a name="argument-argumentname-must-be-greater-than-or-equal-to-zero"></a>Argumento \<argumentname > debe ser mayor o igual a cero.
Un valor de argumento es menor que cero. Los valores de argumento deben ser mayores o iguales a cero.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que el valor del argumento sea mayor o igual a cero.  
  
2.  Si se facilita el argumento mediante cálculo, compruebe los valores que va a procesar el cálculo.
