# Se esperaba el fin de la expresi&#243;n
Se encontraron caracteres adicionales al final de otra expresión completa.  
  
 **Identificador de error:** BC30710  
  
### Para corregir este error  
  
-   Compruebe la sintaxis del objeto o la instrucción.
