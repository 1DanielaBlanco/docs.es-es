# <a name="of-required-when-specifying-type-arguments-for-a-generic-type-or-method"></a>Se requiere 'Of' cuando se especifican argumentos de tipo para un tipo o un método genéricos
Una instrucción intenta construir un tipo a partir de un tipo genérico, o llama a un método genérico, sin usar una cláusula [Of](../../visual-basic/language-reference/statements/of-clause.md) .  
  
 Llama la sintaxis de Visual Basic para tipos genéricos para que los parámetros de tipo y los argumentos de tipo que se introducen mediante la `Of` palabra clave. Además, la lista de parámetros de tipo o la lista de argumentos de tipo se debe incluir entre paréntesis.  
  
 **Identificador de error:** BC32093  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Incluya la palabra clave `Of` al principio de la lista de argumentos de tipo y agregue la lista completa entre paréntesis.
