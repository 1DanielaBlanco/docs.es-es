# <a name="implicit-variable-ltvariablenamegt-is-invalid-because-of-ltmessagegt"></a>Variable implícita '&lt;variablename&gt;'no es válida porque'&lt;mensaje&gt;'
Se ha generado una variable implícita no válida durante la depuración.  
  
 **Identificador de error:** BC36016  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Examine el mensaje para determinar el motivo y la posible solución.  
  
2.  Si el error persiste, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
