# <a name="39for39-loop-control-variable-already-in-use-by-an-enclosing-39for39-loop"></a>&#39;Para&#39; la variable de control ya está en uso envolvente &#39;para&#39; bucles
Todos los bucles `For` anidados deben usar variables de control únicas.  
  
 **Identificador de error:** BC30975  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie uno de los nombres de variable de control.
