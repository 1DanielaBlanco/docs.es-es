# <a name="too-few-type-arguments-to-extension-method-39ltmethodnamegt39-defined-in-39lttypenamegt39"></a>Insuficientes argumentos de tipo al método de extensión &#39; &lt;methodname&gt; &#39; definido en &#39; &lt;typename&gt;&#39;
Se ha invocado un método de extensión genérico con menos argumentos de tipo que los parámetros de tipo que hay en la definición del método.  
  
 Al invocar un método genérico, debe proporcionar un argumento de tipo para cada parámetro de tipo definido por el método.  
  
 **Identificador de error:** BC36590  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue argumentos de tipo a la lista de argumentos de tipo para que haya uno para cada parámetro de tipo del método de extensión que se está invocando.
