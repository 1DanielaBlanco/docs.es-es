# <a name="39structure39-statement-must-end-with-a-matching-39end-structure39"></a>&#39;Estructura&#39; instrucción debe terminar con una coincidencia &#39;End Structure&#39;
Las estructuras comienzan con una instrucción `Structure` y terminan con una construcción `End Structure` .  
  
 **Identificador de error:** BC30624  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que el bloque de la estructura termina con una instrucción `End Structure` .
