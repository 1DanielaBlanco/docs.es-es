# <a name="type-of-argument-ltargumentnamegt-is-lttypenamegt-which-is-not-numeric"></a>Tipo de argumento '&lt;nombredeargumento&gt;'is'&lt;typename&gt;', que no es numérico
Un argumento no numérico se produce en una situación en la que se espera un argumento numérico.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que el argumento sea numérico.
