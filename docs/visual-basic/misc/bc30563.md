# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>'&lt;nombre&gt;'es ambiguo en los objetos de aplicación'&lt;lista&gt;'
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador no tiene reglas para solucionar conflictos; debe eliminar nombres. La lista proporcionada en el mensaje de error detalla los objetos de aplicación en los que el nombre es ambiguo.  
  
 **Identificador de error:** BC30563  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use el nombre completo.
