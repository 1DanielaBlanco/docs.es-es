# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39; &lt;nombre&gt;&#39; es ambiguo en los objetos de aplicación &#39;&lt; lista&gt;&#39;
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El compilador de [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] no tiene reglas de resolución de conflictos. Es necesario eliminar la ambigüedad en los nombres de forma manual. La lista proporcionada en el mensaje de error detalla los objetos de aplicación en los que el nombre es ambiguo.  
  
 **Identificador de error:** BC30563  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use el nombre completo.
