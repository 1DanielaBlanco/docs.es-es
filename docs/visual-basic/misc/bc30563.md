# <a name="39ltnamegt39-is-ambiguous-in-the-application-objects-39ltlistgt39"></a>&#39;&lt;nombre&gt; &#39; es ambiguo en los objetos de aplicación &#39; &lt;lista&gt;&#39;
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El compilador de Visual Basic no tiene reglas de resolución de conflictos; deben eliminar la ambigüedad de los nombres usted mismo. La lista proporcionada en el mensaje de error detalla los objetos de aplicación en los que el nombre es ambiguo.  
  
 **Identificador de error:** BC30563  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use el nombre completo.
