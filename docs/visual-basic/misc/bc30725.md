# <a name="39throw39-statements-are-not-valid-in-the-immediate-window"></a>&#39; Throw &#39; instrucciones no son válidas en la ventana Inmediato.
Las instrucciones`Throw` solo pueden usarse en el código fuente.  
  
 **Identificador de error:** BC30725  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `Throw` del código de depuración.
