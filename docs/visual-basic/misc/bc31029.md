# <a name="method-ltmethodnamegt-cannot-handle-event-lteventnamegt-because-they-do-not-have-the-same-signature"></a>Método '&lt;methodname&gt;'no se puede controlar el evento'&lt;eventname&gt;' porque no tienen la misma firma
Los parámetros o los tipos de parámetro del controlador de eventos especificado no coinciden con los definidos para el evento especificado.  
  
 **Identificador de error:** BC31029  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Vuelva a definir el evento o el controlador de eventos para que las listas de parámetros coincidan.
