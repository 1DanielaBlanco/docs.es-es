# <a name="structure-39ltstructurenamegt39-must-contain-at-least-one-instance-member-variable-or-event-declaration"></a>Estructura &#39; &lt;structurename&gt; &#39; debe contener al menos una variable miembro de instancia o declaración de evento
Aparece una estructura funcionalmente vacía. Debe declararse al menos un miembro de variable entre la instrucción `Structure` y la instrucción `End Structure` .  
  
 **Id. de error:** BC30281  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue una instrucción `Dim` o una instrucción `Event` a la estructura.
