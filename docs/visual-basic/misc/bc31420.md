# <a name="39ltmembergt39-conflicts-with-the-reserved-member-by-this-name-that-is-implicitly-declared-in-all-enums"></a>&#39; &lt;miembro&gt;&#39; está en conflicto con el miembro reservado por este nombre que está declarado implícitamente en todas las enumeraciones
El nombre de un miembro de tipo está en conflicto con el nombre de un miembro declarado implícitamente en todas las enumeraciones.  
  
 El compilador de [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] crea miembros implícitos que corresponden a determinados elementos de programación que el usuario declara. Las enumeraciones declaran implícitamente el miembro `value__member`.  
  
 **Identificador de error:** BC31420  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Modifique el nombre del miembro.
