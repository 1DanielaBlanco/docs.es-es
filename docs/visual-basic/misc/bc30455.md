# <a name="argument-not-specified-for-parameter-ltparameternamegt-of-ltmethodnamegt"></a>Argumento no especificado para el parámetro '&lt;parametername&gt;'de'&lt;methodname&gt;'
No proporcionó un argumento para un parámetro necesario.  
  
 **Identificador de error:** BC30455  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione un argumento para el parámetro especificado.
