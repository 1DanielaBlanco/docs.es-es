# <a name="argument-not-specified-for-parameter-39ltparameternamegt39-of-39ltmethodnamegt39"></a>Argumento no especificado para el parámetro &#39; &lt;parametername&gt;&#39; de &#39;&lt; MethodName&gt;&#39;
No proporcionó un argumento para un parámetro necesario.  
  
 **Identificador de error:** BC30455  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Proporcione un argumento para el parámetro especificado.
