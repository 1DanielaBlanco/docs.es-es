# <a name="type-parameter-lttypeparametername1gt-must-have-either-a-new-constraint-or-a-structure-constraint-to-satisfy-the-new-constraint-for-type-parameter-lttypeparametername2gt"></a><span data-ttu-id="63341-102">Parámetro de tipo '&lt;typeparametername1&gt;'debe tener una restricción 'New' o 'Structure' para satisfacer la restricción 'New' para el parámetro de tipo'&lt;nombredeparámetrodetipo2&gt;'</span><span class="sxs-lookup"><span data-stu-id="63341-102">Type parameter '&lt;typeparametername1&gt;' must have either a 'New' constraint or a 'Structure' constraint to satisfy the 'New' constraint for type parameter '&lt;typeparametername2&gt;'</span></span>
<span data-ttu-id="63341-103">Una instrucción construye un tipo genérico que pasa un parámetro de tipo que no está restringido para satisfacer una restricción `New` .</span><span class="sxs-lookup"><span data-stu-id="63341-103">A statement constructs a generic type passing a type parameter that is not constrained to satisfy a `New` constraint.</span></span>  
  
 <span data-ttu-id="63341-104">La restricción `New` significa que el argumento de tipo proporcionado para ese parámetro de tipo debe exponer un constructor sin parámetros accesible para el código que crea objetos a partir de ella.</span><span class="sxs-lookup"><span data-stu-id="63341-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="63341-105">Todos los tipos de valor tienen constructores sin parámetros, pero no todos los tipos de referencia los tienen.</span><span class="sxs-lookup"><span data-stu-id="63341-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="63341-106">Por lo tanto, la restricción `Structure` satisface la restricción `New` , pero la restricción `Class` o el nombre de clase o interfaz no.</span><span class="sxs-lookup"><span data-stu-id="63341-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>  
  
 <span data-ttu-id="63341-107">Las instrucciones siguientes pueden generar este error.</span><span class="sxs-lookup"><span data-stu-id="63341-107">The following statements can generate this error.</span></span>  
  
```  
Public Class c1(Of t As New)  
End Class  
Public Class c2(Of u)  
    Public testObject As New c1(Of u)  
End Class  
```  
  
 <span data-ttu-id="63341-108">Si la clase `c2` crea un objeto de `c1`, el parámetro de tipo `u` no satisface la restricción `New` en el parámetro de tipo `t`.</span><span class="sxs-lookup"><span data-stu-id="63341-108">When class `c2` creates an object from `c1`, type parameter `u` does not satisfy the `New` constraint on type parameter `t`.</span></span>  
  
 <span data-ttu-id="63341-109">**Identificador de error:** BC32084</span><span class="sxs-lookup"><span data-stu-id="63341-109">**Error ID:** BC32084</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="63341-110">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="63341-110">To correct this error</span></span>

