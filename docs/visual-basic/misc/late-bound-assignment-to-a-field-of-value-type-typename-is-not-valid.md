# <a name="late-bound-assignment-to-a-field-of-value-type-lttypenamegt-is-not-valid-when-ltnamegt-is-the-result-of-a-late-bound-expression"></a>Asignación de tiempo de ejecución a un campo de tipo de valor '&lt;typename&gt;'no es válida cuando'&lt;nombre&gt;' es el resultado de una expresión enlazada en tiempo de ejecución
Intentó realizar una asignación enlazada en tiempo de ejecución que no es válida.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Realice la asignación en tiempo de compilación.
