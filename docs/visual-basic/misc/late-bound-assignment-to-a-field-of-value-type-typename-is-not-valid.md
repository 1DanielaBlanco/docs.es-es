# <a name="late-bound-assignment-to-a-field-of-value-type-39lttypenamegt39-is-not-valid-when-39ltnamegt39-is-the-result-of-a-late-bound-expression"></a>Asignación de tiempo de ejecución a un campo de tipo de valor &#39; &lt;typename&gt;&#39; no es válido cuando &#39;&lt; nombre&gt;&#39; es el resultado de una expresión en tiempo de ejecución
Intentó realizar una asignación enlazada en tiempo de ejecución que no es válida.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Realice la asignación en tiempo de compilación.
