# <a name="cannot-inherit-interface-39ltinterfacename1gt39-because-the-interface-39ltinterfacename2gt39-from-which-it-inherits-could-be-identical-to-interface-39ltinterfacename3gt39-from-which-the-interface-39ltinterfacename4gt39-inherits-for-some-type-arguments"></a><span data-ttu-id="bef18-102">No se puede heredar la interfaz &#39; &lt;interfacename1&gt;&#39; porque la interfaz &#39;&lt; interfacename2&gt;&#39; del que hereda podría ser idéntica a la interfaz &#39;&lt; interfacename3&gt;&#39; desde el que la interfaz &#39;&lt; interfacename4&gt;&#39; hereda para algunos argumentos de tipo</span><span class="sxs-lookup"><span data-stu-id="bef18-102">Cannot inherit interface &#39;&lt;interfacename1&gt;&#39; because the interface &#39;&lt;interfacename2&gt;&#39; from which it inherits could be identical to interface &#39;&lt;interfacename3&gt;&#39; from which the interface &#39;&lt;interfacename4&gt;&#39; inherits for some type arguments</span></span>
<span data-ttu-id="bef18-103">Una interfaz genérica hereda de dos o más interfaces genéricas, y dos de las herencias podrían entrar en conflicto para ciertos valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="bef18-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="bef18-104">Las instrucciones siguientes pueden generar este error.</span><span class="sxs-lookup"><span data-stu-id="bef18-104">The following statements can generate this error.</span></span>  
  
```  
Public Interface interfaceA(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
    Inherits interfaceA(Of v)  
End Interface  
Public Interface interfaceY(Of w)  
    Inherits interfaceA(Of w)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceX(Of t1), interfaceY(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="bef18-105">Si `derivedInterface` se construye o se implementa al proporcionar el mismo tipo tanto a `t1` como a `t2`, debe heredar dos versiones de `interfaceA` con argumentos de tipo idénticos.</span><span class="sxs-lookup"><span data-stu-id="bef18-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceA` with identical type arguments.</span></span> <span data-ttu-id="bef18-106">Esto produciría ambigüedad respecto a la versión a la que se debe acceder.</span><span class="sxs-lookup"><span data-stu-id="bef18-106">Doing so would produce an ambiguity about which version to access.</span></span>  
  
 <span data-ttu-id="bef18-107">**Id. de error:** BC32122</span><span class="sxs-lookup"><span data-stu-id="bef18-107">**Error ID:** BC32122</span></span>
