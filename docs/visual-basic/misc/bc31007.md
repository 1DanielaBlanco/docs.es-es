# <a name="unable-to-open-module-file-39ltfilenamegt39-lterrorgt"></a>No se puede abrir el archivo de módulo &#39; &lt;filename&gt;&#39;: &lt;error&gt;
No se pudo abrir un archivo que contiene un módulo de este proyecto.  
  
 **Identificador de error:** BC31007  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe que el archivo está en la ubicación especificada y no está bloqueado por otra aplicación.
