# <a name="ltmethodgt-39ltmethodname1gt39-must-be-declared-39overloads39-because-another-39ltmethodname2gt39-is-declared-39overloads39"></a>&lt;método&gt; &#39; &lt;nombredemétodo1&gt; &#39; debe declararse &#39;sobrecargas&#39; porque otro &#39; &lt;nombredemétodo2&gt; &#39; se declara &#39; Sobrecargas&#39;
Otra propiedad u otro método con el mismo nombre se declaró con el modificador `Overloads` .  
  
 **Identificador de error:** BC31409  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Agregue el modificador `Overloads` a la propiedad o al método.  
  
2.  Quite el modificador `Overloads` de todas las propiedades o los métodos con el mismo nombre.
