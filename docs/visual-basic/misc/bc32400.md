# <a name="class-39ltclassnamegt39-could-not-be-created-lterrorgt"></a>La clase&lt;classname&gt;' no se pudo crear: &lt;error&gt;
Error de inicialización de una clase. Una posible causa es un dañado [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] instalación.  
  
 **Identificador de error:** BC32400  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Examine el mensaje de error indicado y tome las medidas adecuadas.  
  
2.  Vuelva a instalar el [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort_md.md)] y vuelva a compilar el programa.  
  
3.  Si el error se repite, reinstale el compilador de [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)].  
  
4.  Si el error persiste después de la reinstalación, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
