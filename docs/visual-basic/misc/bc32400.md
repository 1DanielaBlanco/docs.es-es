# No se pudo crear la clase &#39;&lt;nombreClase&gt;&#39;: &lt;error&gt;
Error de inicialización de una clase. Una posible causa es una instalación de [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] dañada.  
  
 **Identificador de error:** BC32400  
  
### Para corregir este error  
  
1.  Examine el mensaje de error indicado y tome las medidas adecuadas.  
  
2.  Reinstale [!INCLUDE[dnprdnshort](../../csharp/getting-started/includes/dnprdnshort-md.md)] y vuelva a compilar el programa.  
  
3.  Si el error se repite, reinstale el compilador de [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
4.  Si el error persiste después de la reinstalación, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
