# <a name="implicit-conversion-from-39lttypename1gt39-to-39lttypename2gt39"></a>Conversión implícita de '&lt;NombreTipo1&gt;'to'&lt;en NombreTipo2&gt;'
Una expresión o una instrucción de asignación toma un valor de un tipo de datos y lo convierte a otro tipo. Dado que no se usa ninguna palabra clave de conversión, la conversión es *implícita*.  
  
 De forma predeterminada, este mensaje es una advertencia. Para obtener información sobre cómo ocultar las advertencias o cómo tratarlas como errores, vea [Configuring Warnings in Visual Basic](https://docs.microsoft.com/visualstudio/ide/configuring-warnings-in-visual-basic).  
  
 **Identificador de error:** BC42016  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Si es posible, utilizar valores del mismo tipo de datos, de modo que [!INCLUDE[vbprvb](../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] no es necesario realizar ninguna conversión.  
  
-   Use `CType` o una de las otras palabras clave de conversión para que la conversión sea *explícita*.
