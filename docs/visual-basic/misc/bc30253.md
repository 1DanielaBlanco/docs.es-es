# <a name="39interface39-must-end-with-a-matching-39end-interface39"></a>&#39;Interfaz&#39; debe terminar con una coincidencia &#39;End Interface&#39;
Una instrucción `Interface` aparece sin ninguna instrucción `End Interface` correspondiente. Debe usar una instrucción `End Interface` para finalizar la definición de interfaz.  
  
 **Identificador de error:** BC30253  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe que los miembros `Interface` tengan el formato correcto.  
  
2.  Agregue una instrucción `End Interface` al final de la definición de interfaz.
