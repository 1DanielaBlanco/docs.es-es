# La versi&#243;n de destino de .NET Compact Framework no admite el uso del modificador Ansi, Auto o Unicode.
La versión de .NET Compact Framework con la que está trabajando no admite los modificadores `Ansi`, `Auto` y `Unicode`.  
  
 **Identificador de error:** BC30763  
  
### Para corregir este error  
  
-   Quite el modificador.
