# <a name="39if39-operands-cannot-be-named-arguments"></a><span data-ttu-id="215b5-102">&#39;Si&#39; operandos no pueden ser argumentos con nombre</span><span class="sxs-lookup"><span data-stu-id="215b5-102">&#39;If&#39; operands cannot be named arguments</span></span>

```

```
