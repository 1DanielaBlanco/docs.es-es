# <a name="unable-to-create-assembly-linker-object-lterror-messagegt"></a>No se puede crear el objeto Assembly Linker: &lt;mensaje de error&gt;
El compilador de [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] llama a Assembly Linker (Al.exe, también denominado Alink) para generar un ensamblado con un manifiesto. El enlazador informa de un error al crear el objeto de trabajo. Podría tratarse de un problema al generar una referencia a un archivo de metadatos o un error al llamar a un archivo de importación.  
  
 **Identificador de error:** BC30141  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Examine el mensaje de error citado y consulte el tema [Errores y advertencias de la herramienta Al.exe](http://msdn.microsoft.com/en-us/7f125d49-0a03-47a6-9ba9-d61a679a7d4b) para obtener una explicación más detallada y consejos.  
  
2.  Si el error persiste, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
