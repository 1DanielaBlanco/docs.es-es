# Se esperaba coma o &#39;)&#39; (lista de argumentos de tipo)
Una lista de argumentos de tipo contiene un elemento inesperado. Use comas \(`,`\) para separar los argumentos de tipo en esta clase de listas y use un paréntesis de cierre \(`)`\) al final de la lista.  
  
 **Identificador de error:** BC32094  
  
### Para corregir este error  
  
-   Agregue una coma y continúe la lista de argumentos de tipo o agregue un paréntesis de cierre para terminar la lista.
