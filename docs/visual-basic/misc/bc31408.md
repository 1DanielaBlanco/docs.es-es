# <a name="specifier1-and-specifier2-cannot-be-combined"></a>'\<specifier1 >' y '\<specifier2 >' no se pueden combinar
Dos especificadores compatibles, como `Shadows` y `Overrides` se usan juntos.  
  
 **Identificador de error:** BC31408  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite un especificador u otro.
