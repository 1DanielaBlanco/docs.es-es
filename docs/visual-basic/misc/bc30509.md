# <a name="ltspecifier1gt-lttypegt-cannot-inherit-from-a-ltspecifier2gt-lttypegt-because-it-expands-the-access-of-the-base-lttypegt"></a>&lt;especificador1&gt; &lt;tipo&gt; no puede heredar de un &lt;especificador2&gt; &lt;tipo&gt; porque amplía el acceso de la base de &lt;tipo&gt;
Ha intentado que una clase pública herede de una clase base marcada como `Private` o `Friend`.  
  
 **Identificador de error:** BC30509  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare la clase base `Public` o declare la clase de herencia `Private` o `Friend`.
