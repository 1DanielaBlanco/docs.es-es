# <a name="too-many-type-arguments-to-genericmethodname"></a>Demasiados argumentos de tipo para '\<Nombredemétodogenérico >'
Se invocó un método genérico con más argumentos de tipo que los parámetros de tipo que hay.  
  
 Al invocar un método genérico, debe proporcionar un argumento de tipo para cada parámetro de tipo definido por el método.  
  
 **Identificador de error:** BC32043  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite argumentos de tipo de la lista de argumentos de tipo para que haya uno para cada parámetro de tipo del método genérico que se está invocando.
