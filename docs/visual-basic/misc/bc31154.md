# <a name="xml-declaration-does-not-allow-attribute-39attributename39"></a>Declaración XML no admite el atributo &#39;attributeName&#39;
Se ha especificado un atributo no válido para un literal de documento XML. Los valores de atributo válido para un literal de documento XML se basan en la especificación XML 1.0.  
  
 **Identificador de error:** BC31154  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el atributo no válido del literal de documento XML.
