# <a name="loop-control-variable-of-type-lttypenamegt-does-not-implement-the-systemicomparable-interface"></a>Variable de control de bucles de tipo '&lt;typename&gt;' no implementa la interfaz 'System.IComparable'
Una variable es de un tipo que no implementa la interfaz `System.IComparable` . `System.IComparable` debe implementarse mediante un tipo de valor o una clase.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que la variable es un tipo de valor.
