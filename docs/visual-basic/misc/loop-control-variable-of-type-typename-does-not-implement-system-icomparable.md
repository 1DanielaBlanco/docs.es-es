# <a name="loop-control-variable-of-type-39lttypenamegt39-does-not-implement-the-39systemicomparable39-interface"></a>Variable de control de bucles de tipo &#39; &lt;typename&gt; &#39; no implementa la &#39;System.IComparable&#39; (interfaz)
Una variable es de un tipo que no implementa la interfaz `System.IComparable` . `System.IComparable` debe implementarse mediante un tipo de valor o una clase.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Asegúrese de que la variable es un tipo de valor.
