# <a name="property-39ltpropertynamegt39-is-of-an-unsupported-type"></a>Propiedad &#39; &lt;propertyname&gt;&#39; es un tipo no admitido
Una propiedad de un ensamblado de referencia contiene un tipo que no se puede representar.  
  
 **Identificador de error:** BC30643  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Especifique un tipo que se pueda usar con la aplicación.
