# <a name="the-value-ltvaluegt-is-invalid-for-option-ltoptionnamegt"></a>el valor '&lt;valor&gt;'no es válido para la opción'&lt;optionname&gt;'
Se proporcionó un valor para una opción de línea de comandos que no era válida.  
  
 **Identificador de error:** BC2014  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe los valores para encontrar el que no es válido.
