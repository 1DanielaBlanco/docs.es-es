# <a name="lambda-expression-cannot-be-converted-to-lttypenamegt-because-lttypenamegt-is-not-a-delegate-type"></a><span data-ttu-id="50a85-102">Expresión lambda no se puede convertir a '&lt;typename&gt;'porque'&lt;typename&gt;' no es un tipo delegado</span><span class="sxs-lookup"><span data-stu-id="50a85-102">Lambda expression cannot be converted to '&lt;typename&gt;' because '&lt;typename&gt;' is not a delegate type</span></span>
<span data-ttu-id="50a85-103">Las expresiones lambda se pueden usar cuando los delegados son válidos.</span><span class="sxs-lookup"><span data-stu-id="50a85-103">Lambda expressions can be used where delegates are valid.</span></span> <span data-ttu-id="50a85-104">Se pueden convertir a tipos de delegado compatibles, pero no a ningún otro tipo.</span><span class="sxs-lookup"><span data-stu-id="50a85-104">They can be converted to compatible delegate types, but not to any other type.</span></span> <span data-ttu-id="50a85-105">Por ejemplo, puede definir un tipo de delegado y asignarle una expresión lambda o enviar una expresión lambda como argumento para un parámetro <xref:System.Func%601> .</span><span class="sxs-lookup"><span data-stu-id="50a85-105">For example, you can define a delegate type and assign a lambda expression to it, or send a lambda expression as the argument to a <xref:System.Func%601> parameter.</span></span> <span data-ttu-id="50a85-106">Estos ejemplos se muestran en el código siguiente.</span><span class="sxs-lookup"><span data-stu-id="50a85-106">These examples are shown in the following code.</span></span>  
  
```vb  
Module Module1  
  
    Delegate Function FunDel(ByVal m As Integer) As Boolean  
  
    Sub Main()  
  
        ' Assign a lambda expression to a function delegate.  
        Dim negative As FunDel = Function(n As Integer) n < 0  
        Console.WriteLine(negative(-3))  
  
        ' Send a lambda as the argument to a delegate parameter.  
        Dim numbers() As Integer = {3, 4, 2, 8, 1, 0, 9, 13, 42}  
        Dim evens = numbers.Where(Function(n) n Mod 2 = 0)  
  
        For Each even In evens  
            Console.WriteLine(even)  
        Next  
  
    End Sub  
  
End Module  
```
