# <a name="ltmodifiergt-is-not-valid-on-a-declare"></a>'&lt;modificador&gt;' no es válido en una instrucción Declare
Una instrucción `Declare` contiene una palabra clave no válida, como `ReadOnly`.  
  
 **Identificador de error:** BC30244  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la palabra clave no válida de la instrucción `Declare` .
