# <a name="property-ltpropertynamegt-must-be-declared-default-because-it-overrides-a-default-property-on-the-base-ltnamegt"></a>Propiedad '&lt;propertyname&gt;'debe declararse 'Default' porque reemplaza una propiedad predeterminada en la base'&lt;nombre&gt;'
Una instrucción `Property` del código reemplaza una propiedad `Default` en la clase base, la estructura o la interfaz y, por tanto, debe declararse como `Default`.  
  
 **Identificador de error:** BC30360  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare la propiedad `Default`.
