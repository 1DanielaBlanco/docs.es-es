# <a name="property-39ltpropertynamegt39-must-be-declared-39default39-because-it-overrides-a-default-property-on-the-base-39ltnamegt39"></a>Propiedad &#39; &lt;propertyname&gt;&#39; debe declararse &#39; predeterminado &#39; porque reemplaza una propiedad predeterminada en la base de &#39;&lt; nombre&gt;&#39;
Una instrucción `Property` del código reemplaza una propiedad `Default` en la clase base, la estructura o la interfaz y, por tanto, debe declararse como `Default`.  
  
 **Id. de error:** BC30360  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Declare la propiedad `Default`.
