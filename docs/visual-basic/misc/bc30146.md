# <a name="unable-to-sign-assembly-lterror-messagegt"></a>No se puede firmar el ensamblado: &lt;mensaje de error&gt;
El compilador de Visual Basic llama a la herramienta Assembly Linker (Al.exe, también denominado Alink) para generar un ensamblado con un manifiesto. El enlazador ha informado de un error al firmar el ensamblado después de su creación. Puede deberse a una clave no válida.  
  
 **Identificador de error:** BC30146  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Examine el mensaje de error citado y consulte el tema [Al.exe (Assembly Linker)](../../framework/tools/al-exe-assembly-linker.md) para obtener más información y consejos.  
  
2.  Si el error persiste, reúna información sobre las circunstancias y notifíquelo a los Servicios de soporte técnico de Microsoft.
