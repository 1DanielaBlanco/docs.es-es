# <a name="39lteventnamegt39-is-not-an-event-of-39ltcontainernamegt39"></a>&#39; &lt;eventname&gt;&#39; no es un evento de &#39;&lt; ContainerName&gt;&#39;
El evento especificado no está declarado para el objeto.  
  
 **Identificador de error:** BC30676  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Compruebe que el nombre del evento esté bien escrito.  
  
2.  Compruebe que está accediendo al objeto correcto. Use referencias completas al objeto o la instrucción `Imports` para importar el espacio de nombres adecuado si es necesario.
