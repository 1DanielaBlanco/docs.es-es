# <a name="xml-comment-tag-39returns39-is-not-permitted-on-a-39writeonly39-property"></a>Etiqueta de comentario XML &#39;devuelve&#39; no se permite en un &#39;WriteOnly&#39; propiedad
No se permite la etiqueta de comentario XML 'returns' en una propiedad 'WriteOnly'. Se omitirá el comentario XML.  
  
 Un comentario XML de una declaración de propiedad de solo escritura no puede contener una \<devuelve > etiqueta.  
  
 **Identificador de error:** BC42313  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quitar no admitida \<devuelve > etiqueta.
