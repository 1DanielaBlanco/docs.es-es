# <a name="instance-of-restricted-type-39lttypenamegt39-cannot-be-used-in-a-query-expression"></a>Instancia de tipo restringido &#39; &lt;typename&gt; &#39; no se puede usar en una expresión de consulta
Las instancias de tipos restringidos como, por ejemplo, <xref:System.ArgIterator>, <xref:System.RuntimeArgumentHandle>y <xref:System.TypedReference>, no se pueden usar en una expresión lambda o una expresión de consulta.  
  
 **Identificador de error:** BC36598
