# <a name="shared-39sub-new39-cannot-have-any-parameters"></a>Compartido &#39; Sub New &#39; no puede tener ningún parámetro
Se han incluido uno o más parámetros en una instrucción `Sub New` compartida. Esto no es válido.  
  
 **Id. de error:** BC30479  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el parámetro o los parámetros.
