# <a name="shared-sub-new-cannot-have-any-parameters"></a>El constructor 'Sub New' compartido no puede tener ningún parámetro
Se han incluido uno o más parámetros en una instrucción `Sub New` compartida. Esto no es válido.  
  
 **Identificador de error:** BC30479  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el parámetro o los parámetros.
