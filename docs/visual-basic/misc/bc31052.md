# <a name="modules-cannot-be-declared-ltspecifiergt"></a>Los módulos no se pueden declarar '&lt;especificador&gt;'
Se declaró un módulo con un modificador de acceso no válido.  
  
 **Identificador de error:** BC31052  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Reemplace el modificador de acceso por un modificador de acceso válido, como `Public` o `Friend`.
