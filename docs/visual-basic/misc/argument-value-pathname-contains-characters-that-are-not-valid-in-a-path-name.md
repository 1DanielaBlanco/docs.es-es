# <a name="argument-value-ltpathnamegt-contains-characters-that-are-not-valid-in-a-path-name"></a>El valor del argumento '&lt;pathname&gt;' contiene caracteres que no son válidos en el nombre de ruta de acceso
Un nombre de ruta de acceso contiene uno o varios caracteres no válidos.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite el carácter del nombre de ruta de acceso.
