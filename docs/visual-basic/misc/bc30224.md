# <a name="39is39-expected"></a>&#39; es &#39; se esperaba
Un operador `TypeOf` aparece sin cláusula `Is` .  
  
 **Identificador de error:** BC30224  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue una cláusula `Is` después del operador `TypeOf` ; por ejemplo, `TypeOf MyControl Is ComboBox`.
