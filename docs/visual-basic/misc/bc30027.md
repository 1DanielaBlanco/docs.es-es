# <a name="39end-function39-expected"></a>&#39;End (función)&#39; esperado
Una instrucción `End` debe coincidir con el bloque que termina. Una definición de procedimiento `Function` no se terminó correctamente.  
  
 **Identificador de error:** BC30027  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue una instrucción `End Function` al final del procedimiento.
