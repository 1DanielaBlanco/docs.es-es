# <a name="39finally39-statements-are-not-valid-in-the-immediate-window"></a>&#39;Por último,&#39; instrucciones no son válidas en la ventana Inmediato.
Las instrucciones`Finally` solo pueden usarse en el código fuente.  
  
 **Id. de error:** BC30716  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite las instrucciones `Finally` del código de depuración.
