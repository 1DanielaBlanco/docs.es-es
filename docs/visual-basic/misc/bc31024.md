# <a name="unable-to-create-a-net-runtime-interface-lterrorgt"></a>No se puede crear una interfaz en tiempo de ejecución. NET: &lt;error&gt;
Hubo un error interno del compilador al crear una interfaz o un error de llamada a un método en una interfaz.  
  
 **Identificador de error:** BC31024  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Guarde el trabajo y reinicie [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
2.  Si el error se repite, reinstale [!INCLUDE[vsprvs](~/includes/vsprvs-md.md)].  
  
3.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
