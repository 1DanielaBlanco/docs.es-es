# <a name="lterrorgt-39ltstructurename1gt39-contains-39ltstructurename2gt39"></a>&lt;error&gt;: &#39;&lt; structurename1&gt;&#39; contiene &#39;&lt; structurename2&gt;&#39;
Una definición de estructura circular se produce cuando una estructura se designa como uno de sus miembros.  
  
 **Id. de error:** BC30293  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el nombre del miembro de la estructura.
