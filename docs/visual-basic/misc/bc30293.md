# <a name="lterrorgt-ltstructurename1gt-contains-ltstructurename2gt"></a>&lt;error&gt;: '&lt;structurename1&gt;"contains"&lt;structurename2&gt;'
Una definición de estructura circular se produce cuando una estructura se designa como uno de sus miembros.  
  
 **Identificador de error:** BC30293  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el nombre del miembro de la estructura.
