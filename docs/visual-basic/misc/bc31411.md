# <a name="39lttypenamegt39-must-be-declared-39mustinherit39-because-it-contains-methods-declared-39mustoverride39"></a>&#39;&lt;TypeName&gt; &#39; debe declararse &#39;MustInherit&#39; porque contiene los métodos declarados &#39;MustOverride&#39;
Los tipos que contienen miembros `MustOverride` deben declararse como `MustInherit`.  
  
 **Identificador de error:** BC31411  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue el modificador `MustInherit` al tipo.
