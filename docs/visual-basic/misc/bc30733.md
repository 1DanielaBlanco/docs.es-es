# <a name="39ltmembernamegt39-is-already-declared-by-39ltprocedurenamegt39-which-was-generated-for-this-ltcontainernamegt"></a>&#39;&lt;MemberName&gt; &#39; ya se declaró por &#39; &lt;nombreProcedimiento&gt;&#39;, que se generó para este &lt;containername&gt;
Se declaró un miembro que tiene el mismo nombre que un miembro que había declarado Common Language Runtime.  
  
 **Id. de error:** BC30733  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Cambie el nombre del miembro para quitar el conflicto.
