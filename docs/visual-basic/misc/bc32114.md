# <a name="type-parameter-with-a-39structure39-constraint-cannot-be-used-as-a-constraint"></a><span data-ttu-id="7c309-102">Parámetro de tipo con un &#39; Estructura &#39; restricción no se puede usar como una restricción</span><span class="sxs-lookup"><span data-stu-id="7c309-102">Type parameter with a &#39;Structure&#39; constraint cannot be used as a constraint</span></span>
<span data-ttu-id="7c309-103">Un parámetro de tipo con una restricción `Structure` se usa como restricción para otro parámetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="7c309-103">A type parameter with a `Structure` constraint is used as the constraint for another type parameter.</span></span>  
  
 <span data-ttu-id="7c309-104">La restricción `Structure` necesita que el argumento de tipo pasado a su parámetro de tipo sea un tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="7c309-104">The `Structure` constraint requires that the type argument passed to its type parameter be a value type.</span></span> <span data-ttu-id="7c309-105">Sin embargo, un tipo de valor no puede implementarse ni heredarse, por lo que no tiene sentido usarlo como restricción, que necesitaría el otro parámetro de tipo para implementarlo o heredarlo.</span><span class="sxs-lookup"><span data-stu-id="7c309-105">However, a value type cannot be implemented or inherited, so it is not meaningful to use it as a constraint, which would require the other type parameter to implement it or inherit from it.</span></span>  
  
 <span data-ttu-id="7c309-106">La única interpretación significativa de esta situación es que ambos argumentos de tipo deben ser del mismo tipo exacto.</span><span class="sxs-lookup"><span data-stu-id="7c309-106">The only meaningful interpretation of this situation is that both type arguments must be of the exact same type.</span></span> <span data-ttu-id="7c309-107">Si ese es el caso, el tipo genérico necesita solo un parámetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="7c309-107">If that is the case, your generic type needs only one type parameter.</span></span>  
  
 <span data-ttu-id="7c309-108">La instrucción siguiente puede generar este error.</span><span class="sxs-lookup"><span data-stu-id="7c309-108">The following statement can generate this error.</span></span>  
  
 `Class c1(Of t1 As Structure, t2 As t1)`  
  
 <span data-ttu-id="7c309-109">El tipo pasado a `t2` no puede implementar ni heredar el tipo pasado a `t1`, porque el tipo pasado a `t1` debe ser un tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="7c309-109">The type passed to `t2` cannot implement or inherit the type passed to `t1`, because the type passed to `t1` must be a value type.</span></span>  
  
 <span data-ttu-id="7c309-110">**Identificador de error:** BC32114</span><span class="sxs-lookup"><span data-stu-id="7c309-110">**Error ID:** BC32114</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7c309-111">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="7c309-111">To correct this error</span></span>  
  
-   <span data-ttu-id="7c309-112">Elimine el parámetro de tipo restringido a `Structure` de la lista de restricciones en el otro parámetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="7c309-112">Remove the type parameter constrained to `Structure` from the constraint list on the other type parameter.</span></span>  
  
-   <span data-ttu-id="7c309-113">Si ambos parámetros de tipo necesitan el mismo tipo de valor, defina el tipo genérico con un solo parámetro de tipo.</span><span class="sxs-lookup"><span data-stu-id="7c309-113">If both type parameters require the same value type, define the generic type with only one type parameter.</span></span>
