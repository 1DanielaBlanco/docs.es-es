# <a name="security-attribute-ltattributenamegt-cannot-be-applied-to-a-module"></a>Atributo de seguridad '&lt;attributename&gt;' no se puede aplicar a un módulo
Common Language Runtime no acepta el atributo de seguridad en un módulo.

**Identificador de error:** BC30131

## <a name="to-correct-this-error"></a>Para corregir este error

- Quite el atributo de la declaración.
