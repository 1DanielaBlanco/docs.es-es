# <a name="39continue39-statements-are-not-valid-in-the-immediate-window"></a>Las instrucciones 'Continue' no son válidas en la ventana Inmediato
`Continue`instrucciones sólo pueden usarse en código fuente.  
  
 **Identificador de error:** BC30780  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la instrucción `Continue` del código de depuración.
