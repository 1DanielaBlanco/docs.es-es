# Las instrucciones &#39;Continue&#39; no son v&#225;lidas en la ventana Inmediato.
Las instrucciones `Continue` solo pueden usarse en el código fuente.  
  
 **Identificador de error:** BC30780  
  
### Para corregir este error  
  
-   Quite la instrucción `Continue` del código de depuración.
