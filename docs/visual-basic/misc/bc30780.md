# <a name="continue-statements-are-not-valid-in-the-immediate-window"></a>Las instrucciones 'Continue' no son válidas en la ventana Inmediato.
Las instrucciones 'Continue' solo pueden usarse en el código fuente.  
  
 **Identificador de error:** BC30780  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Quite la instrucción 'Continue' del código de depuración.
