# <a name="security-attribute-39ltattributenamegt39-is-not-valid-lterrorgt"></a>Atributo de seguridad &#39; &lt;attributename&gt;&#39; no es válido: &lt;error&gt;
Common Language Runtime no acepta el atributo en un contexto de seguridad.

**Identificador de error:** BC30128

## <a name="to-correct-this-error"></a>Para corregir este error

1. Quite el atributo de la declaración o reemplácelo por un atributo que sea válido para el contexto.

2. Examine el mensaje de error indicado y tome las medidas adecuadas.
