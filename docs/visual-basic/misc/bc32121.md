<span data-ttu-id="5ae4d-103">Una interfaz genérica hereda de dos o más interfaces genéricas, y dos de las herencias podrían entrar en conflicto para ciertos valores de argumentos de tipo.</span><span class="sxs-lookup"><span data-stu-id="5ae4d-103">A generic interface inherits from two or more generic interfaces, and two of the inheritances could conflict for certain values of type arguments.</span></span>  
  
 <span data-ttu-id="5ae4d-104">Las instrucciones siguientes pueden generar este error.</span><span class="sxs-lookup"><span data-stu-id="5ae4d-104">The following statements can generate this error.</span></span>  
  
```

    Inherits interfaceX(Of u)  
End Interface  
Public Interface interfaceX(Of v)  
End Interface  
Public Interface derivedInterface(Of t1, t2)  
    Inherits interfaceA(Of t1), interfaceX(Of t2)  
End Interface  
```  
  
 <span data-ttu-id="5ae4d-105">Si `derivedInterface` se construye o implementa suministrando el mismo tipo para `t1` y `t2`, debe heredar dos versiones de `interfaceX` con argumentos de tipo idénticos.</span><span class="sxs-lookup"><span data-stu-id="5ae4d-105">If `derivedInterface` is constructed or implemented supplying the same type to both `t1` and `t2`, it must inherit two versions of `interfaceX` with identical type arguments.</span></span> <span data-ttu-id="5ae4d-106">Esto produciría ambigüedad sobre a qué versión se debe acceder.</span><span class="sxs-lookup"><span data-stu-id="5ae4d-106">Doing so would produce an ambiguity about which version to access.</span></span>
