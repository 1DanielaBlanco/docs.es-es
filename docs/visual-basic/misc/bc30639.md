# <a name="properties-cannot-be-declared-ltmodifiernamegt"></a>No se puede declarar propiedades '&lt;modifiername&gt;'
Hay un error en la sintaxis usada en la declaración `Property` . Este error suele indicar que una palabra clave determinada, como `WithEvents`, no se puede usar en una declaración `Property` .  
  
 **Identificador de error:** BC30639  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe la sintaxis de la declaración `Property` .
