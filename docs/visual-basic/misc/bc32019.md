# <a name="39resume39-or-39goto39-expected"></a>&#39;Reanudar&#39; o &#39;GoTo&#39; esperado
Una instrucción `On Error` no contiene una cláusula `Resume` ni una cláusula `GoTo` . Debe usar una cláusula o la otra.  
  
 **Identificador de error:** BC32019  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Agregue una cláusula `Resume` o una cláusula `GoTo` a la instrucción `On Error` .
