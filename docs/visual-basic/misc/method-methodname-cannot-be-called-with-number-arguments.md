# <a name="method-39ltmethodnamegt39-cannot-be-called-with-ltnumbergt-arguments"></a>Método &#39; &lt;methodname&gt; &#39; no se puede llamar con &lt;número&gt; argumentos
Una llamada de método contiene un número de argumentos incorrecto.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Cuente los argumentos que ha proporcionado.  
  
2.  Haga coincidir el número con el número de parámetros en el método que quiere llamar.  
  
3.  Ajuste el número de argumentos si los dos números no coinciden.
