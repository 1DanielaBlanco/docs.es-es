# Disco no preparado
No hay ningún disco para la unidad especificada o la unidad está abierta.  
  
### Para corregir este error  
  
1.  Compruebe la unidad especificada e inserte un disco si es necesario.  
  
2.  Cierre la puerta de la unidad.
