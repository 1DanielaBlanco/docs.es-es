# <a name="overload-resolution-failed-because-no-accessible-ltmethodgt-can-be-called-with-these-argumentslterrorgt"></a>Error en la resolución de sobrecarga no accesible '&lt;método&gt;' se puede llamar con estos argumentos:&lt;error&gt;
Ha realizado una llamada a un método sobrecargado, pero el compilador no encuentra ningún método con una lista de argumentos coincidente.  
  
 **Identificador de error:** BC30518  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe la lista de argumentos proporcionada.
