<span data-ttu-id="e8541-103">Se ha especificado un argumento de tipo en una llamada a un método de extensión que no tiene parámetros genéricos o no tiene parámetros genéricos para los que aún no se ha especificado un tipo.</span><span class="sxs-lookup"><span data-stu-id="e8541-103">A type argument has been specified in a call to an extension method that either has no generic parameters or has no generic parameters for which a type is not already specified.</span></span> <span data-ttu-id="e8541-104">Por ejemplo, el código siguiente causa este error.</span><span class="sxs-lookup"><span data-stu-id="e8541-104">For example, the following code causes this error.</span></span>  
  
```vb  
' The extension method is not generic.  
<Extension()> _

    ' Body of the Sub.  
End Sub  
```  
  
```vb  
Dim str = "hi"  
'' The call to Example specifies a type argument.  
'' Not valid.  
'str.Example(Of String)()  
```
