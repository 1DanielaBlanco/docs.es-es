# <a name="visual-basic-settings"></a>Configuración de Visual Basic
Configuración de Visual Basic está diseñada para optimizar la experiencia del usuario y maximizar la productividad de los desarrolladores de Visual Basic. Forma parte de la [personalizar la configuración de desarrollo de Visual Studio](http://msdn.microsoft.com/en-us/22c4debb-4e31-47a8-8f19-16f328d7dcd3) característica que le permite personalizar y guardar la configuración en el entorno de desarrollo integrado (IDE) de Visual Studio.  
  
 También puede trasladar o volver a cargar la configuración de IDE en otros equipos. Para obtener más información, consulte [Cómo: Share Settings Between Computers](http://msdn.microsoft.com/en-us/1131fb10-35c1-42da-9cd8-91aa3235b882).

## <a name="dialog-boxes"></a>Cuadros de diálogo  
  
|Característica|Comportamiento|
|---|---|  
|**Nuevo proyecto** cuadro de diálogo|Cuando se instalan otras plataformas en el equipo, el IDE habilita el cuadro combinado que permite especificar la plataforma de destino.|  
|[Opciones (cuadro de diálogo)](https://docs.microsoft.com/visualstudio/ide/reference/options-dialog-box-visual-studio)|Habilita Visual Basic simplificado **opciones** páginas. Haga clic en el **mostrar todas las configuraciones** casilla de verificación para ver la lista completa de opciones.|
