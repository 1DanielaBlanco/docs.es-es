# <a name="visual-basic-intellisense-code-snippets"></a>Fragmentos de código de IntelliSense de Visual Basic
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)]incluye una biblioteca de código que consta de varios cientos de elementos de código, denominadas fragmentos de código de IntelliSense, que están listos para insertarse en la aplicación. Cada fragmento de código realiza una tarea de programación completa, como crear una excepción personalizada, enviar un mensaje de correo electrónico o dibujar un círculo. Puede insertar un fragmento de código en el código fuente con unos pocos clics del mouse. También puede crear sus propios fragmentos de código que satisfagan sus necesidades comerciales, agregarlos a la biblioteca y usarlas cuando sea necesario. Para obtener más información, vea [Fragmentos de código](https://docs.microsoft.com/visualstudio/ide/code-snippets).  
  
## <a name="a-productivity-tool-for-visual-basic-users"></a>Una herramienta de productividad para usuarios de Visual Basic  
 Estos fragmentos de código aumentan la productividad al reducir la cantidad de tiempo que necesario para buscar ejemplos de código, reduciendo el tiempo necesario para aprender a utilizar características desconocidas y admitiendo la reutilización de código.  
  
 La biblioteca de código es compatible con las siguientes características:  
  
-   Inserte uno de los fragmentos de código en el **Editor de código**.  
  
-   Crear nuevas tareas que puede reutilizar en sus proyectos.  
  
-   Crear nuevas tareas para compartirlas con su grupo de trabajo y sus colegas.  
  
-   Editar las tareas.  
  
-   Descargar más tareas de terceros.  
  
 Estos bloques de código están disponibles a lo largo de [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)]:  
  
-   Puede agregarlos a través del menú contextual de la **Editor de código**.  
  
-   Puede arrastrar los archivos de código XML de **Explorador de archivos** en el archivo de código fuente.
