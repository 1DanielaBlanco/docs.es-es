# Acceso a archivos con Visual Basic
[!INCLUDE[vs2017banner](../../../../csharp/includes/vs2017banner.md)]

El objeto `My.Computer.FileSystem` proporciona herramientas para trabajar con archivos y carpetas.  Sus propiedades, métodos y eventos permiten crear, copiar, mover, examinar y eliminar archivos y carpetas.  `My.Computer.FileSystem` proporciona mejor rendimiento que las funciones heredadas \(`FileOpen`, `FileClose`, `Input`, `InputString`, `LineInput`, etc.\) que se proporcionan en [!INCLUDE[vbprvb](../../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] a efectos de compatibilidad con versiones anteriores.
