<span data-ttu-id="2d3e0-103">Los controles de Visual Basic PowerPacks líneas y formas son un conjunto de tres controles gráficos que le permiten dibujar líneas y formas en formularios y contenedores.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-103">The Visual Basic Power Packs Line and Shape controls are a set of three graphical controls that enable you to draw lines and shapes on forms and containers.</span></span> <span data-ttu-id="2d3e0-104">El <xref:Microsoft.VisualBasic.PowerPacks.LineShape> control se utiliza para dibujar líneas horizontales, verticales y diagonales.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-104">The <xref:Microsoft.VisualBasic.PowerPacks.LineShape> control is used to draw horizontal, vertical, and diagonal lines.</span></span> <span data-ttu-id="2d3e0-105">El <xref:Microsoft.VisualBasic.PowerPacks.OvalShape> control se utiliza para dibujar círculos y óvalos y el <xref:Microsoft.VisualBasic.PowerPacks.RectangleShape> control se utiliza para dibujar rectángulos y cuadrados.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-105">The <xref:Microsoft.VisualBasic.PowerPacks.OvalShape> control is used to draw circles and ovals, and the <xref:Microsoft.VisualBasic.PowerPacks.RectangleShape> control is used to draw rectangles and squares.</span></span>  
  
## <a name="line-and-shape-controls"></a><span data-ttu-id="2d3e0-106">Controles de líneas y formas</span><span class="sxs-lookup"><span data-stu-id="2d3e0-106">Line and Shape Controls</span></span>  
 <span data-ttu-id="2d3e0-107">Controles Line y Shape encapsulan muchos de los métodos de gráficos que se encuentran en el <xref:System.Drawing> espacio de nombres.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-107">Line and Shape controls encapsulate many of the graphics methods that are contained in the <xref:System.Drawing> namespace.</span></span> <span data-ttu-id="2d3e0-108">Esto le permite dibujar líneas y formas en un solo paso sin tener que crear pinceles, lápices y objetos de gráficos.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-108">This enables you to draw lines and shapes in a single step without having to create graphics objects, pens, and brushes.</span></span> <span data-ttu-id="2d3e0-109">Técnicas de gráficos complejos, como los degradados pueden realizarse con solo establecer algunas propiedades.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-109">Complex graphics techniques such as gradient fills can be accomplished by just setting some properties.</span></span>  
  
 <span data-ttu-id="2d3e0-110">Aunque también es posible dibujar líneas y formas utilizando métodos gráficos, hay varias ventajas con respecto al uso de los controles Line y Shape:</span><span class="sxs-lookup"><span data-stu-id="2d3e0-110">Although it is also possible to draw lines and shapes by using graphics methods, there are several advantages to using the Line and Shape controls:</span></span>  
  
-   <span data-ttu-id="2d3e0-111">Pueden llamar a métodos de gráficos en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-111">Graphics methods can be called only at run time.</span></span> <span data-ttu-id="2d3e0-112">Controles Line y Shape pueden agregarse a un formulario en tiempo de diseño.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-112">Line and Shape controls can be added to a form at design time.</span></span> <span data-ttu-id="2d3e0-113">Esto le permite ver su apariencia y colóquelos exactamente; También se pueden agregar en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-113">This enables you to see what they look like and to position them exactly; they can also be added at run time.</span></span>  
  
-   <span data-ttu-id="2d3e0-114">Controles Line y Shape son seleccionables en tiempo de ejecución proporcionar eventos como <xref:Microsoft.VisualBasic.PowerPacks.Shape.Click> y <xref:Microsoft.VisualBasic.PowerPacks.Shape.OnDoubleClick%2A>.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-114">Line and Shape controls are selectable at run time, providing events such as <xref:Microsoft.VisualBasic.PowerPacks.Shape.Click> and <xref:Microsoft.VisualBasic.PowerPacks.Shape.OnDoubleClick%2A>.</span></span> <span data-ttu-id="2d3e0-115">Las salidas de los métodos de gráficos no se pueden seleccionables y no proporcionan eventos.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-115">The outputs of graphics methods are not selectable and do not provide events.</span></span>  
  
-   <span data-ttu-id="2d3e0-116">Proporcionan controles Line y Shape <xref:Microsoft.VisualBasic.PowerPacks.Shape.BringToFront%2A> y <xref:Microsoft.VisualBasic.PowerPacks.Shape.SendToBack%2A> métodos que le permiten controlar el orden z en tiempo de diseño y en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-116">Line and Shape controls provide <xref:Microsoft.VisualBasic.PowerPacks.Shape.BringToFront%2A> and <xref:Microsoft.VisualBasic.PowerPacks.Shape.SendToBack%2A> methods that enable you to control their z-order at design time and at run time.</span></span> <span data-ttu-id="2d3e0-117">El orden z de los métodos de gráficos puede controlarse cambiando su orden de ejecución en tiempo de ejecución.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-117">The z-order of graphics methods can be controlled only by changing their order of execution at run time.</span></span>  
  
-   <span data-ttu-id="2d3e0-118">Controles Line y Shape son controles sin ventana; no tienen ningún identificador de ventana y, por tanto, utiliza menos recursos del sistema.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-118">Line and Shape controls are windowless controls; they have no window handles and therefore use less system resources.</span></span>  
  
### <a name="object-model"></a><span data-ttu-id="2d3e0-119">Modelo de objetos</span><span class="sxs-lookup"><span data-stu-id="2d3e0-119">Object Model</span></span>  
 <span data-ttu-id="2d3e0-120">Controles Line y Shape se derivan de una base de <xref:Microsoft.VisualBasic.PowerPacks.Shape> clase que define sus propiedades compartidas, métodos y eventos.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-120">Line and Shape controls derive from a base <xref:Microsoft.VisualBasic.PowerPacks.Shape> class that defines their shared properties, methods, and events.</span></span>  
  
 <span data-ttu-id="2d3e0-121">En la siguiente ilustración muestra la jerarquía de objetos de líneas y formas.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-121">The following illustration shows the Line and Shape object hierarchy.</span></span>

<span data-ttu-id="2d3e0-123">Jerarquía de objetos de línea y forma</span><span class="sxs-lookup"><span data-stu-id="2d3e0-123">Line and Shape object hierarchy</span></span>

 <span data-ttu-id="2d3e0-128">Cuando un control de línea o forma se agrega a un formulario o contenedor, un invisible <xref:Microsoft.VisualBasic.PowerPacks.ShapeContainer> se crea el objeto.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-128">When a Line or Shape control is added to a form or container, an invisible <xref:Microsoft.VisualBasic.PowerPacks.ShapeContainer> object is created.</span></span> <span data-ttu-id="2d3e0-129">El <xref:Microsoft.VisualBasic.PowerPacks.ShapeContainer> actúa como un lienzo para las formas dentro de cada control contenedor; cada <xref:Microsoft.VisualBasic.PowerPacks.ShapeContainer> le corresponde una <xref:Microsoft.VisualBasic.PowerPacks.ShapeCollection> que le permite recorrer en iteración los controles Line y Shape.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-129">The <xref:Microsoft.VisualBasic.PowerPacks.ShapeContainer> acts as a canvas for the shapes within each container control; each <xref:Microsoft.VisualBasic.PowerPacks.ShapeContainer> has a corresponding <xref:Microsoft.VisualBasic.PowerPacks.ShapeCollection> that enables you to iterate through the Line and Shape controls.</span></span> <span data-ttu-id="2d3e0-130">Puede mover formas de un contenedor a otro mediante cortar y pegar o arrastrar y colocar.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-130">You can move shapes from one container to another by using cut and paste or by dragging and dropping.</span></span> <span data-ttu-id="2d3e0-131">Cuando se quita la última forma de un contenedor, el <xref:Microsoft.VisualBasic.PowerPacks.ShapeContainer> también se quita.</span><span class="sxs-lookup"><span data-stu-id="2d3e0-131">When the last shape is removed from a container, the <xref:Microsoft.VisualBasic.PowerPacks.ShapeContainer> is removed also.</span></span>
