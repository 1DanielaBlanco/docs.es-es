# /help, /? (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Muestra las opciones del compilador.  
  
## Sintaxis  
  
```  
/help  
' -or-  
/?  
```  
  
## Comentarios  
 Si incluye esta opción en una compilación, no se creará ningún archivo de salida ni se producirá ningún proceso de compilación.  
  
> [!NOTE]
>  La opción `/help` no está disponible en el entorno de desarrollo de [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)]; solo está disponible cuando se compila desde la línea de comandos.  
  
## Ejemplo  
 La siguiente línea muestra ayuda desde la línea de comandos.  
  
```  
vbc /help  
```
