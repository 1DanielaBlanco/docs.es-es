# Visual Basic
[!INCLUDE[vs2017banner](../csharp/includes/vs2017banner.md)]

Visual Basic está diseñado para crear de manera productiva aplicaciones con seguridad de tipos orientadas a objetos. Visual Basic permite a los desarrolladores establecer como destino dispositivos móviles, web y Windows. Al igual que todos los lenguajes que tienen como destino Microsoft [!INCLUDE[dnprdnshort](../csharp/getting-started/includes/dnprdnshort_md.md)], los programas escritos en Visual Basic se benefician de la seguridad y la interoperabilidad entre lenguajes.  
  
 Esta generación de [!INCLUDE[vbprvb](../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] sigue ofreciendo una manera rápida y sencilla de crear aplicaciones basadas en [!INCLUDE[dnprdnshort](../csharp/getting-started/includes/dnprdnshort_md.md)].  
  
 Si no dispone de Visual Basic, en el sitio de [Visual Studio](https://www.visualstudio.com/products/free-developer-offers-vs) puede adquirir una versión de Visual Studio que incluye Visual Basic de forma gratuita.
