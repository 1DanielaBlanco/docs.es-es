# My.Computer.Clock (Objeto)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Proporciona las propiedades para obtener acceso a la hora local actual y a la hora universal coordinada \(equivalente a la hora media de Greenwich\) del reloj del sistema.  
  
## Comentarios  
 Para obtener información sobre los métodos y las propiedades del objeto de `My.Computer.Clock` , vea <xref:Microsoft.VisualBasic.Devices.Clock>.  
  
## Requisitos  
 **Espacio de nombres:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Clase:** <xref:Microsoft.VisualBasic.Devices.Clock>  
  
 **Ensamblado:** biblioteca en tiempo de ejecución de Visual Basic \(en Microsoft.VisualBasic.dll\)
