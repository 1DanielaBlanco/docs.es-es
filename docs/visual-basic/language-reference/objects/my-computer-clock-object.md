# <a name="mycomputerclock-object"></a>My.Computer.Clock (Objeto)
Proporciona propiedades para obtener acceso a la hora local actual y al Horario universal coordinado (equivalente a la Hora del meridiano de Greenwich) desde el reloj del sistema.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer.Clock` de objetos, consulte <xref:Microsoft.VisualBasic.Devices.Clock>.  
  
## <a name="requirements"></a>Requisitos  
 **Namespace:**<xref:Microsoft.VisualBasic.Devices>  
  
 **Clase:**<xref:Microsoft.VisualBasic.Devices.Clock>  
  
 **Ensamblado:** biblioteca de tiempo de ejecución de Visual Basic (en Microsoft.VisualBasic.dll)
