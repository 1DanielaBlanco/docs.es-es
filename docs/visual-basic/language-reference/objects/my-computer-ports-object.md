# <a name="mycomputerports-object"></a>My.Computer.Ports (Objeto)
Proporciona una propiedad y un método para tener acceso a los puertos serie del equipo.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer.Ports` objeto, vea <xref:Microsoft.VisualBasic.Devices.Ports>.</xref:Microsoft.VisualBasic.Devices.Ports>  
  
 Para obtener más información, consulte [acceso a los puertos del equipo](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md).  
  
> [!NOTE]
>  También puede utilizar los métodos y propiedades de la <xref:System.IO.Ports.SerialPort?displayProperty=fullName>clase para obtener acceso a los puertos serie del equipo.</xref:System.IO.Ports.SerialPort?displayProperty=fullName>
