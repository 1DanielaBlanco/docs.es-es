# My.Computer.Ports (Objeto)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Proporciona una propiedad y un método para obtener acceso a los puertos serie del equipo.  
  
## Comentarios  
 Para obtener información sobre los métodos y las propiedades del objeto de `My.Computer.Ports` , vea <xref:Microsoft.VisualBasic.Devices.Ports>.  
  
 Para obtener más información, vea [Acceso a los puertos del equipo](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-computer-s-ports.md).  
  
> [!NOTE]
>  También puede utilizar propiedades y métodos de la clase de <xref:System.IO.Ports.SerialPort?displayProperty=fullName> para tener acceso a los puertos serie del equipo.
