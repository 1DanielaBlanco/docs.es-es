# <a name="mycomputernetwork-object"></a>My.Computer.Network (Objeto)
Proporciona una propiedad, evento y métodos para interactuar con la red al que está conectado el equipo.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer.Network` de objetos, consulte <xref:Microsoft.VisualBasic.Devices.Network>.  
  
 Para obtener más información, consulte [realizar operaciones de red](../../../visual-basic/developing-apps/programming/computer-resources/performing-network-operations.md).  
  
## <a name="requirements"></a>Requisitos  
 **Namespace:** <xref:Microsoft.VisualBasic.Devices?displayProperty=nameWithType>  
  
 **Clase:** <xref:Microsoft.VisualBasic.Devices.Network>  
  
 **Ensamblado:** biblioteca en tiempo de ejecución de Visual Basic (en Microsoft.VisualBasic.dll)
