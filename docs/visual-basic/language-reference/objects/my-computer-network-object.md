# <a name="mycomputernetwork-object"></a>My.Computer.Network (Objeto)
Proporciona una propiedad, evento y métodos para interactuar con la red a la que está conectado el equipo.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer.Network` de objetos, consulte <xref:Microsoft.VisualBasic.Devices.Network>.  
  
 Para obtener más información, consulte [realizar operaciones de red](../../../visual-basic/developing-apps/programming/computer-resources/performing-network-operations.md).
