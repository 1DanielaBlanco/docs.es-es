# <a name="myuser-object"></a>My.User (Objeto)

 Para obtener información acerca de los métodos y propiedades de la `My.User` de objetos, consulte <xref:Microsoft.VisualBasic.ApplicationServices.User?displayProperty=nameWithType>.  
  
 Para obtener más información, consulte [acceso a datos de usuario](../../../visual-basic/developing-apps/programming/accessing-user-data.md).
