# <a name="myuser-object"></a>My.User (Objeto)
Proporciona acceso a información sobre el usuario actual.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.User` de objetos, consulte <xref:Microsoft.VisualBasic.ApplicationServices.User?displayProperty=nameWithType>.  
  
 Para obtener más información, consulte [acceso a los datos de usuario](../../../visual-basic/developing-apps/programming/accessing-user-data.md).  
  
## <a name="requirements"></a>Requisitos  
 **Ensamblado:** biblioteca en tiempo de ejecución de Visual Basic (en Microsoft.VisualBasic.dll)
