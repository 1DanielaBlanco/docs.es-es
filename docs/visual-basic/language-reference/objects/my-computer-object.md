# <a name="mycomputer-object"></a>My.Computer (Objeto)
Proporciona propiedades para manipular componentes del equipo, como el audio, el reloj, el teclado, el sistema de archivos, etc.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer` de objetos, consulte <xref:Microsoft.VisualBasic.Devices.Computer>. La clase base <xref:Microsoft.VisualBasic.Devices.ServerComputer> proporciona los miembros que están disponibles en todos los proyectos.  
  
## <a name="requirements"></a>Requisitos  
 **Namespace:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Clase:** <xref:Microsoft.VisualBasic.Devices.Computer> (la clase base <xref:Microsoft.VisualBasic.Devices.ServerComputer> proporciona los miembros que están disponibles en todos los proyectos).  
  
 **Ensamblado:** biblioteca de tiempo de ejecución de Visual Basic (en Microsoft.VisualBasic.dll)
