# My.Application.Log (Objeto)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Proporciona una propiedad y métodos para escribir información de eventos y excepciones en los agentes de escucha de registro de la aplicación.  
  
## Comentarios  
 Para obtener información acerca de los métodos y las propiedades del objeto `My.Application.Log`, vea <xref:Microsoft.VisualBasic.Logging.Log>.  
  
 Para obtener más información, consulta [Registrar información de la aplicación](../../../visual-basic/developing-apps/programming/log-info/logging-information-from-the-application.md).  
  
> [!NOTE]
>  También puede usar clases de .NET Framework para registrar información desde la aplicación. Para obtener más información, consulta [Tracing and Instrumenting Applications](../Topic/Tracing%20and%20Instrumenting%20Applications.md).
