# My.Computer.Info (Objeto)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Proporciona las propiedades para obtener información sobre la memoria del equipo, los ensamblados cargados, el nombre y el sistema operativo.  
  
## Comentarios  
 Para obtener información sobre las propiedades del objeto de `My.Computer.Info` , vea <xref:Microsoft.VisualBasic.Devices.ComputerInfo>.  
  
## Requisitos  
 **Espacio de nombres:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Clase:** <xref:Microsoft.VisualBasic.Devices.ComputerInfo>  
  
 **Ensamblado:** biblioteca en tiempo de ejecución de Visual Basic \(en Microsoft.VisualBasic.dll\)
