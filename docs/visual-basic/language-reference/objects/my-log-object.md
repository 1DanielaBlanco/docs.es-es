# My.Log (Objeto)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Proporciona una propiedad y métodos para escribir información sobre excepciones y eventos en los agentes de escucha de registro de la aplicación.  
  
## Comentarios  
 Para obtener información sobre los métodos y las propiedades del objeto de `My.Log` , vea <xref:Microsoft.VisualBasic.Logging.AspLog>.  
  
 El objeto de `My.Log` está disponible para las aplicaciones ASP.NET sólo.  Para las aplicaciones cliente, utilice [My.Application.Log \(Objeto\)](../../../visual-basic/language-reference/objects/my-application-log-object.md).  
  
## Requisitos  
 **Espacio de nombres:** <xref:Microsoft.VisualBasic.Logging>  
  
 **Clase:** <xref:Microsoft.VisualBasic.Logging.AspLog>
