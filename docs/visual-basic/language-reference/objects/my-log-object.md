# <a name="mylog-object"></a>My.Log (Objeto)
Proporciona una propiedad y métodos para escribir información de eventos y excepciones en los agentes de escucha de registro de la aplicación.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Log` de objetos, consulte <xref:Microsoft.VisualBasic.Logging.AspLog>.  
  
 La `My.Log` objeto está disponible para las aplicaciones ASP.NET. Para las aplicaciones de cliente, utilice [My.Application.Log (objeto)](../../../visual-basic/language-reference/objects/my-application-log-object.md).
