# <a name="mycomputermouse-object"></a>My.Computer.Mouse (Objeto)
Proporciona propiedades para obtener información sobre el formato y la configuración del mouse instalado en el equipo local.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer.Mouse` de objetos, consulte <xref:Microsoft.VisualBasic.Devices.Mouse>.  
  
 Para obtener más información, consulte [acceso al Mouse](../../../visual-basic/developing-apps/programming/computer-resources/accessing-the-mouse.md).  
  
## <a name="requirements"></a>Requisitos  
 **Namespace:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Clase:** <xref:Microsoft.VisualBasic.Devices.Mouse>  
  
 **Ensamblado:** biblioteca en tiempo de ejecución de Visual Basic (en Microsoft.VisualBasic.dll)
