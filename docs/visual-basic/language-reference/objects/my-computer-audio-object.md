# <a name="mycomputeraudio-object"></a>My.Computer.Audio (Objeto)
Proporciona métodos para reproducir sonidos.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer.Audio` de objetos, consulte <xref:Microsoft.VisualBasic.Devices.Audio>.  
  
 Para obtener más información, consulte [sonidos de juego](../../../visual-basic/developing-apps/programming/computer-resources/playing-sounds.md).  
  
## <a name="requirements"></a>Requisitos  
 **Namespace:** <xref:Microsoft.VisualBasic.Devices>  
  
 **Clase:** <xref:Microsoft.VisualBasic.Devices.Audio>  
  
 **Ensamblado:** biblioteca de tiempo de ejecución de Visual Basic (en Microsoft.VisualBasic.dll)
