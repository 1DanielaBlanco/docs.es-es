# <a name="mycomputerclipboard-object"></a>My.Computer.Clipboard (Objeto)
Proporciona métodos para manipular el Portapapeles.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer.Clipboard` objeto, vea <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>.</xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>  
  
 Para obtener más información, consulte [almacenar y leer datos desde el Portapapeles](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md).  
  
> [!NOTE]
>  También puede utilizar métodos de la <xref:System.Windows.Forms.Clipboard?displayProperty=fullName>clase para manipular el Portapapeles.</xref:System.Windows.Forms.Clipboard?displayProperty=fullName>
