# <a name="mycomputerclipboard-object"></a>My.Computer.Clipboard (Objeto)
Proporciona métodos para manipular el Portapapeles.  
  
## <a name="remarks"></a>Comentarios  
 Para obtener información acerca de los métodos y propiedades de la `My.Computer.Clipboard` de objetos, consulte <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>.  
  
 Para obtener más información, consulte [almacenar y leer datos desde el Portapapeles](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md).  
  
> [!NOTE]
>  También puede utilizar métodos de la <xref:System.Windows.Forms.Clipboard?displayProperty=nameWithType> clase para manipular el Portapapeles.  
  
## <a name="requirements"></a>Requisitos  
 **Namespace:**<xref:Microsoft.VisualBasic.MyServices>  
  
 **Clase:** <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy> (proporciona acceso a <xref:System.Windows.Forms.Clipboard>)  
  
 **Ensamblado:** [!INCLUDE[vbprvbruntime](~/includes/vbprvbruntime-md.md)]
