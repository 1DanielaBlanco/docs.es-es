# My.Computer.Clipboard (Objeto)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Proporciona los métodos para manipular el Portapapeles.  
  
## Comentarios  
 Para obtener información sobre los métodos y las propiedades del objeto de `My.Computer.Clipboard` , vea <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy>.  
  
 Para obtener más información, vea [Almacenar y leer datos en el Portapapeles](../../../visual-basic/developing-apps/programming/computer-resources/storing-data-to-and-reading-from-the-clipboard.md).  
  
> [!NOTE]
>  También puede utilizar los métodos de la clase de <xref:System.Windows.Forms.Clipboard?displayProperty=fullName> para manipular el portapapeles.  
  
## Requisitos  
 **Espacio de nombres:** <xref:Microsoft.VisualBasic.MyServices>  
  
 **clase:** <xref:Microsoft.VisualBasic.MyServices.ClipboardProxy> \(proporciona acceso a <xref:System.Windows.Forms.Clipboard>\)  
  
 **Ensamblado:** [!INCLUDE[vbprvbruntime](../../../visual-basic/language-reference/objects/includes/vbprvbruntime-md.md)]
