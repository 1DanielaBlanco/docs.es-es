# &#39;Is&#39; requiere operandos que tienen tipos de referencia, pero este operando tiene el tipo de valor &#39;&lt;tipo&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

El operador de comparación `Is` determina si dos variables de objeto se refieren a la misma instancia.  Esta comparación no está definida para tipos de valor.  
  
 **Identificador de error:** BC30020  
  
### Para corregir este error  
  
-   Utilice el operador de comparación aritmética adecuado o el operador `Like` para comparar dos tipos de valor.
