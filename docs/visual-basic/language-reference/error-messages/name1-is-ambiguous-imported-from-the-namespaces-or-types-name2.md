# &#39;&lt;nombre1&gt;&#39; es ambiguo y se ha importado de los espacios de nombres o tipos &#39;&lt;nombre2&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Ha proporcionado un nombre que es ambiguo y, por tanto, tiene problemas con otro nombre.  El compilador de [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)] no tiene reglas para solucionar conflictos; debe eliminar la ambigüedad de los nombres.  
  
 **Identificador de error:** BC30561  
  
### Para corregir este error  
  
1.  Elimine la ambigüedad del nombre quitando las importaciones del espacio de nombres.  
  
2.  Dé un nombre completo al nombre.
