# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>&#39; &lt;nombre1&gt;&#39; es ambiguo y se ha importado de los espacios de nombres o tipos de &#39;&lt; name2&gt;&#39;
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El compilador de [!INCLUDE[vbprvb](~/includes/vbprvb-md.md)] no tiene reglas de resolución de conflictos. Es necesario eliminar la ambigüedad en los nombres de forma manual.  
  
 **Id. de error:** BC30561  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Eliminar la ambigüedad del nombre quitando las importaciones de espacio de nombres.  
  
2.  Use el nombre completo.
