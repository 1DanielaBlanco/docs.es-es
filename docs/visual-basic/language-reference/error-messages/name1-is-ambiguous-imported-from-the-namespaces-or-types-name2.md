# <a name="39ltname1gt39-is-ambiguous-imported-from-the-namespaces-or-types-39ltname2gt39"></a>'&lt;nombre1&gt;'es ambiguo, importado de los espacios de nombres o tipos'&lt;nombre2&gt;'
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador no tiene reglas para solucionar conflictos; debe eliminar nombres.  
  
 **Id. de error:** BC30561  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Eliminar la ambigüedad del nombre quitando las importaciones de espacio de nombres.  
  
2.  Use el nombre completo.
