# No se pueden obtener los nombres de los puertos serie a causa de un error interno del sistema
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Se produjo un error interno al llamar a la propiedad `My.Computer.Ports.SerialPortNames`.  
  
### Para corregir este error  
  
1.  Vea [Conceptos básicos del depurador](/visual-studio/debugger/debugger-basics) para obtener más información para solucionar el problema.  
  
2.  Anote las circunstancias en las que se ha produjo el error y llame a los Servicios de soporte técnico de Microsoft.
