# <a name="unable-to-write-temporary-file-because-temporary-path-is-not-available"></a>No se puede escribir el archivo temporal porque la ruta de acceso temporal no está disponible
[!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] no pudo determinar la ruta de acceso donde se almacenan los archivos temporales.  
  
 **Id. de error:** BC30698  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Reinicie [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)].  
  
2.  Si el problema persiste, reinstale [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs_md.md)].
