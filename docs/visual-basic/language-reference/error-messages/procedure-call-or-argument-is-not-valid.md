# El argumento o la llamada al procedimiento no es v&#225;lido (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Parte de la llamada no se puede completar.  
  
### Para corregir este error  
  
-   Compruebe los intervalos permitidos para argumentos a fin de asegurarse de que ninguna disposición supera los valores permitidos.
