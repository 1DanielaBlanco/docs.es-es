# Esta clave ya est&#225; asociada con un elemento de esta colecci&#243;n
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

La clave especificada para un miembro de la colección ya identifica a otro miembro de la colección.  Una clave es una cadena especificada en el método `Add` que identifica de forma única un miembro específico de una colección.  
  
### Para corregir este error  
  
-   Utilice otra clave para este miembro.
