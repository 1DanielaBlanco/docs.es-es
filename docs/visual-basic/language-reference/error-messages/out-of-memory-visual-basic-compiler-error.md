# Memoria insuficiente (error del compilador de Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Se necesita más memoria de la disponible.  
  
 **Identificador de error:** BC2004  
  
### Para corregir este error  
  
-   Cierre las aplicaciones, documentos y archivos de código fuente innecesarios.  
  
-   Elimine los controles y formularios innecesarios de modo que se carguen menos a la vez  
  
-   Reduzca el número de variables `Public`.  
  
-   Compruebe si hay suficiente espacio en disco.  
  
-   Aumente la memoria RAM disponible instalando memoria adicional o reasignando memoria.  
  
-   Asegúrese de que se libera esa memoria cuando ya no se necesita.
