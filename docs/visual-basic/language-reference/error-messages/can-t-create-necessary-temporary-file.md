# <a name="can39t-create-necessary-temporary-file"></a>CAN &#39; t crear archivo temporal necesario
Está lleno o la unidad que contiene el directorio especificado por la variable de entorno TEMP o la variable de entorno TEMP especifica una unidad no válida o de solo lectura o un directorio.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Eliminar archivos de la unidad, si está completa.  
  
2.  Especifique una unidad diferente en la variable de entorno TEMP.  
  
3.  Especifique una unidad válida para la variable de entorno TEMP.  
  
4.  Quite la restricción de solo lectura de la unidad especificada actualmente o el directorio.
