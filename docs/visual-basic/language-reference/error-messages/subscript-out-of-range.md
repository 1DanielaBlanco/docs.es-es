# El sub&#237;ndice est&#225; fuera del intervalo (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

El subíndice de la matriz no es válido porque está fuera del intervalo permitido.  El valor más bajo de subíndice para una dimensión es siempre 0 y el más alto lo devuelve el método `GetUpperBound`.  
  
### Para corregir este error  
  
-   Cambie el subíndice para que esté dentro del intervalo válido.
