# No se puede escribir el resultado en memoria
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Se ha producido un problema al escribir una salida en la memoria.  
  
 **Identificador del error:** BC31020  
  
### Para corregir este error  
  
1.  Vuelva a compilar el programa para ver si el error vuelve a suceder.  
  
2.  Si el error continúa, guarde el trabajo y reinicie [!INCLUDE[vsprvs](../../../csharp/includes/vsprvs-md.md)].  
  
3.  Si el error se repite, reinstale [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb-md.md)].  
  
4.  Si el error persiste después de la reinstalación, informe al respecto a los Servicios de soporte técnico de Microsoft.
