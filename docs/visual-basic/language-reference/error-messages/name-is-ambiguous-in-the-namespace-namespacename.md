# <a name="39ltnamegt39-is-ambiguous-in-the-namespace-39ltnamespacenamegt39"></a>'&lt;nombre&gt;'es ambiguo en el espacio de nombres'&lt;namespacename&gt;'
Se proporcionó un nombre que es ambiguo y, por tanto, entra en conflicto con otro nombre. El [!INCLUDE[vbprvb](../../../csharp/programming-guide/concepts/linq/includes/vbprvb_md.md)] compilador no tiene reglas para solucionar conflictos; debe eliminar nombres.  
  
 **Id. de error:** BC30560  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Use el nombre completo.
