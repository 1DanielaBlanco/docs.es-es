# El URI de espacio de nombres XML &#39;http://www.w3.org/XML/1998/namespace&#39; s&#243;lo se puede enlazar a &#39;xmlns&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

El URI de espacio de nombres XML http:\/\/www.w3.org\/XML\/1998\/namespace se está utilizando en una declaración de espacio de nombres XML.  Este URI es un espacio de nombres reservado y no se puede incluir en una declaración de espacio de nombres XML.  
  
 **Id. de error:** BC31183  
  
### Para corregir este error  
  
-   Quite la declaración de espacio de nombres XML o reemplace el URI de espacio de nombres http:\/\/www.w3.org\/XML\/1998\/namespace por un URI de espacio de nombres válido.
