# Se requiere un objeto (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Las referencias a propiedades o métodos a menudo requieren un calificador de objetos explícito.  Éste es uno de los casos.  
  
### Para corregir este error  
  
1.  Compruebe que las referencias a una propiedad de objeto o a un método tienen un calificador de objeto válido.  Especifique un calificador de objeto si no ha proporcionado ninguno.  
  
2.  Compruebe si ha escrito correctamente el calificador del objeto y asegúrese de que el objeto está visible en la parte del programa en la que se hace referencia al mismo.  
  
3.  Si se proporciona una ruta de acceso a un comando **Abrir archivo** de la aplicación del host, compruebe que los argumentos son correctos.  
  
4.  Compruebe la documentación del objeto y asegúrese de que la acción es válida.
