# <a name="property-or-method-not-found"></a>No se encuentra la propiedad ni el método
El método de objeto que se hace referencia o la propiedad de objeto no está definido.  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Es posible que haya escrito correctamente el nombre del objeto. Para ver qué propiedades y métodos definidos para un objeto, muestre el Examinador de objetos. Seleccione la biblioteca de objetos correspondiente para ver una lista de propiedades y métodos disponibles.
