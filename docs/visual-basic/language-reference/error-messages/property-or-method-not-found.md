# No se encuentra la propiedad ni el m&#233;todo
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

El método o la propiedad del objeto al que se hace referencia no está definido.  
  
### Para corregir este error  
  
-   Es posible que haya escrito mal el nombre del objeto.  Para ver las propiedades y métodos que están definidos para un objeto, muestre el Examinador de objetos.  Seleccione la biblioteca de objetos apropiada para ver una lista de propiedades y métodos disponibles.
