# La cl&#225;usula Handles requiere una variable WithEvents definida en el tipo contenedor o en uno de sus tipos base
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

No ha suministrado una variable `WithEvents` en la cláusula `Handles`.  La palabra clave `Handles` al final de una declaración de procedimiento permite controlar eventos provocados por una variable de objeto declarada con la palabra clave `WithEvents`.  
  
 **Identificador de error:** BC30506  
  
### Para corregir este error  
  
-   Suministre la variable `WithEvents` necesaria.
