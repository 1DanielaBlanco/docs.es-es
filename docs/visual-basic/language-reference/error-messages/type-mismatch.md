# No coinciden los tipos (Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Intentó convertir un valor a otro tipo de un modo que no es válido.  
  
### Para corregir este error  
  
1.  Compruebe la asignación para asegurarse de que es válida.  
  
2.  Asegúrese de que no ha pasado un objeto a un procedimiento en el que se esperaba una sola propiedad o valor.  
  
3.  Asegúrese de que no ha utilizado un nombre de módulo o proyecto donde se esperaba una expresión.
