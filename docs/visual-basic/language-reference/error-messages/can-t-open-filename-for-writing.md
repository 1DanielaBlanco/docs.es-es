# No se puede abrir &#39;&lt;nombreDeArchivo&gt;&#39; para escritura
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

El archivo especificado no se puede abrir para escribir, quizás porque ya se ha abierto.  
  
 **Identificador de error:** BC2012  
  
### Para corregir este error  
  
1.  Cierre el archivo y vuelva a abrirlo  
  
2.  Compruebe los permisos del archivo.
