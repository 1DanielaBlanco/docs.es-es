# No se puede cargar informaci&#243;n para la clase &#39;&lt;nombredeclase&gt;&#39;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Se hizo referencia a una clase que no está disponible.  
  
 **Identificador de error:** BC30712  
  
### Para corregir este error  
  
1.  Compruebe que la clase está definida y que se ha escrito correctamente su nombre.  
  
2.  Intente obtener acceso a alguno de los miembros declarados en el módulo.  En algunos casos, el entorno de depuración no encuentra los miembros, porque los módulos donde se declararon aún no se habían cargado.
