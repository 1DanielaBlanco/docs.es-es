# Se ha producido un error interno en &lt;ubicaci&#243;n&gt;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Error interno.  La línea en la que apareció se indica en el mensaje de error.  
  
### Para corregir este error  
  
-   Asegúrese de que este error no se produjo por la instrucción `Error` o el método `Raise`; si no lo fue, póngase en contacto con los servicios de soporte técnico de Microsoft para informar de las condiciones en las que apareció el mensaje de error.
