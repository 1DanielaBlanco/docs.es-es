# Algunas subclaves no se pueden eliminar
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Se ha intentado eliminar una clave del Registro, pero se produjo un error en la operación porque no se puede eliminar algunas subclaves.  Normalmente esto se debe a la falta de permisos.  
  
### Para corregir este error  
  
-   Asegúrese de que tiene los permisos necesarios para eliminar las subclaves especificadas.
