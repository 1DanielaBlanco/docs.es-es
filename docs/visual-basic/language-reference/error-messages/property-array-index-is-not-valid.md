# El &#237;ndice de la matriz de propiedades no es v&#225;lido
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

El valor proporcionado no es válido para un índice de la matriz de propiedades.  
  
### Para corregir este error  
  
-   Compruebe la documentación del componente para asegurarse de que el índice está comprendido en el intervalo válido para la propiedad especificada.
