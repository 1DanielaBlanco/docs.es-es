# <a name="39optional39-expected"></a>&#39; opcional &#39; se esperaba
Un argumento opcional de una declaración de procedimiento va seguido de un argumento requerido. Cada argumento que sigue a un argumento opcional también debe ser opcional.  
  
 **Id. de error:** BC30202  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
1.  Si el argumento está destinado a ser necesario, muévalo para que preceda al primer argumento opcional en la lista de argumentos.  
  
2.  Si el argumento debe ser opcional, use la `Optional` palabra clave.
