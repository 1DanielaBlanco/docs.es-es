# <a name="39class39-statement-must-end-with-a-matching-39end-class39"></a>&#39; Clase &#39; la instrucción debe terminar con una coincidencia &#39; End Class &#39;
`Class`se usa para iniciar un `Class` bloque; por lo tanto, solo puede aparecer al principio del bloque, con su correspondiente `End Class` final del bloque de instrucción. O bien hay una redundancia `Class` instrucción, o no haya finalizado la `Class` bloque con `End Class`.  
  
 **Id. de error:** BC30481  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Busque y quite el innecesarios `Class` instrucción.  
  
-   Concluir la `Class` bloque con su correspondiente `End Class`.
