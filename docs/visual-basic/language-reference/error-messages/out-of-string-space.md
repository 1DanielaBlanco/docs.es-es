# Espacio para cadenas insuficiente (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Con Visual Basic, puede usar cadenas muy grandes.  No obstante, los requisitos de otros programas y el modo en que trabaje con las cadenas puede producir aún este error.  
  
### Para corregir este error  
  
1.  Asegúrese de que una expresión que requiera la creación de una cadena temporal durante la evaluación no esté produciendo el error.  
  
2.  Quite las aplicaciones innecesarias de memoria para crear más espacio.
