# Las constantes de cadena deben terminar en comillas dobles
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Las constantes de cadena deben comenzar y finalizar con comillas.  
  
 **Identificador de error:** BC30648  
  
### Para corregir este error  
  
-   Asegúrese de que el literal de cadena finaliza con una comilla \("\).  Si pega los valores desde otros editores de texto, asegúrese de que los caracteres pegados son comillas dobles válidas y no caracteres que solo lo parecen, como "comillas tipográficas" \(" o "\) o dos comillas simples \(''\).
