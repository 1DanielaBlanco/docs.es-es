# <a name="string-constants-must-end-with-a-double-quote"></a>Las constantes de cadena deben terminar en comillas dobles
Las constantes de cadena deben empezar y terminar con comillas.  
  
 **Identificador del error:** BC30648  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Asegúrese de que el literal de cadena finaliza con un signo de comillas ("). Si pega los valores desde otros editores de texto, asegúrese de que los caracteres pegados son comillas dobles válidas y no uno de los caracteres que sólo lo parecen, como "inteligente" o "llave" entre comillas ("o") o dos simples entre comillas (").
