# Desbordamiento (Error de Visual Basic)
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Un literal representa un valor situado fuera los límites del tipo de datos al se está asignando.  
  
 **Identificador de error:** BC30036  
  
### Para corregir este error  
  
-   Consulte el intervalo de valores para el tipo de datos de destino y vuelva a escribir el literal para que se conforme a dicho intervalo.
