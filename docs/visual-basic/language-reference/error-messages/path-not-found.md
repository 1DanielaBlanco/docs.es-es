# No se encuentra la ruta de acceso
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

El sistema operativo no ha podido encontrar la ruta de acceso especificada durante la operación de acceso al archivo o de acceso al disco.  La ruta de acceso a un archivo incluye la especificación de la unidad y los directorios y subdirectorios que deben recorrerse para localizar el archivo.  Una ruta de acceso puede ser relativa o absoluta.  
  
### Para corregir este error  
  
-   Compruebe la ruta de acceso y vuelva a especificarla.
