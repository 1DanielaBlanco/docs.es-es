# Los eventos no se pueden declarar con un tipo delegado que tenga un tipo de valor devuelto
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Se especificó un delegado para un procedimiento de función.  
  
 **Identificador de error:** BC31084  
  
### Para corregir este error  
  
-   Especifique un delegado para un procedimiento `Sub`.
