# La expresi&#243;n no genera un valor
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Ha intentado utilizar una expresión que no genera un valor en un contexto que genera valores, como llamar a `Sub` en un contexto donde se espera `Function`.  
  
 **Identificador de error:** BC30491  
  
### Para corregir este error  
  
-   Cambie la expresión a una que genere un valor.
