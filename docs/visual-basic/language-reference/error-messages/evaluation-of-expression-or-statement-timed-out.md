# Se super&#243; el tiempo de espera de evaluaci&#243;n de la expresi&#243;n o instrucci&#243;n
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

La evaluación de una expresión no se ha completado a tiempo.  
  
 **Identificador de error:** BC30722  
  
### Para corregir este error  
  
1.  Compruebe que el código introducido es correcto.  
  
2.  Simplifique su expresión para que tarde menos en ejecutarse.
