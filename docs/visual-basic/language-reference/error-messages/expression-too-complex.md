# Expresi&#243;n demasiado compleja
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Una expresión de punto flotante contiene demasiadas subexpresiones anidadas.  
  
### Para corregir este error  
  
-   Divida la expresión en varias expresiones independientes según sea necesario para evitar que se produzca el error.
