# TextFieldParser no puede finalizar la operaci&#243;n de lectura porque se ha superado el tama&#241;o m&#225;ximo de b&#250;fer
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

No se puede finalizar la operación porque se ha superado el tamaño de búfer máximo \(10.000.000 bytes\).  
  
### Para corregir este error  
  
-   Asegúrese de que no hay ningún campo incorrecto en el archivo.
