# No se encontr&#243; el nombre del archivo o de la clase durante la operaci&#243;n de automatizaci&#243;n (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

El nombre especificado para el nombre de archivo o clase en una llamada a la función `GetObject` no se ha podido encontrar.  
  
### Para corregir este error  
  
-   Compruebe los nombres y vuelva a intentarlo.  Asegúrese de que el nombre utilizado para el parámetro `class` coincide con el registrado con el sistema.
