# No se encontr&#243; &#39;Sub Main&#39; en &#39;&lt;nombre&gt;&#39;
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Falta `Sub Main` o se ha especificado una ubicación incorrecta para la instrucción.  
  
 **Id. de error:** BC30420  
  
### Para corregir este error  
  
1.  Indique la instrucción `Sub Main` que falta o, si existe, muévala a la ubicación adecuada del código.  Para obtener más información sobre `Sub Main`, vea [Procedimiento Main en Visual Basic](../../../visual-basic/programming-guide/program-structure/main-procedure.md).  
  
2.  Especifique la ubicación del objeto de inicio del proyecto en el cuadro **Formulario de inicio** del **Diseñador de proyectos**.
