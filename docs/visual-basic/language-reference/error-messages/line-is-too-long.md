# La l&#237;nea es demasiado larga
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Las líneas del texto fuente no pueden superar los 65535 caracteres.  
  
 **Identificador de error:** BC30494  
  
### Para corregir este error  
  
-   Disminuya la longitud de la línea a 65535 caracteres o menos.
