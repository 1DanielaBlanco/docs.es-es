# No se puede crear el ensamblado con nombre desde el archivo de clave &lt;nombreDeArchivo&gt;: &lt;error&gt;
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

No se ha podido crear un ensamblado con nombre seguro a partir del archivo de clave especificado.  
  
 **Identificador de error:** BC31026  
  
### Para corregir este error  
  
1.  Compruebe que se ha especificado el archivo de clave correcto y no está bloqueado por otra aplicación.
