# Error de E/S del dispositivo
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Se ha producido un error de entrada o salida cuando el programa utilizaba un dispositivo como una impresora o una unidad de disco.  
  
### Para corregir este error  
  
-   Asegúrese de que el dispositivo está funcionando correctamente y vuelva a intentar la operación.
