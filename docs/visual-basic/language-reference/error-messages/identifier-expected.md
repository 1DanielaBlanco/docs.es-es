# <a name="identifier-expected"></a>Se esperaba un identificador
Un elemento de programación que no es un nombre de elemento declarado reconocible se produce en el contexto requiere un nombre de elemento. Una posible causa es que un atributo se ha especificado en otro distinto al principio de la instrucción.  
  
 **Id. de error:** BC30203  
  
## <a name="to-correct-this-error"></a>Para corregir este error  
  
-   Compruebe que todos los atributos de la instrucción se colocan al principio.  
  
-   Compruebe que todos los nombres de elemento en la instrucción se ha escrito correctamente.
