# Se esperaba un identificador
[!INCLUDE[vs2017banner](../../../visual-basic/developing-apps/includes/vs2017banner.md)]

Un elemento de programación que no es un nombre de elemento declarado reconocible aparece en un contexto que requiere un nombre de elemento.  Una posible causa es que se haya especificado un atributo en una ubicación diferente del comienzo de la instrucción.  
  
 **Identificador de error:** BC30203  
  
### Para corregir este error  
  
-   Compruebe los atributos de la instrucción están todos colocados al comienzo.  
  
-   Compruebe que todos los nombres de elemento de la instrucción están correctamente escritos.
