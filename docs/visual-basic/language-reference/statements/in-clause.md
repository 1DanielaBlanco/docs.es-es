# In (Cl&#225;usula, Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Especifica el grupo que recorre la variable de bucle en un bucle `For Each` o especifica la colección para consultar en una cláusula `From`, `Join` o `Group Join`.

 La palabra clave `In` se puede usar en los siguientes contextos:
