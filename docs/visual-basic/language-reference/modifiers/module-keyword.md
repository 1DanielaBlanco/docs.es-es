# <a name="module-ltkeywordgt-visual-basic"></a>Módulo &lt;palabra clave&gt; (Visual Basic)
Especifica que un atributo al principio de un archivo de código fuente se aplica al módulo de ensamblado actual.  
  
## <a name="remarks"></a>Comentarios  
 Muchos atributos pertenecen a un elemento de programación individual, como una clase o propiedad. Se aplica un atributo de este tipo si se adjunta el bloque de atributos, entre corchetes angulares (`< >`), directamente a la instrucción de declaración.  
  
 Si un atributo se aplica no solo para el siguiente elemento sino al módulo de ensamblado actual, se sitúa el bloque de atributos al principio del archivo de origen e identifique el atributo con el `Module` palabra clave. Si se aplica a todo el ensamblado, utilice la [ensamblado](../../../visual-basic/language-reference/modifiers/assembly.md) palabra clave.  
  
 El `Module` modificador no es el mismo que el [Module (instrucción)](../../../visual-basic/language-reference/statements/module-statement.md).
