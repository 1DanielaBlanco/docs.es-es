# <a name="notinheritable-visual-basic"></a>NotInheritable (Visual Basic)
Especifica que una clase no puede utilizarse como una clase base.  
  
## <a name="remarks"></a>Comentarios  
 **Alternar términos**. A veces se denomina una clase que no se puede heredar un *sellado* clase.  
  
 El `NotInheritable` modificador se puede usar en este contexto:
