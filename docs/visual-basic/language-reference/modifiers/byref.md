# ByRef (Visual Basic)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Indica que un argumento se pasa de tal forma que el procedimiento al que se ha llamado puede cambiar el valor de una variable subyacente al argumento en el código que realiza la llamada.  
  
## Comentarios  
 El modificador `ByRef` se puede utilizar en estos contextos:
