# new (Referencia de C#)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

En C\#, la palabra clave `new` se puede utilizar como operador, modificador o restricción.  
  
 [new \(Operador\)](../../../csharp/language-reference/keywords/new-operator.md)  
 Se utiliza para crear objetos e invocar constructores.  
  
 [new \(Modificador\)](../../../csharp/language-reference/keywords/new-modifier.md)  
 Se utiliza para ocultar un miembro heredado de un miembro de clase base.  
  
 [new \(Restricción\)](../../../csharp/language-reference/keywords/new-constraint.md)  
 Se utiliza para restringir tipos que se podrían utilizar como argumentos para un parámetro de tipo en una declaración genérica.
