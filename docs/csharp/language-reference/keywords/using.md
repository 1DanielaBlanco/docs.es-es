# using (Referencia de C#)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

La palabra clave `using` tiene dos usos principales:  
  
-   Como directiva, cuando se utiliza para crear un alias para un espacio de nombres o para importar tipos definidos en otros espacios de nombres.  Vea [Directiva using](../../../csharp/language-reference/keywords/using-directive.md).  
  
-   Como instrucción, cuando define un ámbito al final del cual el objeto se destruye.  Vea [Instrucción using](../../../csharp/language-reference/keywords/using-statement.md).
