# <a name="using-c-reference"></a>using (Referencia de C#)
La palabra clave `using` tiene dos usos principales:  
  
-   Como directiva, cuando se usa para crear un alias para un espacio de nombres o para importar tipos definidos en otros espacios de nombres. Vea [using (Directiva, Referencia de C#)](../../../csharp/language-reference/keywords/using-directive.md).  
  
-   Como instrucción, cuando define un ámbito al final del cual se eliminará un objeto. Vea [using (Instrucción, Referencia de C#)](../../../csharp/language-reference/keywords/using-statement.md).  
  
Además, la directiva [using static](using-static.md) permite definir un tipo a cuyos miembros estáticos se puede acceder sin especificar un nombre de tipo.
