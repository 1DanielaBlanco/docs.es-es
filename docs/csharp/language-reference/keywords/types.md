# Tipos (Referencia de C#)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

El sistema de tipos de C\# contiene las siguientes categorías:  
  
-   [Tipos de valor](../../../csharp/language-reference/keywords/value-types.md)  
  
-   [Tipos de referencia](../../../csharp/language-reference/keywords/reference-types.md)  
  
-   [Tipos de puntero](../../../csharp/programming-guide/unsafe-code-pointers/pointer-types.md)  
  
 Las variables de son tipos de valor almacenan datos, mientras que las que son tipos de referencia almacenan referencias a los datos reales.  Los tipos de referencia también se denominan objetos.  Los tipos de puntero sólo se pueden utilizar en modo [unsafe](../../../csharp/language-reference/keywords/unsafe.md).  
  
 Un tipo de valor se puede convertir en un tipo de referencia y de nuevo en un tipo de valor mediante [boxing y unboxing](../../../csharp/programming-guide/types/boxing-and-unboxing.md).  No se puede convertir un tipo de referencia en un tipo de valor, excepto en tipos de valor a los que se les ha aplicado una conversión boxing.  
  
 Esta sección también presenta [void](../../../csharp/language-reference/keywords/void.md).  
  
 Los tipos de valor también aceptan valores null, por tanto pueden almacenar un estado sin valor adicional.  Para obtener más información, vea [Tipos que aceptan valores NULL \(Guía de programación de C\#\)](../../../csharp/programming-guide/nullable-types/index.md).
