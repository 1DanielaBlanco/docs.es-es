# <a name="integral-types-table-c-reference"></a>Tabla de tipos enteros (Referencia de C#)
En la siguiente tabla, se muestran los tamaños e intervalos de los tipos enteros, que constituyen un subconjunto de tipos simples.  
  
|Tipo|Intervalo|Tamaño|  
|----------|-----------|----------|  
|[sbyte](../../../csharp/language-reference/keywords/sbyte.md)|De -128 a 127|Entero de 8 bits con signo|  
|[byte](../../../csharp/language-reference/keywords/byte.md)|De 0 a 255|Entero de 8 bits sin signo|  
|[char](../../../csharp/language-reference/keywords/char.md)|U+0000 a U+ffff|Carácter Unicode de 16 bits|  
|[short](../../../csharp/language-reference/keywords/short.md)|De -32 768 a 32 767|Entero de 16 bits con signo|  
|[ushort](../../../csharp/language-reference/keywords/ushort.md)|De 0 a 65.535|Entero de 16 bits sin signo|  
|[int](../../../csharp/language-reference/keywords/int.md)|De -2.147.483.648 a 2.147.483.647|Entero de 32 bits con signo|  
|[uint](../../../csharp/language-reference/keywords/uint.md)|De 0 a 4.294.967.295|Entero de 32 bits sin signo|  
|[long](../../../csharp/language-reference/keywords/long.md)|De -9.223.372.036.854.775.808 a 9.223.372.036.854.775.807|Entero de 64 bits con signo|  
|[ulong](../../../csharp/language-reference/keywords/ulong.md)|De 0 a 18.446.744.073.709.551.615|Entero de 64 bits sin signo|  
  
 Si el valor que representa un literal entero supera el intervalo de `ulong`, se producirá un error de compilación.
