# <a name="iteration-statements-c-reference"></a>Instrucciones de iteración (Referencia de C#)

Puede crear bucles con las instrucciones de iteración. Las instrucciones de iteración producen instrucciones insertadas que se ejecutarán varias veces, según los criterios de la finalización de bucle. Estas instrucciones se ejecutan en orden, excepto cuando se detecta una [instrucción de salto](jump-statements.md).

Las siguientes palabras clave se usan en instrucciones de iteración:

- [do](do.md)

- [for](for.md)

- [foreach, in](foreach-in.md)

- [while](while.md)
