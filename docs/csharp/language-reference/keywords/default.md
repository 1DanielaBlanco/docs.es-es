# default (Referencia de C#)
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

Palabra clave `default`.  
  
 La palabra clave `default` se puede utilizar en la instrucción `switch` o en código genérico:  
  
-   [Instrucción switch](../../../csharp/language-reference/keywords/switch.md): Especifica la etiqueta predeterminada.  
  
-   [Código genérico](../../../csharp/programming-guide/generics/default-keyword-in-generic-code.md): Especifica el valor predeterminado del parámetro de tipo.  Éste será null para los tipos de referencia y cero para los tipos de valor.
