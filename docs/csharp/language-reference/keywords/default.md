# <a name="default-c-reference"></a>default (Referencia de C#)
Palabra clave `default`.  
  
 La palabra clave `default` puede usarse en la instrucción `switch` o en código genérico:  
  
-   [La instrucción switch](../../../csharp/language-reference/keywords/switch.md): especifica la etiqueta predeterminada.  
  
-   [Código genérico](../../../csharp/programming-guide/generics/default-keyword-in-generic-code.md): especifica el valor predeterminado del parámetro de tipo. Este será NULL para los tipos de referencia y cero para los tipos de valor.
