# <a name="default-c-reference"></a>default (Referencia de C#)

La palabra clave `default` puede usarse en la instrucción `switch` o en una expresión de valor predeterminado:

- [Instrucción switch](switch.md): especifica la etiqueta predeterminada.

- [Expresiones de valor predeterminado](../../programming-guide/statements-expressions-operators/default-value-expressions.md): genera el valor predeterminado de un tipo.
