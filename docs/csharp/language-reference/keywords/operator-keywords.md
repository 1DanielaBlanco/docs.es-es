# <a name="operator-keywords-c-reference"></a>Palabras clave de operadores (Referencia de C#)

Se usa para realizar acciones diferentes como crear objetos, comprobar el tipo en tiempo de ejecución de un objeto, obtener el tamaño de un tipo y otras acciones. En esta sección se presentan las siguientes palabras clave:

- [as](as.md) Convierte un objeto en un tipo compatible.

- [await](await.md) Suspende un método [asincrónico](async.md) hasta que finalice una tarea en espera.

- [is](is.md) Comprueba el tipo de tiempo de ejecución de un objeto o (a partir de C# 7.0) prueba una expresión en un patrón.

- [new](new.md)

  - [new (Operador)](new-operator.md) Crea objetos.

  - [new (Modificador)](new-modifier.md) Oculta un miembro heredado.

  - [Restricción new](new-constraint.md) Califica un parámetro de tipo.

- [nameof](nameof.md) Obtiene el nombre de cadena (incompleto) simple de una variable, tipo o miembro.

- [sizeof](sizeof.md) Obtiene el tamaño de un tipo no administrado.  

- [typeof](typeof.md) Obtiene el objeto <xref:System.Type?displayProperty=nameWithType> de un tipo.  

- [true](true.md)  

  - [true (Operator)](true-false-operators.md) Devuelve el valor [bool](bool.md) `true` para indicar que el operando es definitivamente true.

  - [Literal True](true-literal.md) Representa el valor [bool](bool.md) `true`.

- [false](false.md)  

  - [false (Operator)](true-false-operators.md) Devuelve el valor [bool](bool.md) `true` para indicar que el operando es definitivamente false.

  - [Literal False](false-literal.md) Representa el valor [bool](bool.md) `false`.

- [stackalloc](stackalloc.md) Asigna un bloque de memoria en la pila.  

Las siguientes palabras clave, que se pueden usar como operadores e instrucciones, se describen en la sección [Instrucciones](statement-keywords.md):

- [checked](checked.md) Especifica un contexto comprobado.  

- [unchecked](unchecked.md) Especifica un contexto no comprobado.
