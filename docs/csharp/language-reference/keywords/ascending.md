# ascending (Referencia de C#)
La palabra clave contextual `ascending` se utiliza en la [cláusula orderby](../../../csharp/language-reference/keywords/orderby-clause.md) de expresiones de consulta para especificar que el criterio de ordenación es de menor a mayor.  Dado que `ascending` es el criterio de ordenación predeterminado, no tiene que especificarlo.
