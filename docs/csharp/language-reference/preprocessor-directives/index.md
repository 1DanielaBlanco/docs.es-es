 Vea los temas individuales para obtener más información y ejemplos.  
  
 Aunque el compilador no tiene un preprocesador independiente, directivas descritas en esta sección se procesan como si hubiera una.  se utilizan para ayudar en la compilación condicional.  A diferencia de las directivas de C y C\+\+, no es posible utilizar estas directivas para crear macros.  
  
 Una directiva de preprocesador debe ser la única instrucción de una línea.
