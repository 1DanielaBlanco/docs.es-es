# #endregion (Referencia de C#)
`#endregion` marca el final de un bloque [\#region](../../../csharp/language-reference/preprocessor-directives/preprocessor-region.md).  Por ejemplo:  
  
```  
  
      #region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
