# C# Compiler Options
El compilador genera archivos ejecutables \(.exe\), bibliotecas de vínculos dinámicos \(.dll\), o módulos de código \(.netmodule\).  
  
 Todas las opciones del compilador están disponibles de dos formas: **\-option** y **\/option**.  En la documentación se muestra únicamente el formato **\/option**.  
  
 En Visual Web Developer 2008, las opciones del compilador se establecen en el archivo web.config.  Para obtener más información, vea [Elemento \<compiler\>](../Topic/%3Ccompiler%3E%20Element.md).
