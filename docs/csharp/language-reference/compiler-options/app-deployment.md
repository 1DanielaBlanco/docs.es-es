# <a name="deployment-of-c-applications"></a>Implementación de aplicaciones de C#
Cuando haya terminado de compilar la aplicación de C#, el paso siguiente es distribuirla. C# es un lenguaje .NET; por consiguiente, para distribuir los ejecutables de C# en otras máquinas, es necesario que .NET Framework esté instalado en cada una de las máquinas de ejecución (y, posiblemente, en otras dependencias para la aplicación). Dispone de varias opciones para distribuir .NET Framework. Para obtener información general, vea [Guía de implementación de .NET Framework para desarrolladores](https://msdn.microsoft.com/library/ee942965).  
  
 El traslado de aplicaciones completas a otros equipos se suele denominar implementación. El entorno de desarrollo de Microsoft proporciona mecanismos para la implementación. Para obtener más información, vea [Deploying Applications and Components](/visualstudio/deployment/deploying-applications-services-and-components) (Implementar aplicaciones y componentes).  
  
 Si compila y distribuye principalmente desde la línea de comandos, puede que tenga que contemplar otros métodos de implementación y redistribución de dependencias.
