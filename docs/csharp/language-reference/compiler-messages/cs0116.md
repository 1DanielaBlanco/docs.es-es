<span data-ttu-id="64e8f-103">Un espacio de nombres no contiene directamente miembros como campos o métodos</span><span class="sxs-lookup"><span data-stu-id="64e8f-103">A namespace does not directly contain members such as fields or methods</span></span>  
  
 <span data-ttu-id="64e8f-104">Dentro de un [espacio de nombres](../../../csharp/language-reference/keywords/namespace.md), el compilador solo acepta clases, estructuras, uniones, enumeraciones, interfaces y delegados.</span><span class="sxs-lookup"><span data-stu-id="64e8f-104">Inside a [namespace](../../../csharp/language-reference/keywords/namespace.md), the compiler only accepts classes, structures, unions, enumerations, interfaces, and delegates.</span></span> <span data-ttu-id="64e8f-105">Este error se genera a menudo por los desarrolladores desde un fondo de C/C++ porque se olvidan de que en C#, los métodos y las variables deben declararse y definirse en un struct o clase.</span><span class="sxs-lookup"><span data-stu-id="64e8f-105">This error is often generated by developers from a C/C++ background who forget that in C#, methods and variables must be declared and defined within a struct or class.</span></span> <span data-ttu-id="64e8f-106">Para obtener más información, vea [Estructura general de un programa de C#](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span><span class="sxs-lookup"><span data-stu-id="64e8f-106">For more information, see [General Structure of a C# Program](../../../csharp/programming-guide/inside-a-program/general-structure-of-a-csharp-program.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="64e8f-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="64e8f-107">Example</span></span>  
 <span data-ttu-id="64e8f-108">En el ejemplo siguiente se genera el error CS0116:</span><span class="sxs-lookup"><span data-stu-id="64e8f-108">The following sample generates CS0116:</span></span>  
  
```csharp  
// CS0116.cs  
namespace x  
{  
    using System;  
  
    // method must be in class/struct  
    void Method(string str) // CS0116  
    {  
        Console.WriteLine(str);  
    }  
    // To fix the error, you must  
    // enclose a method in a class:  
    class Program  
    {  
        void Method2(string str)  
        {  
            Console.WriteLine(str);  
        }  
    }  
}  
```
