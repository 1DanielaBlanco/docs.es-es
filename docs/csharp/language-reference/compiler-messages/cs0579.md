# <a name="compiler-error-cs0579"></a>Error del compilador CS0579
Atributo "attribute" duplicado  
  
 No es posible especificar el mismo atributo m√°s de una vez, a menos que el atributo especifique **AllowMultiple=true** en su [AttributeUsage](../../../csharp/programming-guide/concepts/attributes/attributeusage.md).  
  
## <a name="example"></a>Ejemplo  
 En el ejemplo siguiente se genera el error CS0579.  
  
```csharp  
// CS0579.cs  
using System;  
public class MyAttribute : Attribute  
{  
}  
  
[AttributeUsage(AttributeTargets.All,AllowMultiple=true)]  
public class MyAttribute2 : Attribute  
{  
}  
  
public class z  
{  
    [MyAttribute, MyAttribute]     // CS0579  
    public void zz()  
    {  
    }  
  
    [MyAttribute2, MyAttribute2]   // OK  
    public void zzz()  
    {  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
