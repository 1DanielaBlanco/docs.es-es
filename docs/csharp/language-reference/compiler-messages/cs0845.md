# <a name="compiler-error-cs0845"></a><span data-ttu-id="06fc2-102">Error del compilador CS0845</span><span class="sxs-lookup"><span data-stu-id="06fc2-102">Compiler Error CS0845</span></span>

## <a name="example"></a><span data-ttu-id="06fc2-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="06fc2-107">Example</span></span>  
 <span data-ttu-id="06fc2-108">En el código siguiente se genera el error CS0845:</span><span class="sxs-lookup"><span data-stu-id="06fc2-108">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
