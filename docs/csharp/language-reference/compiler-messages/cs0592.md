## <a name="example"></a><span data-ttu-id="46cbb-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="46cbb-108">Example</span></span>

 <span data-ttu-id="46cbb-109">En el ejemplo siguiente se genera la advertencia CS0592:</span><span class="sxs-lookup"><span data-stu-id="46cbb-109">The following sample generates CS0592:</span></span>  

```csharp
// CS0592.cs  
using System;  
  
[AttributeUsage(AttributeTargets.Interface)]  
public class MyAttribute : Attribute
{  
}  
  
[MyAttribute]  
// Generates CS0592 because MyAttribute is not valid for a class.

{  
    public static void Main()  
    {  
    }  
}  
```
