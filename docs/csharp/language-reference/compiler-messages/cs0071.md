## <a name="example"></a><span data-ttu-id="82c0e-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="82c0e-107">Example</span></span>  
 <span data-ttu-id="82c0e-108">En el ejemplo siguiente se genera la advertencia CS0071.</span><span class="sxs-lookup"><span data-stu-id="82c0e-108">The following sample generates CS0071.</span></span>  
  
```csharp  
// CS0071.cs  
public delegate void MyEvent(object sender);  
  
interface ITest  
{  
    event MyEvent Clicked;  
}  
  
class Test : Itest  
{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead  
/*  
private MyEvent clicked;  
  
    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
