 <span data-ttu-id="93ef4-104">Al implementar de manera explícita un [evento](../../../csharp/language-reference/keywords/event.md) que se ha declarado en una interfaz, debe proporcionar manualmente los descriptores de acceso de eventos `add` y `remove` que normalmente se proporcionan mediante el compilador.</span><span class="sxs-lookup"><span data-stu-id="93ef4-104">When explicitly implementing an [event](../../../csharp/language-reference/keywords/event.md) that was declared in an interface, you must use manually provide the `add` and `remove` event accessors that are typically provided by the compiler.</span></span> <span data-ttu-id="93ef4-105">El código de acceso puede conectar el evento de interfaz a otro evento en su clase (se muestra más adelante en este tema), o a su propio tipo de delegado.</span><span class="sxs-lookup"><span data-stu-id="93ef4-105">The accessor code can connect the interface event to another event in your class (shown later in this topic), or to its own delegate type.</span></span> <span data-ttu-id="93ef4-106">Para obtener más información, vea [Cómo: Implementar eventos de interfaz](../../../csharp/programming-guide/events/how-to-implement-interface-events.md).</span><span class="sxs-lookup"><span data-stu-id="93ef4-106">For more information, see [How to:  Implement Interface Events](../../../csharp/programming-guide/events/how-to-implement-interface-events.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="93ef4-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="93ef4-107">Example</span></span>  
 <span data-ttu-id="93ef4-108">En el ejemplo siguiente se genera la advertencia CS0071.</span><span class="sxs-lookup"><span data-stu-id="93ef4-108">The following sample generates CS0071.</span></span>  
  
```csharp  
// CS0071.cs  
public delegate void MyEvent(object sender);  
  
interface ITest  
{  
    event MyEvent Clicked;  
}  
  
class Test : Itest  
{  
    event MyEvent ITest.Clicked;  // CS0071  
  
    // try the following code instead  
/*  
private MyEvent clicked;  
  
    event MyEvent Itest.Clicked  
    {  
        add  
        {  
            clicked += value;  
        }  
        remove  
        {  
            clicked -= value;  
        }  
    }  
*/  
    public static void Main() { }  
}  
```
