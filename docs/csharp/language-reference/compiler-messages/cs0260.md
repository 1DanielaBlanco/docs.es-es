# <a name="compiler-error-cs0260"></a><span data-ttu-id="8e5c5-102">Error del compilador CS0260</span><span class="sxs-lookup"><span data-stu-id="8e5c5-102">Compiler Error CS0260</span></span>
<span data-ttu-id="8e5c5-103">Falta el modificador parcial en la declaración de tipo "type"; ya existe otra declaración parcial de este tipo</span><span class="sxs-lookup"><span data-stu-id="8e5c5-103">Missing partial modifier on declaration of type 'type'; another partial declaration of this type exists</span></span>  
  
 <span data-ttu-id="8e5c5-104">Este error indica que ha declarado varias clases con el mismo nombre.</span><span class="sxs-lookup"><span data-stu-id="8e5c5-104">This error indicates that you have declared multiple classes that have the same name.</span></span> <span data-ttu-id="8e5c5-105">Además, por lo menos una de estas declaraciones contiene el modificador `partial`, pero no todas ellas.</span><span class="sxs-lookup"><span data-stu-id="8e5c5-105">In addition, at least one but not all of the declarations contains the `partial` modifier.</span></span> <span data-ttu-id="8e5c5-106">Si quiere definir una clase en varias partes, todas estas partes deberán declararse con la palabra clave `partial`.</span><span class="sxs-lookup"><span data-stu-id="8e5c5-106">If you want to define a class in several parts, you must declare each part by using the keyword `partial`.</span></span>  
  
 <span data-ttu-id="8e5c5-107">Este error también se produce si declara una clase y, accidentalmente, tiene el mismo nombre que una clase parcial declarada en otro lugar del mismo espacio de nombres.</span><span class="sxs-lookup"><span data-stu-id="8e5c5-107">This error also occurs if you declare a class and accidentally give it the same name as a partial class that's declared elsewhere in the same namespace.</span></span>  
  
 <span data-ttu-id="8e5c5-108">En el ejemplo siguiente se genera el error CS0260:</span><span class="sxs-lookup"><span data-stu-id="8e5c5-108">The following sample generates CS0260:</span></span>  
  
```csharp  
// CS0260.cs  
// You must mark both parts of the definition of class C   
// by using the partial keyword.  
  
// The following line causes CS0260. To resolve the error, add  
// the 'partial' keyword to the declaration.  
class C    
{  
}  
  
partial class C  
{  
}  
```
