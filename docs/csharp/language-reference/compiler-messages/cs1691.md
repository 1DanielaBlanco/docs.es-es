# Compiler Warning (level 1) CS1691
'número' no es un número de advertencia válido  
  
 El número pasado a la directiva del preprocesador [\#pragma warning](../../../csharp/language-reference/preprocessor-directives/preprocessor-pragma-warning.md) no era un número de advertencia válido.  Compruebe que el número represente una advertencia, no un error ni otra secuencia de caracteres.  
  
## Ejemplo  
 El siguiente ejemplo genera la advertencia CS1691.  
  
```  
// CS1691.cs  
public class C  
{  
    int i = 1;  
    public static void Main()  
    {  
        C myC = new C();  
#pragma warning disable 151  // CS1691  
// Try the following line instead:  
// #pragma warning disable 1645    
        myC.i++;  
#pragma warning restore 151  // CS1691  
// Try the following line instead:  
// #pragma warning restore 1645    
    }  
}  
```
