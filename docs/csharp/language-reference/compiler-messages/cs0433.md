## <a name="example"></a><span data-ttu-id="447d3-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="447d3-110">Example</span></span>  
 <span data-ttu-id="447d3-111">En el ejemplo siguiente se genera el error CS0433.</span><span class="sxs-lookup"><span data-stu-id="447d3-111">The following example generates CS0433.</span></span>
