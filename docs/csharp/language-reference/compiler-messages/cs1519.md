# <a name="compiler-error-cs1519"></a><span data-ttu-id="bbd07-102">Error del compilador CS1519</span><span class="sxs-lookup"><span data-stu-id="bbd07-102">Compiler Error CS1519</span></span>
<span data-ttu-id="bbd07-103">El token "token" no es válido en una clase, struct o declaración de miembro de interfaz</span><span class="sxs-lookup"><span data-stu-id="bbd07-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="bbd07-104">Este error se genera cada vez que se encuentra un token en una ubicación a la que no pertenece.</span><span class="sxs-lookup"><span data-stu-id="bbd07-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="bbd07-105">Un *token* es una palabra clave; un identificador (el nombre de una clase, un struct, un método, etc.); una cadena, un carácter o un valor literal numérico como 108, "Hello" o "A"; o un operador o signo de puntuación, como `==` o `;`.</span><span class="sxs-lookup"><span data-stu-id="bbd07-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="bbd07-106">Cualquier [clase](../../../csharp/language-reference/keywords/class.md), struct o declaración de miembro de interfaz que contenga modificadores no válidos antes del tipo, producirá este error.</span><span class="sxs-lookup"><span data-stu-id="bbd07-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="bbd07-107">Para corregir el error, quite los modificadores no válidos.</span><span class="sxs-lookup"><span data-stu-id="bbd07-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="bbd07-108">En el siguiente ejemplo se genera CS1519 en cinco lugares porque los tokens se han colocado en ubicaciones donde no son válidos:</span><span class="sxs-lookup"><span data-stu-id="bbd07-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
