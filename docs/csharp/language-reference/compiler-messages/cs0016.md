# Compiler Error CS0016
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

No se puede escribir en el archivo de salida 'archivo' — 'causa'  
  
 El compilador no pudo escribir en un archivo de salida.  Compruebe la ruta de acceso al archivo para asegurarse de que existe.  Si ya hay un archivo compilado anteriormente en esa ubicación, asegúrese de que es grabable y de que ningún proceso tiene bloqueado el archivo actualmente.  Por ejemplo, asegúrese de que su archivo ejecutable no está cargado en memoria cuando intente compilarlo.
