# <a name="compiler-error-cs0151"></a><span data-ttu-id="d2904-102">Error del compilador CS0151</span><span class="sxs-lookup"><span data-stu-id="d2904-102">Compiler Error CS0151</span></span>
<span data-ttu-id="d2904-103">Se espera un valor de tipo entero</span><span class="sxs-lookup"><span data-stu-id="d2904-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="d2904-104">Se ha usado una variable en una situación donde se necesitaba un tipo de datos entero.</span><span class="sxs-lookup"><span data-stu-id="d2904-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="d2904-105">Para obtener más información, vea [Tipos](../../../csharp/programming-guide/types/index.md).</span><span class="sxs-lookup"><span data-stu-id="d2904-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="d2904-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="d2904-106">Example</span></span>  
 <span data-ttu-id="d2904-107">Este error puede producirse cuando no existe ninguna conversión o si las conversiones implícitas disponibles provocan una situación ambigua.</span><span class="sxs-lookup"><span data-stu-id="d2904-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="d2904-108">En el ejemplo siguiente se genera el error CS0151.</span><span class="sxs-lookup"><span data-stu-id="d2904-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```
