## <a name="example"></a><span data-ttu-id="df322-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="df322-107">Example</span></span>  
 <span data-ttu-id="df322-108">En el ejemplo siguiente se muestra una manera en que se genera CS1721 y, a continuación, se muestran dos maneras posibles para evitar el error.</span><span class="sxs-lookup"><span data-stu-id="df322-108">The following example shows one way in which CS1721 is generated, and then shows two possible ways to avoid the error.</span></span>  
  
```csharp  
// CS1721.cs  
public class A {}  
public class B {}  
public class MyClass : A, B {}   // CS1721  
  
// One possible fix is to use the following approach instead:  
public class A {}  
public class B : A {}  
public class C : B {}  
  
// Another possible fix is to use interfaces instead of base classes:  
public class A {}  
public interface B {}  
public class C : A, B {}  
```
