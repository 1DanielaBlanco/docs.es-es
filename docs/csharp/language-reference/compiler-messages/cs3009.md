# Compiler Warning (level 1) CS3009
'tipo': el tipo base 'tipo' no es conforme a CLS  
  
 Se ha marcado un tipo base como si no requiriera conforme a CLS \(Common Language Specification\) en un ensamblado marcado como conforme a CLS.  Quite el atributo que especifica que el ensamblado es conforme a CLS, o bien quite el atributo que indica que el tipo no es conforme a CLS.  Para obtener más información sobre la conformidad con CLS, vea [Escribir código conforme a CLS](http://msdn.microsoft.com/es-es/4c705105-69a2-4e5e-b24e-0633bc32c7f3) y [Independencia del lenguaje y componentes independientes del lenguaje](../Topic/Language%20Independence%20and%20Language-Independent%20Components.md).  
  
## Ejemplo  
 El ejemplo siguiente genera la advertencia CS3009:  
  
```  
// CS3009.cs  
  
using System;  
  
[assembly:CLSCompliant(true)]  
[CLSCompliant(false)]  
public class B  
{  
}  
  
public class C : B   // CS3009  
{  
    public static void Main () {}  
}  
```
