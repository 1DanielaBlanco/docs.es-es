# C# Compiler Errors
Algunos de los errores del compilador de C\# tienen temas correspondientes en los que se explica por qué se genera un error, y en algunos casos, cómo corregirlo.  Siga uno de los pasos que se describen a continuación para ver si existe Ayuda para un mensaje de error concreto.  
  
-   Busque el número de error \(por ejemplo, CS0029\) en [Resultados \(Ventana\)](/visual-studio/ide/reference/output-window)y, a continuación busquelo para él en MSDN.  
  
-   Elija el número de error \(por ejemplo, CS0029\) en [Resultados \(Ventana\)](/visual-studio/ide/reference/output-window), y elija la tecla F1.  
  
-   En el índice, escriba el número de error en el cuadro de **Buscar** .  
  
 Si ninguno de estos pasos conducen a la información sobre el error, vaya al final de esta página, y enviar comentarios que incluye el número o el texto del error.  
  
 Para obtener información sobre cómo configurar las opciones de errores y advertencias en C\#, vea [Compilar \(Página, Diseñador de proyectos\) \(C\#\)](/visual-studio/ide/reference/build-page-project-designer-csharp).
