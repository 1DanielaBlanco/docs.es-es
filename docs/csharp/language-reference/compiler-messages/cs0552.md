 <span data-ttu-id="2d3bc-104">No se puede crear una conversión definida por el usuario a una interfaz o desde una interfaz.</span><span class="sxs-lookup"><span data-stu-id="2d3bc-104">You cannot create a user-defined conversion to or from an interface.</span></span> <span data-ttu-id="2d3bc-105">Si se requiere la rutina de conversión, se puede resolver el error convirtiendo la interfaz en clase o derivando una clase de la interfaz.</span><span class="sxs-lookup"><span data-stu-id="2d3bc-105">If you need the conversion routine, resolve this error by making the interface a class or derive a class from the interface.</span></span>  
  
 <span data-ttu-id="2d3bc-106">En el ejemplo siguiente se genera la advertencia CS0552:</span><span class="sxs-lookup"><span data-stu-id="2d3bc-106">The following sample generates CS0552:</span></span>  
  
```csharp  
// CS0552.cs

{  
}

{  
   // delete the routine to resolve CS0552

      return new ii();  
   }  
  
   public static void Main()  
   {  
   }  
}
```
