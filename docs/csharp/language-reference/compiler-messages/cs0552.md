# <a name="compiler-error-cs0552"></a>Error del compilador CS0552
'conversion routine' : conversión a/de la interfaz definida por el usuario  
  
 No se puede crear una conversión definida por el usuario a una interfaz o desde una interfaz. Si se requiere la rutina de conversión, se puede resolver el error convirtiendo la interfaz en clase o derivando una clase de la interfaz.  
  
 En el ejemplo siguiente se genera la advertencia CS0552:  
  
```csharp  
// CS0552.cs  
public interface ii  
{  
}  
  
public class a  
{  
   // delete the routine to resolve CS0552  
   public static implicit operator ii(a aa) // CS0552  
   {  
      return new ii();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
