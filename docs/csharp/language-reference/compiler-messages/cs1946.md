<span data-ttu-id="e028e-103">Una expresión de método anónimo no se puede convertir en un árbol de expresión.</span><span class="sxs-lookup"><span data-stu-id="e028e-103">An anonymous method expression cannot be converted to an expression tree.</span></span>
