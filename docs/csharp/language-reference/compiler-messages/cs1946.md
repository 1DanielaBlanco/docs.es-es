# <a name="compiler-error-cs1946"></a><span data-ttu-id="ae921-102">Error del compilador CS1946</span><span class="sxs-lookup"><span data-stu-id="ae921-102">Compiler Error CS1946</span></span>

<span data-ttu-id="ae921-103">Una expresión de método anónimo no se puede convertir en un árbol de expresión.</span><span class="sxs-lookup"><span data-stu-id="ae921-103">An anonymous method expression cannot be converted to an expression tree.</span></span>  
  
<span data-ttu-id="ae921-104">Un método anónimo representa un conjunto de instrucciones pero un árbol de expresión no debe contener una instrucción.</span><span class="sxs-lookup"><span data-stu-id="ae921-104">An anonymous method represents a set of statements but an expression tree must not contain a statement.</span></span> <span data-ttu-id="ae921-105">Por lo tanto, un método anónimo no puede representarse mediante un árbol de expresión.</span><span class="sxs-lookup"><span data-stu-id="ae921-105">Therefore an anonymous method cannot be represented by an expression tree.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="ae921-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="ae921-106">To correct this error</span></span>

1. <span data-ttu-id="ae921-107">Cambie el método anónimo a una expresión lambda.</span><span class="sxs-lookup"><span data-stu-id="ae921-107">Change the anonymous method to a lambda expression.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ae921-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="ae921-108">Example</span></span>

<span data-ttu-id="ae921-109">El ejemplo siguiente genera el error CS1946:</span><span class="sxs-lookup"><span data-stu-id="ae921-109">The following example generates CS1946:</span></span>  

```csharp
// cs1946.cs  
using System;  
    using System.Linq.Expressions;  
  
    public delegate void D();  
  
    class Test  
    {  
        static void Main()  
        {  
            Expression<D> tree = delegate() { }; //CS1946  
            // Try using a lambda expression instead.  
            // Expression<D> tree = (x) => x + 1;  
        }  
    }  
```
