<span data-ttu-id="b84cd-103">Una expresión de método anónimo no se puede convertir en un árbol de expresión.</span><span class="sxs-lookup"><span data-stu-id="b84cd-103">An anonymous method expression cannot be converted to an expression tree.</span></span>

## <a name="example"></a><span data-ttu-id="b84cd-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="b84cd-108">Example</span></span>  
 <span data-ttu-id="b84cd-109">El ejemplo siguiente genera el error CS1946:</span><span class="sxs-lookup"><span data-stu-id="b84cd-109">The following example generates CS1946:</span></span>

    public delegate void D();

        static void Main()

            Expression<D> tree = delegate() { }; //CS1946

    }
