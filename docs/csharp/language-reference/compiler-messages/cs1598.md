# Compiler Warning (level 1) CS1598
[!INCLUDE[vs2017banner](../../../csharp/includes/vs2017banner.md)]

El analizador XML no se puede cargar por la siguiente razón: 'razón'.El archivo de documentación XML 'archivo' no se generará.  
  
 Se especificó la opción [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md), pero el compilador no pudo encontrar y cargar msxml3.dll.  Asegúrese de que el archivo msxml3.dll está instalado y registrado en el sistema.
