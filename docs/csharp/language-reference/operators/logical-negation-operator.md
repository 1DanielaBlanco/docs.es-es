# Operador ! (Referencia de C#)
El operador lógico de negación \(`!`\) es un operador unario que niega su operando.  Está definido para el tipo `bool` y devuelve `true` si, y sólo si, su operando es `false`.  
  
## Comentarios  
 Los tipos definidos por el usuario pueden sobrecargar el operador `!` \(vea [operador](../../../csharp/language-reference/keywords/operator.md)\).
