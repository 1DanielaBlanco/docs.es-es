# <a name="properties-c-programming-guide"></a><span data-ttu-id="e139a-102">Propiedades (Guía de programación de C#)</span><span class="sxs-lookup"><span data-stu-id="e139a-102">Properties (C# Programming Guide)</span></span>

<span data-ttu-id="e139a-103">Una propiedad es un miembro que proporciona un mecanismo flexible para leer, escribir o calcular el valor de un campo privado.</span><span class="sxs-lookup"><span data-stu-id="e139a-103">A property is a member that provides a flexible mechanism to read, write, or compute the value of a private field.</span></span> <span data-ttu-id="e139a-104">Las propiedades se pueden usar como si fueran miembros de datos públicos, pero en realidad son métodos especiales denominados *descriptores de acceso*.</span><span class="sxs-lookup"><span data-stu-id="e139a-104">Properties can be used as if they are public data members, but they are actually special methods called *accessors*.</span></span> <span data-ttu-id="e139a-105">Esto permite acceder fácilmente a los datos a la vez que proporciona la seguridad y la flexibilidad de los métodos.</span><span class="sxs-lookup"><span data-stu-id="e139a-105">This enables data to be accessed easily and still helps promote the safety and flexibility of methods.</span></span>  

## <a name="properties-overview"></a><span data-ttu-id="e139a-106">Información general sobre propiedades</span><span class="sxs-lookup"><span data-stu-id="e139a-106">Properties overview</span></span>  
  
- <span data-ttu-id="e139a-107">Las propiedades permiten que una clase exponga una manera pública de obtener y establecer valores, a la vez que se oculta el código de implementación o comprobación.</span><span class="sxs-lookup"><span data-stu-id="e139a-107">Properties enable a class to expose a public way of getting and setting values, while hiding implementation or verification code.</span></span>  
  
- <span data-ttu-id="e139a-108">Para devolver el valor de la propiedad se usa un descriptor de acceso de propiedad [get](../../../csharp/language-reference/keywords/get.md), mientras que para asignar un nuevo valor se emplea un descriptor de acceso de propiedad [set](../../../csharp/language-reference/keywords/set.md).</span><span class="sxs-lookup"><span data-stu-id="e139a-108">A [get](../../../csharp/language-reference/keywords/get.md) property accessor is used to return the property value, and a [set](../../../csharp/language-reference/keywords/set.md) property accessor is used to assign a new value.</span></span> <span data-ttu-id="e139a-109">Estos descriptores de acceso pueden tener diferentes niveles de acceso.</span><span class="sxs-lookup"><span data-stu-id="e139a-109">These accessors can have different access levels.</span></span> <span data-ttu-id="e139a-110">Para más información, vea [Restringir la accesibilidad del descriptor de acceso](../../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md).</span><span class="sxs-lookup"><span data-stu-id="e139a-110">For more information, see [Restricting Accessor Accessibility](../../../csharp/programming-guide/classes-and-structs/restricting-accessor-accessibility.md).</span></span>  
  
- <span data-ttu-id="e139a-111">La palabra clave [value](../../../csharp/language-reference/keywords/value.md) se usa para definir el valor que va a asignar el descriptor de acceso `set`.</span><span class="sxs-lookup"><span data-stu-id="e139a-111">The [value](../../../csharp/language-reference/keywords/value.md) keyword is used to define the value being assigned by the `set` accessor.</span></span>  
- <span data-ttu-id="e139a-112">Las propiedades pueden ser *de lectura y escritura* (en ambos casos tienen un descriptor de acceso `get` y `set`), *de solo lectura* (tienen un descriptor de acceso `get`, pero no `set`) o *de solo escritura* (tienen un descriptor de acceso `set`, pero no `get`).</span><span class="sxs-lookup"><span data-stu-id="e139a-112">Properties can be *read-write* (they have both a `get` and a `set` accessor), *read-only* (they have a `get` accessor but no `set` accessor), or *write-only* (they have a `set` accessor, but no `get` accessor).</span></span> <span data-ttu-id="e139a-113">Las propiedades de solo escritura son poco frecuentes y se suelen usar para restringir el acceso a datos confidenciales.</span><span class="sxs-lookup"><span data-stu-id="e139a-113">Write-only properties are rare and are most commonly used to restrict access to sensitive data.</span></span>

- <span data-ttu-id="e139a-114">Las propiedades simples que no necesitan ningún código de descriptor de acceso personalizado se pueden implementar como definiciones de cuerpos de expresión o como [propiedades implementadas automáticamente](../../../csharp/programming-guide/classes-and-structs/auto-implemented-properties.md).</span><span class="sxs-lookup"><span data-stu-id="e139a-114">Simple properties that require no custom accessor code can be implemented either as expression body definitions or as [auto-implemented properties](../../../csharp/programming-guide/classes-and-structs/auto-implemented-properties.md).</span></span>

<span data-ttu-id="e139a-116">Un patrón básico para implementar una propiedad conlleva el uso de un campo de respaldo privado para establecer y recuperar el valor de la propiedad.</span><span class="sxs-lookup"><span data-stu-id="e139a-116">One basic pattern for implementing a property involves using a private backing field for setting and retrieving the property value.</span></span> <span data-ttu-id="e139a-117">El descriptor de acceso `get` devuelve el valor del campo privado y el descriptor de acceso `set` puede realizar alguna validación de datos antes de asignar un valor al campo privado.</span><span class="sxs-lookup"><span data-stu-id="e139a-117">The `get` accessor returns the value of the private field, and the `set` accessor may perform some data validation before assigning a value to the private field.</span></span> <span data-ttu-id="e139a-118">Ambos descriptores de acceso además pueden realizar algún cálculo o conversión en los datos antes de que se almacenen o se devuelvan.</span><span class="sxs-lookup"><span data-stu-id="e139a-118">Both accessors may also perform some conversion or computation on the data before it is stored or returned.</span></span>
