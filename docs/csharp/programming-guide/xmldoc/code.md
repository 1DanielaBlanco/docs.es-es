# &lt;code&gt; (Gu&#237;a de programaci&#243;n de C#)
## Sintaxis  
  
```  
<code>content</code>  
```  
  
#### Parámetros  
 `content`  
 Texto que se desea marcar como código.  
  
## Comentarios  
 La etiqueta \<code\> proporciona un modo de marcar varias líneas como código.  Utilice [\<c\>](../../../csharp/programming-guide/xmldoc/code-inline.md) para indicar que el texto de una descripción se debería marcar como código.  
  
 Compile con el parámetro [\/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) para procesar los comentarios de documentación y generar un archivo con ellos.
