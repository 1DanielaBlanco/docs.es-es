# <a name="ltcodegt-c-programming-guide"></a>&lt;code&gt; (Guía de programación de C#)
## <a name="syntax"></a>Sintaxis  
  
```xml  
<code>content</code>  
```  
  
#### <a name="parameters"></a>Parámetros  
 `content`  
 El texto que quiere marcar como código.  
  
## <a name="remarks"></a>Comentarios  
 La etiqueta \<code> le proporciona una manera de indicar varias líneas como código. Use [\<c>](../../../csharp/programming-guide/xmldoc/code-inline.md) para indicar que el texto dentro de una descripción debe marcarse como código.  
  
 Compile con [/doc](../../../csharp/language-reference/compiler-options/doc-compiler-option.md) para procesar los comentarios de documentación a un archivo.
