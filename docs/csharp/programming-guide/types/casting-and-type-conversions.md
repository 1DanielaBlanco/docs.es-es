<span data-ttu-id="00b79-103">Dado que C# tiene tipos estáticos en tiempo de compilación, después de declarar una variable, no se puede volver a declarar ni usar para almacenar valores de otro tipo a menos que ese tipo sea convertible al tipo de la variable.</span><span class="sxs-lookup"><span data-stu-id="00b79-103">Because C# is statically-typed at compile time, after a variable is declared, it cannot be declared again or used to store values of another type unless that type is convertible to the variable's type.</span></span> <span data-ttu-id="00b79-104">Por ejemplo, no hay ninguna conversión de un entero a una cadena arbitraria.</span><span class="sxs-lookup"><span data-stu-id="00b79-104">For example, there is no conversion from an integer to any arbitrary string.</span></span> <span data-ttu-id="00b79-105">Por tanto, después de declarar `i` como un entero, no se le puede asignar la cadena "Hello", como se muestra en el código siguiente.</span><span class="sxs-lookup"><span data-stu-id="00b79-105">Therefore, after you declare `i` as an integer, you cannot assign the string "Hello" to it, as is shown in the following code.</span></span>
