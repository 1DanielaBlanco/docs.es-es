# <a name="serializing-xml-trees-c"></a>Serializar árboles XML (C#)
Serializar un árbol XML significa generar XML a partir de un árbol XML. Puede serializar a un archivo, a una implementación concreta de la clase <xref:System.IO.TextWriter> o a una implementación concreta de <xref:System.Xml.XmlWriter>.  
  
 Puede controlar diversos aspectos del proceso de serialización. Por ejemplo, puede decidir si desea agregar una sangría al XML serializado y si desea escribe una declaración XML.
