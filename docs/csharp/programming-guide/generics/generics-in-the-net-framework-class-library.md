# <a name="generics-in-the-net-framework-class-library-c-programming-guide"></a>Tipos genéricos en la biblioteca de clases de .NET Framework (Guía de programación de C#)
La versión 2.0 de la biblioteca de clases .NET Framework proporciona un nuevo espacio de nombres, <xref:System.Collections.Generic>, que incluye varias clases de colección genérica listas para usar e interfaces asociadas. Otros espacios de nombres, como <xref:System>, también proporcionan nuevas interfaces genéricas como <xref:System.IComparable%601>. Estas clases e interfaces tienen seguridad de tipos y son más eficientes que las clases de colección no genérica incluidas en versiones anteriores de .NET Framework. Antes de diseñar e implementar sus propias clases de colección personalizada, considere si puede usar o derivar una clase de una de las clases proporcionadas en la biblioteca de clases de .NET Framework.
