## <a name="example"></a><span data-ttu-id="a8636-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="a8636-110">Example</span></span>  
 <span data-ttu-id="a8636-111">El código siguiente generará la advertencia CS0728.</span><span class="sxs-lookup"><span data-stu-id="a8636-111">The following code will generate warning CS0728.</span></span>  
  
```  
// CS0728.cs  
  
using System;  
public class ValidBase : IDisposable  
{  
    public void Dispose() {  }  
}  
  
public class Logger  
{  
    public static void dummy()  
    {  
        ValidBase vb = null;  
        using (vb)   
        {  
            vb = null;  // CS0728  
        }  
        vb = null;  
    }  
    public static void Main() { }  
}  
```
