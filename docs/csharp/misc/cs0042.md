# <a name="compiler-error-cs0042"></a>Error del compilador CS0042
Error inesperado al crear el archivo de información de depuración 'archivo': 'motivo'.  
  
 El compilador no pudo crear la información de depuración; el *motivo* contiene información adicional sobre la condición del error.
