# Error del compilador CS1556
La construcción 'construcción' que se especificó para el método Main debe ser una clase o una estructura válida  
  
 Se pasó a la opción del compilador [\/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) un identificador que no es un nombre de clase.
