# <a name="compiler-error-cs0025"></a>Error del compilador CS0025
No se encuentra el archivo de biblioteca estándar 'file'.  
  
 No se encontró un archivo necesario para el compilador. Compruebe que la ruta de acceso es correcta y que el archivo existe.  
  
 Si el archivo es un archivo de sistema de Visual Studio, debe reparar la instalación de Visual Studio o volver a instalarlo completamente.
