# Error del compilador CS0025
No se encuentra el archivo de biblioteca estándar 'file'.  
  
 No se encontró un archivo necesario para el compilador. Compruebe que la ruta de acceso es correcta y que el archivo existe.  
  
 Si el archivo es un archivo del sistema de [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)], puede que necesite reparar la instalación de [!INCLUDE[vsprvs](../../csharp/includes/vsprvs-md.md)] o volver a instalarlo completamente.
