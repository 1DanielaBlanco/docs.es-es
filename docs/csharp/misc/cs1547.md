# Error del compilador CS1547
La palabra clave 'void' no se puede usar en este contexto  
  
 El compilador detectó un uso no válido de la palabra clave [void](../../csharp/language-reference/keywords/void.md).  
  
 El ejemplo siguiente genera la advertencia CS1547:  
  
```  
// CS1547.cs public class MyClass { void BadMethod() { void i;   // CS1547, cannot have variables of type void } public static void Main() { } }  
```
