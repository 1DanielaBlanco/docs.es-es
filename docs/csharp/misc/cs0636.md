<span data-ttu-id="79ad9-103">El atributo FieldOffset solo se puede colocar en miembros de tipos marcados con StructLayout(LayoutKind.Explicit)</span><span class="sxs-lookup"><span data-stu-id="79ad9-103">The FieldOffset attribute can only be placed on members of types marked with the StructLayout(LayoutKind.Explicit)</span></span>  
  
 <span data-ttu-id="79ad9-104">Debe usar el atributo **StructLayout (LayoutKind.Explicit)** en la declaración struct, si contiene algún miembro marcado con el atributo **FieldOffset** .</span><span class="sxs-lookup"><span data-stu-id="79ad9-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="79ad9-105">Para obtener más información, vea [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="79ad9-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>  
  
 <span data-ttu-id="79ad9-106">El ejemplo siguiente genera la advertencia CS0636:</span><span class="sxs-lookup"><span data-stu-id="79ad9-106">The following sample generates CS0636:</span></span>  
  
```csharp  
// CS0636.cs

using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]

{  
   [FieldOffset(4)]public int i;   // CS0636   
}

{  
   public static void Main ()  
   {  
   }  
}  
```
