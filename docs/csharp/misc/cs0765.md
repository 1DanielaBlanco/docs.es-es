# <a name="compiler-error-cs0765"></a><span data-ttu-id="03648-102">Error del compilador CS0765</span><span class="sxs-lookup"><span data-stu-id="03648-102">Compiler Error CS0765</span></span>
<span data-ttu-id="03648-103">En los árboles de expresión no se pueden usar métodos parciales con solo una declaración de definición ni métodos condicionales quitados</span><span class="sxs-lookup"><span data-stu-id="03648-103">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="03648-104">Aunque una llamada a un método parcial quitado es una expresión, no es una expresión aceptable en un árbol de expresión.</span><span class="sxs-lookup"><span data-stu-id="03648-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="03648-105">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="03648-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="03648-106">Agregue una declaración de implementación para el método parcial o quite el código que está causando que el método condicional se excluya de la compilación.</span><span class="sxs-lookup"><span data-stu-id="03648-106">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="03648-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="03648-107">Example</span></span>  
 <span data-ttu-id="03648-108">El código siguiente genera el error CS0765 en dos ubicaciones:</span><span class="sxs-lookup"><span data-stu-id="03648-108">The following code generates CS0765 in two locations:</span></span>  
  
```csharp  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
