# <a name="compiler-error-cs1627"></a><span data-ttu-id="938f6-102">Error del compilador CS1627</span><span class="sxs-lookup"><span data-stu-id="938f6-102">Compiler Error CS1627</span></span>

 <span data-ttu-id="938f6-104">Este error se produce si `yield` se usa sin una expresión.</span><span class="sxs-lookup"><span data-stu-id="938f6-104">This error occurs if `yield` is used without an expression.</span></span> <span data-ttu-id="938f6-105">Para evitar este error, inserte la expresión adecuada en la instrucción.</span><span class="sxs-lookup"><span data-stu-id="938f6-105">To avoid this error, insert the appropriate expression in the statement.</span></span>  
  
 <span data-ttu-id="938f6-106">El ejemplo siguiente genera la advertencia CS1627:</span><span class="sxs-lookup"><span data-stu-id="938f6-106">The following sample generates CS1627:</span></span>  
  
```  
// CS1627.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      yield return;   // CS1627  
      // To resolve, add the following line:  
      // yield return 0;  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
