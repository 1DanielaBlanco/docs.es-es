<span data-ttu-id="44bee-103">No se puede adquirir la dirección de la expresión dada</span><span class="sxs-lookup"><span data-stu-id="44bee-103">Cannot take the address of the given expression</span></span>  
  
 <span data-ttu-id="44bee-104">Puede tomar la dirección de campos, variables locales y el direccionamiento indirecto de punteros, pero no puede tomar, por ejemplo, la dirección de la suma de dos variables locales.</span><span class="sxs-lookup"><span data-stu-id="44bee-104">You can take the address of fields, local variables, and indirection of pointers, but you cannot take, for example, the address of the sum of two local variables.</span></span> <span data-ttu-id="44bee-105">Para obtener más información, vea [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md) (Código no seguro y punteros [Guía de programación de C#]).</span><span class="sxs-lookup"><span data-stu-id="44bee-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="44bee-106">El ejemplo siguiente genera la advertencia CS0211:</span><span class="sxs-lookup"><span data-stu-id="44bee-106">The following sample generates CS0211:</span></span>  
  
```csharp  
// CS0211.cs  
// compile with: /unsafe  
  
public class MyClass  
{  
   unsafe public void M()  
   {  
      int a = 0, b = 0;  
      int *i = &(a + b);   // CS0211, the addition of two local variables  
      // try the following line instead  
      // int *i = &a;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
