# <a name="compiler-error-cs1639"></a>Error del compilador CS1639
La firma 'signature' de la clase contenedora de la coclase administrada para la interfaz 'interface' no es una firma de nombre de clase válida  
  
 Este error se produce si los metadatos de la interfaz no son válidos. Esto no debería suceder nunca si la interfaz se ha generado mediante C# o un lenguaje .NET compatible. Póngase en contacto con el proveedor de la interfaz o compruebe que el ensamblado de la interfaz se ha generado correctamente.
