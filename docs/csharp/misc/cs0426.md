# <a name="compiler-error-cs0426"></a><span data-ttu-id="87544-102">Error del compilador CS0426</span><span class="sxs-lookup"><span data-stu-id="87544-102">Compiler Error CS0426</span></span>
<span data-ttu-id="87544-103">El nombre de tipo 'identificador' no existe en el tipo 'tipo'</span><span class="sxs-lookup"><span data-stu-id="87544-103">The type name 'identifier' does not exist in the type 'type'</span></span>  
  
 <span data-ttu-id="87544-104">Este error se genera cuando se hace referencia a un tipo anidado dentro de otro tipo, pero dicho tipo anidado no existe.</span><span class="sxs-lookup"><span data-stu-id="87544-104">This error occurs when you reference a type nested within another type, but no such nested type exists.</span></span> <span data-ttu-id="87544-105">Esto puede ocurrir si se escribe incorrectamente el nombre del tipo anidado.</span><span class="sxs-lookup"><span data-stu-id="87544-105">This can occur if you mistype the name of the nested type.</span></span> <span data-ttu-id="87544-106">Compruebe la ortografía de los nombres usados y compruebe que el tipo envolvente tenga el miembro esperado.</span><span class="sxs-lookup"><span data-stu-id="87544-106">Check the spelling of the names used, and verify that the enclosing type has the expected member.</span></span>  
  
 <span data-ttu-id="87544-107">El ejemplo siguiente genera la advertencia CS0426, porque la clase C no tiene ningún tipo anidado A:</span><span class="sxs-lookup"><span data-stu-id="87544-107">The following sample generates CS0426 because class C has no nested type A:</span></span>  
  
```  
// CS0426.cs  
  
class C  
{  
    // No nested types are declared.     
}  
  
class D  
{  
   public static void Main()  
   {  
       C c = new C();  
       // Attempt to reference a nested type A:  
       C.A a; // CS0426 because no such type C.A  
   }  
}  
```
