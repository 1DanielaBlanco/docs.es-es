# <a name="compiler-error-cs1615"></a><span data-ttu-id="11071-102">Error del compilador CS1615</span><span class="sxs-lookup"><span data-stu-id="11071-102">Compiler Error CS1615</span></span>
<span data-ttu-id="11071-103">No se debe pasar el argumento 'número' con la palabra clave 'palabra clave'.</span><span class="sxs-lookup"><span data-stu-id="11071-103">Argument 'number' should not be passed with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="11071-104">Se usó una de las palabras clave `ref` u **out** cuando la función no tomaba un parámetro `ref` u **out** para ese argumento.</span><span class="sxs-lookup"><span data-stu-id="11071-104">One of the keywords `ref` or **out** was used when the function did not take a `ref` or **out** parameter for that argument.</span></span> <span data-ttu-id="11071-105">Para resolver este error, quite la palabra clave incorrecta y use la palabra clave adecuada que coincida con la declaración de función, si existe.</span><span class="sxs-lookup"><span data-stu-id="11071-105">To resolve this error, remove the incorrect keyword and use the appropriate keyword that matches the function declaration, if any.</span></span>  
  
 <span data-ttu-id="11071-106">El ejemplo siguiente genera la advertencia CS1615:</span><span class="sxs-lookup"><span data-stu-id="11071-106">The following sample generates CS1615:</span></span>  
  
```csharp  
// CS1615.cs  
class C  
{  
   public void f(int i) {}  
   public static void Main()  
   {  
      int i = 1;  
      f(ref i);  // CS1615  
   }  
}  
```
