## <a name="example"></a><span data-ttu-id="58e3e-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="58e3e-109">Example</span></span>

```

using System.Linq;  
public class TestClass  
{  
    public string Message { get; set; }  
    public static int Number { get; set; }      
}  
class Test  
{  
    static void Main()  
    {  
        TestClass b = new TestClass() { Message = "Hello", Number = "555-1212" }; // CS1914  
  
    }  
}  
```
