# <a name="compiler-error-cs0066"></a><span data-ttu-id="0499d-102">Error del compilador CS0066</span><span class="sxs-lookup"><span data-stu-id="0499d-102">Compiler Error CS0066</span></span>
<span data-ttu-id="0499d-103">'evento': el evento debe ser de tipo delegado</span><span class="sxs-lookup"><span data-stu-id="0499d-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="0499d-104">La palabra clave event requiere un tipo [delegate](../../csharp/language-reference/keywords/delegate.md) .</span><span class="sxs-lookup"><span data-stu-id="0499d-104">The event keyword requires a [delegate](../../csharp/language-reference/keywords/delegate.md) type.</span></span> <span data-ttu-id="0499d-105">Para obtener más información, consulte [eventos](../../csharp/programming-guide/events/index.md) y [delegados](../../csharp/programming-guide/delegates/index.md).</span><span class="sxs-lookup"><span data-stu-id="0499d-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="0499d-106">El ejemplo siguiente genera la advertencia CS0066:</span><span class="sxs-lookup"><span data-stu-id="0499d-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
