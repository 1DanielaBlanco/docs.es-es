## <a name="example"></a><span data-ttu-id="71b45-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="71b45-105">Example</span></span>

```csharp
// CS0135.cs  
public class MyClass2  
{  
   public static int i = 0;  
  
   public static void Main()  
   {  
      {  
         int i = 4;  
         i++;  
      }  
      i = 0;   // CS0135  
   }  
}
```  
