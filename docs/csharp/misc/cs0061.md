 <span data-ttu-id="81bc3-104">Una construcción [public](../../csharp/language-reference/keywords/public.md) debe devolver un objeto accesible públicamente.</span><span class="sxs-lookup"><span data-stu-id="81bc3-104">A [public](../../csharp/language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="81bc3-107">El ejemplo siguiente genera la advertencia CS0061.</span><span class="sxs-lookup"><span data-stu-id="81bc3-107">The following sample generates CS0061.</span></span>  
  
```  
// CS0061.cs  
// compile with: /target:library  
internal interface A {}  
public interface AA : A {}  // CS0061  
  
// OK  
public interface B {}  
internal interface BB : B {}  
  
internal interface C {}  
internal interface CC : C {}  
```
