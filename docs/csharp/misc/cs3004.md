# <a name="compiler-warning-level-1-cs3004"></a>Advertencia del compilador (nivel 1) CS3004
Los caracteres Unicode mixtos y descompuestos no son conformes a CLS  
  
 Se permiten caracteres UNICODE compuestos sólo en [público](../../csharp/language-reference/keywords/public.md), [protegido](../../csharp/language-reference/keywords/protected.md), o `protected internal` identificadores para ser compatible con Common Language Specification (CLS). Para obtener más información sobre la conformidad con CLS, vea [Escribir código conforme con CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3) e [Independencia del lenguaje y componentes independientes del lenguaje](../../../docs/standard/language-independence-and-language-independent-components.md).
