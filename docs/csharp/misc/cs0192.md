 <span data-ttu-id="7f982-104">Un campo (variable) marcado con la [readonly](../../csharp/language-reference/keywords/readonly.md) palabra clave no se puede pasar a un [ref](../../csharp/language-reference/keywords/ref.md) o [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parámetro excepto dentro de un constructor.</span><span class="sxs-lookup"><span data-stu-id="7f982-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="7f982-105">Para obtener más información, vea [Campos (Guía de programación de C#)](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="7f982-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>

## <a name="example"></a><span data-ttu-id="7f982-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="7f982-107">Example</span></span>  
 <span data-ttu-id="7f982-108">El ejemplo siguiente genera la advertencia CS0192:</span><span class="sxs-lookup"><span data-stu-id="7f982-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
