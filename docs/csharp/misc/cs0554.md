# <a name="compiler-error-cs0554"></a><span data-ttu-id="c986a-102">Error del compilador CS0554</span><span class="sxs-lookup"><span data-stu-id="c986a-102">Compiler Error CS0554</span></span>
<span data-ttu-id="c986a-103">'rutina de conversión': conversión definida por el usuario a clase derivada o desde ella</span><span class="sxs-lookup"><span data-stu-id="c986a-103">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="c986a-104">No se permiten las conversiones definidas por el usuario a valores de una clase derivada; no es necesario este tipo de operador.</span><span class="sxs-lookup"><span data-stu-id="c986a-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="c986a-105">Consulte el capítulo 6 en la especificación del lenguaje C# para obtener más información sobre las conversiones definidas por el usuario.</span><span class="sxs-lookup"><span data-stu-id="c986a-105">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="c986a-106">El ejemplo siguiente genera la advertencia CS0554:</span><span class="sxs-lookup"><span data-stu-id="c986a-106">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
