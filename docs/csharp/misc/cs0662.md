<span data-ttu-id="9ffa1-103">'method' no puede especificar solo el atributo Out en un parámetro ref.</span><span class="sxs-lookup"><span data-stu-id="9ffa1-103">'method' cannot specify only Out attribute on a ref parameter.</span></span> <span data-ttu-id="9ffa1-104">Use ambos atributos In y Out, o bien ninguno.</span><span class="sxs-lookup"><span data-stu-id="9ffa1-104">Use both In and Out attributes, or neither.</span></span>  
  
 <span data-ttu-id="9ffa1-105">Un método de interfaz tiene un parámetro que usa [ref](../../csharp/language-reference/keywords/ref.md) con el atributo [Out](xref:System.Runtime.InteropServices.OutAttribute) solamente.</span><span class="sxs-lookup"><span data-stu-id="9ffa1-105">An interface method has a parameter that uses [ref](../../csharp/language-reference/keywords/ref.md) with just the [Out](xref:System.Runtime.InteropServices.OutAttribute) attribute.</span></span> <span data-ttu-id="9ffa1-106">Un parámetro `ref` que usa el atributo **Out** también debe usar el atributo [In](xref:System.Runtime.InteropServices.InAttribute) .</span><span class="sxs-lookup"><span data-stu-id="9ffa1-106">A `ref` parameter that uses the **Out** attribute must also use the [In](xref:System.Runtime.InteropServices.InAttribute) attribute.</span></span>  
  
 <span data-ttu-id="9ffa1-107">El ejemplo siguiente genera la advertencia CS0662:</span><span class="sxs-lookup"><span data-stu-id="9ffa1-107">The following sample generates CS0662:</span></span>  
  
```csharp
// CS0662.cs  
using System.Runtime.InteropServices;  
  
interface I  
{  
   void method([Out] ref int i);   // CS0662  
   // try one of the following lines instead  
   // void method(ref int i);  
   // void method([Out, In]ref int i);  
}  
  
class test  
{  
   public static void Main()  
   {  
   }  
}  
```
