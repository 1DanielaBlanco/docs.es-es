# <a name="compiler-error-cs1583"></a>Error del compilador CS1583
'archivo' no es un archivo de recursos Win32 válido.  
  
 Este error se produce cuando se especifica un nombre de archivo con el [/win32res](../../csharp/language-reference/compiler-options/win32res-compiler-option.md) opción del compilador que no es un archivo de recursos Win32 válido o con el formato correcto. En Visual Studio, el nombre de archivo se especifica en el panel Aplicación del Diseñador de proyectos.
