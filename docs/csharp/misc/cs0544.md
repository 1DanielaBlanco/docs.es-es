 <span data-ttu-id="23086-104">Se intentó reemplazar un tipo de datos que no era de propiedad como una [propiedad](../../csharp/programming-guide/classes-and-structs/properties.md), lo que no está permitido.</span><span class="sxs-lookup"><span data-stu-id="23086-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>

{  
   public int i;

      }
