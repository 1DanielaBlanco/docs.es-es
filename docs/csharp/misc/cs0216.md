# <a name="compiler-error-cs0216"></a><span data-ttu-id="8b686-102">Error del compilador CS0216</span><span class="sxs-lookup"><span data-stu-id="8b686-102">Compiler Error CS0216</span></span>
<span data-ttu-id="8b686-103">El operador 'operator' requiere que también se defina un operador coincidente 'missing_operator'.</span><span class="sxs-lookup"><span data-stu-id="8b686-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="8b686-104">El operador [true](../../csharp/language-reference/keywords/true.md) definido por el usuario requiere un operador [false](../../csharp/language-reference/keywords/false.md) definido por el usuario y viceversa.</span><span class="sxs-lookup"><span data-stu-id="8b686-104">A user-defined [true](../../csharp/language-reference/keywords/true.md) operator requires a user-defined [false](../../csharp/language-reference/keywords/false.md) operator, and vice versa.</span></span> <span data-ttu-id="8b686-105">Para obtener más información, vea [Operadores (Guía de programación de C#)](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span><span class="sxs-lookup"><span data-stu-id="8b686-105">For more information, see [Operators](../../csharp/programming-guide/statements-expressions-operators/operators.md).</span></span>  
  
 <span data-ttu-id="8b686-106">El ejemplo siguiente genera la advertencia CS0216:</span><span class="sxs-lookup"><span data-stu-id="8b686-106">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
