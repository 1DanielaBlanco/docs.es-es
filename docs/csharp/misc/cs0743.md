# <a name="compiler-error-cs0743"></a><span data-ttu-id="c1954-102">Error del compilador CS0743</span><span class="sxs-lookup"><span data-stu-id="c1954-102">Compiler Error CS0743</span></span>
<span data-ttu-id="c1954-103">Se esperaba la palabra clave contextual 'on'</span><span class="sxs-lookup"><span data-stu-id="c1954-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="c1954-104">El patrón para una cláusula `join` es `join`...`in`...`on`...`equals`, como se muestra en este ejemplo:</span><span class="sxs-lookup"><span data-stu-id="c1954-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y  
            select x;

## <a name="example"></a><span data-ttu-id="c1954-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="c1954-107">Example</span></span>  
 <span data-ttu-id="c1954-108">El código siguiente genera el error CS0743:</span><span class="sxs-lookup"><span data-stu-id="c1954-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
