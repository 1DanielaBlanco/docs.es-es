# Error del compilador CS0076
El nombre de enumerador 'value\_\_' está reservado y no se puede usar.  
  
 Un elemento de una [enumeración](../../csharp/language-reference/keywords/enum.md) no puede tener un identificador denominado **value\_\_**.
