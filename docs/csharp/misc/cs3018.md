# <a name="compiler-warning-level-1-cs3018"></a>Advertencia del compilador (nivel 1) CS3018
El tipo 'type' no se puede marcar como conforme con CLS porque es miembro de un tipo 'type' no conforme con CLS  
  
 Esta advertencia se produce si una clase anidada con el atributo CLSCompliant establecido en `true` se declara como un miembro de una clase declarada con el atributo CLSCompliant establecido en `false`. Esto no se permite, ya que una clase anidada no puede ser conforme con CLS si es miembro de una clase externa que no es conforme con CLS. ## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera la advertencia CS3018.  
  
```  
// CS3018.cs  
// compile with: /target:library  
using System;  
  
[assembly: CLSCompliant(true)]  
[CLSCompliant(false)]  
public class Outer  
{  
   [CLSCompliant(true)]   // CS3018  
   public class Nested {}  
  
   // OK  
   public class Nested2 {}  
  
   [CLSCompliant(false)]  
   public class Nested3 {}  
}  
```
