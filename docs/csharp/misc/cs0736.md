# <a name="compiler-error-cs0736"></a><span data-ttu-id="a4676-102">Error del compilador CS0736</span><span class="sxs-lookup"><span data-stu-id="a4676-102">Compiler Error CS0736</span></span>

## <a name="example"></a><span data-ttu-id="a4676-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="a4676-110">Example</span></span>  
 <span data-ttu-id="a4676-111">El código siguiente genera CS0736 porque `Program.testMethod` está declarado como estático:</span><span class="sxs-lookup"><span data-stu-id="a4676-111">The following code generates CS0736 because `Program.testMethod` is declared as static:</span></span>  
  
```csharp

namespace CS0736  
{     
  
    interface ITest  
    {  
        int testMethod(int x);  
    }  
  
    class Program : ITest // CS0736  
    {  
        public static int testMethod(int x) { return 0; }  
        // Try the following line instead.  
        // public int testMethod(int x) { return 0; }  
        public static void Main() { }  
    }      
}  
```
