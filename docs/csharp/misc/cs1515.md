# <a name="compiler-error-cs1515"></a>Error del compilador CS1515
Se esperaba 'in'  
  
 En una instrucci√≥n [foreach, in](../../csharp/language-reference/keywords/foreach-in.md) , falta la parte "in".  
  
## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera la advertencia CS1515:  
  
```  
using System;  
  
class Driver  
{  
   static void Main()  
   {  
      int[] arr = new int[] {1, 2, 3};  
  
      // try the following line instead  
      // foreach (int x in arr)  
      foreach (int x arr)      // CS1515, "in" is missing  
      {  
         Console.WriteLine(x);  
      }  
   }  
}  
```
