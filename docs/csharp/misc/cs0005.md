# <a name="compiler-error-cs0005"></a>Error del compilador CS0005
La opción 'compiler_option' del compilador debe ir seguida de un argumento  
  
 Algunas opciones del compilador requieren parámetros. Si no pasa los argumentos requeridos por la opción del compilador, se genera CS0005.  
  
 Para obtener más información, vea [Opciones del compilador C#](../../csharp/language-reference/compiler-options/index.md).
