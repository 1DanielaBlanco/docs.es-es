# <a name="compiler-error-cs1101"></a><span data-ttu-id="8aa25-102">Error del compilador CS1101</span><span class="sxs-lookup"><span data-stu-id="8aa25-102">Compiler Error CS1101</span></span>
<span data-ttu-id="8aa25-103">El modificador de parámetro 'ref' no se puede usar con 'this'</span><span class="sxs-lookup"><span data-stu-id="8aa25-103">The parameter modifier 'ref' cannot be used with 'this'.</span></span>  
  
 <span data-ttu-id="8aa25-104">Cuando la palabra clave `this` modifica el primer parámetro de un método estático, indica al compilador que el método es un método de extensión.</span><span class="sxs-lookup"><span data-stu-id="8aa25-104">When the `this` keyword modifies the first parameter of a static method, it signals to the compiler that the method is an extension method.</span></span> <span data-ttu-id="8aa25-105">No se necesiten ni se permiten otros modificadores en el primer parámetro de un método de extensión.</span><span class="sxs-lookup"><span data-stu-id="8aa25-105">No other modifiers are needed or allowed on the first parameter of an extension method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8aa25-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="8aa25-106">Example</span></span>  
 <span data-ttu-id="8aa25-107">El ejemplo siguiente genera el error CS1101:</span><span class="sxs-lookup"><span data-stu-id="8aa25-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
