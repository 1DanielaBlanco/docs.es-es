## <a name="example"></a><span data-ttu-id="aad1d-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="aad1d-106">Example</span></span>  
 <span data-ttu-id="aad1d-107">El ejemplo siguiente genera el error CS1101:</span><span class="sxs-lookup"><span data-stu-id="aad1d-107">The following example generates CS1101:</span></span>  
  
```csharp  
// cs1101.cs  
// Compile with: /target:library  
public static class Extensions  
{  
    // No type parameters.  
        public static void Test(ref this int i) {} // CS1101  
  
    // Single type parameter.  
        public static void Test<T>(ref this T t) {}// CS1101  
  
    // Multiple type parameters.  
        public static void Test<T,U,V>(ref this U u) {}// CS1101  
}  
```
