# <a name="compiler-warning-level-1-cs1720"></a><span data-ttu-id="b6326-102">Advertencia del compilador (nivel 1) CS1720</span><span class="sxs-lookup"><span data-stu-id="b6326-102">Compiler Warning (level 1) CS1720</span></span>
<span data-ttu-id="b6326-103">La expresión siempre producirá la excepción System.NullReferenceException porque el valor predeterminado de 'tipo genérico' es null</span><span class="sxs-lookup"><span data-stu-id="b6326-103">Expression will always cause a System.NullReferenceException because the default value of 'generic type' is null</span></span>  
  
 <span data-ttu-id="b6326-104">Si escribe una expresión que contiene el valor predeterminado de una variable de tipo genérico que es un tipo de referencia (por ejemplo, una clase), se producirá este error.</span><span class="sxs-lookup"><span data-stu-id="b6326-104">If you write an expression involving the default of a generic type variable that is a reference type (for example, a class), this error will occur.</span></span> <span data-ttu-id="b6326-105">Observe la siguiente expresión:</span><span class="sxs-lookup"><span data-stu-id="b6326-105">Consider the following expression:</span></span>
