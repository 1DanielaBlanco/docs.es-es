# <a name="compiler-error-cs1918"></a><span data-ttu-id="9df72-102">Error del compilador CS1918</span><span class="sxs-lookup"><span data-stu-id="9df72-102">Compiler Error CS1918</span></span>
<span data-ttu-id="9df72-103">Los miembros de la propiedad 'nombre' de tipo 'tipo' no se pueden asignar con un inicializador de objeto porque es de un tipo de valor.</span><span class="sxs-lookup"><span data-stu-id="9df72-103">Members of property 'name' of type 'type' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="9df72-104">Este error se produce al intentar usar un inicializador de objeto para inicializar las propiedades de un tipo de estructura que a su vez es una propiedad de la clase que se va a inicializar.</span><span class="sxs-lookup"><span data-stu-id="9df72-104">This error occurs when you try to use an object initializer to initialize the properties of a struct type that is itself a property of the class that is being initialized.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="9df72-105">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="9df72-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="9df72-106">Si debe inicializar completamente los campos de la propiedad en el inicializador de objeto, cambie la estructura a un tipo de clase.</span><span class="sxs-lookup"><span data-stu-id="9df72-106">If you must fully initialize the fields of the property in the object initializer, change the struct to a class type.</span></span> <span data-ttu-id="9df72-107">De lo contrario, inicialice los miembros de la estructura en una llamada a método independiente después de crear el objeto mediante el inicializador de objeto.</span><span class="sxs-lookup"><span data-stu-id="9df72-107">Otherwise, initialize the struct members in a separate method call after you create the object by using the object initializer.</span></span>  
  
## <a name="example"></a><span data-ttu-id="9df72-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="9df72-108">Example</span></span>  
 <span data-ttu-id="9df72-109">El ejemplo siguiente genera el error CS1918:</span><span class="sxs-lookup"><span data-stu-id="9df72-109">The following example generates CS1918:</span></span>  
  
```csharp  
// cs1918.cs  
public struct MyStruct  
{  
    public int i;  
  
}  
public class Test  
{  
    private MyStruct str = new MyStruct();  
    public MyStruct Str  
    {  
        get  
        {  
            return str;  
        }  
    }  
    public static int Main()  
    {  
        Test t = new Test { Str = { i = 1 } }; // CS1918  
        return 0;  
    }  
}  
```
