# <a name="compiler-error-cs0549"></a><span data-ttu-id="43096-102">Error del compilador CS0549</span><span class="sxs-lookup"><span data-stu-id="43096-102">Compiler Error CS0549</span></span>

 <span data-ttu-id="43096-106">El ejemplo siguiente genera la advertencia CS0549:</span><span class="sxs-lookup"><span data-stu-id="43096-106">The following sample generates CS0549:</span></span>

```csharp  
{  
   virtual public void TestMethod() {}   // CS0549

}  
```
