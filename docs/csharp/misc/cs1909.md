## <a name="example"></a><span data-ttu-id="6963c-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="6963c-105">Example</span></span>  
 <span data-ttu-id="6963c-106">El ejemplo siguiente genera la advertencia CS1909.</span><span class="sxs-lookup"><span data-stu-id="6963c-106">The following sample generates CS1909.</span></span>  
  
```csharp  
// CS1909.cs

public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909  
  
   void Test2([DefaultParameterValue("Test String")] string s);   // OK  
}  
```
