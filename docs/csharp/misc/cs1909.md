## <a name="example"></a><span data-ttu-id="8423d-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="8423d-105">Example</span></span>

```csharp

public interface ISomeInterface  
{  
   void Test1([DefaultParameterValue(new int[] {1, 2})] int[] arr1);   // CS1909

}  
```
