# <a name="compiler-error-cs0447"></a><span data-ttu-id="83ba3-102">Error del compilador CS0447</span><span class="sxs-lookup"><span data-stu-id="83ba3-102">Compiler Error CS0447</span></span>
<span data-ttu-id="83ba3-103">Los atributos no se pueden utilizar en argumentos de tipo, sólo en parámetros de tipo</span><span class="sxs-lookup"><span data-stu-id="83ba3-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="83ba3-104">Este error se produce cuando se aplica un atributo a un argumento de tipo que se produce en una instrucción de invocación.</span><span class="sxs-lookup"><span data-stu-id="83ba3-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="83ba3-105">Es aceptable aplicar un atributo a un parámetro de tipo en una instrucción de declaración de clase o método como la que se muestra a continuación:</span><span class="sxs-lookup"><span data-stu-id="83ba3-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

```  
  
 <span data-ttu-id="83ba3-106">La siguiente línea de código generará este error.</span><span class="sxs-lookup"><span data-stu-id="83ba3-106">The following line of code will generate this error.</span></span> <span data-ttu-id="83ba3-107">Se supone que la clase `C`, definida en la línea anterior de código, tiene un método estático llamado `MyStaticMethod`.</span><span class="sxs-lookup"><span data-stu-id="83ba3-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

```
