## <a name="example"></a><span data-ttu-id="30b92-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="30b92-107">Example</span></span>  
 <span data-ttu-id="30b92-108">El ejemplo siguiente genera la advertencia CS3019:</span><span class="sxs-lookup"><span data-stu-id="30b92-108">The following sample generates CS3019:</span></span>  
  
```csharp  
// CS3019.cs  
// compile with: /W:2  
  
using System;  
  
[assembly: CLSCompliant(true)]  
  
// To fix the error, remove the next line  
[CLSCompliant(true)]  // CS3019

{  
    [CLSCompliant(false)]  // CS3019  
    void Foo()  
    {  
    }  
  
    static void Main()  
    {  
    }  
}  
```
