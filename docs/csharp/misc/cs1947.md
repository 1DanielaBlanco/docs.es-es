## <a name="example"></a><span data-ttu-id="23fee-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="23fee-109">Example</span></span>  
 <span data-ttu-id="23fee-110">El código siguiente genera el error CS1947:</span><span class="sxs-lookup"><span data-stu-id="23fee-110">The following code generates CS1947:</span></span>

```csharp  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array

    }  
}  
```
