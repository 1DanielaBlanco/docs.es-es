# <a name="compiler-error-cs1626"></a><span data-ttu-id="5f70e-102">Error del compilador CS1626</span><span class="sxs-lookup"><span data-stu-id="5f70e-102">Compiler Error CS1626</span></span>
<span data-ttu-id="5f70e-103">No se puede proporcionar un valor en el cuerpo de un bloque try con una cláusula catch</span><span class="sxs-lookup"><span data-stu-id="5f70e-103">Cannot yield a value in the body of a try block with a catch clause</span></span>  
  
 <span data-ttu-id="5f70e-104">Una instrucción yield no se permite en un bloque try si hay una cláusula catch asociada al bloque try.</span><span class="sxs-lookup"><span data-stu-id="5f70e-104">A yield statement is not allowed in a try block if there is a catch clause associated with the try block.</span></span> <span data-ttu-id="5f70e-105">Para evitar este error, mueva la instrucción yield fuera de la cláusula catch.</span><span class="sxs-lookup"><span data-stu-id="5f70e-105">To avoid this error, move the yield statement out of the catch clause.</span></span>  
  
 <span data-ttu-id="5f70e-106">El ejemplo siguiente genera la advertencia CS1626:</span><span class="sxs-lookup"><span data-stu-id="5f70e-106">The following sample generates CS1626:</span></span>  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
