# <a name="compiler-error-cs0013"></a>Error del compilador CS0013
Error inesperado al escribir metadatos en el archivo 'ModelStore.dll' -- 'No queda espacio lógico para crear más cadenas de usuario.  
  
 .NET Framework Common Language Runtime no emitió metadatos. Compruebe que la ruta de acceso sea correcta y que el disco no esté lleno. Si el problema persiste, tendrá que reparar o reinstalar Visual Studio o .NET Framework.
