 <span data-ttu-id="fe207-107">El ejemplo siguiente genera la advertencia CS0505:</span><span class="sxs-lookup"><span data-stu-id="fe207-107">The following sample generates CS0505:</span></span>  
  
```

{  
   public int i;  
}

   public override int i() { return 0; }   // CS0505  
}  
```
