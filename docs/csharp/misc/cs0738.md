## <a name="example"></a><span data-ttu-id="a909a-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="a909a-108">Example</span></span>  
 <span data-ttu-id="a909a-109">El código siguiente genera el error CS0738 porque el método de clase devuelve `void` y el miembro de interfaz del mismo nombre devuelve `int`:</span><span class="sxs-lookup"><span data-stu-id="a909a-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

```csharp  
interface ITest  
{  
    int TestMethod();  
}  
public class Test: ITest  
{  
    public void TestMethod() { } // CS0738  
    // Try the following line instead.  
    // public int TestMethod();  
}  
```
