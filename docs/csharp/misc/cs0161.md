 <span data-ttu-id="1071a-104">Un método que devuelve un valor debe tener una instrucción `return` en todas las rutas de código.</span><span class="sxs-lookup"><span data-stu-id="1071a-104">A method that returns a value must have a `return` statement in all code paths.</span></span> <span data-ttu-id="1071a-105">Para más información, vea [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="1071a-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="1071a-106">El ejemplo siguiente genera la advertencia CS0161:</span><span class="sxs-lookup"><span data-stu-id="1071a-106">The following sample generates CS0161:</span></span>  
  
```csharp  
// CS0161.cs  
public class Test  
{  
   public static int Main() // CS0161  
   {  
      int i = 10;  
      if (i < 10)  
      {  
         return i;  
      }  
      else  
      {  
         // uncomment the following line to resolve  
         // return 1;  
      }  
   }  
}  
```
