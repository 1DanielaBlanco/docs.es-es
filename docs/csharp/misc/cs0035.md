<span data-ttu-id="c31e3-103">El operador 'operator' es ambiguo en un operando del tipo 'type'</span><span class="sxs-lookup"><span data-stu-id="c31e3-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="c31e3-104">El compilador tiene más de una conversión disponible y no sabe cuál debe realizar antes de aplicar el operador.</span><span class="sxs-lookup"><span data-stu-id="c31e3-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="c31e3-105">Para obtener más información, consulte [conversiones de definidas por el usuario con plantilla](/cpp/misc/templated-user-defined-conversions) y [operadores de conversión](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="c31e3-105">For more information, see [Templated User Defined Conversions](/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="c31e3-106">El ejemplo siguiente genera la advertencia CS0035:</span><span class="sxs-lookup"><span data-stu-id="c31e3-106">The following sample generates CS0035:</span></span>  
  
```  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
