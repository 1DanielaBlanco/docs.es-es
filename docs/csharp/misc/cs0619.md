# <a name="compiler-error-cs0619"></a><span data-ttu-id="74942-102">Error del compilador CS0619</span><span class="sxs-lookup"><span data-stu-id="74942-102">Compiler Error CS0619</span></span>
<span data-ttu-id="74942-103">'member' está obsoleto: 'text'</span><span class="sxs-lookup"><span data-stu-id="74942-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="74942-104">Se marcó un miembro de clase con la [obsoleto](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) atributo, por ejemplo, que se emitirá un error cuando se haga referencia al miembro de clase.</span><span class="sxs-lookup"><span data-stu-id="74942-104">A class member was marked with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="74942-105">El ejemplo siguiente genera la advertencia CS0619:</span><span class="sxs-lookup"><span data-stu-id="74942-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
