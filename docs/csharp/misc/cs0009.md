# <a name="compiler-error-cs0009"></a>Error del compilador CS0009
No se pudo abrir el archivo de metadatos 'file' — 'description'  
  
 El archivo especificado con la opción del compilador [/reference](../../csharp/language-reference/compiler-options/reference-compiler-option.md) no contiene metadatos válidos.
