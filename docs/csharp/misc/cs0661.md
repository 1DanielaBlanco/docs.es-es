# <a name="compiler-warning-level-3-cs0661"></a><span data-ttu-id="f2d6d-102">Advertencia del compilador (nivel 3) CS0661</span><span class="sxs-lookup"><span data-stu-id="f2d6d-102">Compiler Warning (level 3) CS0661</span></span>
<span data-ttu-id="f2d6d-103">'class' define el operador == o el operador != pero no reemplaza Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="f2d6d-103">'class' defines operator == or operator != but does not override Object.GetHashCode()</span></span>  
  
 <span data-ttu-id="f2d6d-104">El compilador detectó la igualdad definida por el usuario o el operador de desigualdad, pero ningún reemplazo para la función **GetHashCode** .</span><span class="sxs-lookup"><span data-stu-id="f2d6d-104">The compiler detected the user-defined equality or inequality operator, but no override for the **GetHashCode** function.</span></span> <span data-ttu-id="f2d6d-105">Un operador de igualdad o desigualdad definido por el usuario implica que también desea reemplazar la función **GetHashCode** .</span><span class="sxs-lookup"><span data-stu-id="f2d6d-105">A user-defined equality or inequality operator implies that you also want to override the **GetHashCode** function.</span></span>  
  
 <span data-ttu-id="f2d6d-106">El ejemplo siguiente genera la advertencia CS0661:</span><span class="sxs-lookup"><span data-stu-id="f2d6d-106">The following sample generates CS0661:</span></span>  
  
```csharp  
// CS0661.cs  
// compile with: /W:3  
class Test   // CS0661  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   public static bool operator != (object o, Test t)  
   {  
      return true;  
   }  
  
   public override bool Equals(object o)  
   {  
      return true;  
   }  
  
   // uncomment the GetHashCode function to resolve  
   // public override int GetHashCode()  
   // {  
   //    return 0;  
   // }  
  
   public static void Main()  
   {  
   }  
}  
```
