<span data-ttu-id="042c7-103">El atributo Conditional no es válido en 'método' porque es un método de invalidación.</span><span class="sxs-lookup"><span data-stu-id="042c7-103">The Conditional attribute is not valid on 'method' because it is an override method</span></span>  
  
 <span data-ttu-id="042c7-104">El [condicional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) atributo no se permite en un método que se marca con la [invalidar](../../csharp/language-reference/keywords/override.md) palabra clave.</span><span class="sxs-lookup"><span data-stu-id="042c7-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute is not allowed on a method that is marked with the [override](../../csharp/language-reference/keywords/override.md) keyword.</span></span> <span data-ttu-id="042c7-105">Para obtener más información, consulte [saber cuándo se debe invalidar el uso y las nuevas palabras clave](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span><span class="sxs-lookup"><span data-stu-id="042c7-105">For more information, see [Knowing When to Use Override and New Keywords](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).</span></span>  
  
 <span data-ttu-id="042c7-106">El compilador nunca se enlaza a métodos de invalidación; solo se enlaza al método base y Common Language Runtime llama a la invalidación, según corresponda.</span><span class="sxs-lookup"><span data-stu-id="042c7-106">The compiler never binds to override methods; it only binds to the base method and the common language runtime calls the override, as appropriate.</span></span>  
  
 <span data-ttu-id="042c7-107">El ejemplo siguiente genera la advertencia CS0243:</span><span class="sxs-lookup"><span data-stu-id="042c7-107">The following sample generates CS0243:</span></span>  
  
```csharp  
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
