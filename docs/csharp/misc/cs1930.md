## <a name="example"></a><span data-ttu-id="5db24-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="5db24-108">Example</span></span>  
 <span data-ttu-id="5db24-109">El ejemplo siguiente genera el error CS1930 porque el identificador `num` se usa para la variable de rango en la cláusula `from` y para la variable de rango que introduce la cláusula `let` .</span><span class="sxs-lookup"><span data-stu-id="5db24-109">The following example generates CS1930 because the identifier `num` is used for the range variable in the `from` clause and for the range variable introduced by the `let` clause.</span></span>  
  
```  
// cs1930.cs  
using System.Linq;  
class Program  
{  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        var query = from num in nums  
                    let num = 3 // CS1930  
                    select num;   
    }  
}  
```
