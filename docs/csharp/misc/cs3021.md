 <span data-ttu-id="1ba75-104">Esta advertencia se produce si `[CLSCompliant(false)]` aparece en una clase de un ensamblado que no tiene un atributo de nivel de ensamblado CLSCompliant establecido en true (es decir, la línea `[assembly: CLSCompliant(true)]`).</span><span class="sxs-lookup"><span data-stu-id="1ba75-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="1ba75-105">Dado que el ensamblado no se declara a sí mismo conforme a CLS, no hay ninguna necesidad de que nada dentro del ensamblado que se declare no conforme, ya que se supone que no es conforme.</span><span class="sxs-lookup"><span data-stu-id="1ba75-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="1ba75-106">Para obtener más información sobre la conformidad con CLS, vea [escribir código conforme a CLS](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span><span class="sxs-lookup"><span data-stu-id="1ba75-106">For more information on CLS Compliance, see [Writing CLS-Compliant Code](http://msdn.microsoft.com/library/4c705105-69a2-4e5e-b24e-0633bc32c7f3).</span></span>  
  
 <span data-ttu-id="1ba75-107">Para hacer desaparecer esta advertencia, quite el atributo o agregue el atributo de nivel de ensamblado.</span><span class="sxs-lookup"><span data-stu-id="1ba75-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1ba75-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="1ba75-108">Example</span></span>  
 <span data-ttu-id="1ba75-109">El ejemplo siguiente genera el error CS3021:</span><span class="sxs-lookup"><span data-stu-id="1ba75-109">The following example generates CS3021:</span></span>  
  
```  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
