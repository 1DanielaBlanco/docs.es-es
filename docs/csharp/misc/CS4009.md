# <a name="compiler-error-cs4009"></a><span data-ttu-id="83fca-102">CS4009 de Error del compilador</span><span class="sxs-lookup"><span data-stu-id="83fca-102">Compiler Error CS4009</span></span>

<span data-ttu-id="83fca-103">'Type.Method': no se puede marcar un punto de entrada con el `async` modificador.</span><span class="sxs-lookup"><span data-stu-id="83fca-103">'Type.Method': an entry point cannot be marked with the `async` modifier.</span></span>

<span data-ttu-id="83fca-104">No se puede utilizar el `async` palabra clave en el punto de entrada de la aplicación (normalmente el `Main` método).</span><span class="sxs-lookup"><span data-stu-id="83fca-104">You cannot use the `async` keyword in the application entry point (typically the `Main` method).</span></span>  

<span data-ttu-id="83fca-105">El ejemplo siguiente genera CS4009:</span><span class="sxs-lookup"><span data-stu-id="83fca-105">The following example produces CS4009:</span></span>

```csharp
using System;
using System.Threading.Tasks;

public class Example
{
   public static async void Main()
   {
       Console.WriteLine("About to wait two seconds");
       await WaitTwoSeconds();
       Console.WriteLine("About to exit the program");
   }

   private static async Task WaitTwoSeconds()
   {
      await Task.Delay(2000);
      Console.WriteLine("Returning from an asynchronous method");
   } 
}
```
