# <a name="compiler-error-cs1605"></a>Error del compilador CS1605
No se puede pasar 'var' como un argumento out o ref porque es de solo lectura  
  
 Pasa una variable como un [ref](../../csharp/language-reference/keywords/ref.md) o [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parámetro debe modificarse en el método llamado. Por lo tanto, no es posible pasar un parámetro de solo lectura como `ref` o `out`.
