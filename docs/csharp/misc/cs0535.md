## <a name="example"></a><span data-ttu-id="3f573-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="3f573-106">Example</span></span>  
 <span data-ttu-id="3f573-107">El ejemplo siguiente genera la advertencia CS0535.</span><span class="sxs-lookup"><span data-stu-id="3f573-107">The following sample generates CS0535.</span></span>  
  
```  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
