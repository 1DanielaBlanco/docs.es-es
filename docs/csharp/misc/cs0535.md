 <span data-ttu-id="32477-104">Se derivó una [clase](../../csharp/language-reference/keywords/class.md) de una [interfaz](../../csharp/language-reference/keywords/interface.md), pero la clase no implementó uno o varios de los miembros de la interfaz.</span><span class="sxs-lookup"><span data-stu-id="32477-104">A [class](../../csharp/language-reference/keywords/class.md) derived from an [interface](../../csharp/language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="32477-105">Una clase debe implementar todos los miembros de las interfaces de la que deriva, o bien se la debe declarar como `abstract`.</span><span class="sxs-lookup"><span data-stu-id="32477-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="32477-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="32477-106">Example</span></span>  
 <span data-ttu-id="32477-107">El ejemplo siguiente genera la advertencia CS0535.</span><span class="sxs-lookup"><span data-stu-id="32477-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
