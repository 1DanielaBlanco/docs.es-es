# <a name="compiler-error-cs1719"></a>Error del compilador CS1719
Error al leer el archivo de recursos Win32 'File Name' -- 'reason'  
  
 Se produjo un error al intentar leer el archivo de recursos Win32 por el motivo indicado en el mensaje, por le general, algo como "archivo no encontrado" o "acceso denegado". Este error se resuelve corrigiendo el problema descrito en el motivo.
