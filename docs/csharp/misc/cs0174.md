# <a name="compiler-error-cs0174"></a>Error del compilador CS0174
Clase base requerida para una referencia 'base'  
  
 Este error solo se produce cuando Common Language Runtime de .NET Framework compila el código fuente de la clase `System.Object` , que es la única clase que no tiene clase base.
