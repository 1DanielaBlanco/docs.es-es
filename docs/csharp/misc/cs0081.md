<span data-ttu-id="b0b87-103">La declaración de parámetros de tipo debe ser un identificador, no un tipo</span><span class="sxs-lookup"><span data-stu-id="b0b87-103">Type parameter declaration must be an identifier not a type</span></span>  
  
 <span data-ttu-id="b0b87-104">Cuando declare un tipo o método genérico, especifique el parámetro de tipo como un identificador, por ejemplo, "T" o "inputType".</span><span class="sxs-lookup"><span data-stu-id="b0b87-104">When you declare a generic method or type, specify the type parameter as an identifier, for example "T" or "inputType".</span></span> <span data-ttu-id="b0b87-105">Cuando el código de cliente llama al método, proporciona el tipo, que reemplaza cada aparición del identificador en el cuerpo del método o la clase.</span><span class="sxs-lookup"><span data-stu-id="b0b87-105">When client code calls the method, it supplies the type, which replaces each occurrence of the identifier in the method or class body.</span></span> <span data-ttu-id="b0b87-106">Para obtener más información, consulte [parámetros de tipo genérico](../../csharp/programming-guide/generics/generic-type-parameters.md).</span><span class="sxs-lookup"><span data-stu-id="b0b87-106">For more information, see [Generic Type Parameters](../../csharp/programming-guide/generics/generic-type-parameters.md).</span></span>  
  
```  
// CS0081.cs  
class MyClass  
{  
   public void F<int>() {}   // CS0081  
   public void F<T>(T input) {}   // OK  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
      a.F<int>(2);  
      a.F<double>(.05);  
   }  
}  
```
