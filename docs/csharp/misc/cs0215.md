 <span data-ttu-id="2b7dc-104">Los operadores [true](../../csharp/language-reference/keywords/true.md) y [false](../../csharp/language-reference/keywords/false.md) definidos por el usuario deben tener un tipo de valor devuelto de [bool](../../csharp/language-reference/keywords/bool.md).</span><span class="sxs-lookup"><span data-stu-id="2b7dc-104">User-defined [true](../../csharp/language-reference/keywords/true.md) and [false](../../csharp/language-reference/keywords/false.md) operators must have a return type of [bool](../../csharp/language-reference/keywords/bool.md).</span></span> <span data-ttu-id="2b7dc-105">Para obtener más información, vea [Operadores sobrecargables](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span><span class="sxs-lookup"><span data-stu-id="2b7dc-105">For more information, see [Overloadable Operators](../../csharp/programming-guide/statements-expressions-operators/overloadable-operators.md).</span></span>  
  
 <span data-ttu-id="2b7dc-106">El ejemplo siguiente genera la advertencia CS0215:</span><span class="sxs-lookup"><span data-stu-id="2b7dc-106">The following sample generates CS0215:</span></span>  
  
```csharp  
// CS0215.cs  
class MyClass  
{  
   public static int operator true (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator true (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static int operator false (MyClass MyInt)   // CS0215  
   // try the following line instead  
   // public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
