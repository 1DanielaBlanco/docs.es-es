# <a name="compiler-error-cs0156"></a>Error del compilador CS0156
No se permite una instrucción throw sin argumentos en una cláusula finally anidada en la cláusula catch más cercana  
  
 Una instrucción [throw](../../csharp/language-reference/keywords/throw.md) sin parámetros solo puede aparecer en una cláusula **catch** que no tome parámetros.  
  
 Para obtener más información, consulte [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) y [excepciones y control de excepciones](../../csharp/programming-guide/exceptions/index.md).  
  
 El ejemplo siguiente genera la advertencia CS0156:  
  
```csharp  
// CS0156.cs  
using System;  
  
namespace MyNamespace  
{  
   public class MyClass2 : Exception  
   {  
   }  
  
   public class MyClass  
   {  
      public static void Main()  
      {  
         try  
         {  
            throw;   // CS0156  
         }  
  
         catch(MyClass2)  
         {  
            throw;   // this throw is valid  
         }  
      }  
   }  
}  
```
