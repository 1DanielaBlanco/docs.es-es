# Error del compilador CS0228
'tipo' no contiene una definición para 'miembro' o no es accesible.  
  
 Puede que haya reemplazado la versión del sistema de los elementos **System.Hashtable**, **System.String** o **System.Array** y el reemplazo no contiene `member`.  
  
 En otras circunstancias, debe reparar o reinstalar Visual Studio.
