## <a name="example"></a><span data-ttu-id="0cf3b-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="0cf3b-107">Example</span></span>  
 <span data-ttu-id="0cf3b-108">El código siguiente produce el error CS1952:</span><span class="sxs-lookup"><span data-stu-id="0cf3b-108">The following code produces CS1952:</span></span>  
  
```csharp

using System;  
using System.Linq.Expressions;  
  
class Test  
{  
    public static int M(__arglist)  
    {  
        return 1;  
    }  
  
    static int Main()  
    {  
        Expression<Func<int, int>> f = x => Test.M(__arglist(x)); // CS1952  
        return 1;  
    }  
}  
```
