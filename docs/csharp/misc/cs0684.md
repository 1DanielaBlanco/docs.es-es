# <a name="compiler-warning-level-1-cs0684"></a><span data-ttu-id="9cfb5-102">Advertencia del compilador (nivel 1) CS0684</span><span class="sxs-lookup"><span data-stu-id="9cfb5-102">Compiler Warning (level 1) CS0684</span></span>
<span data-ttu-id="9cfb5-103">La interfaz 'interfaz' marcada con 'CoClassAttribute' no está marcada con 'ComImportAttribute'</span><span class="sxs-lookup"><span data-stu-id="9cfb5-103">'interface' interface marked with 'CoClassAttribute' not marked with 'ComImportAttribute'</span></span>  
  
 <span data-ttu-id="9cfb5-104">Si especifica **CoClassAttribute** en una interfaz, también debe especificar **ComImportAttribute**.</span><span class="sxs-lookup"><span data-stu-id="9cfb5-104">If you specify **CoClassAttribute** on an interface, you must also specify **ComImportAttribute**.</span></span>  
  
 <span data-ttu-id="9cfb5-105">El ejemplo siguiente genera la advertencia CS0684:</span><span class="sxs-lookup"><span data-stu-id="9cfb5-105">The following sample generates CS0684:</span></span>  
  
```  
// CS0684.cs  
// compile with: /W:1  
using System;  
using System.Runtime.InteropServices;  
  
[CoClass(typeof(C))] // CS0684  
// try the following line instead  
// [CoClass(typeof(C)), ComImport]  
interface I  
{  
}  
  
class C  
{  
   static void Main() {}  
}  
```
