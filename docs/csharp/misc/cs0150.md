# <a name="compiler-error-cs0150"></a><span data-ttu-id="2bb1d-102">Error del compilador CS0150</span><span class="sxs-lookup"><span data-stu-id="2bb1d-102">Compiler Error CS0150</span></span>
<span data-ttu-id="2bb1d-103">Se espera un valor constante</span><span class="sxs-lookup"><span data-stu-id="2bb1d-103">A constant value is expected</span></span>  
  
 <span data-ttu-id="2bb1d-104">Se encontró una variable donde se esperaba una constante.</span><span class="sxs-lookup"><span data-stu-id="2bb1d-104">A variable was found where a constant was expected.</span></span> <span data-ttu-id="2bb1d-105">Para obtener más información, consulta [switch](../../csharp/language-reference/keywords/switch.md).</span><span class="sxs-lookup"><span data-stu-id="2bb1d-105">For more information, see [switch](../../csharp/language-reference/keywords/switch.md).</span></span>  
  
 <span data-ttu-id="2bb1d-106">El ejemplo siguiente genera la advertencia CS0150:</span><span class="sxs-lookup"><span data-stu-id="2bb1d-106">The following sample generates CS0150:</span></span>  
  
```  
// CS0150.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         int j = 0;  
  
         switch(i)  
         {  
            case j:   // CS0150, j is a variable int, not a constant int  
            // try the following line instead  
            // case 1:  
         }  
      }  
   }  
}  
```  
  
 <span data-ttu-id="2bb1d-107">Este error también se genera cuando se especifica un tamaño de matriz con un valor variable y se inicializa con un inicializador de matriz.</span><span class="sxs-lookup"><span data-stu-id="2bb1d-107">This error is also produced when an array size is specified with a variable value and initialized with an array initializer.</span></span> <span data-ttu-id="2bb1d-108">Para quitar el error, inicialice la matriz en una o varias instrucciones independientes.</span><span class="sxs-lookup"><span data-stu-id="2bb1d-108">To remove the error, initialize the array in a separate statement or statements.</span></span>  
  
```  
// CS0150.cs  
    namespace MyNamespace  
    {  
        public class MyClass  
        {  
            public static void Main()  
            {  
                int size = 2;  
                double[] nums = new double[size] { 46.9, 89.4 }; //CS0150  
                // Try the following lines instead  
                // double[] nums = new double[size];  
                // nums[0] = 46.9;   
                // nums[1] = 89.4;  
            }  
        }  
  
    }  
```
