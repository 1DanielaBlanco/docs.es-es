# <a name="compiler-error-cs0631"></a>Error del compilador CS0631
ref y out no son válidos en este contexto  
  
 La declaración de un [indexador](../../csharp/programming-guide/indexers/index.md) no puede incluir el uso de los parámetros [out](../../csharp/language-reference/keywords/ref.md) o [ref](../../csharp/language-reference/keywords/out.md) .  
  
## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera la advertencia CS0631:  
  
```  
// CS0631.cs  
public class MyClass  
{  
    public int this[ref int i]   // CS0631  
    // try the following line instead  
    // public int this[int i]  
    {  
        get  
        {  
            return 0;  
        }  
    }  
}  
  
public class MyClass2  
{  
    public static void Main()  
    {  
    }  
}  
```
