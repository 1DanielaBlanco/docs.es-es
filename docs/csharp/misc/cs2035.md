# <a name="compiler-error-cs2035"></a>Error del compilador CS2035
Error de sintaxis de línea de comandos: falta ':\<número >' para la opción 'compiler_option'  
  
 Algunas opciones del compilador requieren un valor.  
  
## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera la advertencia CS2035.  
  
```  
// CS2035.cs  
// compile with: /baseaddress  
// CS2035 expected  
```
