 <span data-ttu-id="f728d-104">Cuando declara una [propiedad](../../csharp/programming-guide/classes-and-structs/using-properties.md), también debe declarar los métodos de descriptor de acceso.</span><span class="sxs-lookup"><span data-stu-id="f728d-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="f728d-105">Sin embargo, una propiedad no puede tener más de un método de descriptor de acceso `get` o más de un método de descriptor de acceso `set` .</span><span class="sxs-lookup"><span data-stu-id="f728d-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f728d-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="f728d-106">Example</span></span>  
 <span data-ttu-id="f728d-107">El ejemplo siguiente genera la advertencia CS1007:</span><span class="sxs-lookup"><span data-stu-id="f728d-107">The following sample generates CS1007:</span></span>

```csharp  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
