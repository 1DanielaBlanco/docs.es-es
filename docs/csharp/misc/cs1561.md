# <a name="compiler-error-cs1561"></a>Error del compilador CS1561
Nombre de archivo de salida demasiado largo o no válido  
  
 El nombre del archivo de salida no puede tener más de 256 caracteres y no debe contener caracteres no válidos, como el símbolo del euro, un signo de interrogación o una barra diagonal inversa, entre otros.
