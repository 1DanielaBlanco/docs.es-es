# <a name="compiler-error-cs1714"></a>Error del compilador CS1714
La clase base o la interfaz de TypeName1 no se ha podido resolver o no es válida  
  
 Está implementando TypeName1 desde una clase o interfaz que no se pudo resolver como cuando, por ejemplo, el compilador no encuentra la clase o interfaz, o bien cuando no es válida. La solución consiste en determinar cuál de estos dos casos es y especificar correctamente la ubicación del tipo, o bien corregir los errores del compilador en la clase base.
