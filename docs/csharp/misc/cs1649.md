# <a name="compiler-error-cs1649"></a><span data-ttu-id="5d70d-102">Error del compilador CS1649</span><span class="sxs-lookup"><span data-stu-id="5d70d-102">Compiler Error CS1649</span></span>
<span data-ttu-id="5d70d-103">Los miembros del campo de solo lectura 'identifier' no se pueden pasar como out o ref (excepto en un constructor)</span><span class="sxs-lookup"><span data-stu-id="5d70d-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="5d70d-104">Este error se produce si se pasa una variable a una función que sea miembro de un campo `readonly` como un argumento `ref` o `out` .</span><span class="sxs-lookup"><span data-stu-id="5d70d-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="5d70d-105">Como la función puede modificar los parámetros `ref` y `out` , esto no está permitido.</span><span class="sxs-lookup"><span data-stu-id="5d70d-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="5d70d-106">Para resolver este error, quite la palabra clave `readonly` en el campo o no pase los miembros del campo `readonly` a la función.</span><span class="sxs-lookup"><span data-stu-id="5d70d-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="5d70d-107">Por ejemplo, puede intentar crear una variable temporal que se pueda modificar y pasarla como un argumento `ref` , como se muestra en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="5d70d-107">For example, you might try creating a temporary variable which can be modifed and passing the temporary as a `ref` argument, as shown in the following example.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5d70d-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="5d70d-108">Example</span></span>  
 <span data-ttu-id="5d70d-109">El ejemplo siguiente genera la advertencia CS1649:</span><span class="sxs-lookup"><span data-stu-id="5d70d-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();   
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
