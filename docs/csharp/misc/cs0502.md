# <a name="compiler-error-cs0502"></a><span data-ttu-id="a4857-102">Error del compilador CS0502</span><span class="sxs-lookup"><span data-stu-id="a4857-102">Compiler Error CS0502</span></span>
<span data-ttu-id="a4857-103">'member' no puede ser abstract y sealed</span><span class="sxs-lookup"><span data-stu-id="a4857-103">'member' cannot be both abstract and sealed</span></span>  
  
 <span data-ttu-id="a4857-104">Un miembro de clase no puede ser [abstract](../../csharp/language-reference/keywords/abstract.md) y [sealed](../../csharp/language-reference/keywords/sealed.md).</span><span class="sxs-lookup"><span data-stu-id="a4857-104">A class member cannot be both [abstract](../../csharp/language-reference/keywords/abstract.md) and [sealed](../../csharp/language-reference/keywords/sealed.md).</span></span>  
  
 <span data-ttu-id="a4857-105">El ejemplo siguiente genera la advertencia CS0502:</span><span class="sxs-lookup"><span data-stu-id="a4857-105">The following sample generates CS0502:</span></span>  
  
```csharp  
// CS0502.cs  
public class B  
{  
   abstract public void F();  
}  
  
public class C : B  
{  
   abstract sealed override public void F()   // CS0502  
   {  
   }  
}  
  
public class CMain  
{  
    public static void Main()  
    { }  
}  
```
