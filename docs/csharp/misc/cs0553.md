 <span data-ttu-id="4af6f-104">No se permiten las conversiones definidas por el usuario a valores de una clase base; no es necesario este tipo de operador.</span><span class="sxs-lookup"><span data-stu-id="4af6f-104">User-defined conversions to values of a base class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="4af6f-105">El ejemplo siguiente genera la advertencia CS0553:</span><span class="sxs-lookup"><span data-stu-id="4af6f-105">The following sample generates CS0553:</span></span>  
  
```csharp  
// CS0553.cs  
namespace x  
{  
   public class ii  
   {  
   }  
  
   public class a : ii  
   {  
      // delete the conversion routine to resolve CS0553  
      public static implicit operator ii(a aa) // CS0553  
      {  
         return new ii();  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
