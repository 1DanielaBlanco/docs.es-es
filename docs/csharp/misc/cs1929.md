# <a name="compiler-error-cs1929"></a><span data-ttu-id="07464-102">Error del compilador CS1929</span><span class="sxs-lookup"><span data-stu-id="07464-102">Compiler Error CS1929</span></span>
<span data-ttu-id="07464-103">Argumento de instancia: no se puede convertir de 'typeA' a 'typeB'.</span><span class="sxs-lookup"><span data-stu-id="07464-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="07464-104">Este error se genera cuando se intenta invocar un método de extensión de una clase que no se extiende.</span><span class="sxs-lookup"><span data-stu-id="07464-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="07464-105">En este ejemplo, se define el método de extensión de la clase derivada `A`, pero no de la clase base `B`.</span><span class="sxs-lookup"><span data-stu-id="07464-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="07464-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="07464-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="07464-107">Cree un nuevo método de extensión para el tipo donde tiene que invocarlo, o bien mueva la llamada a un objeto del tipo que el método existente extiende.</span><span class="sxs-lookup"><span data-stu-id="07464-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="07464-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="07464-108">Example</span></span>  
 <span data-ttu-id="07464-109">El código siguiente genera los errores CS1928 y CS1929:</span><span class="sxs-lookup"><span data-stu-id="07464-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
