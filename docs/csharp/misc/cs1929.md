<span data-ttu-id="0d225-103">Argumento de instancia: no se puede convertir de 'typeA' a 'typeB'.</span><span class="sxs-lookup"><span data-stu-id="0d225-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="0d225-104">Este error se genera cuando se intenta invocar un método de extensión de una clase que no se extiende.</span><span class="sxs-lookup"><span data-stu-id="0d225-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="0d225-105">En este ejemplo, se define el método de extensión de la clase derivada `A`, pero no de la clase base `B`.</span><span class="sxs-lookup"><span data-stu-id="0d225-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>

## <a name="example"></a><span data-ttu-id="0d225-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="0d225-108">Example</span></span>  
 <span data-ttu-id="0d225-109">El código siguiente genera los errores CS1928 y CS1929:</span><span class="sxs-lookup"><span data-stu-id="0d225-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
