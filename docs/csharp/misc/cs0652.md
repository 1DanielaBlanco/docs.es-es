 <span data-ttu-id="3aac8-105">El ejemplo siguiente genera la advertencia CS0652:</span><span class="sxs-lookup"><span data-stu-id="3aac8-105">The following sample generates CS0652:</span></span>  
  
```

// compile with: /W:2

{  
   private static byte i = 0;  
   public static void Main()  
   {  
      short j = 256;  
      if (i == 256)   // CS0652, 256 is out of range for byte  
         i = 0;  
   }  
}  
```
