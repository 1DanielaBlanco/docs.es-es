# <a name="compiler-error-cs1660"></a><span data-ttu-id="5943e-102">Error del compilador CS1660</span><span class="sxs-lookup"><span data-stu-id="5943e-102">Compiler Error CS1660</span></span>
<span data-ttu-id="5943e-103">No se puede convertir el bloque de método anónimo al tipo 'type' porque no es un tipo de delegado.</span><span class="sxs-lookup"><span data-stu-id="5943e-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="5943e-104">Este error se produce si intenta asignar o convertir un bloque de método anónimo a un tipo que no sea un tipo de delegado.</span><span class="sxs-lookup"><span data-stu-id="5943e-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="5943e-105">El ejemplo siguiente genera la advertencia CS1660:</span><span class="sxs-lookup"><span data-stu-id="5943e-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
