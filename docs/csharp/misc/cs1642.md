## <a name="example"></a><span data-ttu-id="22a8c-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="22a8c-106">Example</span></span>  
 <span data-ttu-id="22a8c-107">El ejemplo siguiente genera la advertencia CS1642.</span><span class="sxs-lookup"><span data-stu-id="22a8c-107">The following sample generates CS1642.</span></span>  
  
```  
// CS1642.cs

unsafe class C  
{  
   fixed int a[10];   // CS1642  
}

{  
    fixed int a[10];  
}

{  
   public int[] a = null;  
}  
```
