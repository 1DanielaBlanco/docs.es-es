# <a name="compiler-error-cs1648"></a><span data-ttu-id="d0af2-102">Error del compilador CS1648</span><span class="sxs-lookup"><span data-stu-id="d0af2-102">Compiler Error CS1648</span></span>
<span data-ttu-id="d0af2-103">Los miembros del campo de solo lectura 'identificador' no se pueden modificar (excepto en un constructor o en un inicializador de variable)</span><span class="sxs-lookup"><span data-stu-id="d0af2-103">Members of readonly field 'identifier' cannot be modified (except in a constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="d0af2-104">Este error se produce cuando se intenta modificar un miembro de un campo que es de solo lectura donde no se permite su modificación.</span><span class="sxs-lookup"><span data-stu-id="d0af2-104">This error occurs when you attempt to modify a member of a field which is readonly where it is not allowed to be modified.</span></span> <span data-ttu-id="d0af2-105">Para resolver este error, limite las asignaciones en campos de solo lectura al constructor o al inicializador de variable, o bien quite la palabra clave readonly de la declaración del campo.</span><span class="sxs-lookup"><span data-stu-id="d0af2-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the readonly keyword from the declaration of the field.</span></span>  
  
 <span data-ttu-id="d0af2-106">El ejemplo siguiente genera la advertencia CS1648:</span><span class="sxs-lookup"><span data-stu-id="d0af2-106">The following sample generates CS1648:</span></span>  
  
```csharp  
// CS1648.cs  
public struct Inner  
  {  
    public int i;  
  }  
  
class Outer  
{    
  public readonly Inner inner = new Inner();  
}  
  
class D  
{  
   static void Main()  
   {  
      Outer outer = new Outer();  
      outer.inner.i = 1;  // CS1648  
   }  
}  
```
