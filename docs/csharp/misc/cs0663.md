# <a name="compiler-error-cs0663"></a><span data-ttu-id="f85dc-102">Error del compilador CS0663</span><span class="sxs-lookup"><span data-stu-id="f85dc-102">Compiler Error CS0663</span></span>
<span data-ttu-id="f85dc-103">No se pueden definir métodos sobrecargados que difieran solo en ref y out.</span><span class="sxs-lookup"><span data-stu-id="f85dc-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="f85dc-104">Los métodos que únicamente se diferencian en el uso de [ref](../../csharp/language-reference/keywords/ref.md) y [out](../../csharp/language-reference/keywords/out.md) en un parámetro no se permiten.</span><span class="sxs-lookup"><span data-stu-id="f85dc-104">Methods that differ only on their use of [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="f85dc-105">El ejemplo siguiente genera la advertencia CS0663:</span><span class="sxs-lookup"><span data-stu-id="f85dc-105">The following sample generates CS0663:</span></span>  
  
```  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
