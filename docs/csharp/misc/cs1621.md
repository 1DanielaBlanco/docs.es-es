# <a name="compiler-error-cs1621"></a><span data-ttu-id="ad8a7-102">Error del compilador CS1621</span><span class="sxs-lookup"><span data-stu-id="ad8a7-102">Compiler Error CS1621</span></span>
<span data-ttu-id="ad8a7-103">La instrucción yield no se puede usar dentro de un método anónimo o una expresión lambda</span><span class="sxs-lookup"><span data-stu-id="ad8a7-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
 <span data-ttu-id="ad8a7-104">La instrucción yield no puede ser un bloque de método anónimo en un iterador.</span><span class="sxs-lookup"><span data-stu-id="ad8a7-104">The yield statement cannot be in an anonymous method block in an iterator.</span></span>  
  
## <a name="example"></a><span data-ttu-id="ad8a7-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="ad8a7-105">Example</span></span>  
 <span data-ttu-id="ad8a7-106">El ejemplo siguiente genera la advertencia CS1621:</span><span class="sxs-lookup"><span data-stu-id="ad8a7-106">The following sample generates CS1621:</span></span>  
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
