## <a name="example"></a><span data-ttu-id="868e5-105">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="868e5-105">Example</span></span>  
 <span data-ttu-id="868e5-106">El ejemplo siguiente genera la advertencia CS1722.</span><span class="sxs-lookup"><span data-stu-id="868e5-106">The following sample generates CS1722.</span></span>

interface I {}  
  
public class MyClass : I, A {}   // CS1722
