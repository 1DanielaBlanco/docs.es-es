 <span data-ttu-id="6dc34-104">Se debe declarar e inicializar la variable en una [instrucción fixed](../../csharp/language-reference/keywords/fixed-statement.md).</span><span class="sxs-lookup"><span data-stu-id="6dc34-104">You must declare and initialize the variable in a [fixed statement](../../csharp/language-reference/keywords/fixed-statement.md).</span></span> <span data-ttu-id="6dc34-105">Para obtener más información, vea [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md) (Código no seguro y punteros [Guía de programación de C#]).</span><span class="sxs-lookup"><span data-stu-id="6dc34-105">For more information, see [Unsafe Code and Pointers](../../csharp/programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="6dc34-106">El ejemplo siguiente genera la advertencia CS0210:</span><span class="sxs-lookup"><span data-stu-id="6dc34-106">The following sample generates CS0210:</span></span>  
  
```  
// CS0210a.cs  
// compile with: /unsafe  
  
class Point  
{  
   public int x, y;  
}  
  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      Point pt = new Point();  
  
      fixed (int i)    // CS0210  
      {  
      }  
      // try the following lines instead  
      /*  
      fixed (int* p = &pt.x)  
      {  
      }  
      fixed (int* q = &pt.y)  
      {  
      }  
      */  
   }  
}  
```  
  
 <span data-ttu-id="6dc34-107">En el ejemplo siguiente también genera el error CS0210 porque la [instrucción using](../../csharp/language-reference/keywords/using-statement.md) no tiene inicializador.</span><span class="sxs-lookup"><span data-stu-id="6dc34-107">The following sample also generates CS0210 because the [using statement](../../csharp/language-reference/keywords/using-statement.md) has no initializer.</span></span>

```  
         w.WriteLine("Hello there");  
      }  
   }  
}  
```
