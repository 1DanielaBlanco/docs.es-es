## <a name="example"></a><span data-ttu-id="b4037-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="b4037-106">Example</span></span>  
 <span data-ttu-id="b4037-107">El ejemplo siguiente genera la advertencia CS0136:</span><span class="sxs-lookup"><span data-stu-id="b4037-107">The following sample generates CS0136:</span></span>  
  
```csharp
// CS0136.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static void Main()  
      {  
         int i = 0;  
         {  
            char i = 'a';   // CS0136, hides int i  
         }  
         i++;  
      }  
   }  
}  
```
