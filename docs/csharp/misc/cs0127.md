# <a name="compiler-error-cs0127"></a><span data-ttu-id="db482-102">Error del compilador CS0127</span><span class="sxs-lookup"><span data-stu-id="db482-102">Compiler Error CS0127</span></span>
<span data-ttu-id="db482-103">Debido a que 'función' devuelve void, una palabra clave return no debe ir seguida de una expresión de objeto.</span><span class="sxs-lookup"><span data-stu-id="db482-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="db482-104">Un método con un tipo de valor devuelto [void](../../csharp/language-reference/keywords/void.md) no puede devolver un valor.</span><span class="sxs-lookup"><span data-stu-id="db482-104">A method with a [void](../../csharp/language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="db482-105">Para más información, vea [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="db482-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="db482-106">El ejemplo siguiente genera la advertencia CS0127:</span><span class="sxs-lookup"><span data-stu-id="db482-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
