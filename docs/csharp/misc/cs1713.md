# <a name="compiler-error-cs1713"></a>Error del compilador CS1713
Error inesperado al crear un nombre de metadatos para el tipo nombreDeTipo1 — Motivo  
  
 A menudo, este error se debe a un error interno del compilador. Realizar algunos cambios menores en el código, como acortar la longitud del nombre y, después, volver a compilarlo, puede solucionar este error.
