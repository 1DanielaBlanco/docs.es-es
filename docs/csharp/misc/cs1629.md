# <a name="compiler-error-cs1629"></a>Error del compilador CS1629
No puede aparecer código no seguro en iteradores  
  
 La especificación del lenguaje C# no permite código no seguro en iteradores.  
  
 El ejemplo siguiente genera la advertencia CS1629:  
  
```csharp  
// CS1629.cs  
// compile with: /unsafe    
using System.Collections.Generic;  
class C   
{  
   IEnumerator<int> IteratorMeth() {  
      int i;  
      unsafe  // CS1629  
      {  
         int *p = &i;  
         yield return *p;  
      }  
   }  
}  
```
