# <a name="compiler-error-cs0572"></a><span data-ttu-id="fd015-102">Error del compilador CS0572</span><span class="sxs-lookup"><span data-stu-id="fd015-102">Compiler Error CS0572</span></span>
<span data-ttu-id="fd015-103">'type': no se puede hacer referencia a un tipo a través de una expresión; pruebe con 'path_to_type' en su lugar</span><span class="sxs-lookup"><span data-stu-id="fd015-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="fd015-104">Se intentó acceder a un miembro de una clase mediante un identificador, lo cual no está permitido en esta situación.</span><span class="sxs-lookup"><span data-stu-id="fd015-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="fd015-105">El ejemplo siguiente genera la advertencia CS0572:</span><span class="sxs-lookup"><span data-stu-id="fd015-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
