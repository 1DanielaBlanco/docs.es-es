<span data-ttu-id="fa167-103">No se puede devolver un valor a partir de un iterador.</span><span class="sxs-lookup"><span data-stu-id="fa167-103">Cannot return a value from an iterator.</span></span> <span data-ttu-id="fa167-104">Utilice la instrucción yield return para devolver un valor o yield break para terminar la iteración.</span><span class="sxs-lookup"><span data-stu-id="fa167-104">Use the yield return statement to return a value, or yield break to end the iteration.</span></span>  
  
 <span data-ttu-id="fa167-105">Un iterador es una función especial que devuelve un valor a través de la instrucción yield en lugar de la instrucción return.</span><span class="sxs-lookup"><span data-stu-id="fa167-105">An iterator is a special function that returns a value via the yield statement rather than the return statement.</span></span> <span data-ttu-id="fa167-106">Para más información, vea **Iteradores**.</span><span class="sxs-lookup"><span data-stu-id="fa167-106">For more information, see **iterators**.</span></span>  
  
 <span data-ttu-id="fa167-107">El ejemplo siguiente genera la advertencia CS1622:</span><span class="sxs-lookup"><span data-stu-id="fa167-107">The following sample generates CS1622:</span></span>  
  
```csharp  
// CS1622.cs  
// compile with: /target:library  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      return (IEnumerator) this;  // CS1622  
      yield return this;   // OK  
   }  
}  
```
