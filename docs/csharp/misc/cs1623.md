# <a name="compiler-error-cs1623"></a><span data-ttu-id="f0618-102">Error del compilador CS1623</span><span class="sxs-lookup"><span data-stu-id="f0618-102">Compiler Error CS1623</span></span>
<span data-ttu-id="f0618-103">Los iteradores no pueden tener parámetros ref ni out</span><span class="sxs-lookup"><span data-stu-id="f0618-103">Iterators cannot have ref or out parameters</span></span>  
  
 <span data-ttu-id="f0618-104">Este error se produce si un método iterador toma un parámetro `ref` o `out` .</span><span class="sxs-lookup"><span data-stu-id="f0618-104">This error occurs if an iterator method takes a `ref` or `out` parameter.</span></span> <span data-ttu-id="f0618-105">Para evitar este error, elimine la palabra clave `ref` o `out` de la signatura del método.</span><span class="sxs-lookup"><span data-stu-id="f0618-105">To avoid this error, remove the `ref` or `out` keyword from the method signature.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f0618-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="f0618-106">Example</span></span>  
 <span data-ttu-id="f0618-107">El ejemplo siguiente genera la advertencia CS1623:</span><span class="sxs-lookup"><span data-stu-id="f0618-107">The following sample generates CS1623:</span></span>  
  
```  
// CS1623.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        yield return 0;  
    }  
  
    // To resolve the error, remove ref  
    public IEnumerator GetEnumerator(ref int i)  // CS1623  
    {  
        yield return i;  
    }  
  
    // To resolve the error, remove out  
    public IEnumerator GetEnumerator(out float f)  // CS1623  
    {  
        f = 0.0F;  
        yield return f;  
    }  
}  
```
