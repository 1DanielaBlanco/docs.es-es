## <a name="example"></a><span data-ttu-id="61d4c-106">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="61d4c-106">Example</span></span>  
 <span data-ttu-id="61d4c-107">El ejemplo siguiente genera la advertencia CS0704:</span><span class="sxs-lookup"><span data-stu-id="61d4c-107">The following sample generates CS0704:</span></span>  
  
```  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}

{  
    public static void Main() {}  
}  
```
