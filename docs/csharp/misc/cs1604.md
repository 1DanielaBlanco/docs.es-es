# <a name="compiler-error-cs1604"></a>Error del compilador CS1604
No se puede asignar a 'variable' porque es de solo lectura.  
  
 Se realizó una asignación a una variable de solo lectura. Para evitar este error, no asigne ni intente modificar esta variable en este contexto.
