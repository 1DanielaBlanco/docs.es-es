# Advertencia del compilador (nivel 1) CS1592
XML con formato incorrecto en el archivo de comentarios de inclusión: 'reason'  
  
 Se encontró un problema notificado como ***reason*** en el archivo especificado por la etiqueta [\<include\>](../../csharp/programming-guide/xmldoc/include.md).
