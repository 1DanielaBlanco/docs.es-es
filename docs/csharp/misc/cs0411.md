## <a name="example"></a><span data-ttu-id="05ec4-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="05ec4-107">Example</span></span>  
 <span data-ttu-id="05ec4-108">El ejemplo siguiente genera la advertencia CS0411:</span><span class="sxs-lookup"><span data-stu-id="05ec4-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="05ec4-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="05ec4-109">Example</span></span>

```csharp  
{  
    public void F<T>(T t) where T : C   
    {  
    }  
  
    public static void Main()  
    {  
        C c = new C();  
        c.F(null);  // CS0411  
    }

```
