## <a name="example"></a><span data-ttu-id="a44be-107">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="a44be-107">Example</span></span>  
 <span data-ttu-id="a44be-108">El ejemplo siguiente genera la advertencia CS0411:</span><span class="sxs-lookup"><span data-stu-id="a44be-108">The following sample generates CS0411:</span></span>  
  
```csharp  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
