<span data-ttu-id="614e6-103">Un miembro estático 'function' no se puede marcar como override, virtual o abstract</span><span class="sxs-lookup"><span data-stu-id="614e6-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="614e6-104">Una declaración de método que utilice la palabra clave `override`, **virtual**o **abstract** no puede usar también la palabra clave **static** .</span><span class="sxs-lookup"><span data-stu-id="614e6-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="614e6-105">Para más información, vea [Métodos](../../csharp/programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="614e6-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="614e6-106">El ejemplo siguiente genera la advertencia CS0112:</span><span class="sxs-lookup"><span data-stu-id="614e6-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
