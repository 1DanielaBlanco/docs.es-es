<span data-ttu-id="82c72-103">'clase' define el operador == o el operador !=, pero no invalida Object.Equals(object o)</span><span class="sxs-lookup"><span data-stu-id="82c72-103">'class' defines operator == or operator != but does not override Object.Equals(object o)</span></span>  
  
 <span data-ttu-id="82c72-104">El compilador detectó el operador de igualdad o desigualdad definido por el usuario, pero no detectó ninguna invalidación para la función **Equals** .</span><span class="sxs-lookup"><span data-stu-id="82c72-104">The compiler detected the user-defined equality or inequality operator, but no override for the **Equals** function.</span></span> <span data-ttu-id="82c72-105">Un operador de igualdad o desigualdad definido por el usuario implica que también se desea invalidar la función **Equals** .</span><span class="sxs-lookup"><span data-stu-id="82c72-105">A user-defined equality or inequality operator implies that you also want to override the **Equals** function.</span></span> <span data-ttu-id="82c72-106">Para obtener más información, consulte [NIB: directrices para invalidar Equals() y el operador == (Guía de programación de C#)](http://msdn.microsoft.com/library/7e4c24c5-7693-4c45-88fb-ba5204fbcb20).</span><span class="sxs-lookup"><span data-stu-id="82c72-106">For more information, see [NIB - Guidelines for Overriding Equals() and Operator == (C# Programming Guide)](http://msdn.microsoft.com/library/7e4c24c5-7693-4c45-88fb-ba5204fbcb20).</span></span>  
  
 <span data-ttu-id="82c72-107">El ejemplo siguiente genera la advertencia CS0660:</span><span class="sxs-lookup"><span data-stu-id="82c72-107">The following sample generates CS0660:</span></span>

```csharp  
class Test   // CS0660  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   // uncomment the Equals function to resolve  
   // public override bool Equals(object o)  
   // {  
   //    return true;  
   // }  
  
   public override int GetHashCode()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
