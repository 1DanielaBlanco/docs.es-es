 <span data-ttu-id="00703-105">Un programa puede tener un solo método [Main](../../csharp/programming-guide/main-and-command-args/index.md) .</span><span class="sxs-lookup"><span data-stu-id="00703-105">A program can only have one [Main](../../csharp/programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="00703-106">Para resolver este error, puede eliminar todos los métodos Main del código, salvo uno, o bien puede usar la opción del compilador [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) para especificar qué método Main que se va a usar.</span><span class="sxs-lookup"><span data-stu-id="00703-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="00703-107">El ejemplo siguiente genera la advertencia CS0017:</span><span class="sxs-lookup"><span data-stu-id="00703-107">The following sample generates CS0017:</span></span>  
  
```csharp

{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
