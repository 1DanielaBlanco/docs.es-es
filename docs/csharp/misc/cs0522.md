# <a name="compiler-error-cs0522"></a><span data-ttu-id="b80a3-102">Error del compilador CS0522</span><span class="sxs-lookup"><span data-stu-id="b80a3-102">Compiler Error CS0522</span></span>
<span data-ttu-id="b80a3-103">'constructor': las estructuras no pueden llamar a constructores de clase base</span><span class="sxs-lookup"><span data-stu-id="b80a3-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="b80a3-104">Una [estructura](../../csharp/language-reference/keywords/struct.md) no puede llamar a un constructor de clase base; quite la llamada al constructor de clase base.</span><span class="sxs-lookup"><span data-stu-id="b80a3-104">A [struct](../../csharp/language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="b80a3-105">El ejemplo siguiente genera la advertencia CS0522:</span><span class="sxs-lookup"><span data-stu-id="b80a3-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
