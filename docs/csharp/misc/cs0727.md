# <a name="compiler-error-cs0727"></a>Error del compilador CS0727
Especificador de formato no válido  
  
 Este error se produce en el depurador. Cuando escriba un nombre de variable en una de las ventanas del depurador, puede agregarle una coma y, a continuación, un especificador de formato. Algunos ejemplos son: myInt, h; o myString, nq. Este error se produce cuando el compilador no puede analizar completamente lo que ha escrito. Para resolver este error, vuelva a escribir el nombre de variable, seguido opcionalmente por una coma y un [especificador de formato válido](https://docs.microsoft.com/visualstudio/debugger/format-specifiers-in-csharp).
