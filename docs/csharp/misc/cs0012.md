 <span data-ttu-id="7922c-108">La siguiente secuencia de compilaciones dará como resultado CS0012:</span><span class="sxs-lookup"><span data-stu-id="7922c-108">The following sequence of compilations will result in CS0012:</span></span>

  
 <span data-ttu-id="7922c-109">Entonces:</span><span class="sxs-lookup"><span data-stu-id="7922c-109">Then:</span></span>

```csharp  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="7922c-110">Entonces:</span><span class="sxs-lookup"><span data-stu-id="7922c-110">Then:</span></span>

```csharp  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```
