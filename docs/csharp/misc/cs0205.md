 <span data-ttu-id="151fa-106">El ejemplo siguiente genera la advertencia CS0205:</span><span class="sxs-lookup"><span data-stu-id="151fa-106">The following sample generates CS0205:</span></span>

```csharp  
   abstract public void M();  
}

      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
