 <span data-ttu-id="60789-104">Error al intentar reemplazar uno de los métodos de descriptor de acceso para una propiedad porque no se puede reemplazar el descriptor de acceso.</span><span class="sxs-lookup"><span data-stu-id="60789-104">An attempt to override one of the accessor methods for a property failed because the accessor cannot be overridden.</span></span> <span data-ttu-id="60789-105">Este error puede producirse en los casos siguientes:</span><span class="sxs-lookup"><span data-stu-id="60789-105">This error can occur if:</span></span>

## <a name="example"></a><span data-ttu-id="60789-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="60789-110">Example</span></span>  
 <span data-ttu-id="60789-111">El ejemplo siguiente genera la advertencia CS0546 porque la clase base no declara ningún descriptor de acceso set para la propiedad.</span><span class="sxs-lookup"><span data-stu-id="60789-111">The following sample generates CS0546 because the base class does not declare a set accessor for the property.</span></span>  
  
```csharp  
// CS0546.cs  
// compile with: /target:library  
public class a  
{  
   public virtual int i  
   {  
      get  
      {  
         return 0;  
      }  
   }  
  
   public virtual int i2  
   {  
      get  
      {  
         return 0;  
      }  
  
      set {}  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      set {}   // CS0546 error no set  
   }  
  
   public override int i2  
   {  
      set {}   // OK  
   }  
}  
```
