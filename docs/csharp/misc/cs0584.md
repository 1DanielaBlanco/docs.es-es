# <a name="compiler-error-cs0584"></a>Error del compilador CS0584
Error interno del compilador: fase 'stage' símbolo 'symbol'  
  
 Intente determinar si se están produciendo errores en el compilador porque no puede analizar una sintaxis inesperada. Si no es el caso, pruebe [charle con nosotros](/visualstudio/ide/talk-to-us).
