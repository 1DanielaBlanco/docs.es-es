# <a name="type-providers"></a><span data-ttu-id="0245e-103">Proveedores de tipos</span><span class="sxs-lookup"><span data-stu-id="0245e-103">Type Providers</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="0245e-112">Proveedores de tipos generativa y borrados</span><span class="sxs-lookup"><span data-stu-id="0245e-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="0245e-113">Los proveedores de tipo tienen dos formas: Generativa y borrados.</span><span class="sxs-lookup"><span data-stu-id="0245e-113">Type Providers come in two forms: Generative and Erased.</span></span>
