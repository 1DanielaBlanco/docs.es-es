# <a name="getting-started-with-f"></a>Introducción a F# #

Existen tres formas principales de empezar a trabajar con F#:

Si usa Windows, consulte [Getting Started with F# in Visual Studio](getting-started-visual-studio.md) (Introducción a F# en Visual Studio).

Si le interesa escribir código en un IDE ligero en Windows, Linux o macOS, consulte [Getting Started with F# in Visual Studio Code with Ionide](getting-started-vscode.md) (Introducción a F# en Visual Studio Code con Ionide).

Si prefiere usar la línea de comandos en cualquier sistema operativo, consulte [Getting Started with F# with Command-line Tools](getting-started-command-line.md) (Introducción a F# con las herramientas de línea de comandos).
