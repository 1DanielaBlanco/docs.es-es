## <a name="self-contained-deployments-scd"></a><span data-ttu-id="66435-132">Implementaciones autocontenidas (SCD)</span><span class="sxs-lookup"><span data-stu-id="66435-132">Self-contained deployments (SCD)</span></span>

<span data-ttu-id="66435-137">La implementación de una implementación autocontenida tiene dos ventajas principales:</span><span class="sxs-lookup"><span data-stu-id="66435-137">Deploying a Self-contained deployment has two major advantages:</span></span>
