## <a name="increase-the-maximum-open-file-limit-net-core-versions-before-net-core-sdk-202"></a><span data-ttu-id="97fe0-127">Aumentar el límite máximo de archivos abiertos (versiones de .NET Core antes del SDK de .NET Core 2.0.2)</span><span class="sxs-lookup"><span data-stu-id="97fe0-127">Increase the maximum open file limit (.NET Core versions before .NET Core SDK 2.0.2)</span></span>

<span data-ttu-id="97fe0-129">Puede aumentar este límite si sigue estos pasos:</span><span class="sxs-lookup"><span data-stu-id="97fe0-129">You can increase this limit by following these steps:</span></span>

        "http://www.apple.com/DTDs/PropertyList-1.0.dtd">

    <key>ServiceIPC</key>
