<span data-ttu-id="6c46e-129">El límite predeterminado de archivos abiertos en macOS puede no ser suficiente para algunas cargas de trabajo de .NET Core, como la restauración de proyectos o la ejecución de pruebas unitarias.</span><span class="sxs-lookup"><span data-stu-id="6c46e-129">The default open file limit on macOS may not be sufficient for some .NET Core workloads, such as restoring projects or running unit tests.</span></span>

<span data-ttu-id="6c46e-130">Puede aumentar este límite si sigue estos pasos:</span><span class="sxs-lookup"><span data-stu-id="6c46e-130">You can increase this limit by following these steps:</span></span>

        "http://www.apple.com/DTDs/PropertyList-1.0.dtd">

  <dict>
    <key>Label</key>
    <string>limit.maxfiles</string>
