# <a name="using-a-brush-to-fill-shapes"></a>Utilizar un pincel para rellenar formas
A [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Brush> objeto se usa para rellenar el interior de una forma cerrada. [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)]define varios estilos de relleno: color sólido, patrón de trama, textura de imagen y degradado de color.
