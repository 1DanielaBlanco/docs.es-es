# Utilizar un pincel para rellenar formas
Se utiliza un objeto <xref:System.Drawing.Brush> de [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] para rellenar el interior de una forma cerrada.  [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] define varios estilos de relleno: color sólido, modelo de trama, textura de imagen y degradado de color.

 [Cómo: Rellenar una forma con un patrón de trama](../../../../docs/framework/winforms/advanced/how-to-fill-a-shape-with-a-hatch-pattern.md)
