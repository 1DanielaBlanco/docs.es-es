# Usar transformaciones en la interfaz GDI+ administrada
Las transformaciones afines son: giro, ajuste de tamaño, reflexión, sesgo y traslado.  En [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)], la clase <xref:System.Drawing.Drawing2D.Matrix> proporciona la base para realizar transformaciones afines en dibujos vectoriales, imágenes y texto.
