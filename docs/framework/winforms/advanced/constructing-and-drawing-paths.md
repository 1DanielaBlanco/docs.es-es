# Crear y dibujar trazados
Un trazado es una secuencia de tipos primitivos de gráficos \(líneas, rectángulos, curvas, texto, etc.\) que se puede manipular y dibujar como una sola unidad.  Un trazado se puede dividir en *figuras* que están abiertas o cerradas.  Una figura puede contener varios tipos primitivos.  
  
 Se puede dibujar un trazado llamado al método <xref:System.Drawing.Graphics.DrawPath%2A> de la clase <xref:System.Drawing.Graphics> y el trazado se puede rellenar llamando al método <xref:System.Drawing.Graphics.FillPath%2A> de la clase <xref:System.Drawing.Graphics>.
