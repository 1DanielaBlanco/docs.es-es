# Networking in Windows Forms Applications
.NET Framework proporciona clases para mostrar páginas web, descargar contenido web, interactuar con sitios de protocolo de transferencia de archivos \(FTP\) y utilizar servicios Web, lo que facilita la creación de funciones de red en la aplicación.  Los recursos siguientes le ayudarán a entender las tecnologías de red de .NET Framework y cómo puede integrarlas en formularios Windows Forms.  
  
## Referencia  
 <xref:System.Net>  
 El espacio de nombres raíz para las clases de .NET Framework que controlan la conectividad de red.  
  
 <xref:System.Net.WebClient>  
 Una clase cómoda para recuperar mediante programación contenido Web o basado en HTTP.  
  
 <xref:System.Net.FtpWebRequest>  
 Clase para recuperar y enviar archivos con FTP.  
  
 <xref:System.Windows.Forms.WebBrowser>  
 Clase contenedora administrada para el control `WebBrowser` que se incluye con Windows.
