# Herencia visual de formularios Windows Forms
En ocasiones, quizá decida que un proyecto debe llamar a un formulario similar a uno que ya creó en un proyecto anterior.  O bien, quizá desee crear un formulario básico con configuraciones tales como una marca de agua o determinado diseño de controles que utilizará de nuevo dentro de un proyecto, donde cada una de las iteraciones contendrá modificaciones respecto a la plantilla de formulario original.  La herencia de formularios permite crear un formulario base, heredar de él y hacer modificaciones mientras se conserva la configuración original necesaria.  
  
 Puede crear mediante programación los formularios de clase derivada o mediante el Selector de herencia visual.
