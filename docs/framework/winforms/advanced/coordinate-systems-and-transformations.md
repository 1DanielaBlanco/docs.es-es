# Sistemas de coordenadas y transformaciones
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] ofrece la posibilidad de realizar transformaciones de coordenadas universales y de página, de modo que es posible transformar \(rotar, ajustar la escala, convertir, etcétera\) los elementos que se dibujan.  Estos dos tipos de transformaciones permiten, a su vez, trabajar con varios sistemas de coordenadas.
