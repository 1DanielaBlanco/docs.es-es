# Keyboard Input in a Windows Forms Application
Los formularios Windows Forms incluyen eventos de teclado estándar que permiten responder a presiones de tecla concretas y además proporcionan maneras de interceptar, modificar y utilizar presiones de tecla en el nivel de la aplicación, formulario y control.

 Muestra cómo averiguar si se ha presionado MAYÚS, ALT o CTRL además de otra tecla.  
  
 [How to: Handle Keyboard Input at the Form Level](../../../docs/framework/winforms/how-to-handle-keyboard-input-at-the-form-level.md)
