### <a name="to-disable-a-menu-item-at-design-time"></a><span data-ttu-id="eb174-110">Para deshabilitar un elemento de menú en tiempo de diseño</span><span class="sxs-lookup"><span data-stu-id="eb174-110">To disable a menu item at design time</span></span>  
  
1.  <span data-ttu-id="eb174-111">Con el elemento de menú seleccionado en el formulario, establezca la <xref:System.Windows.Forms.ToolStripMenuItem.Enabled%2A> propiedad `false`.</span><span class="sxs-lookup"><span data-stu-id="eb174-111">With the menu item selected on the form, set the <xref:System.Windows.Forms.ToolStripMenuItem.Enabled%2A> property to `false`.</span></span>  
  
    > [!TIP]
    >  <span data-ttu-id="eb174-112">Si se deshabilita el elemento de menú de primer o de nivel superior en un menú, deshabilitan todos los elementos de menú contenidos en el menú.</span><span class="sxs-lookup"><span data-stu-id="eb174-112">Disabling the first or top-level menu item in a menu disables all the menu items contained within the menu.</span></span> <span data-ttu-id="eb174-113">Del mismo modo, si se deshabilita un elemento de menú que tiene elementos de submenú, deshabilitan los elementos de submenú.</span><span class="sxs-lookup"><span data-stu-id="eb174-113">Likewise, disabling a menu item that has submenu items disables the submenu items.</span></span> <span data-ttu-id="eb174-114">Si todos los comandos de un menú no están disponibles para el usuario, se considera buena práctica de programación ocultar y deshabilitar todo el menú, tal y como se presenta una interfaz de usuario limpia.</span><span class="sxs-lookup"><span data-stu-id="eb174-114">If all the commands on a given menu are unavailable to the user, it is considered good programming practice to both hide and disable the entire menu, as this presents a clean user interface.</span></span> <span data-ttu-id="eb174-115">Se debe ocultar y deshabilitar el menú, como si sólo se oculta no impide el acceso a un comando de menú a través de una tecla de método abreviado.</span><span class="sxs-lookup"><span data-stu-id="eb174-115">You should both hide and disable the menu, as hiding alone does not prevent access to a menu command via a shortcut key.</span></span> <span data-ttu-id="eb174-116">Establecer el <xref:System.Windows.Forms.ToolStripItem.Visible%2A> propiedad de un elemento de menú de nivel superior para `false` para ocultar todo el menú.</span><span class="sxs-lookup"><span data-stu-id="eb174-116">Set the <xref:System.Windows.Forms.ToolStripItem.Visible%2A> property of a top-level menu item to `false` to hide the entire menu.</span></span>
