# Desarrollar un control de formularios Windows Forms compuesto
Puede desarrollar un control de Windows Forms compuesto mediante la combinación de otros controles de Windows Forms.  Los controles compuestos que derivan de [System.Windows.Forms.UserControl](frlrfSystemWebUIUserControlClassTopic) se denominan controles de usuario.  La clase base, <xref:System.Windows.Forms.UserControl>, proporciona el enrutamiento de teclado de los controles secundarios, lo que asegura que puedan recibir el foco.  Para obtener un ejemplo de un control de usuario, consulte <xref:System.Windows.Forms.UserControl> en [Cómo: Aplicar atributos en controles de formularios Windows Forms](../../../../docs/framework/winforms/controls/how-to-apply-attributes-in-windows-forms-controls.md).
