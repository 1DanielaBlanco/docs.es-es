# <a name="picturebox-control-overview-windows-forms"></a>Información general del control PictureBox (Formularios Windows Forms)
Los formularios Windows Forms <xref:System.Windows.Forms.PictureBox> control se utiliza para mostrar gráficos en formato de mapa de bits, GIF, JPEG, metarchivo o icono.  
  
## <a name="key-properties-and-methods"></a>Métodos y propiedades claves  
 La imagen que se muestra está determinada por la <xref:System.Windows.Forms.PictureBox.Image%2A> propiedad, que se puede establecer en tiempo de ejecución o en tiempo de diseño. O bien puede especificar la imagen estableciendo la <xref:System.Windows.Forms.PictureBox.ImageLocation%2A> propiedad y, a continuación, cargue la imagen de forma sincrónica mediante la <xref:System.Windows.Forms.PictureBox.Load%2A> método o forma asincrónica con el <xref:System.Windows.Forms.PictureBox.LoadAsync%2A> método. El <xref:System.Windows.Forms.PictureBox.SizeMode%2A> propiedad controla cómo la imagen y el control se ajustan entre sí. Para obtener más información, consulte [Cómo: modificar el tamaño o la ubicación de una imagen en tiempo de ejecución](../../../../docs/framework/winforms/controls/how-to-modify-the-size-or-placement-of-a-picture-at-run-time-windows-forms.md).
