# Entrada de datos en el control DataGridView de formularios Windows Forms
El control `DataGridView` proporciona varias características que le permiten cambiar la manera en que los usuarios agregan o modifican los datos en el control.  Por ejemplo, puede hacer más eficaz la entrada de datos proporcionando valores predeterminados para las nuevas filas y avisando a los usuarios en el momento en que aparecen errores.
