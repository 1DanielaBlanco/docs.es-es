# LinkLabel (Control, formularios Windows Forms)
El control `LinkLabel` de formularios Windows Forms permite agregar vínculos de estilo Web a aplicaciones de Windows Forms.  Puede utilizar el control `LinkLabel` para todo aquello para lo que pueda utilizar el control <xref:System.Windows.Forms.Label>. También puede establecer parte del texto como un vínculo a un objeto o una página Web.
