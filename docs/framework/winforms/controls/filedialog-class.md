# FileDialog (Clase)
La clase de Windows Forms <xref:System.Windows.Forms.FileDialog> es la clase base común de los componentes <xref:System.Windows.Forms.OpenFileDialog> y <xref:System.Windows.Forms.SaveFileDialog>.  Puede realizar modificaciones a la clase <xref:System.Windows.Forms.FileDialog> que afectan al aspecto y comportamiento de estos cuadros de diálogo, dependiendo de la versión de Windows en la que la aplicación se está ejecutando.
