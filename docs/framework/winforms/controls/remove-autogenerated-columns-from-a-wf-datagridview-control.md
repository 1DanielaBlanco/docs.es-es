# <a name="how-to-remove-autogenerated-columns-from-a-windows-forms-datagridview-control"></a>Cómo: Quitar columnas generadas automáticamente desde un control DataGridView de formularios Windows Forms
Cuando el <xref:System.Windows.Forms.DataGridView> control está establecido para generar automáticamente las columnas basadas en datos de su origen de datos, puede omitir ciertas columnas de forma selectiva. Puede hacerlo mediante una llamada a la <xref:System.Windows.Forms.DataGridViewColumnCollection.Remove%2A> método en el <xref:System.Windows.Forms.DataGridView.Columns%2A> colección. Como alternativa, puede ocultar las columnas de vista estableciendo la <xref:System.Windows.Forms.DataGridViewColumn.Visible%2A> propiedad `false`. Esta técnica es útil cuando desea mostrar las columnas ocultas en determinadas condiciones, o cuando necesite tener acceso a los datos de las columnas sin mostrarlo.  
  
### <a name="to-remove-autogenerated-columns"></a>Para quitar columnas generadas automáticamente  
  
-   Llame a la <xref:System.Windows.Forms.DataGridViewColumnCollection.Remove%2A> método en el <xref:System.Windows.Forms.DataGridView.Columns%2A> colección.

### <a name="to-hide-autogenerated-columns"></a>Para ocultar las columnas generadas automáticamente
