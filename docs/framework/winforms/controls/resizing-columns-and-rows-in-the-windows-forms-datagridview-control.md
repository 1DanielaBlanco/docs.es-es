# Cambiar el tama&#241;o de columnas y filas en el control DataGridView de formularios Windows Forms
El control `DataGridView` proporciona numerosas opciones para personalizar el comportamiento del ajuste de tamaño de las columnas y filas.  Normalmente, las celdas `DataGridView` no cambian el tamaño basándose en su contenido.  En su lugar, recortan cualquier valor de presentación que sea mayor que la celda.  Si el contenido se puede mostrar como una cadena, la celda lo muestra en la información sobre herramientas.  
  
 De forma predeterminada, los usuarios pueden arrastrar divisores de fila, columna y encabezado con el mouse para mostrar más información.  Los usuarios también pueden hacer doble clic en un divisor para cambiar el tamaño automáticamente de la fila, columna o banda de encabezado asociada basándose en su contenido.  
  
 El control `DataGridView` proporciona las propiedades, métodos y eventos que permiten personalizar o deshabilitar todos estos comportamientos dirigidos por el usuario.  De forma adicional, puede cambiar mediante programación el tamaño de filas, columnas y encabezados para que ajusten su contenido o puede configurarlos para que cambien el tamaño automáticamente cada vez que cambia su contenido.  También puede configurar las columnas de manera que se divida automáticamente el ancho disponible del control en la proporción que se especifique.
