# C&#243;mo: Situar un control ToolStripItem en un control ToolStrip
Se puede desplazar o agregar un control <xref:System.Windows.Forms.ToolStripItem> a la izquierda o a la derecha de un control <xref:System.Windows.Forms.ToolStrip>.  
  
### Para desplazar o agregar un control ToolStripItem a la izquierda de un control ToolStrip  
  
1.  Establezca la propiedad <xref:System.Windows.Forms.ToolStripItemAlignment> del control <xref:System.Windows.Forms.ToolStripItem> en <xref:System.Windows.Forms.ToolStripItemAlignment>.  
  
### Para mover o agregar un control ToolStripItem a la derecha de un control ToolStrip  
  
1.  Establezca la propiedad <xref:System.Windows.Forms.ToolStripItemAlignment> del control <xref:System.Windows.Forms.ToolStripItem> en <xref:System.Windows.Forms.ToolStripItemAlignment>.
