# Control Panel (Windows Forms)
Los controles `Panel` de Windows Forms se utilizan para proporcionar un agrupamiento identificable para otros controles.  Normalmente, los paneles se utilizan para subdividir un formulario por funciones.  Los controles `Panel` y <xref:System.Windows.Forms.GroupBox> son similares; sin embargo, el control `Panel` es el único de los dos que puede tener barras de desplazamiento y el control <xref:System.Windows.Forms.GroupBox> es el único de los dos que muestra una leyenda.
