# <a name="fontdialog-component-windows-forms"></a>FontDialog (Componente, formularios Windows Forms)
El componente <xref:System.Windows.Forms.FontDialog> de Windows Forms es un cuadro de diálogo preconfigurado. Es el mismo **fuente** cuadro de diálogo expuesto por el sistema operativo Windows. El componente hereda de la <xref:System.Windows.Forms.CommonDialog> clase.  
  
## <a name="in-this-section"></a>En esta sección  
 [Información general del componente FontDialog](../../../../docs/framework/winforms/controls/fontdialog-component-overview-windows-forms.md)  
 Presenta los conceptos generales de la <xref:System.Windows.Forms.FontDialog> componente, que se utiliza para mostrar un cuadro de diálogo preconfigurado. Los usuarios pueden utilizar el cuadro de diálogo para manipular las fuentes y su configuración.  
  
 [Mostrar una lista de fuentes con el componente FontDialog](../../../../docs/framework/winforms/controls/how-to-show-a-font-list-with-the-fontdialog-component.md)  
 Explica cómo elegir una fuente en tiempo de ejecución a través de una instancia de la <xref:System.Windows.Forms.FontDialog> componente.
