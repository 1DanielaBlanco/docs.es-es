# FontDialog (Componente, formularios Windows Forms)
El componente <xref:System.Windows.Forms.FontDialog> de Windows Forms es un cuadro de diálogo preconfigurado.  Es el mismo cuadro de diálogo **Fuente** que expone el sistema operativo Windows.  El componente hereda de la clase <xref:System.Windows.Forms.CommonDialog>.  
  
## En esta sección  
 [Información general sobre el componente FontDialog](../../../../docs/framework/winforms/controls/fontdialog-component-overview-windows-forms.md)  
 Proporciona información sobre los conceptos generales del componente <xref:System.Windows.Forms.FontDialog>, que se utiliza para mostrar un cuadro de diálogo preconfigurado.  Los usuarios pueden utilizar el cuadro de diálogo para manipular las fuentes y su configuración.  
  
 [Cómo: Mostrar una lista de fuentes con el componente FontDialog](../../../../docs/framework/winforms/controls/how-to-show-a-font-list-with-the-fontdialog-component.md)  
 Explica cómo elegir una fuente en tiempo de ejecución con una instancia del componente <xref:System.Windows.Forms.FontDialog>.
