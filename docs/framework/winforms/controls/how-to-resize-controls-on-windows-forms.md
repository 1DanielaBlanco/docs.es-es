# <a name="how-to-resize-controls-on-windows-forms"></a><span data-ttu-id="4f801-102">Cómo: Cambiar el tamaño de los controles en formularios Windows Forms</span><span class="sxs-lookup"><span data-stu-id="4f801-102">How to: Resize Controls on Windows Forms</span></span>
<span data-ttu-id="4f801-103">Puede cambiar el tamaño de los controles individuales, y puede cambiar el tamaño varios controles del mismo o diferente tipo, como <xref:System.Windows.Forms.Button> y <xref:System.Windows.Forms.GroupBox> controles.</span><span class="sxs-lookup"><span data-stu-id="4f801-103">You can resize individual controls, and you can resize multiple controls of the same or different kind, such as <xref:System.Windows.Forms.Button> and <xref:System.Windows.Forms.GroupBox> controls.</span></span>  
  
### <a name="to-resize-a-control"></a><span data-ttu-id="4f801-104">Para cambiar el tamaño de un control</span><span class="sxs-lookup"><span data-stu-id="4f801-104">To resize a control</span></span>  
  
1.  <span data-ttu-id="4f801-105">Haga clic en el control para cambiar de tamaño y arrastre uno de los controladores de tamaño de ocho.</span><span class="sxs-lookup"><span data-stu-id="4f801-105">Click the control to be resized and drag one of the eight sizing handles.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="4f801-106">Seleccione el control y presione las teclas de dirección mientras mantiene presionada la tecla MAYÚS para cambiar el tamaño del control un píxel a la vez.</span><span class="sxs-lookup"><span data-stu-id="4f801-106">Select the control and press the ARROW keys while holding down the SHIFT key to resize the control one pixel at a time.</span></span> <span data-ttu-id="4f801-107">Presione las teclas de flecha hacia abajo o hacia la derecha manteniendo presionadas las teclas MAYÚS y CTRL para cambiar el tamaño del control en incrementos grandes.</span><span class="sxs-lookup"><span data-stu-id="4f801-107">Press the DOWN or RIGHT arrow keys while holding down the SHIFT and CTRL keys to resize the control in large increments.</span></span>  
  
### <a name="to-resize-multiple-controls-on-a-form"></a><span data-ttu-id="4f801-108">Para cambiar el tamaño de varios controles en un formulario</span><span class="sxs-lookup"><span data-stu-id="4f801-108">To resize multiple controls on a form</span></span>  
  
1.  <span data-ttu-id="4f801-109">Mantenga presionada la tecla CTRL o MAYÚS y seleccione los controles que desea cambiar el tamaño.</span><span class="sxs-lookup"><span data-stu-id="4f801-109">Hold down the CTRL or SHIFT key and select the controls you want to resize.</span></span> <span data-ttu-id="4f801-110">El tamaño del primer control que seleccione se utiliza para los demás controles.</span><span class="sxs-lookup"><span data-stu-id="4f801-110">The size of the first control you select is used for the other controls.</span></span>  
  
2.  <span data-ttu-id="4f801-111">En el **formato** menú, elija **Igualar tamaño**y seleccione una de las cuatro opciones.</span><span class="sxs-lookup"><span data-stu-id="4f801-111">On the **Format** menu, choose **Make Same Size**, and select one of the four options.</span></span> <span data-ttu-id="4f801-112">Los tres primeros comandos cambian las dimensiones de los controles para que coincida con el primer control seleccionado.</span><span class="sxs-lookup"><span data-stu-id="4f801-112">The first three commands change the dimensions of the controls to match the first-selected control.</span></span>
