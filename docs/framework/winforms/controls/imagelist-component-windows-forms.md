# <a name="imagelist-component-windows-forms"></a>ImageList (Componente, formularios Windows Forms)
El componente `ImageList` de Windows Forms se usa para almacenar imágenes que posteriormente se pueden mostrar en los controles. Una lista de imágenes permite escribir el código para un catálogo único y coherente de imágenes.
