# Selecci&#243;n y uso del Portapapeles con el control DataGridView de formularios Windows Forms
El control `DataGridView` proporciona distintas opciones para configurar la manera en que los usuarios pueden seleccionar celdas, filas y columnas.  Por ejemplo, puede habilitar la selección única y múltiple, la selección de filas o columnas completas cuando los usuarios hacen clic en celdas o la selección de filas y columnas completas sólo cuando el usuario hace clic en sus encabezados, que permite la selección de celdas también.  Si desea proporcionar su propia interfaz de usuario para la selección, puede deshabilitar la selección ordinaria y controlar mediante programación toda la selección.  Además, puede permitir que los usuarios copien los valores seleccionados al Portapapeles.
