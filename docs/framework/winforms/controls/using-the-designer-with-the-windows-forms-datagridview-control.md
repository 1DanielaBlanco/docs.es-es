# Utilizar el Dise&#241;ador con el control DataGridView de formularios Windows Forms
Visual Studio proporciona la función del diseñador para el control `DataGridView` que le permite realizar muchas tareas de configuración sin escribir código.  Entre estas tareas se incluye enlazar el control a un origen de datos, modificar las columnas utilizadas para mostrar datos y ajustar la apariencia y comportamiento básico del control.
