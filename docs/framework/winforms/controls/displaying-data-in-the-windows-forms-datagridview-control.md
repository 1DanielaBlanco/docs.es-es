# Mostrar datos en el control DataGridView de formularios Windows Forms
Se utiliza el control `DataGridView` para mostrar los datos de distintos orígenes de datos externos.  Alternativamente, puede agregar filas y columnas al control y rellenarlo manualmente con datos.  
  
 Cuando enlaza el control a un origen de datos, puede generar automáticamente columnas basándose en el esquema del origen de datos.  Si estas columnas no aparecen tal como desea, puede ocultarlas, quitarlas o reorganizarlas.  También puede agregar columnas independientes para mostrar datos complementarios que no proceden del origen de datos.  
  
 De forma adicional, puede mostrar los datos con formatos estándar \(como el formato de divisa\) o puede personalizar el formato de presentación para presentar los datos, no obstante, que necesita \(como cambiar el color de fondo de números negativos o reemplazar valores de cadena con las imágenes correspondientes\).
