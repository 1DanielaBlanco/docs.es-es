# <a name="displaying-data-in-the-windows-forms-datagridview-control"></a>Mostrar datos en el control DataGridView de formularios Windows Forms
El `DataGridView` control se usa para mostrar datos desde una variedad de orígenes de datos externos. Como alternativa, puede agregar filas y columnas al control y rellenarlo manualmente con datos.  
  
 Al enlazar el control a un origen de datos, puede generar automáticamente columnas basándose en el esquema del origen de datos. Si estas columnas no aparecen tal como desea que puedan, puede ocultar, quitar o reorganizar ellos. También puede agregar columnas sin enlazar para mostrar datos complementarios que no proceden del origen de datos.  
  
 Además, puede mostrar los datos con formatos estándares (por ejemplo, el formato de moneda) o puede personalizar el formato de presentación para presentar los datos, no obstante, necesita (como cambiar el color de fondo para los números negativos o reemplazar valores de cadena con las imágenes correspondientes).
