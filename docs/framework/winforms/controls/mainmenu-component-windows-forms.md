# MainMenu (Componente, formularios Windows Forms)
> [!NOTE]
>  Aunque los controles `MenuStrip` y `ContextMenuStrip` reemplazan los controles `MainMenu` y <xref:System.Windows.Forms.ContextMenu> de versiones anteriores y les agregan funcionalidad, los controles `MainMenu` y <xref:System.Windows.Forms.ContextMenu> se conservan a efectos de compatibilidad con versiones anteriores y, en su caso, de uso futuro.  
  
 El componente `MainMenu` de formularios Windows Forms muestra un menú en tiempo de ejecución.
