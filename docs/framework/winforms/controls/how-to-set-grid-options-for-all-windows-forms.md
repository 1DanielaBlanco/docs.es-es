# C&#243;mo: Establecer opciones de cuadr&#237;cula para todos los formularios Windows Forms
A medida que se habitúe a trabajar con el entorno de desarrollo de [!INCLUDE[vsprvs](../../../../includes/vsprvs-md.md)], podrá establecer preferencias para todos los formularios y proyectos con los que trabaje en el Diseñador de Windows Forms.  
  
> [!NOTE]
>  Los cuadros de diálogo y comandos de menú que se ven pueden diferir de los descritos en la Ayuda, en función de los valores de configuración o de edición activos.  Para cambiar la configuración, elija **Importar y exportar configuraciones** en el menú **Herramientas**.  Para obtener más información, vea [Customizing Development Settings in Visual Studio](http://msdn.microsoft.com/es-es/22c4debb-4e31-47a8-8f19-16f328d7dcd3).  
  
### Para establecer opciones globales de formularios Windows Forms  
  
1.  En el menú **Herramientas**, seleccione **Opciones**.  
  
2.  En el panel izquierdo del cuadro de diálogo **Opciones**, haga clic en **Diseñador de Windows Forms**.  
  
     En el panel derecho, bajo el encabezado **Configuración de diseño**, puede establecer las configuraciones predeterminadas de la cuadrícula para todos los formularios nuevos que cree.  Entre estas configuraciones se incluyen el tamaño de la cuadrícula, si los controles se ajustan a ella y si está activa de forma predeterminada.  Además, puede seleccionar entre los modos de diseño **AjustarALaCuadrícula** y **Líneas de ajuste**.  Para obtener más información sobre líneas de ajuste, vea [Tutorial: Organizar controles en formularios Windows Forms mediante líneas de ajuste](../../../../docs/framework/winforms/controls/walkthrough-arranging-controls-on-windows-forms-using-snaplines.md).
