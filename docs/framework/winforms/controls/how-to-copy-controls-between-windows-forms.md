# C&#243;mo: Copiar controles entre formularios Windows Forms
Puede copiar un control al mismo formulario, a otro formulario del proyecto o al Portapapeles para utilizarlo en otras soluciones.  
  
### Para copiar un control  
  
1.  Seleccione el control y elija **Copiar** en el menú **Edición**.  
  
 Ahora podrá pegar el control en cualquier formulario que acepte este tipo de control.  Además, el control se habrá agregado al Portapapeles.
