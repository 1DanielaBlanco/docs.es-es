# Resumen del control ToolStripStatusLabel
<xref:System.Windows.Forms.ToolStripStatusLabel> es una etiqueta para <xref:System.Windows.Forms.StatusStrip>.  Como <xref:System.Windows.Forms.Label> o <xref:System.Windows.Forms.ToolStripLabel>, <xref:System.Windows.Forms.ToolStripStatusLabel> proporciona un área de presentación no seleccionable para el texto, imágenes o ambos.  <xref:System.Windows.Forms.StatusStrip> hospeda <xref:System.Windows.Forms.ToolStripStatusLabel>.  
  
### Miembros de ToolStripStatusLabel importantes  
  
|Name|Descripción|  
|----------|-----------------|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.Spring%2A>|Obtiene o establece un valor que indica si <xref:System.Windows.Forms.ToolStripStatusLabel> rellena automáticamente el espacio disponible en <xref:System.Windows.Forms.StatusStrip> cuando se cambia el tamaño del formulario|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderSides%2A>|Obtiene o establece un valor que indica qué lados del control <xref:System.Windows.Forms.ToolStripStatusLabel> tienen bordes.|  
|<xref:System.Windows.Forms.ToolStripStatusLabel.BorderStyle%2A>|Obtiene o establece el estilo de los bordes del control <xref:System.Windows.Forms.ToolStripStatusLabel>.|
