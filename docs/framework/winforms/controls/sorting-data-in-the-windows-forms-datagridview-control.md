# Ordenar datos en el control DataGridView de formularios Windows Forms
De forma predeterminada, los usuarios pueden ordenar los datos en un control `DataGridView` haciendo clic en el encabezado de la columna de cuadro de texto.  Puede modificar la propiedad `SortMode` de columnas concretas para permitir que los usuarios ordenen por otros tipos de columna cuando tiene sentido hacerlo.  También puede ordenar los datos mediante programación por cualquier columna o por varias columnas.
