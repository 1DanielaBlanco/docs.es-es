# Control GroupBox (formularios Windows Forms)
Los controles <xref:System.Windows.Forms.GroupBox> de Windows Forms se utilizan para proporcionar un agrupamiento identificable para otros controles.  Normalmente, los cuadros de grupo se utilizan para subdividir un formulario por funciones.  Por ejemplo, podría tener un formulario de pedido que especifique opciones de envío, como el servicio de transporte urgente que se va a utilizar.  La agrupación de todas las opciones en un cuadro de grupo ofrece al usuario una pista visual lógica.  Los controles <xref:System.Windows.Forms.GroupBox> y <xref:System.Windows.Forms.Panel> son similares; sin embargo, el control <xref:System.Windows.Forms.GroupBox> es el único de los dos que muestra una leyenda, y el control <xref:System.Windows.Forms.Panel> es el único de los dos que puede tener barras de desplazamiento.
