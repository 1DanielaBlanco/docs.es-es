# Dise&#241;o de los controles de formularios Windows Forms
La posición precisa de los controles en el formulario es de alta prioridad para muchas aplicaciones.  El espacio de nombres <xref:System.Windows.Forms?displayProperty=fullName> proporciona muchas herramientas de diseño para conseguir esto.
