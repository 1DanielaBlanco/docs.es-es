# Informaci&#243;n general del control ToolStripPanel
<xref:System.Windows.Forms.ToolStripPanel> proporciona un área única para colocar los controles <xref:System.Windows.Forms.ToolStrip>, <xref:System.Windows.Forms.MenuStrip> y <xref:System.Windows.Forms.StatusStrip> y compartir el espacio.  Varios controles <xref:System.Windows.Forms.ToolStrip> se apilan vertical u horizontalmente en función de la propiedad <xref:System.Windows.Forms.ToolStripPanelRow.Orientation%2A> del panel <xref:System.Windows.Forms.ToolStripPanel>.  
  
### Miembros importantes de ToolStripPanel  
  
|Name|Descripción|  
|----------|-----------------|  
|<xref:System.Windows.Forms.ToolStripPanel.Orientation%2A>|Obtiene o establece un valor que indica la orientación vertical u horizontal de los paneles <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.Renderer%2A>|Obtiene o establece una clase <xref:System.Windows.Forms.ToolStripRenderer> que se utiliza para personalizar la apariencia de paneles <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.RenderMode%2A>|Obtiene o establece los estilos de dibujo que van a aplicarse al <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.RowMargin%2A>|Obtiene o establece el espaciado, en píxeles, entre <xref:System.Windows.Forms.ToolStripPanelRow> y <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.Rows%2A>|Obtiene <xref:System.Windows.Forms.ToolStripPanelRow> de este panel <xref:System.Windows.Forms.ToolStripPanel>.|  
|<xref:System.Windows.Forms.ToolStripPanel.Join%2A>|Agrega un objeto <xref:System.Windows.Forms.ToolStrip> a <xref:System.Windows.Forms.ToolStripPanel>.|
