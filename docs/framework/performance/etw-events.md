# ETW Events in the .NET Framework
El Seguimiento de eventos para Windows \(ETW\) es un sistema de seguimiento de alto rendimiento y escasa sobrecarga proporcionado por los sistemas operativos Windows.  Complementa la compatibilidad de generación de perfiles y de depuración proporcionada por .NET Framework y se puede utilizar para solucionar problemas en numerosos escenarios.  Para obtener información general sobre ETW, vea [Mejorar la optimización de depuración y de rendimiento con ETW](http://go.microsoft.com/fwlink/?LinkID=161142) en MSDN Library.  
  
 En .NET Framework, el seguimiento de eventos ETW está disponible para Common Language Runtime \(CLR\), la [biblioteca TPL \(Task Parallel Library, biblioteca de procesamiento paralelo basado en tareas\)](../../../docs/standard/parallel-programming/task-parallel-library-tpl.md) y [Parallel LINQ \(PLINQ\)](../../../docs/standard/parallel-programming/parallel-linq-plinq.md).
