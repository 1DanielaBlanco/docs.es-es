## <a name="resources-in-text-files"></a><span data-ttu-id="28c24-123">Recursos en archivos de texto</span><span class="sxs-lookup"><span data-stu-id="28c24-123">Resources in Text Files</span></span>  
 <span data-ttu-id="28c24-124">Solo puede usar archivos de texto (.txt o .restext) para almacenar los recursos de cadena.
