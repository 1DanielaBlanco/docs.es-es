# UI Automation Specification and Community Promise
La automatización de la interfaz de usuario proporciona acceso mediante programación a la mayoría de los elementos de la interfaz de usuario del escritorio, lo que permite que los productos de tecnología de asistencia, tales como los lectores de pantalla, proporcionen información sobre ella a los usuarios finales y permitan manipularla mediante recursos distintos de la entrada estándar.
