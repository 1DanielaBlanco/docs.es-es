# Accessibility
> [!NOTE]
>  Esta documentación está dirigida a desarrolladores de .NET Framework que desean usar las clases administradas de [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] definidas en el espacio de nombres <xref:System.Windows.Automation>.  Para obtener información actualizada sobre [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)], vea [Windows Automation API: UI Automation](http://go.microsoft.com/fwlink/?LinkID=156746).  
  
 La [!INCLUDE[TLA#tla_uiautomation](../../../includes/tlasharptla-uiautomation-md.md)] es el nuevo marco de accesibilidad para [!INCLUDE[TLA#tla_win](../../../includes/tlasharptla-win-md.md)].  Se ocupa de los requisitos de los productos de tecnología de ayuda y marcos de pruebas automatizadas al proporcionar acceso mediante programación a información acerca de la [!INCLUDE[TLA#tla_ui](../../../includes/tlasharptla-ui-md.md)].  Además, la [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] permite a los desarrolladores de controles y aplicaciones crear productos accesibles.  
  
 En esta documentación se describe la [!INCLUDE[TLA#tla_api](../../../includes/tlasharptla-api-md.md)] de [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] para código administrado.  Para obtener información acerca de la programación para la [!INCLUDE[TLA2#tla_uiautomation](../../../includes/tla2sharptla-uiautomation-md.md)] en C\+\+, vea [Automatizar la interfaz de usuario para aplicaciones Win32](entry_uiauto_win32).
