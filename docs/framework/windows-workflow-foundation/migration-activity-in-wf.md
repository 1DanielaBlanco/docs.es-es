# Actividad de migraci&#243;n en WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)] proporciona el <xref:> System.Activities.Statements.Interop?qualifyHint=False&autoUpgrade=True actividad para ejecutar actividades que derivan de <xref:System.WorkflowModel.Activities.Activity> dentro de un flujo de trabajo que se basa en <xref:System.Activities.Activity>. [!INCLUDE[crdefault](../../../includes/crdefault-md.md)] el [Guía de migración](../../../docs/framework/windows-workflow-foundation//migration-guidance.md) sección.  
  
> [!NOTE]
>  El <xref:System.Activities.Statements.Interop> actividad no aparecerá en el cuadro de herramientas Diseñador de flujo de trabajo a menos que el proyecto del flujo de trabajo tiene su **.NET Framework de destino** definida en **.Net Framework 4** o superior.
