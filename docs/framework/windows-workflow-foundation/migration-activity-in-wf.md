# <a name="migration-activity-in-wf"></a>Actividad de migración en WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)]proporciona el <xref:System.Activities.Statements.Interop> actividad para ejecutar actividades que derivan de la actividad dentro de un flujo de trabajo que se basa en <xref:System.Activities.Activity>. [!INCLUDE[crdefault](../../../includes/crdefault-md.md)]el [Guía de migración](../../../docs/framework/windows-workflow-foundation/migration-guidance.md) sección.  
  
> [!NOTE]
>  El <xref:System.Activities.Statements.Interop> actividad no aparece en el cuadro de herramientas Diseñador de flujo de trabajo a menos que el proyecto del flujo de trabajo tiene su **.NET Framework de destino** definida en **.Net Framework 4** o superior.
