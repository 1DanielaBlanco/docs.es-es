# Cadena de conexi&#243;n y nombre de cadena de conexi&#243;n
La propiedad **Cadena de conexión** especifica la cadena de conexión que el almacén de instancias de flujo de trabajo de SQL debe usar para conectarse a una base de datos de persistencia.Este parámetro es opcional.La propiedad **Nombre de cadena de conexión** especifica el nombre de la cadena de conexión con nombre que el almacén de instancias de flujo de trabajo de SQL debería usar para conectarse a la base de datos de persistencia.Este parámetro es opcional.Debería especificar un valor para la propiedad Nombre de cadena de conexión o la propiedad Cadena de conexión si no desea que el almacén de instancias de flujo de trabajo de SQL use la cadena de conexión con nombre predeterminado **DefaultSqlWorkflowInstanceStoreConnectionString**.
