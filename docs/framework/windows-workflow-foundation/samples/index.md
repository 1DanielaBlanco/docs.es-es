# <a name="windows-workflow-wf-samples"></a>Ejemplos de Windows Workflow (WF)

Estos ejemplos proporcionan instrucciones sobre distintos aspectos de Windows Workflow Foundation (WF).

Los ejemplos se pueden descargar desde el [Centro de descarga de Microsoft](https://go.microsoft.com/fwlink/?LinkId=150780).

> [!NOTE]
> Estos ejemplos se crearon con Visual Studio 2010 y [!INCLUDE[netfx40_short](../../../../includes/netfx40-short-md.md)], pero son compatibles con versiones posteriores de Visual Studio y .NET Framework. Ejemplos adicionales para Windows Workflow Foundation en [!INCLUDE[netfx_current_short](../../../../includes/netfx-current-short-md.md)] pueden encontrarse en [ejemplos de código MSDN](https://aka.ms/WF45Samples).
