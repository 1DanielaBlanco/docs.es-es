# Ejemplos de Windows Workflow (WF)
Estos ejemplos proporcionan instrucciones sobre distintos aspectos de [!INCLUDE[wf](../../../../includes/wf-md.md)].  
  
 Los ejemplos se pueden descargar desde el [Centro de descarga de Microsoft](http://go.microsoft.com/fwlink/?LinkId=150780).  
  
> [!NOTE]
>  Estos ejemplos se crearon con [!INCLUDE[vs2010](../../../../includes/vs2010-md.md)] y [!INCLUDE[netfx40_short](../../../../includes/netfx40-short-md.md)], pero son compatibles con [!INCLUDE[vs_current_long](../../../../includes/vs-current-long-md.md)] y [!INCLUDE[netfx_current_short](../../../../includes/netfx-current-short-md.md)].Ejemplos adicionales para [!INCLUDE[wf2](../../../../includes/wf2-md.md)] en [!INCLUDE[netfx_current_short](../../../../includes/netfx-current-short-md.md)] se pueden encontrar en [Codeplex](http://aka.ms/WF45Samples).
