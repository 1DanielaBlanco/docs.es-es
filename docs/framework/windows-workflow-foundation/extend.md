# <a name="extending-windows-workflow-foundation"></a>Ampliar Windows Workflow Foundation
En la siguiente sección se describe cómo ampliar [!INCLUDE[wf](../../../includes/wf-md.md)] con actividades y diseñadores personalizados en entornos que se han rehospedado fuera de [!INCLUDE[vs2010](../../../includes/vs2010-md.md)].  
  
## <a name="in-this-section"></a>En esta sección  
 [Personalización de la experiencia de diseño del flujo de trabajo](../../../docs/framework/windows-workflow-foundation/customizing-the-workflow-design-experience.md)  
 Indica cómo los escenarios para diseñar actividades personalizadas y para re-hospedar [!INCLUDE[wfd1](../../../includes/wfd1-md.md)] se han simplificado enormemente en [!INCLUDE[netfx40_short](../../../includes/netfx40-short-md.md)]. Las tareas de desarrollo e implementación resultan ahora más fáciles y más flexibles porque el nuevo modelo de programación del diseñador de actividad se basa en [!INCLUDE[avalon1](../../../includes/avalon1-md.md)].
