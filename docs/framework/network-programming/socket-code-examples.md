# Ejemplos de c&#243;digo de socket
Los ejemplos de código siguientes muestran cómo utilizar la clase de <xref:System.Net.Sockets.Socket> como cliente para conectarse a los servicios de red remotos y como servidor para realizar escuchas para conexiones de clientes remotos.  
  
## En esta sección  
 [Ejemplo de sockets de cliente sincrónicos](../../../docs/framework/network-programming/synchronous-client-socket-example.md)  
 Muestra cómo implementar un cliente sincrónico de <xref:System.Net.Sockets.Socket> para conectar con un servidor y muestra los datos devueltos del servidor.  
  
 [Ejemplo de sockets de servidor sincrónicos](../../../docs/framework/network-programming/synchronous-server-socket-example.md)  
 Las escrituras cómo implementar un servidor sincrónico de <xref:System.Net.Sockets.Socket> que acepte las conexiones de un cliente y compilaciones de repetición admiten los datos recibidos del cliente.  
  
 [Ejemplo de sockets de cliente asincrónicos](../../../docs/framework/network-programming/asynchronous-client-socket-example.md)  
 Muestra cómo implementar un cliente asincrónico de <xref:System.Net.Sockets.Socket> para conectar con un servidor y muestra los datos devueltos del servidor.  
  
 [Ejemplo de sockets de servidor asincrónicos](../../../docs/framework/network-programming/asynchronous-server-socket-example.md)  
 Las escrituras cómo implementar un servidor asincrónico de <xref:System.Net.Sockets.Socket> que acepte las conexiones de un cliente y compilaciones de repetición admiten los datos recibidos del cliente.
