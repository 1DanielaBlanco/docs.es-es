# Elementos base
Cuatro clases clave, que son <xref:System.Windows.UIElement>, <xref:System.Windows.ContentElement>, <xref:System.Windows.FrameworkElement> y <xref:System.Windows.FrameworkContentElement>, implementan un porcentaje importante de la funcionalidad común de elementos disponible en la programación de WPF.  En este SDK nos referimos a estas cuatro clases como las clases de elementos base.
