# Entrada manuscrita digital
En esta sección se analiza el uso de entrada de lápiz digital en [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)].  Aunque tradicionalmente solo se encontraba en el SDK de Tablet PC, la entrada de lápiz digital ahora está disponible en Windows Presentation Foundation.  Esto significa que se pueden desarrollar aplicaciones para Tablet PC completas aprovechando la capacidad de Windows Presentation Foundation.
