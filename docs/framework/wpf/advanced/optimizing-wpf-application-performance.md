# Optimizar WPF: Rendimiento de aplicaciones
Esta sección está diseñada como referencia para los programadores de aplicaciones de [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] que buscan maneras de mejorar el rendimiento de sus aplicaciones.  Si usted es un programador que desconoce las plataformas [!INCLUDE[TLA#tla_avalonwinfx](../../../../includes/tlasharptla-avalonwinfx-md.md)] y [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)], primero deberá familiarizarse con ambas.  En esta sección se da por supuesto un conocimiento práctico de las dos y está escrita para programadores que ya tienen conocimientos suficientes para programar aplicaciones que funcionan.  
  
> [!NOTE]
>  Los datos de rendimiento proporcionados en esta sección se basan en aplicaciones de [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] que se ejecutan en un PC a 2,8 GHz con 512 de RAM y una tarjeta gráfica ATI Radeon 9700.
