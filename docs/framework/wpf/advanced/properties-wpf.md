# Propiedades
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] proporciona un conjunto de servicios que se pueden utilizar para extender la funcionalidad de una propiedad de [!INCLUDE[TLA#tla_clr](../../../../includes/tlasharptla-clr-md.md)].  Colectivamente, estos servicios se conocen como el sistema de propiedades de WPF.  Una propiedad que está respaldada por el sistema de propiedades de [!INCLUDE[TLA#tla_titlewinclient](../../../../includes/tlasharptla-titlewinclient-md.md)] se conoce como una [propiedad de dependencia](GTMT).
