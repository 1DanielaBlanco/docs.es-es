# Glifos
Los glifos son una representación de bajo nivel de un carácter que se va a dibujar en pantalla.  [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] proporciona acceso directo a los glifos para los clientes que desean interceptar y conservar el texto después de darle formato.
