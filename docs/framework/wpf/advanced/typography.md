# Tipograf&#237;a
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] incluye compatibilidad con la presentación enriquecida de contenido de texto.  En [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)], se presenta el texto utilizando [!INCLUDE[TLA#tla_ct](../../../../includes/tlasharptla-ct-md.md)], que mejora su claridad y legibilidad.  [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] también admite las fuentes [!INCLUDE[TLA#tla_opentype](../../../../includes/tlasharptla-opentype-md.md)], que ofrecen funciones adicionales a las definidas por el formato [!INCLUDE[TLA#tla_truetype](../../../../includes/tlasharptla-truetype-md.md)].
