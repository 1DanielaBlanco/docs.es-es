# Recursos (WPF)
Un recurso es un objeto que se puede reutilizar en distintos lugares de la aplicación.  WPF admite diferentes tipos de recursos.  Estos recursos son principalmente dos tipos de recursos: Recursos XAML y archivos de datos de recursos.  Ejemplos de los pinceles y los estilos de inclusión de recursos XAML.  Los archivos de datos de recursos son archivos de datos no ejecutables que necesita la aplicación.
