# C&#243;mo: Utilizar un objeto MatrixTransform para crear transformaciones personalizadas
En este ejemplo se muestra cómo utilizar <xref:System.Windows.Media.MatrixTransform> para trasladar \(mover\) la posición, el ajuste y el [sesgo](GTMT) de un control <xref:System.Windows.Controls.Button>.
