# Programaci&#243;n de capas visuales
El objeto <xref:System.Windows.Media.Visual> es un objeto básico de [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)], cuyo rol principal es proporcionar la compatibilidad con la representación.  Los controles de interfaz de usuario, tales como <xref:System.Windows.Controls.Button> y <xref:System.Windows.Controls.TextBox>, se derivan de la clase <xref:System.Windows.Media.Visual> y la utilizan para conservar sus datos de representación.
