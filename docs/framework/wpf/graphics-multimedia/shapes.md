# Formas
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] proporciona compatibilidad para dibujar formas básicas mediante la clase <xref:System.Windows.Shapes.Shape>.  Los objetos de forma derivan de <xref:System.Windows.FrameworkElement> y, como tal, participan en el sistema de diseño y pueden dibujarse simplemente utilizando [!INCLUDE[TLA#tla_xaml](../../../../includes/tlasharptla-xaml-md.md)].
