# Dibujos
Los objetos <xref:System.Windows.Media.Drawing> se utilizan para dibujar eficazmente formas, imágenes o texto.  Los dibujos se utilizan al pintar con un <xref:System.Windows.Media.DrawingBrush> o programar con objetos <xref:System.Windows.Media.Visual>.
