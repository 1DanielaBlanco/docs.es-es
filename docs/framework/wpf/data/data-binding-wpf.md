# Enlace de datos (WPF)
El enlace de datos de Windows Presentation Foundation \(WPF\) proporciona un método simple y coherente para que las aplicaciones presenten e interactúen con datos.  Los elementos se pueden enlazar a los datos de diversos orígenes de datos en forma de objetos CLR \(Common Language Runtime\) y XML.
