# Fotograma
El control <xref:System.Windows.Controls.Frame> admite la navegación de contenido dentro del contenido.  <xref:System.Windows.Controls.Frame> puede estar hospedado en un elemento raíz como <xref:System.Windows.Window>, <xref:System.Windows.Navigation.NavigationWindow>, <xref:System.Windows.Controls.Page>, <xref:System.Windows.Controls.UserControl> o <xref:System.Windows.Documents.FlowDocument>, o como una isla dentro de un árbol de contenido que pertenece a un elemento raíz.
