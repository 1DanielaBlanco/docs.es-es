# FlowDocumentScrollViewer
El control <xref:System.Windows.Controls.FlowDocumentScrollViewer> se utiliza para ver el contenido de <xref:System.Windows.Documents.FlowDocument> en un contenedor de desplazamiento.  Contrasta con <xref:System.Windows.Controls.FlowDocumentPageViewer>, que ve el contenido en páginas individuales.
