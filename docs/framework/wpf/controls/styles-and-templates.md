# Estilos y plantillas
Los estilos y plantillas de Windows Presentation Foundation \(WPF\) hacen referencia a un conjunto de características \(estilos, plantillas, desencadenadores y guiones gráficos\) que permiten a los diseñadores de aplicaciones, documentos o interfaces de usuario crear aplicaciones visualmente atractivas y estandarizar la apariencia del producto.
