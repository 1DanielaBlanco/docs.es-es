# BulletDecorator
<xref:System.Windows.Controls.Primitives.BulletDecorator> tiene dos propiedades de contenido: <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> y <xref:System.Windows.Controls.Decorator.Child%2A>.  La propiedad <xref:System.Windows.Controls.Primitives.BulletDecorator.Bullet%2A> define <xref:System.Windows.UIElement> para que se utilice como una viñeta.  La propiedad <xref:System.Windows.Controls.Decorator.Child%2A> define <xref:System.Windows.UIElement>, que se alinea visualmente con la viñeta.
