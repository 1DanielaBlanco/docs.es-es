# PrintDialog
El control <xref:System.Windows.Controls.PrintDialog> se utiliza para crear instancias de un cuadro de diálogo de impresión estándar que automáticamente configura <xref:System.Printing.PrintTicket> y <xref:System.Printing.PrintQueue> según los datos proporcionados por el usuario.
