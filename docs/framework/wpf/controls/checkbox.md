# CheckBox
Puede utilizar un control <xref:System.Windows.Controls.CheckBox> en la [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] de su aplicación para representar opciones que un usuario puede activar o desactivar.  Puede utilizar una sola casilla, o bien agrupar dos o más.
