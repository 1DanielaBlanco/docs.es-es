# RepeatButton
<xref:System.Windows.Controls.Primitives.RepeatButton> es parecido a un control <xref:System.Windows.Controls.Button>.  Sin embargo, los elementos <xref:System.Windows.Controls.Primitives.RepeatButton> proporcionan control sobre cuándo y cómo se produce el evento <xref:System.Windows.Controls.Primitives.ButtonBase.Click>.  
  
 En la ilustración siguiente se muestra un ejemplo de los tres estados de un control de botón repetido: predeterminado, con el foco del puntero y presionado.  El primer botón muestra el estado predeterminado de <xref:System.Windows.Controls.Primitives.RepeatButton>.  El segundo muestra cómo cambia su aspecto cuando se desplaza el puntero del mouse sobre él, y recibe el foco.  El último botón muestra el aspecto de <xref:System.Windows.Controls.Primitives.RepeatButton> cuando el usuario presiona el botón del mouse sobre el control.
