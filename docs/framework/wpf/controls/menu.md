# Men&#250;
Un menú \(<xref:System.Windows.Controls.Menu>\) es un control que permite organizar jerárquicamente los elementos asociados a comandos o controladores de eventos.  Cada <xref:System.Windows.Controls.Menu> puede contener varios controles <xref:System.Windows.Controls.MenuItem>.  Cada <xref:System.Windows.Controls.MenuItem> puede invocar un comando o un controlador de eventos `Click`.  Un <xref:System.Windows.Controls.MenuItem> también puede tener varios elementos <xref:System.Windows.Controls.MenuItem> como elementos secundarios, que constituyen un submenú.  
  
 En la ilustración siguiente se muestran los tres estados diferentes de un control de menú.  El estado predeterminado es aquél que se produce cuando ningún dispositivo \(como un puntero del mouse\) está situado sobre <xref:System.Windows.Controls.Menu>.  El estado del foco se produce cuando se desplaza el puntero del mouse sobre <xref:System.Windows.Controls.Menu> y el estado presionado, cuando se hace clic en un botón del mouse sobre <xref:System.Windows.Controls.Menu>.
