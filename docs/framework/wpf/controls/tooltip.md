# Informaci&#243;n sobre herramientas
Una información sobre herramientas es una pequeña ventana emergente que aparece cuando el usuario detiene el puntero del mouse sobre un elemento, por ejemplo sobre un control <xref:System.Windows.Controls.Button>.  
  
 En la ilustración siguiente se muestra un puntero del mouse situado sobre el control **Cerrar**<xref:System.Windows.Controls.Button>, que muestra su <xref:System.Windows.Controls.ToolTip> identificativa.
