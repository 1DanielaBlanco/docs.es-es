# ComboBox
El control <xref:System.Windows.Controls.ComboBox> presenta a los usuarios una lista de opciones.  La lista se muestra y oculta cuando el control se expande y contrae.  En su estado predeterminado, la lista está contraída y únicamente muestra una opción.  El usuario hace clic en un botón para ver la lista completa de opciones.
