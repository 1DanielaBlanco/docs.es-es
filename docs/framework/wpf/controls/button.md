# Bot&#243;n
Un control <xref:System.Windows.Controls.Button> reacciona a los datos proporcionados por el usuario desde un mouse, teclado, lápiz u otro dispositivo de entrada y provoca un evento <xref:System.Windows.Controls.Primitives.ButtonBase.Click>.  <xref:System.Windows.Controls.Button> es un componente básico de la [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)] que puede incluir contenido simple, como texto, pero también complejo, como imágenes y controles <xref:System.Windows.Controls.Panel>.
