# ProgressBar
<xref:System.Windows.Controls.ProgressBar> indica el progreso de una operación.  El control <xref:System.Windows.Controls.ProgressBar> está compuesto de una ventana que se llena gradualmente del color de resaltado del sistema a medida que una operación progresa.
