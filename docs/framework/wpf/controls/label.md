# Etiqueta
Los controles <xref:System.Windows.Controls.Label> suelen proporcionar información en la [!INCLUDE[TLA#tla_ui](../../../../includes/tlasharptla-ui-md.md)].  Históricamente, <xref:System.Windows.Controls.Label> sólo contenía texto, pero debido a que el control <xref:System.Windows.Controls.Label> que se distribuye con [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] es <xref:System.Windows.Controls.ContentControl>, puede contener texto o elementos <xref:System.Windows.UIElement>.  
  
 <xref:System.Windows.Controls.Label> proporciona compatibilidad funcional y visual para las teclas de acceso.  Se utiliza con frecuencia para permitir el acceso rápido desde el teclado a controles como <xref:System.Windows.Controls.TextBox>.  Para asignar un control <xref:System.Windows.Controls.Label> a un control <xref:System.Windows.Controls.Control>, establezca la propiedad <xref:System.Windows.Controls.Label.Target%2A?displayProperty=fullName> en el control que debe obtener el foco cuando el usuario presione la tecla de acceso.  
  
 En la ilustración siguiente se muestra un control <xref:System.Windows.Controls.Label> denominado "Themes" cuyo destino es un control <xref:System.Windows.Controls.ComboBox>.  Cuando el usuario presiona, <xref:System.Windows.Controls.ComboBox> recibe el foco.  Para obtener más información, vea [How to: Set the Target Property of a Label](http://msdn.microsoft.com/es-es/b24c6977-ebcb-4855-a9bb-3fd4435af8f8).
