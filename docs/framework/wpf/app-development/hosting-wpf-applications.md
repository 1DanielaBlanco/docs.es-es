# Hospedar aplicaciones de WPF
Las aplicaciones de explorador XAML de WPF \(XBAP\) son aplicaciones cliente enriquecidas que se pueden implementar en un servidor web e iniciar en un explorador.  El host de WPF \(PresentationHost.exe\) se registra como el shell y el controlador MIME de los archivos XBAP y XAML.  Por consiguiente, Internet Explorer sabe iniciar el host de WPF cuando se inicia una aplicación XBAP.  Los usuarios de Firefox pueden instalar complementos de Firefox que permiten a Firefox hospedar también aplicaciones XBAP.  Una aplicación XBAP puede hospedarse en otros exploradores o aplicaciones independientes utilizando el explorador nativo que hospeda las API proporcionadas por WPF.
