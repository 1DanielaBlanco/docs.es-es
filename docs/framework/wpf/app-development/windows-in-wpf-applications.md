# Windows en aplicaciones para WPF
Los usuarios interactúan con las aplicaciones a través de las ventanas.  El propósito fundamental de una ventana es hospedar y mostrar contenido.  El tipo de contenido que una ventana hospeda depende del tipo de datos con que opera que una aplicación, y puede incluir multimedia, páginas XAML \(Extensible Application Markup Language\), páginas web, documentos, tablas y registros de base de datos e información del sistema.
