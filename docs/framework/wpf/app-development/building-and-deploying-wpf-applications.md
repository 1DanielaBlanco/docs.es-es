# Compilar e implementar aplicaciones WPF
La compilación y el modelo de implementación proporciona funciones para generar e implementar aplicaciones local y remotamente, incluido lo siguiente:  
  
-   MSBuild: el sistema de compilación de .NET situado en el espacio de nombres Microsoft.Build.Tasks.Windows.  
  
-   Recursos: trabajar con recursos de interfaz de usuario.  
  
-   Implementación de ClickOnce: sistema de publicación e implementación de .NET.
