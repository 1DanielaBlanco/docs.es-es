# <a name="application-compatibility-in-the-net-framework"></a>Compatibilidad de aplicaciones en .NET Framework
En los artículos siguientes se describen los problemas de compatibilidad de aplicaciones de las versiones recientes de .NET Framework:  
  
-   Entre las versiones 4 y 4.5: [Compatibilidad de aplicaciones en 4.5](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-5.md)  
  
-   Entre las versiones 4.5 y 4.5.1: [Compatibilidad de aplicaciones en 4.5.1](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-5-1.md)  
  
-   Entre las versiones 4.5.1 y 4.5.2: [Compatibilidad de aplicaciones en 4.5.2](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-5-2.md)  
  
-   Entre las versiones 4.5.2 y 4.6: [Compatibilidad de aplicaciones en 4.6](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-6.md)  
  
-   Entre las versiones 4.6 y [!INCLUDE[net_v461](../../../includes/net-v461-md.md)]: [Compatibilidad de aplicaciones en 4.6.1](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-6-1.md)  
  
-   Entre las versiones 4.6.1 y [!INCLUDE[net_v462](../../../includes/net-v462-md.md)]: [Compatibilidad de aplicaciones en 4.6.2](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-6-2.md)  

- Entre las versiones 4.6.2 y .NET Framework 4.7: [Compatibilidad de aplicaciones en 4.7](../../../docs/framework/migration-guide/application-compatibility-in-the-net-framework-4-7.md)  

En general, hemos intentado mantener un nivel muy alto de compatibilidad con versiones anteriores entre las versiones de .NET Framework, de modo que las aplicaciones que tienen como destino una versión concreta se ejecutarán sin cambios en versiones posteriores. La mayoría de los problemas de compatibilidad de aplicaciones tienen un impacto reducido y no requieren modificaciones de programación en las aplicaciones. Para aquellos que sí requieren modificaciones, vea la columna Impacto de las tablas de estos vínculos.
