# <a name="icordebugstepper2-interface1"></a>ICorDebugStepper2 Interfaz1
Proporciona compatibilidad con solo mi código ("JMC) de depuración.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[SetJMC (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugstepper2-setjmc-method.md)|Establece un valor que especifica si este ICorDebugStepper recorre paso a paso solo código creado por el desarrollador de una aplicación.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
>  Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
