# <a name="icordebugassembly2-interface1"></a>ICorDebugAssembly2 Interfaz1
Representa un ensamblado. Esta interfaz es una extensión de ICorDebugAssembly (interfaz).  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[IsFullyTrusted (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugassembly2-isfullytrusted-method.md)|Obtiene un valor que indica si el ensamblado se ha concedido plena confianza por el sistema de seguridad en tiempo de ejecución.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
>  Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
