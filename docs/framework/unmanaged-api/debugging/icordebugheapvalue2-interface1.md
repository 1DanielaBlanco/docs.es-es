# <a name="icordebugheapvalue2-interface1"></a>ICorDebugHeapValue2 Interfaz1
Una extensión de ICorDebugHeapValue que proporciona compatibilidad con common language runtime (CLR) administra.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[CreateHandle (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugheapvalue2-createhandle-method.md)|Crea un identificador del tipo especificado para este `ICorDebugHeapValue2` objeto.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
>  Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
