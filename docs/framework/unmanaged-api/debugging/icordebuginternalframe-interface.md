# <a name="icordebuginternalframe-interface1"></a>ICorDebugInternalFrame Interfaz1
Representa un marco de tiempo de ejecución interno en la pila. Esta interfaz es una subclase de ICorDebugFrame (interfaz).  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[GetFrameType (método)](../../../../docs/framework/unmanaged-api/debugging/icordebuginternalframe-getframetype-method.md)|Obtiene el tipo de este marco interno.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
>  Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
