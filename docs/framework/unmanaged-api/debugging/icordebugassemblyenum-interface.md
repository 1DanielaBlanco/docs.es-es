# <a name="icordebugassemblyenum-interface1"></a>ICorDebugAssemblyEnum Interfaz1
Implementa los métodos de ICorDebugEnum y enumera las matrices de ICorDebugAssembly.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Next (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugassemblyenum-next-method.md)|Obtiene el número especificado de `ICorDebugAssembly` instancias en la enumeración, comenzando desde la posición actual.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
>  Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
