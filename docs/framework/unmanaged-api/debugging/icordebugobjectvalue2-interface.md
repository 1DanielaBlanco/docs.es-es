# <a name="icordebugobjectvalue2-interface"></a>ICorDebugObjectValue2 (Interfaz)

Extiende la interfaz "ICorDebugObjectValue" para admitir la herencia y la reemplaza.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[GetVirtualMethodAndType (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugobjectvalue2-getvirtualmethodandtype-method.md)|`GetVirtualMethodAndType` no se ha implementado.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
>  Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
