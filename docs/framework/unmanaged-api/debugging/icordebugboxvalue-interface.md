# <a name="icordebugboxvalue-interface1"></a>ICorDebugBoxValue Interfaz1
Una subclase de "ICorDebugHeapValue" que representa un objeto de clase de valor con conversión boxing.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[GetObject (método)](../../../../docs/framework/unmanaged-api/debugging/icordebugboxvalue-getobject-method.md)|Obtiene un puntero de interfaz a la instancia de "ICorDebugObjectValue" conversión boxing.|  
  
## <a name="remarks"></a>Comentarios  
  
> [!NOTE]
>  Esta interfaz no admite que se la llame de forma remota, ya sea entre procesos o entre equipos.
