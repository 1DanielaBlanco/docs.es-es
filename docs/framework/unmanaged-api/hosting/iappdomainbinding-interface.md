# <a name="iappdomainbinding-interface"></a>IAppDomainBinding (Interfaz)
Proporciona un método invocado por common language runtime (CLR) para notificar a la aplicación host que se ha creado un dominio de aplicación.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[OnAppDomain (método)](../../../../docs/framework/unmanaged-api/hosting/iappdomainbinding-onappdomain-method.md)|Llamado por common language runtime (CLR) para notificar al host que se ha creado un dominio de aplicación.|
