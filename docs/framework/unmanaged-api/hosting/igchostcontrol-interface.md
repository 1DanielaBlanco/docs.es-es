# <a name="igchostcontrol-interface"></a>IGCHostControl (Interfaz)
Proporciona un método que permite que el recolector de elementos no utilizados solicitar el host para cambiar los límites de memoria virtual.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[RequestVirtualMemLimit (método)](../../../../docs/framework/unmanaged-api/hosting/igchostcontrol-requestvirtualmemlimit-method.md)|Solicita que el host para cambiar los límites de memoria virtual.|
