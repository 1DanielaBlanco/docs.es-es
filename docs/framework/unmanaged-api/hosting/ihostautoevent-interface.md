# <a name="ihostautoevent-interface"></a>IHostAutoEvent (Interfaz)
Proporciona una representación de la implementación del host de un evento de restablecimiento automático.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Set (método)](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-set-method.md)|Establece la actual `IHostAutoEvent` instancia en un estado señalado.|  
|[Wait (método)](../../../../docs/framework/unmanaged-api/hosting/ihostautoevent-wait-method.md)|Hace que el actual `IHostAutoEvent` instancia para esperar hasta que es propiedad de evento o una cantidad especificada de tiempo determinado.|
