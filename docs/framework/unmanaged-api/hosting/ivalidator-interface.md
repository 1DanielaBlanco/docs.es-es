# <a name="ivalidator-interface"></a>IValidator (Interfaz)
Proporciona métodos para validar imágenes portables de ejecutable (PE) e informar de errores de validación.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|Validar|Valida el archivo de lenguaje intermedio (MSIL) PE o Microsoft especificado.|  
|FormatEventInfo|Obtiene el mensaje de error correspondiente al error de validación especificado.|
