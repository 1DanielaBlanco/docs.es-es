# <a name="iassemblyenum-interface"></a>IAssemblyEnum (Interfaz)
Representa un enumerador para una matriz de `IAssemblyName` objetos.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[Clone (método)](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-clone-method.md)|Crea una copia superficial de este `IAssemblyEnum` objeto.|  
|[GetNextAssembly (método)](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-getnextassembly-method.md)|Obtiene un puntero a la siguiente `IAssemblyName` contenida en esta instancia `IAssemblyEnum` objeto.|  
|[Reset (método)](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-reset-method.md)|Restablece esta `IAssemblyEnum` objeto a su posición inicial.|
