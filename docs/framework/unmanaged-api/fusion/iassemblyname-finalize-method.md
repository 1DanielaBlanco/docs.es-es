# <a name="iassemblynamefinalize-method"></a>IAssemblyName::Finalize (Método)
Esto permite [IAssemblyName](../../../../docs/framework/unmanaged-api/fusion/iassemblyname-interface.md) objeto para liberar recursos y realizar otras operaciones de limpieza antes de llama a su destructor.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
HRESULT Finalize ();  
```
