# <a name="imetadataemitsavetomemory-method"></a>IMetaDataEmit::SaveToMemory (Método)
Guarda todos los metadatos en el ámbito actual en el área de memoria especificada.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
HRESULT SaveToMemory (   
    [out]  void        *pbData,   
    [in]   ULONG       cbData   
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `pbData`  
 [out] La dirección donde se comienzan a escribir metadatos.  
  
 `cbData`  
 [in] El tamaño, en bytes, de la memoria asignada.
