# <a name="imetadataemitsetfieldrva-method"></a>IMetaDataEmit::SetFieldRVA (Método)
Establece un valor de la variable global para la dirección virtual relativa del campo al que hace referencia el token especificado.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
HRESULT SetFieldRVA (   
    [in]  mdFieldDef  fd,   
    [in]  ULONG       ulRVA   
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `fd`  
 [in] El token para el campo de destino.  
  
 `ulRVA`  
 [in] La dirección de un área de código o datos.
