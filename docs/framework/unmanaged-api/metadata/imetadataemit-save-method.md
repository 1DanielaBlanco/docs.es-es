# <a name="imetadataemitsave-method"></a>IMetaDataEmit::Save (Método)
Guarda todos los metadatos en el ámbito actual en el archivo en la dirección especificada.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
HRESULT Save (   
    [in]  LPCWSTR     szFile,   
    [in]  DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `wzFile`  
 [in] El nombre del archivo que se guarde en. Si este valor es null, la copia en memoria se guardará a la última ubicación que se utilizó.  
  
 `dwSaveFlags`  
 [in] Reservado. Debe ser cero.
