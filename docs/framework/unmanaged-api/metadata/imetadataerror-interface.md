# <a name="imetadataerror-interface"></a>IMetaDataError (Interfaz)
Proporciona un mecanismo de devolución de llamada para notificar errores durante la combinación de metadatos.  
  
> [!NOTE]
>  El `IMetaDataError` debe implementar la interfaz por el cliente.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[OnError (método)](../../../../docs/framework/unmanaged-api/metadata/imetadataerror-onerror-method.md)|Proporciona una notificación de errores que se producen durante la combinación de metadatos.|
