# <a name="ibindingdisplay-interface"></a>IBindingDisplay (Interfaz)
Proporciona métodos que muestran la información de enlace actual sobre la aplicación en ejecución.  
  
## <a name="methods"></a>Métodos  
  
|Método|Descripción|  
|------------|-----------------|  
|[GetCurrentDisplay (método)](../../../../docs/framework/unmanaged-api/diagnostics/ibindingdisplay-getcurrentdisplay-method.md)|Devuelve la información de visualización de enlace actual.|  
|[InitializeForProcess (método)](../../../../docs/framework/unmanaged-api/diagnostics/ibindingdisplay-initializeforprocess-method.md)|Inicializa el `IBindingDisplay` objeto.|
