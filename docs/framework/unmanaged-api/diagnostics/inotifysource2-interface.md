|Método|Descripción|  
|------------|-----------------|  
|[SetNotifyFilter (método)](../../../../docs/framework/unmanaged-api/diagnostics/inotifysource2-setnotifyfilter-method.md)|Asigna un filtro de notificación para su uso con este origen.|
