# <a name="isymunmanageddocumentwritersetsource-method"></a>ISymUnmanagedDocumentWriter::SetSource (Método)
Conjuntos de código fuente incrustan para un documento que se está escribiendo.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
HRESULT SetSource(  
    [in]  ULONG32  sourceSize,  
    [in, size_is(sourceSize)] BYTE  source[]);  
```  
  
#### <a name="parameters"></a>Parámetros  
 `sourceSize`  
 [in] A `ULONG32` que contiene el tamaño de la `source` búfer.  
  
 `source`  
 [in] El búfer que almacena el código fuente incrustado.  
  
## <a name="return-value"></a>Valor devuelto  
 S_OK si el método tiene éxito; en caso contrario, E_FAIL u otro código de error.  
  
## <a name="requirements"></a>Requisitos  
 **Encabezado:** CorSym.idl, CorSym.h
