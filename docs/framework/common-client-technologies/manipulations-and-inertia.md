<span data-ttu-id="8b698-103">Las clases de procesador *manipulations* e *inertia* ayudan a la aplicación a mover elementos de la interfaz de usuario (UI) en la pantalla de manera natural e intuitiva.</span><span class="sxs-lookup"><span data-stu-id="8b698-103">The *manipulations* and *inertia* processor classes assist your application in moving user interface (UI) elements on the screen in a natural and intuitive way.</span></span> <span data-ttu-id="8b698-104">Las manipulaciones permiten a los usuarios mover, girar y cambiar el tamaño de los componentes mediante el uso de manipuladores, como un mouse, un lápiz o los dedos (en un escenario habilitado para toque directo).</span><span class="sxs-lookup"><span data-stu-id="8b698-104">Manipulations enable users to move, rotate, and resize components by using manipulators, such as a mouse device, a stylus, or (in a direct touch-enabled scenario) their fingers.</span></span> <span data-ttu-id="8b698-105">La *inercia* emula el comportamiento real de los elementos de la UI que están en movimiento, ya que simula fuerzas de fricción en los elementos.</span><span class="sxs-lookup"><span data-stu-id="8b698-105">*Inertia* emulates real-world behavior for UI elements that are in motion by simulating friction forces on the elements.</span></span> <span data-ttu-id="8b698-106">Esto permite que el movimiento (lineal y angular) de los elementos se ralentice gradualmente antes de detenerse.</span><span class="sxs-lookup"><span data-stu-id="8b698-106">This enables elements to gradually slow their movement (both linear and angular) before coming to a stop.</span></span>
