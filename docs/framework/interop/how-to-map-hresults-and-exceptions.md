### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="71b1e-114">Para crear una nueva clase de excepción y asignarla a un valor HRESULT</span><span class="sxs-lookup"><span data-stu-id="71b1e-114">To create a new exception class and map it to an HRESULT</span></span>  
  
1.  <span data-ttu-id="71b1e-115">Use el código siguiente para crear una nueva clase de excepción denominada `NoAccessException` y asignarla al valor HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="71b1e-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>

    ```cpp  
        NoAccessException () {  
        HResult = E_ACCESSDENIED;

    {  
    throw new NoAccessException();  
    }  
    ```
