 <span data-ttu-id="b621b-106">Las clases de excepción definidas por el usuario pueden especificar el valor HRESULT que sea necesario.</span><span class="sxs-lookup"><span data-stu-id="b621b-106">User-defined exception classes can specify whatever HRESULT is appropriate.</span></span> <span data-ttu-id="b621b-107">Estas clases de excepción pueden cambiar dinámicamente el valor HRESULT que se devuelve cuando se genera la excepción si se establece el campo **HResult** en el objeto de excepción.</span><span class="sxs-lookup"><span data-stu-id="b621b-107">These exception classes can dynamically change the HRESULT to be returned when the exception is generated by setting the **HResult** field on the exception object.</span></span> <span data-ttu-id="b621b-108">Se proporciona información adicional sobre la excepción al cliente a través de la interfaz **IErrorInfo**, que se implementa en el objeto .NET en el proceso no administrado.</span><span class="sxs-lookup"><span data-stu-id="b621b-108">Additional information about the exception is provided to the client through the **IErrorInfo** interface, which is implemented on the .NET object in the unmanaged process.</span></span>  
  
 <span data-ttu-id="b621b-109">Si crea una clase que extienda **System.Exception**, debe establecer el campo HRESULT durante la construcción.</span><span class="sxs-lookup"><span data-stu-id="b621b-109">If you create a class that extends **System.Exception**, you must set the HRESULT field during construction.</span></span> <span data-ttu-id="b621b-110">En caso contrario, la clase base asigna el valor HRESULT.</span><span class="sxs-lookup"><span data-stu-id="b621b-110">Otherwise, the base class assigns the HRESULT value.</span></span> <span data-ttu-id="b621b-111">Puede asignar nuevas clases de excepción a un valor HRESULT existente proporcionando el valor en el constructor de la excepción.</span><span class="sxs-lookup"><span data-stu-id="b621b-111">You can map new exception classes to an existing HRESULT by supplying the value in the exception's constructor.</span></span>

### <a name="to-create-a-new-exception-class-and-map-it-to-an-hresult"></a><span data-ttu-id="b621b-114">Para crear una nueva clase de excepción y asignarla a un valor HRESULT</span><span class="sxs-lookup"><span data-stu-id="b621b-114">To create a new exception class and map it to an HRESULT</span></span>  
  
1.  <span data-ttu-id="b621b-115">Use el código siguiente para crear una nueva clase de excepción denominada `NoAccessException` y asignarla al valor HRESULT `E_ACCESSDENIED`.</span><span class="sxs-lookup"><span data-stu-id="b621b-115">Use the following code to create a new exception class called `NoAccessException` and map it to the HRESULT `E_ACCESSDENIED`.</span></span>  
  
    ```cpp  
    Class NoAccessException : public ApplicationException  
    {  
        NoAccessException () {  
        HResult = E_ACCESSDENIED;   
    }  
    }  
    CMyClass::MethodThatThrows  
    {  
    throw new NoAccessException();  
    }  
    ```
