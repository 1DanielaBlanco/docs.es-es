# <a name="windows-workflow-foundation-configuration-schema"></a>Esquema de configuración de Windows Workflow Foundation
Elementos de configuración de Windows Workflow Foundation (WF) le permiten configurar las aplicaciones de flujo de trabajo. Para una aplicación de flujo de trabajo puede configurar, entre otras cosas, el seguimiento y la traza. Para obtener más información sobre el seguimiento y la traza, vea [Workflow Tracking and Tracing](../../../../../docs/framework/windows-workflow-foundation/workflow-tracking-and-tracing.md) (Seguimiento y traza del flujo de trabajo). Para los servicios de flujo de trabajo, también puede utilizar elementos de configuración de Windows Communication Foundation (WCF). Para obtener más información sobre [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)], vea [WCF Configuration Schema](../../../../../docs/framework/configure-apps/file-schema/wcf/index.md) (Esquema de configuración de WCF).  
  
 Puesto que los archivos de configuración tienen el formato XML, debe familiarizarse con el formato XML si desea modificarlos manualmente con un editor de texto. De lo contrario, puede encontrarse con problemas como no encontrar un atributo o una etiqueta de elemento XML. Esto se debe a que los atributos y las etiquetas del elemento XML distinguen entre mayúsculas y minúsculas.
