# Esquema de secciones de configuraci&#243;n
El esquema de secciones de configuración contiene elementos que colocan valores personalizados en los archivos de configuración.  
  
 [\<configuration\>](../../../../docs/framework/configure-apps/file-schema/configuration-element.md)  
  
 [\<configSections\>](http://msdn.microsoft.com/es-es/8a5cbc84-0257-4c2e-80a9-a064fe7c896b)  
  
 [\<clear\>](http://msdn.microsoft.com/es-es/b0ce3c91-89fa-4bb5-a995-30e21a61036b)  
  
 [\<remove\>](http://msdn.microsoft.com/es-es/6e2b2d45-113d-4d10-9f8e-0b15fc86bb98)  
  
 [\<sección\>](http://msdn.microsoft.com/es-es/51c9d658-c275-4ab2-aa66-c5d5b5289570)  
  
 [\<sectionGroup\>](http://msdn.microsoft.com/es-es/45fbeb5c-26c7-4c68-b732-b3f1e62c53bf)  
  
|Elemento|Descripción|  
|--------------|-----------------|  
|[Elemento clear para configSections \(Esquema de configuración general\)](http://msdn.microsoft.com/es-es/b0ce3c91-89fa-4bb5-a995-30e21a61036b)|Borra todas las secciones y grupos de secciones definidas anteriormente.|  
|[Elemento configSections \(Esquema de configuración general\)](http://msdn.microsoft.com/es-es/8a5cbc84-0257-4c2e-80a9-a064fe7c896b)|Contiene las declaraciones de las secciones de configuración y del espacio de nombres.|  
|[Elemento remove para configSections \(Esquema de configuración general\)](http://msdn.microsoft.com/es-es/6e2b2d45-113d-4d10-9f8e-0b15fc86bb98)|Quita una sección o grupo de secciones predefinidas.|  
|[Elemento section para configSections \(Esquema de configuración general\)](http://msdn.microsoft.com/es-es/51c9d658-c275-4ab2-aa66-c5d5b5289570)|Contiene una declaración de la sección de configuración.|  
|[Elemento sectionGroup para configSections \(Esquema de configuración general\)](http://msdn.microsoft.com/es-es/45fbeb5c-26c7-4c68-b732-b3f1e62c53bf)|Define el espacio de nombres de las secciones de configuración.|
