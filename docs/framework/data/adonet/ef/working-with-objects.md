# <a name="working-with-objects"></a>Trabajar con objetos
[!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] permite consultar, insertar, actualizar y eliminar datos, que se expresan como objetos CLR (Common Language Runtime) con establecimiento de tipos que son instancias de tipos de entidad. Los tipos de entidad representan las entidades definidas en el modelo conceptual. [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] asigna las entidades y las relaciones que se definen en un modelo conceptual a un origen de datos. El [!INCLUDE[adonet_ef](../../../../../includes/adonet-ef-md.md)] ofrece los medios para hacer lo siguiente: materializar datos devueltos desde el origen de datos como objetos; seguimiento de cambios que se realizaron en los objetos; controlar la simultaneidad; propagar los cambios de objeto en el origen de datos; y enlazar objetos a controles.  
  
 Para obtener más información sobre cómo trabajar con objetos en la versión más reciente de Entity Framework Vea [trabajar con objetos](http://go.microsoft.com/fwlink/?LinkId=235289).
