# Descargar bases de datos de ejemplo
En varios ejemplos y tutoriales de la documentación de [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)] se utiliza la base de datos de ejemplo Northwind y SQL Server Express Edition.  Puede descargar estos productos de forma gratuita desde el sitio de descargas de Microsoft.  
  
## Descargar la base de datos Northwind  
  
#### Para descargar e instalar la base de datos de ejemplo Northwind para SQL Server  
  
1.  Inicie Internet Explorer.  
  
2.  Vaya al sitio web de las [bases de datos de ejemplo Northwind y Pubs](http://go.microsoft.com/fwlink?linkid=64296).  
  
3.  Haga clic en **Descargar**.  
  
4.  En el cuadro de diálogo **Descarga de archivos**, seleccione **Guardar**.  
  
5.  Una vez que el archivo se haya descargado, haga doble clic en el archivo **Nwind.exe** para instalar la base de datos.  
  
     De forma predeterminada, la base de datos se instala en *unidad*:\\SQL Server 2000 Sample Databases.  
  
## Descargar SQL Server Express Edition  
 SQL Server Express Edition está disponible gratuitamente y puede redistribuirse con aplicaciones.  Si utiliza [!INCLUDE[vsprvs](../../../../../../includes/vsprvs-md.md)], SQL Server Express Edition se incluye en las ediciones Professional y superiores.  
  
#### Para descargar e instalar SQL Server Express Edition  
  
1.  Inicie Internet Explorer.  
  
2.  Vaya al sitio web del [Centro de descarga de Microsoft](http://go.microsoft.com/fwlink?linkid=74602).  
  
3.  En el campo de palabras clave, escriba SQL Server Express.  
  
4.  Haga clic en **Ir**.  
  
5.  En la página de resultados, haga clic en el vínculo de la página de descarga de **Microsoft SQL Server 2005 Express Edition**.  
  
6.  Siga las instrucciones de instalación del sitio web.  
  
## Descargar Microsoft SQL Server Management Studio Express  
 Si desea modificar una base de datos que ha descargado, puede tener acceso a la base de datos desde el **Explorador de servidores** en el entorno de desarrollo integrado \(IDE\) de [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)] o utilizar Microsoft SQL Server Management Studio Express \(SSMSE\).  
  
#### Para descargar Microsoft SQL Server Management Studio Express  
  
-   Siga las instrucciones que encontrará en el [sitio SSMSE](http://go.microsoft.com/fwlink/?LinkId=95933).
