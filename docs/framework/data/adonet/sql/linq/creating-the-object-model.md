# Crear el modelo de objetos
Puede crear un modelo de objetos a partir de una base de datos existente y utilizar el modelo en su estado predeterminado.  También puede personalizar muchos aspectos del modelo y su comportamiento.  
  
 Si utiliza [!INCLUDE[vs_current_short](../../../../../../includes/vs-current-short-md.md)], puede usar el [!INCLUDE[vs_ordesigner_long](../../../../../../includes/vs-ordesigner-long-md.md)] para crear el modelo de objetos.
