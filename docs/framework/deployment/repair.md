# <a name="repair-the-net-framework"></a>Reparar .NET Framework

En algunas situaciones, la instalación de .NET Framework puede resultar dañada y debe repararse. Este puede ser el caso si la aplicación se bloquea justo después de intentar iniciarla o si no puede instalar versiones más recientes de .NET Framework.

Puede reparar la instalación de .NET Framework con la [herramienta de reparación de .NET Framework](http://download.microsoft.com/download/2/B/D/2BDE5459-2225-48B8-830C-AE19CAF038F1/NetFxRepairTool.exe). 

Si la aplicación sigue sin iniciarse después de reparar .NET Framework, puede que el problema sea de la aplicación. En ese caso, debe ponerse en contacto con el fabricante de la aplicación.
