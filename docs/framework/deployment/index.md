# Implementar .NET Framework
Esta sección de la documentación de .NET Framework ofrece información a los desarrolladores que deseen instalar .NET Framework con sus aplicaciones, y a los administradores que deseen implementar .NET Framework a través de una red.  También describe los problemas de activación y reinicio asociados a la implementación, y explica cómo supervisar el progreso de la instalación de .NET Framework.  
  
## En esta sección  
 [Guía de implementación para desarrolladores](../../../docs/framework/deployment/deployment-guide-for-developers.md)  
 Explica cómo los desarrolladores de software pueden instalar .NET Framework en los equipos de los usuarios con sus aplicaciones.  
  
 [Guía de implementación para administradores](../../../docs/framework/deployment/guide-for-administrators.md)  
 Explica cómo un administrador del sistema puede implementar .NET Framework y sus dependencias del sistema en una red utilizando System Center Configuration Manager \(SCCM\).  
  
 [Reducir los reinicios del sistema durante las instalaciones de .NET Framework 4.5](../../../docs/framework/deployment/reducing-system-restarts.md)  
 Describe el Administrador de reinicio, que evita, en la medida de lo posible, los reinicios del equipo y explica cómo pueden aprovecharlo las aplicaciones que instalan .NET Framework.  
  
 [Cómo: Obtener el progreso del instalador de .NET Framework 4.5](../../../docs/framework/deployment/how-to-get-progress-from-the-dotnet-installer.md)  
 Describe cómo iniciar y seguir en modo silencioso el proceso de instalación de .NET Framework, mientras se muestra la vista del progreso de la instalación.  
  
 [Errores de inicialización de .NET Framework: Administrar la experiencia del usuario](../../../docs/framework/deployment/initialization-errors-managing-the-user-experience.md)  
 Explica qué ocurre cuando una aplicación de .NET Framework requiere una versión de CLR ya expirada o que no está instalada en el equipo del usuario, cómo resolver estos errores, y cómo controlar el mensaje de error que se le muestra al usuario.  
  
 [Cómo: Depurar problemas de activación de CLR](../../../docs/framework/deployment/how-to-debug-clr-activation-issues.md)  
 Explica cómo puede ver y depurar registros de activación de CLR para solucionar los problemas que puede encontrarse al intentar ejecutar la aplicación con la versión correcta de CLR.
