# Seguridad de transporte
La seguridad de transporte en [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] depende del enlace seleccionado.El transporte que el enlace implementa determina el mecanismo de seguridad real.Los temas de esta sección explican los mecanismos que se implementan y sus opciones.
