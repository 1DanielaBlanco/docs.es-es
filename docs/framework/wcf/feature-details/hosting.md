# Hospedaje
Los temas de esta sección describen el hospedaje de servicios.Internet Information Services \(IIS\), el Servicio de activación de procesos de Windows \(WAS\), Windows Server AppFabric, un servicio de Windows o una aplicación administrada pueden hospedar un servicio \(a menudo, esta opción se conoce como *autohospedaje*\).  
  
 Es importante tener en cuenta que ejecutar un servicio o cualquier extensión desde un host que no sea de confianza pone en peligro la seguridad.
