# Requisitos del sistema WCF
[!INCLUDE[indigo1](../../../includes/indigo1-md.md)] es una infraestructura de comunicación que se usa para crear aplicaciones distribuidas.  A continuación se detallan los requisitos que permiten [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] ejecutarse.  
  
## Requisitos del sistema  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] está instalado de manera predeterminada en [!INCLUDE[wv](../../../includes/wv-md.md)].  
  
 [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] también se pueden instalar en [!INCLUDE[wxpsp2](../../../includes/wxpsp2-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)]o [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)].  
  
 **Nota**:   La funcionalidad de Message Queuing \(MSMQ\) de [!INCLUDE[indigo2](../../../includes/indigo2-md.md)] solo se admite en [!INCLUDE[wv](../../../includes/wv-md.md)], [!INCLUDE[ws2003r2](../../../includes/ws2003r2-md.md)], [!INCLUDE[ws2003sp1](../../../includes/ws2003sp1-md.md)] y [!INCLUDE[wxppro](../../../includes/wxppro-md.md)].
