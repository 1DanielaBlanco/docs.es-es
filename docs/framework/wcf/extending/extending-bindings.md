# Extensi&#243;n de enlaces
Los enlaces especifican el transporte, codificación y protocolo requeridos para conectar a un extremo.Las extensiones de enlace y los enlaces personalizados implementan la funcionalidad de la comunicación personalizada exigida para admitir las características de la aplicación.
