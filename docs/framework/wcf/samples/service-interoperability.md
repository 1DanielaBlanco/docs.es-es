# interoperabilidad del servicio
Esta sección contiene ejemplos que muestran la interacción entre [!INCLUDE[indigo1](../../../../includes/indigo1-md.md)] y otras tecnologías de servicios.  
  
## En esta sección  
 [Uso del moniker de WCF con clientes COM](../../../../docs/framework/wcf/samples/using-the-wcf-moniker-with-com-clients.md)  
 Muestra cómo utilizar el moniker de servicio de [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] para integrar los servicios Web en entornos de desarrollo basados en COM, como Microsoft Office Visual Basic para Aplicaciones \(Office VBA\) o Visual Basic 6.0.  
  
 [Cliente ASMX con un servicio WCF](../../../../docs/framework/wcf/samples/asmx-client-with-a-wcf-service.md)  
 Muestra cómo crear un servicio mediante [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)] y después se tiene acceso al servicio desde un cliente que no sea de [!INCLUDE[indigo2](../../../../includes/indigo2-md.md)], como un cliente ASMX.
