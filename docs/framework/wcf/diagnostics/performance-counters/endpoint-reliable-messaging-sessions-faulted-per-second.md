# Extremo: errores en las sesiones de mensajer&#237;a de confianza por segundo
Nombre de contador: errores en las sesiones de mensajería de confianza por segundo.  
  
## Descripción  
 Número de sesiones de mensajería de confianza fallidas en este extremo en un segundo.  
  
 Este es un tipo de contador de rendimiento [PERF\_COUNTER\_COUNTER](http://go.microsoft.com/fwlink/?LinkID=94649), cuyo valor se calcula con la siguiente fórmula.  
  
 \(N 1 \- N 0 \) \/ \( \(D 1 \-D 0 \) \/ F\)
