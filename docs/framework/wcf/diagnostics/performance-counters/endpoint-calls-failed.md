# <a name="endpoint-calls-failed"></a>punto de conexión: Errores de llamadas
Nombre de contador: Errores de llamadas  
  
## <a name="description"></a>Descripción  
 Número de llamadas con excepciones no atendidas en este extremo. Este contador se incrementa siempre que se produce una excepción no controlada en este punto de conexión.
