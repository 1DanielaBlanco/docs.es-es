# Duraci&#243;n de la llamada
Nombre de contador: duración de la llamada  
  
## Descripción  
 La duración media de las llamadas a este operación.  La duración media se calcula aplicando esta ecuación: \(N1\-N0\)\/\(D1\-D0\).  
  
> [!WARNING]
>  Cuando se utiliza en un servicio WCF asincrónico, el contador de duración de la llamada siempre devuelve \-1.
