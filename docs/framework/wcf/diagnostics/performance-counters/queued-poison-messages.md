# Mensajes dudosos en cola
Nombre de contador: Mensajes dudosos en cola.  
  
## Descripción  
 El número de mensajes que son marcados como dudosos por el transporte en cola en este servicio.  
  
 Vea [Control de mensajes dudosos](http://go.microsoft.com/fwlink/?LinkID=96233) para obtener más información sobre cuándo los mensajes se marcan como dudosos.
