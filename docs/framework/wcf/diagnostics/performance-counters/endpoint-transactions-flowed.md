# Extremo: Flujo de transacciones
Nombre de contador: Flujo de transacciones.  
  
## Descripción  
 El número de transacciones que han fluido a las operaciones en este extremo.Este contador se incrementa siempre que se encuentra un id. de transacción en un mensaje enviado al extremo.
