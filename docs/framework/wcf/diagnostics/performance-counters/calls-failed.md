# <a name="calls-failed"></a>Errores de llamadas
Nombre del contador: Errores de llamadas  
  
## <a name="description"></a>Descripción  
 Número de llamadas con excepciones no atendidas en esta operación. El contador se incrementa siempre que se produce una excepción no controlada en esta operación.
