# Extremo: mensajes de mensajer&#237;a de confianza quitados por segundo
Nombre de contador: sesiones de mensajería de confianza quitadas por segundo.  
  
## Descripción  
 Número total de mensajes de mensajería de confianza quitados en este extremo en un segundo.  
  
 Este es un tipo de contador de rendimiento [PERF\_COUNTER\_COUNTER](http://go.microsoft.com/fwlink/?LinkID=94649), cuyo valor se calcula con la siguiente fórmula.  
  
 \(N 1 \- N 0 \) \/ \( \(D 1 \-D 0 \) \/ F\)
