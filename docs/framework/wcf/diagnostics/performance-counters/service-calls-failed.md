# <a name="service-calls-failed"></a>Servicio: Errores de llamadas
Nombre de contador: Errores de llamadas  
  
## <a name="description"></a>Descripción  
 Número de llamadas con excepciones no atendidas en este servicio. El contador se incrementa siempre que se produce una excepción no controlada en este servicio.
