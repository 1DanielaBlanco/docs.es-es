# Operaciones de transacci&#243;n confirmadas por segundo
Nombre del contador: Operaciones de transacción confirmadas por segundo.  
  
## Descripción  
 Número de operaciones transaccionales confirmadas en este servicio cada segundo.  
  
 Este contador es un tipo de contador de rendimiento [PERF\_COUNTER\_COUNTER](http://go.microsoft.com/fwlink/?LinkID=94649), cuyo valor se calcula mediante la fórmula siguiente.  
  
 \(N 1 \- N 0 \) \/ \( \(D 1 \-D 0 \) \/ F\)
