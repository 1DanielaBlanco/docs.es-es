# Servicio: duraci&#243;n de las llamadas
Nombre de contador: duración de la llamada  
  
## Descripción  
 La duración media de las llamadas a este servicio.  La duración media se calcula aplicando esta ecuación: \(N1\-N0\)\/\(D1\-D0\).
