# WmiPropertyMissing
Id: 129  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## Descripción  
 Este evento indica que hay un error de coincidencia en el tipo.La clase CIM especificada no cuenta con la propiedad concreta del tipo específico.El evento enumera el nombre y el id. del proceso.
