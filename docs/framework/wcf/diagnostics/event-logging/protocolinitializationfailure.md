# ProtocolInitializationFailure
Id: 142  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que el servicio de protocolo WS\-Atomic Transaction  no se inicializó.Como resultado, la funcionalidad WS\-AtomicTransaction está deshabilitada.El evento enumera el identificador de protocolo, el nombre de protocolo, la excepción, el nombre del proceso y el identificador del proceso.
