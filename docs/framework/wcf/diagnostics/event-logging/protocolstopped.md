# <a name="protocolstopped"></a>ProtocolStopped
Id: 151  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  
 Este evento indica que se detuvo el servicio de protocolo WS-Atomic Transaction. El evento indica el identificador de protocolo, el nombre del protocolo, el nombre del proceso y el identificador de proceso. .
