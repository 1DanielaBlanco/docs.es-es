# ProtocolStopped
Id: 151  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que se detuvo el servicio de protocolo WS\-Atomic Transaction.  El evento enumera elId. del protocolo, el nombre del protocolo, el nombre de proceso y el id. de proceso.
