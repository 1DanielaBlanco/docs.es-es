# <a name="participantrecoverylogentrycreationfailure"></a>ParticipantRecoveryLogEntryCreationFailure
Id.: 141  
  
 Gravedad: Error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  
 Este evento indica que no se pudo generar una entrada de registro de recuperación para una inscripción de participante. Se anulará la transacción. El evento enumera el id. de transacción, el id. de inscripción, la excepción, el nombre del proceso y el id. de éste.
