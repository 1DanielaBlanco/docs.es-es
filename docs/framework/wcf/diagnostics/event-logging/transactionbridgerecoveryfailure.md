# TransactionBridgeRecoveryFailure
Id: 146  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que se produjo un error en el MSDTC TransactionBridge durante la recuperación.Se trata de una condición grave, por lo que el servicio de MSDTC ha finalizado.El evento enumera la excepción, el nombre del proceso y el identificador del mismo.
