# ProtocolRecoveryBeginningFailure
Id: 144  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que se produjo un error en el protocolo de transacción WS\-Atomic de MSDTC al comienzo de la recuperación.Como resultado, la funcionalidad WS\-AtomicTransaction está deshabilitada.El evento enumera el id. de protocolo, el nombre de protocolo, la excepción, el nombre del proceso y el id. de proceso.
