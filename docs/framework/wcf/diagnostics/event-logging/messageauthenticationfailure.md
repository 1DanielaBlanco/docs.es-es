# MessageAuthenticationFailure
Id: 170  
  
 Gravedad: error  
  
 Categoría: SecurityAudit  
  
## Descripción  
 Este evento indica que se produce un error en la autenticación de mensajes.  El evento hace una lista del servicio, acción, identidad del cliente e id. de actividad.
