# ProtocolRecoveryCompleteFailure
Id: 145  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que el servicio del protocolo de transacción WS\-Atomic no puedo completar el inicio ni la recuperación.Como resultado, la funcionalidad WS\-AtomicTransaction está deshabilitada.El evento enumera el id. de protocolo, el nombre de protocolo, la excepción, el nombre del proceso y el id. de proceso.
