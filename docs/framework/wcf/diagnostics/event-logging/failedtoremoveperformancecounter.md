# <a name="failedtoremoveperformancecounter"></a>FailedToRemovePerformanceCounter
Id: 119  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  
 Este evento indica que no se quitó un contador de rendimiento. El evento enumera el nombre de la categoría, el nombre del contador, el nombre de la excepción, el nombre del proceso y el id. de proceso.
