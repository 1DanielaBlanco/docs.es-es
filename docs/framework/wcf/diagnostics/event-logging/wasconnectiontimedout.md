# WasConnectionTimedout
Id: 163  
  
 Gravedad: error  
  
 Categoría: SMSvcHost  
  
## Descripción  
 Este evento indica a la conexión A que el Servicio de activación de procesos de Windows \(WAS\) requiere tiempo de espera, haciendo una lista del nombre e id. del proceso.
