# <a name="complusservicehoststartingserviceerror"></a>ComPlusServiceHostStartingServiceError
Id.: 130  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  
 Este evento indica que se produjo un error al iniciar el servicio COM+. El evento muestra una lista de AppID, ClsID, suplente, excepción, nombre e ID de proceso.
