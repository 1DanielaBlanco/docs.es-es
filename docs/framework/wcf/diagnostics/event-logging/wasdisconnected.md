# WasDisconnected
Id: 162  
  
 Gravedad: error  
  
 Categoría: SMSvcHost  
  
## Descripción  
 Este evento indica que Windows Process Activation \(WAS\) se desconectó.El evento enumera HRESULT, el nombre del proceso y el id. del proceso.
