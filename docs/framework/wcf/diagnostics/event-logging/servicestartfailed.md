# ServiceStartFailed
Id: 164  
  
 Gravedad: error  
  
 Categoría: SMSvcHost  
  
## Descripción  
 Este evento indica que no se pudo cumplir la solicitud de iniciar el servicio.El evento enumera el código de error, el nombre del proceso y el Id. del proceso.
