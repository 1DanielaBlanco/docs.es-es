# <a name="wmiexecqueryfailed"></a>WmiExecQueryFailed
Id.: 124  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  
 Este evento indica que no se procesó la solicitud ExecQuery de WMI. El evento enumera la excepción, el nombre del proceso y el id. del mismo.
