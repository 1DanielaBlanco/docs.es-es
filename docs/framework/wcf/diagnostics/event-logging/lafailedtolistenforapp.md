# <a name="lafailedtolistenforapp"></a>LAFailedToListenForApp
Id.: 160  
  
 Gravedad: error  
  
 Categoría: SMSvcHost  
  
## <a name="description"></a>Descripción  
 Este evento indica que se produjo un error en el Servicio de activación especificado del protocolo especificado, al intentar realizar escuchas para el sitio especificado, por lo que el protocolo está deshabilitado para el sitio temporalmente. El evento hace una lista de la dirección URL, el estado, la excepción, el nombre de proceso y la id. de proceso.
