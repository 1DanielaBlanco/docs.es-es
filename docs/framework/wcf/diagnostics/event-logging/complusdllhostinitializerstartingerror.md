# ComPlusDllHostInitializerStartingError
Id: 131  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## Descripción  
 Este evento indica que se produjo un error al iniciar el inicializador de DllHost.El evento enumera AppID, la excepción, el nombre del proceso y el id. del mismo.
