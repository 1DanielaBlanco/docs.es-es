# MessageQueueDuplicatedSocketLeak
Id: 165  
  
 Gravedad: error  
  
 Categoría: SMSvcHost  
  
## Descripción  
 Este evento indica que se produjo un error al enviar un socket duplicado.Este identificador se perdió en el proceso.El evento enumera la fuente, la excepción, el nombre de proceso y el id. de proceso.
