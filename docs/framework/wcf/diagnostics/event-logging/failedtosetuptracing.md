# FailedToSetupTracing
Id: 100  
  
 Gravedad: error  
  
 Categoría: seguimiento  
  
## Descripción  
 Se ha producido un error en un intento de configuración de seguimiento.  Seguimiento deshabilitado.  
  
 El evento enumera la excepción, el nombre del proceso y el id. del mismo.
