# SslNoAccessiblePrivateKey
Id: 155  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que un certificado de identidad con el nombre de sujeto concreto y la huella digital no tiene una clave privada accesible.El evento enumera el nombre y el id. del proceso.
