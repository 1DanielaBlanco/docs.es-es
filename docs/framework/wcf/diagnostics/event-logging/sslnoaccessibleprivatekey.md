# <a name="sslnoaccessibleprivatekey"></a>SslNoAccessiblePrivateKey
Id.: 155  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## <a name="description"></a>Descripción  
 Este evento indica que un certificado de identidad con el nombre de sujeto concreto y la huella digital no tiene una clave privada accesible. El evento enumera el nombre y el id. de proceso.
