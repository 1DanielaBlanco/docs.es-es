# WmiPutInstanceFailed
Id: 121  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## Descripción  
 Este evento indica que no se procesó la solicitud PutInstance de WMI.El evento enumera el objeto WMI, excepción, nombre del proceso e id. del proceso.
