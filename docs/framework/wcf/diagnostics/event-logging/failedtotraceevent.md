# <a name="failedtotraceevent"></a>FailedToTraceEvent
Id.: 104  
  
 Gravedad: error  
  
 Categoría: seguimiento  
  
## <a name="description"></a>Descripción  
 No se hizo el seguimiento de ningún evento o eventos. El evento enumera la cadena de evento original, el nombre del proceso y el id. de proceso.
