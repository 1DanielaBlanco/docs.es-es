# WmiDeleteInstanceFailed
Id: 122  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## Descripción  
 Este evento indica que no se procesó la solicitud DeleteInstance de WMI.El evento enumera la excepción, el nombre del proceso y el id. de proceso.
