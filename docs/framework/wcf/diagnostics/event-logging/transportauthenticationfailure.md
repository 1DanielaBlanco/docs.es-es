# TransportAuthenticationFailure
Id: 174  
  
 Gravedad: error  
  
 Categoría: SecurityAudit  
  
## Descripción  
 Este evento indica que se ha producido un error en la autenticación del transporte y detalla la identidad del cliente e id. de actividad.
