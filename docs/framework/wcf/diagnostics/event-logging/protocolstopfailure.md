# ProtocolStopFailure
Id: 147  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que el servicio de protocolo WS\-Atomic Transaction no se pudo detenerse.El evento enumera el id. de protocolo, el nombre de protocolo, la excepción, el nombre del proceso y el id. del proceso.
