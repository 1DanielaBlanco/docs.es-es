# UnknownListenerAdapterError
Id: 161  
  
 Gravedad: error  
  
 Categoría: SMSvcHost  
  
## Descripción  
 Este evento indica que se produjo un error inesperado en el adaptador del agente de escucha al controlar una notificación del servicio de activación de procesos de Windows \(WAS\).  El proceso ha finalizando.  El evento enumera el protocolo, la excepción, el nombre del protocolo, y el id. de proceso.
