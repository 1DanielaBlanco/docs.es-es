# ProtocolRecoveryComplete
Id: 150  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que el servicio del protocolo de transacción WS\-Atomic completó con éxito el inicio y la recuperación.El evento enumera elId. del protocolo, el nombre del protocolo, el nombre de proceso y el id. de proceso.
