# <a name="piiloggingon"></a>PiiLoggingOn
Id.: 107  
  
 Gravedad: error  
  
 Categoría: seguimiento  
  
## <a name="description"></a>Descripción  
 Este evento indica que se ha activado el registro de PII. Se registrará la información confidencial. El evento enumera el nombre y el id. de proceso.
