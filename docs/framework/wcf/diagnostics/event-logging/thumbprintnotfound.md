# ThumbPrintNotFound
Id: 152  
  
 Gravedad: error  
  
 Categoría: TransactionBridge  
  
## Descripción  
 Este evento indica que no se pudo encontrar un certificado de identidad con la huella digital específica.El evento enumera el nombre y el id. del proceso.
