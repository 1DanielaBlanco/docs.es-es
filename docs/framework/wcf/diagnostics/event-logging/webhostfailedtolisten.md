# WebHostFailedToListen
Id: 112  
  
 Gravedad: error  
  
 Categoría: ServiceModel  
  
## Descripción  
 Este evento indica que se produjo un error intentando realizar escuchas en la URL concreta.Se finaliza este proceso de trabajo.El evento enumera la excepción, el nombre del proceso y el id. del proceso.
