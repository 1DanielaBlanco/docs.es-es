# StartErrorPublish
Id: 158  
  
 Gravedad: error  
  
 Categoría: SMSvcHost  
  
## Descripción  
 Este evento indica que el servicio NT no se inició.  No se pudo publicar el extremo de escucha.  El evento enumera el nombre y el id. de proceso.
