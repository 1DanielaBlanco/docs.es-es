# <a name="failedtocreatemessageloggingtracesource"></a>FailedToCreateMessageLoggingTraceSource
Id.: 115  
  
 Gravedad: Error  
  
 Categoría: ServiceModel  
  
## <a name="description"></a>Descripción  
 Este evento indica que el registro de mensajes no pudo crear el origen de seguimiento. El evento enumera la excepción, el nombre del proceso y el id. del mismo.
