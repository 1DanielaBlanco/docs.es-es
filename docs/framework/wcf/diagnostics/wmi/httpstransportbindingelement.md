# <a name="httpstransportbindingelement"></a>HttpsTransportBindingElement
HttpsTransportBindingElement  
  
## <a name="syntax"></a>Sintaxis  
  
```  
class HttpsTransportBindingElement : HttpTransportBindingElement  
{  
  boolean RequireClientCertificate;  
};  
```  
  
## <a name="methods"></a>Métodos  
 La clase HttpsTransportBindingElement no define ningún método.  
  
## <a name="properties"></a>Propiedades  
 La clase HttpsTransportBindingElement tiene la siguiente propiedad:  
  
### <a name="requireclientcertificate"></a>RequireClientCertificate  
 Tipo de datos: booleano  
  
 Tipo de acceso: solo lectura  
  
 Un valor que indica si se requiere o no la autenticación de cliente SSL.
