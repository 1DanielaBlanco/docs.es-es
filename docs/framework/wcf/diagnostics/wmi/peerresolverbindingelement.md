# <a name="peerresolverbindingelement"></a>PeerResolverBindingElement
PeerResolverBindingElement  
  
## <a name="syntax"></a>Sintaxis  
  
```  
class PeerResolverBindingElement : BindingElement  
{  
  string ReferralPolicy;  
};  
```  
  
## <a name="methods"></a>Métodos  
 La clase PeerResolverBindingElement no define ningún método.  
  
## <a name="properties"></a>Propiedades  
 La clase PeerResolverBindingElement tiene la propiedad siguiente:  
  
### <a name="referralpolicy"></a>ReferralPolicy  
 Tipo de datos: cadena  
  
 Tipo de acceso: solo lectura  
  
 Determina cómo se comparten las referencias entre iguales.
