# <a name="4817---innerchannelcreationfailed"></a>4817 - InnerChannelCreationFailed
## <a name="properties"></a>Propiedades  
  
|||  
|-|-|  
|Id.|4817|  
|Palabras clave|Descubrimiento|  
|Nivel|Advertencia|  
|Canal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Descripción  
 El evento se genera cuando DiscoveryClientChannel no pudo crear el canal con un extremo detectado. DiscoveryClientChannel ahora intentará usar el siguiente extremo detectable disponible.  
  
## <a name="message"></a>Mensaje  
 DiscoveryClientChannel no pudo crear el canal con un extremo detectado con EndpointAddress='%1' y Via='%2'. DiscoveryClientChannel ahora intentará usar el siguiente extremo detectable disponible.
