# 3825 - RoutingServiceAbandoningReceiveContext
## Propiedades  
  
|||  
|-|-|  
|Id.|3825|  
|Palabras clave|RoutingServices|  
|Nivel|Advertencia|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Se genera este evento cuando el servicio de enrutamiento llama a ReceiveContext.Abandon para un mensaje con un id. especificado.  
  
## Mensaje  
 El servicio de enrutamiento está llamando a ReceiveContext.Abandon para un mensaje con id.: '%1'.
