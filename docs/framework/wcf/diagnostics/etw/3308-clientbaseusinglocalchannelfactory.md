## <a name="description"></a>Descripción  
 Se genera este evento cuando no se usa un generador de canales desde la memoria caché.  
  
## <a name="message"></a>Mensaje  
 No utiliza el generador de canales de memoria caché, es decir, el almacenamiento en caché deshabilitado para la instancia.
