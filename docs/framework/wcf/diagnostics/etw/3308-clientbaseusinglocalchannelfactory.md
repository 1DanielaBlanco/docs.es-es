# 3308 - ClientBaseUsingLocalChannelFactory
## Propiedades  
  
|||  
|-|-|  
|Id.|3308|  
|Palabras clave|ServiceModel|  
|Nivel|Información|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Se genera este evento cuando no se usa un generador de canales desde la memoria caché.  
  
## Mensaje  
 No se usa el generador de canales de la memoria caché, es decir  almacenamiento en memoria caché deshabilitado para la instancia.
