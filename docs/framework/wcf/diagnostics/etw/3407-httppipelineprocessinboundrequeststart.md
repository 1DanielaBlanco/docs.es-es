# 3407 - HttpPipelineProcessInboundRequestStart
## Propiedades  
  
|||  
|-|-|  
|Id.|3407|  
|Palabras clave|HTTP|  
|Nivel|Detallado|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Se genera este evento cuando el controlador de mensajes de HTTP ha iniciado el procesamiento de la solicitud entrante.  
  
## Mensaje  
 El controlador de mensajes HTTP comenzó el procesamiento de la solicitud entrante.
