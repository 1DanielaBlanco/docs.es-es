|||

|Id.|4819|  
|Palabras clave|Descubrimiento|

## <a name="description"></a>Descripción  
 Este evento se genera cuando DiscoveryClientChannel detecta correctamente un extremo y abre el canal con él.  
  
## <a name="message"></a>Mensaje  
 DiscoveryClientChannel detectó correctamente un extremo y abrió el canal con él. El cliente está conectado a un servicio que usa EndpointAddress='%1' y Via='%2'.
