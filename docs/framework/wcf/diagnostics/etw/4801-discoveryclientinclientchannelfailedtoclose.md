# 4801 - DiscoveryClientInClientChannelFailedToClose
## Propiedades  
  
|||  
|-|-|  
|Id.|4801|  
|Palabras clave|Descubrimiento|  
|Nivel|Advertencia|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Este evento se genera cuando DiscoveryClient creado en DiscoveryClientChannel generó un error al cerrarse y por ese motivo se ha anulado.  
  
## Mensaje  
 No se pudo cerrar el DiscoveryClient creado en DiscoveryClientChannel y, por lo tanto, se ha anulado.
