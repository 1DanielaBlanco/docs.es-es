## Descripción  
 Este evento se genera cuando se agrega una nueva ruta de Asp.Net '%1' con serviceType '%2' y serviceHostFactoryType '%3'.

 Se agregó una nueva ruta de Asp.Net '%1' con serviceType '%2' y serviceHostFactoryType '%3'.
