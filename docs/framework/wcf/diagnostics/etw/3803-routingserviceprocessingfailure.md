# 3803 - RoutingServiceProcessingFailure
## Propiedades  
  
|||  
|-|-|  
|Id.|3803|  
|Palabras clave|RoutingServices|  
|Nivel|Error|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Este evento se genera cuando el servicio de enrutamiento genera un error al procesar un mensaje en el extremo.  
  
## Mensaje  
 Error en el servicio de enrutamiento al procesar un mensaje en el extremo con la dirección '%1'.
