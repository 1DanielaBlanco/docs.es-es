# 4813 - DuplicateDiscoveryMessage

|||

|Palabras clave|Descubrimiento|

## Descripción  
 Se genera este evento cuando el mensaje se quitó porque estaba duplicado.  
  
## Mensaje  
 Se quitó un mensaje %1 con messageId\='%2' porque era un duplicado.
