# 3826 - RoutingServiceUsingExistingTransaction
## Propiedades  
  
|||  
|-|-|  
|Id.|3826|  
|Palabras clave|RoutingServices|  
|Nivel|Detallado|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Este evento se genera cuando el servicio de enrutamiento envíe mensajes con la transacción existente.  
  
## Mensaje  
 El servicio de enrutamiento enviará mensajes con la transacción existente '%1'.
