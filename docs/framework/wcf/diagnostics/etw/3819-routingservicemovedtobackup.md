# 3819 - RoutingServiceMovedToBackup
## Propiedades  
  
|||  
|-|-|  
|Id.|3819|  
|Palabras clave|RoutingServices|  
|Nivel|Información|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Este evento se genera cuando el servicio de enrutamiento se mueve a extremos de copia de seguridad.  
  
## Mensaje  
 Mensaje del servicio de enrutamiento con identificador: '%1' \[operación %2\] movido al extremo auxiliar '%3'.
