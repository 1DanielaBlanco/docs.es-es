## Descripción  
 Este evento se genera cuando DiscoveryClient recibe un mensaje de supresión multidifusión de DiscoveryProxy.  
  
## Mensaje  
 DiscoveryClient recibió un mensaje de supresión multidifusión de DiscoveryProxy.
