# 4815 - EndpointDiscoverabilityEnabled
## Propiedades  
  
|||  
|-|-|  
|Id.|4815|  
|Palabras clave|Descubrimiento|  
|Nivel|Información|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Se genera este evento cuando la detectabilidad del extremo se ha habilitado.  
  
## Mensaje  
 Se ha habilitado la detectabilidad de extremo con EndpointAddress\='%1' y ListenUri\='%2'.
