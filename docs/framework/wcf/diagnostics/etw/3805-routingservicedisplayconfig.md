# 3805 - RoutingServiceDisplayConfig
## Propiedades  
  
|||  
|-|-|  
|Id.|3805|  
|Palabras clave|RoutingServices|  
|Nivel|Detallado|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Este evento se genera cuando el servicio de enrutamiento está configurado con RouteOnHeadersOnly.  
  
## Mensaje  
 El Servicio de enrutamiento está configurado con RouteOnHeadersOnly: %1, SoapProcessingEnabled: %2, EnsureOrderedDispatch: %3.
