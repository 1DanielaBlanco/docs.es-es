# 3816 - RoutingServiceTransmittingMessage
## Propiedades  
  
|||  
|-|-|  
|Id.|3816|  
|Palabras clave|RoutingServices|  
|Nivel|Información|  
|Canal|Microsoft\-Windows\-Application Server\-Applications\/Debug|  
  
## Descripción  
 Este evento se genera cuando el servicio de enrutamiento está transmitiendo un mensaje.  
  
## Mensaje  
 El Servicio de enrutamiento está transmitiendo el mensaje con identificador: '%1' \[operación %2\] a '%3'.
