# <a name="servicemodel-registration-tool"></a>Herramienta de registro ServiceModel
La Herramienta de registro ServiceModel es una herramienta de línea de comandos que le permite administrar el registro de ServiceModel en un único equipo. Para obtener más información acerca de esta herramienta, consulte [herramienta de registro de ServiceModel (ServiceModelReg.exe)](../../../../docs/framework/wcf/servicemodelreg-exe.md).
