# Comunicaciones b&#225;sicas: marco de canales
En este tema se hacen una lista de todas las excepciones generadas por el marco de canales [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
## Lista de excepción  
  
|Código de recurso|Cadena de recurso|  
|-----------------------|-----------------------|  
|ReceiveRequestTimedOut|La solicitud recibida en la dirección local especificada ha agotado su tiempo de espera después de la hora especificada.El tiempo asignado a esta operación puede ser parte de un tiempo de espera más largo.|  
|ReceiveTimedOutNoLocalAddress|La operación de recepción ha agotado su tiempo de espera después de la hora especificada.El tiempo asignado a esta operación puede ser parte de un tiempo de espera más largo.|  
|UnableToDemuxChannel|Ningún canal está disponible para aceptar el mensaje con la acción especificada.|
