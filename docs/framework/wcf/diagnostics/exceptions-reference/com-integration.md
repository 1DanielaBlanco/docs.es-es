# <a name="com-integration"></a>Integración de COM+
En este tema se enumeran todas las excepciones generadas por la integración de COM+.  
  
## <a name="exception-list"></a>Lista de excepciones  
  
|Código de recurso|Cadena de recurso|  
|-------------------|---------------------|  
|CannotResolveTypeForParamInMessageDescription|No se puede resolver el tipo del parámetro especificado dentro del espacio de nombres especificado.|  
|NoEqualSignFound|La palabra clave especificada no tiene ningún signo igual que lo siga. Asegúrese de que un signo igual y un valor siguen a cada palabra clave.|  
|UdtNotFoundInAssembly|No se puede encontrar el tipo definido por el usuario especificado. Asegúrese de que el tipo y biblioteca de tipos correctos se registran y especifican.|
