# <a name="systemservicemodelcommunicationobjectopenfailed"></a>System.ServiceModel.CommunicationObjectOpenFailed
No se pudo abrir un objeto de comunicación.  
  
## <a name="description"></a>Descripción  
 No se pudo abrir un objeto de comunicación debido a que se inició una excepción durante la ejecución de su método Open. Otro seguimiento cercano podría contener detalles sobre la excepción.
