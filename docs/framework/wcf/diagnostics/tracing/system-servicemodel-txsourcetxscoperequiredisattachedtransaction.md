# System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction
La transacción especificada se recibió para la operación especificada desde una transacción de InstanceContext.  
  
## Descripción  
 Seguido si la transacción puesta a disposición del código de usuario se dejó tras un método AutoComplete\=false llamado antes de esta invocación.
