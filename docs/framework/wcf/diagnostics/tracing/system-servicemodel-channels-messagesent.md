# <a name="systemservicemodelchannelsmessagesent"></a>System.ServiceModel.Channels.MessageSent
System.ServiceModel.Channels.MessageSent  
  
## <a name="description"></a>Descripción  
 Envío de un mensaje a través de un canal.  
  
 Este seguimiento se emite cuando se envía un mensaje de [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)] si el origen de seguimiento de `System.ServiceModel` está habilitado a modo informativo o detallado. Este seguimiento es necesario para ver la flecha de correlación de mensajes en la vista gráfica de actividad.
