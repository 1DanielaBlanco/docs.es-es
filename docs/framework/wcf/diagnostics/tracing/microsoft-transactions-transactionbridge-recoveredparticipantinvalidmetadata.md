# <a name="microsofttransactionstransactionbridgerecoveredparticipantinvalidmetadata"></a>Microsoft.Transactions.TransactionBridge.RecoveredParticipantInvalidMetadata
La referencia del punto de conexión encontrada en una entrada de registro de recuperación de participante contiene metadatos inválidos o incompatibles.  
  
## <a name="description"></a>Descripción  
 El sistema no puede crear un proxy para una inscripción de participante recuperada.  
  
## <a name="troubleshooting"></a>Solución de problemas  
 Póngase en contacto con Microsoft si el problema persiste.
