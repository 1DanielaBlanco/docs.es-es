# System.ServiceModel.CommunicationObjectAbortFailed
No se pudo anular un objeto de comunicación.  
  
## Descripción  
 No se pudo anular un objeto de comunicación debido a que se inició una excepción durante la ejecución de su método Abort.  Otro seguimiento cercano podría contener detalles sobre la excepción.
