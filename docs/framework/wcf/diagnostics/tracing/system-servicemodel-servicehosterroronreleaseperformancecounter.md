# System.ServiceModel.ServiceHostErrorOnReleasePerformanceCounter
System.ServiceModel.ServiceHostErrorOnReleasePerformanceCounter  
  
## Descripción  
 Se produjeron errores en el ServiceHost al llamar a ReleasePerformanceCounters.Este seguimiento se puede utilizar para determinar los errores al liberar los contadores de rendimiento.
