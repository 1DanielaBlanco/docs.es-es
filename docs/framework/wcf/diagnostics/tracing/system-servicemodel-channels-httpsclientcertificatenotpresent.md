# <a name="systemservicemodelchannelshttpsclientcertificatenotpresent"></a>System.ServiceModel.Channels.HttpsClientCertificateNotPresent
Se requiere el certificado de cliente. No se encontró ningún certificado en la solicitud.  
  
## <a name="description"></a>Descripción  
 Este seguimiento de traza indica que el agente de escucha HTTPS recibió una solicitud HTTPS que no estaba asociada con un certificado de cliente. Dado que el agente de escucha se configuró para exigir certificados de cliente a todas las solicitudes HTTPS, el agente de escucha no pudo validar la autenticidad del cliente.
