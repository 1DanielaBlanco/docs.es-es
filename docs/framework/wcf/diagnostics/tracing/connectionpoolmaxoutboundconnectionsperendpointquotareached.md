# System.ServiceModel.Channels.ConnectionPoolMaxOutboundConnectionsPerEndpointQuotaReached
System.ServiceModel.Channels.ConnectionPoolMaxOutboundConnectionsPerEndpointQuotaReached  
  
## Descripción  
 Se alcanzó la cuota de MaxOutboundConnectionsPerEndpoint, por lo que se cerró la conexión y no se almacenó en este grupo de conexiones.
