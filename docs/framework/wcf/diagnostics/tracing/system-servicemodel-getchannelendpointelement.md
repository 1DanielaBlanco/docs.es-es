# <a name="systemservicemodelgetchannelendpointelement"></a>System.ServiceModel.GetChannelEndpointElement
System.ServiceModel.GetChannelEndpointElement  
  
## <a name="description"></a>Descripción  
 Se emite esta traza cuando se construye un generador de canales, a modo informativo. Proporciona una descripción del punto de conexión con el que está hablando el cliente (p. ej.: dirección remota, enlace, nombre del contrato).
