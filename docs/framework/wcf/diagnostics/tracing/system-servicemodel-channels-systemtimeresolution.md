# System.ServiceModel.Channels.SystemTimeResolution
System.ServiceModel.Channels.SystemTimeResolution  
  
## Descripción  
 La resolución del temporizador se detectó como *n* tics, lo que equivale, aproximadamente, a *m* milisegundos.  Donde *n* y *m* son valores numéricos.
