# System.ServiceModel.Channels.HttpChannelRequestAborted
No se pudo enviar un mensaje de solicitud sobre HTTP.  
  
## Descripción  
 Este seguimiento de nivel de advertencia indica que se anuló una solicitud de canal HTTP.  La información adicional de este seguimiento revelará la razón o el mensaje pasado al método Abort.
