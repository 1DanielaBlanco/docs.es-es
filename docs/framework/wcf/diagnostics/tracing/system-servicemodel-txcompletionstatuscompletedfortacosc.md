# <a name="systemservicemodeltxcompletionstatuscompletedfortacosc"></a>System.ServiceModel.TxCompletionStatusCompletedForTACOSC
La transacción concreta se completó al cerrarse la sesión.  
  
## <a name="description"></a>Descripción  
 Se realizó un seguimiento cuando se cerró la sesión activa y se definió `TransactionAutoCompleteOnSessionClose` en `true`.
