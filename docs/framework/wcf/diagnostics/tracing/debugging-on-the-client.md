# Depuraci&#243;n del cliente
Para facilitar a los usuarios la escritura de aplicaciones cliente para su servicio [!INCLUDE[indigo2](../../../../../includes/indigo2-md.md)], puede agregar el comportamiento del servicio [\<serviceDebug\>](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) al archivo de configuración de su servicio.Este comportamiento se puede utilizar para publicar páginas de ayuda y devolver información de excepciones administrada en los detalles de errores SOAP devueltos al cliente.
