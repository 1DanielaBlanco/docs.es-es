# System.ServiceModel.OverridingDuplicateConfigurationKey
System.ServiceModel.OverridingDuplicateConfigurationKey  
  
## Descripción  
 El sistema de configuración ha detectado una clave duplicada en un ámbito de configuración diferente y está invalidando con el valor más reciente.
