# System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached
System.ServiceModel.Channels.ServerMaxPooledConnectionsQuotaReached  
  
## Descripción  
 Se alcanzó la cuota de MaxOutboundConnectionsPerEndpoint, por lo que se cerró la conexión y la escucha no la volvió a usar.
