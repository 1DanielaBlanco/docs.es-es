# <a name="systemservicemodeldroppedamessage"></a>System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## <a name="description"></a>Descripción  
 Se recibió un mensaje de respuesta, pero no hay ninguna solicitud pendiente esperando a este mensaje. El mensaje se está colocando en tiempo de ejecución [!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)].  
  
 Este seguimiento se puede utilizar para depurar los mensajes perdidos.
