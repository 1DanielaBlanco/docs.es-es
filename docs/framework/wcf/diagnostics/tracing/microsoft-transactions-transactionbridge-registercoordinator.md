# <a name="microsofttransactionstransactionbridgeregistercoordinator"></a>Microsoft.Transactions.TransactionBridge.RegisterCoordinator
El servicio de protocolo WS-Atomic Transaction se registró correctamente con su coordinador para un protocolo de control.  
  
## <a name="description"></a>Descripción  
 Seguimiento de la traza cuando el administrador de transacciones local se ha registrado correctamente con su administrador de transacciones superior
