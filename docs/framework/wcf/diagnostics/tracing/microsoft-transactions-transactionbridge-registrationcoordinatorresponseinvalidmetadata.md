# System.ServiceModel.Diagnostics.MessageCountLimitExceeded
System.ServiceModel.Diagnostics.MessageCountLimitExceeded  
  
## Descripción  
 Se alcanza el límite de mensajes que se puede registrar.  El registro de mensajes se está deteniendo.
