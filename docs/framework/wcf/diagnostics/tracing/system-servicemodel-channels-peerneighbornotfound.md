# System.ServiceModel.Channels.PeerNeighborNotFound
No se pudo realizar una operación en un vecino porque no se pudo encontrar en la lista de vecinos.  
  
## Descripción  
 Este error se produce cuando se conecta a un vecino o se cierra.
