# System.ServiceModel.Channels.PnrpUnregisteredAddresses
La resolución de PNRP ha eliminado el registro de las direcciones del nodo local.  
  
## Descripción  
 Esta traza tiene lugar después de que el servicio PNRP elimine correctamente el registro de las direcciones del nodo local.
