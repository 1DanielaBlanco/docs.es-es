# <a name="httpwebrequestautoredirects-field"></a>HttpWebRequest. \_AutoRedirects campo

`HttpWebRequest._AutoRedirects`es un <xref:System.Int32> que refleja el número de redirecciones realizados para este <xref:System.Net.HttpWebRequest>.

## <a name="syntax"></a>Sintaxis  
  
```csharp  
private int _AutoRedirects
```

> [!WARNING]
> El `HttpWebRequest._AutoRedirects` campo es interno y no están hechos para usarse directamente en el código.
> 
> Microsoft no admite el uso de este campo en una aplicación de producción bajo ninguna circunstancia.

## <a name="requirements"></a>Requisitos

**Namespace:**<xref:System.Net>

**Ensamblado:** sistema (en System.dll)

**Versiones de .NET framework:** disponible desde la versión 2.0.
